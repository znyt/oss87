　　第346章
<br/>　　自从和珅加入琴川关后，大家发现一切都变得井井有条，秩序井然。而大家也感受到大当家对和珅的重视，几乎每天二人都要见面。
<br/>　　每天三次，每次一炷香，时间卡得刚刚好。
<br/>　　这一天下午，程大雷午睡过后，喝了下午茶，换了一套舒服的袍子，慢条斯理的去找和珅聊天。
<br/>　　这些日子以来，虽然没有具体数值显示，但程大雷能感受到和珅对自己的好感度蹭蹭往上涨。
<br/>　　所谓贪，无非贪财，贪权，贪名，贪色。目前没发现和珅有贪名好色的癖好，但权与利却是真的。
<br/>　　一言以蔽之，贪，无非是私心而已。
<br/>　　只要将好感度刷到一定程度，那么他的私心就是自己，也就无所谓贪或者不贪。
<br/>　　目前山寨正在用人之际，身为一个上位者，要有不拘一格降人才的气魄。
<br/>　　一想到用『上位者』形容自己，程大雷就变得欣欣然起来。
<br/>　　当然，每天和和珅接触，程大雷也不单纯为刷好感度，重点也是为了向和珅学上那么几招。而在和珅身边，程大雷安排了几个有天赋，肯上进的手下，让和珅为自己培养人才。等哪天和珅不可用了，自己也有可接替的人才。
<br/>　　和珅就算管理一切事，他也不是山寨的主管，程大雷放了一个刘悲压着他。刘悲是不贪名不贪利，最大的嗜好是每天长吁短叹：生活太没意思了。
<br/>　　和珅办公的地点在工匠坊，此地囊括铁匠铺，木匠坊，裁缝铺等等建筑，而和珅亲自管理的事情，要比这更多。
<br/>　　程大雷去的时候，和珅正在和几个项目的负责人议事，程大雷将首领光环开启，步入了大厅。
<br/>　　一见程大雷进来，所有人同时起身，和珅本来有些怒容，此刻也立即烟消云散。
<br/>　　“坐，坐。”
<br/>　　程大雷满面春风：“我就是随便过来看看，你们聊什么呢？”
<br/>　　“禀大当家，陆陆续续有人迁入琴川，这些人都是从帝国各地奔来的流民，甚至还有戎族。关于他们，还不知该如何安置，如果让他们在琴川安家，必然要出一笔不小的安家费，可如果将他们逐走，却又有损我们之前的名声。”
<br/>　　刚才今天之前，大家就在吵这件事。大家的意见还是处在留或逐之中，现在山寨刚有些积蓄，可不能糟践在无关人等上。
<br/>　　“大当家，依你看这件事该如何处理？”
<br/>　　“嗳，什么话，我既然交给你做，你不管怎么做我都会支持，何必问我的意见。”
<br/>　　程大雷说完，发现和珅竟忽然眼含热泪起来。
<br/>　　“大当家以国士待之，属下真不知该以何报大当家。”
<br/>　　程大雷眨了眨眼，刷好感度也有符合基本法，就算有好感度加持，也不能你在他面前杀了他全家，糟蹋了他女人，他还对你抱有好感。
<br/>　　看来刚才自己一句话，恰好击中了和珅。
<br/>　　“大当家，我有一言，想单独与大当家谈。”
<br/>　　一炷香时间已到，程大雷已经准备要走，冷不防和珅冒出这样一句话。程大雷挥挥手，屏退左右。
<br/>　　“和先生有话直说。”
<br/>　　见大厅内人都散去，和珅道：“属下自从来到蛤蟆寨，自问兢兢业业，不敢有一刻忘公，不敢有一丝私念，敢问大当家一句话，为何信不过属下？”
<br/>　　程大雷一怔，抬头对上了和珅的双目，随之下意识点点头。
<br/>　　首领光环不是弱者光环，并没有将和珅变成一个傻瓜。程大雷心里一直对和珅提防着，难道这满清第一巨贪会觉察不到。
<br/>　　你以诚待人，人才会以诚待你。若一开始就没有信任，怎能换来别人的信任？
<br/>　　如果是以前的和珅，就算觉察到也绝不会说出口，这是城府的问题。但就是因为他对程大雷的好感与日俱增，所以才肯一吐真言。
<br/>　　对陌生人客气，在朋友面前，自然会略略放肆一些。
<br/>　　只不过，程大雷从未有过一刻将和珅当做朋友。
<br/>　　此时，一炷香时间已过，首领光环失效。
<br/>　　程大雷起身：“汝以国士报之，吾必待以国士之礼。”
<br/>　　和珅单膝跪地：“愿生死跟随大当家。”
<br/>　　程大雷未放心和珅，和珅心里难道就没有防范程大雷？所以，乌鸦落在野猪身上，谁也别说谁黑。
<br/>　　今天二人把话说开了，程大雷以后也不会像防贼一样防着和珅。不再每天盯着和珅，隔三五天过来刷一把好感度就可以了。
<br/>　　余下的时间就是自己的，程大雷偶尔和小桃与柳芷都逗闷子，吃吃苏樱口上的胭脂。或者说与苏樱逗逗闷子，吃吃小桃与柳芷口上的胭脂。日子过得不亦乐乎，身在西北荒凉之地，程大雷渐渐感受到一些江南水乡的软绵味道。
<br/>　　不过程大雷还是提醒自己，万万不可沉迷其中，俗话说温柔乡是英雄冢。自己当然是英雄，就连和珅都说了，若大当家不是英雄，天下还有谁配称得上英雄。
<br/>　　况且还有无数双眼睛看着呢，若让他们见到大当家每天沉迷酒色，心里会如何想。目前的琴川关还是要以军事为主，生活为辅，自己不能提前大家过上小康生活。
<br/>　　而由和珅主持工作后，山寨的大小事务程大雷几乎以不用过问，当一切迈入正轨，工作量自然也可减轻。
<br/>　　程大雷决定把重心转移到提升自己实力上。
<br/>　　现在程大雷拥有绝世的力量，但却没有使用这种能力的力量。他体内像藏着一只猛虎，猛虎放出来可以伤人，也会伤己，只要学会如何控制这头猛虎，这力量才算是真正属于自己。
<br/>　　想要控制这头猛虎，靠的不是身体强度，不是智慧，而是意志。
<br/>　　程大雷站在院子中，轻轻握紧斧头，嘴里轻轻道了一声。
<br/>　　“出来吧……”
<br/>　　恍惚有种莫名的力量在体内觉醒，随着血液流动运转全身。
<br/>　　随之，猛虎开口，龙腾四海。
<br/>　　这动静惊动了正在刺绣的苏樱，她与小桃柳芷来到后院，还未靠近便吓了一跳。
<br/>　　墙倒屋塌，大树倒地，树杈散落，院子中心躺着伤痕累累的程大雷。
