　　第96章
<br/>　　入夜，更鼓已响，大概是夜里三更时分。
<br/>　　黑石城王家大院外，蹲着几个人，都是小衣打扮，黑布蒙脸。
<br/>　　“这天可是真够冷的！”程大雷吸溜一口凉气，出门做事，为了行动方便，皮衣都丢在义庄。
<br/>　　“大当家，如今风声正紧，咱们还出来做事，不会有危险吧？”刘悲。
<br/>　　“风声再紧，本职工作不能丢啊。现在有多少为非作歹的恶霸等着我们去替天行道呢，比如这个王家吧，徐军师你打听出来什么了，和大家说说……”
<br/>　　“这王家的主人今年都六十了，买了个十四岁的小丫头，我打听清楚了，今天刚进门……”
<br/>　　“好啦，够啦，单单这一条就够五马分尸！”
<br/>　　“大当家动手吧！”刘悲五人齐声道。
<br/>　　“军师，刘悲你们守在外面接应，事成之后大家分头走，去下一家！”
<br/>　　“大当家，抢完城里的守军肯定被惊动了，咱还要抢！”
<br/>　　“越乱越好，搞的满城大乱，咱们趁乱出城！”
<br/>　　秦蛮第一个翻过墙头，又把程大雷拉上来，这时候张肥和林少羽已经跳进城。
<br/>　　“大当家，奔哪儿？”
<br/>　　“找那老财主屋子，这么老了还想老牛吃嫩草，真该千刀万剐！”
<br/>　　王家大院，一处厢房内。
<br/>　　炕上卧着一美妇人，用锦被拥着香肩。一男人把酒盏递到她腮边：“三娘，你若有心，就吃我这半盏残酒？”
<br/>　　妇人腮边见红：“把灯灭了，别让人看见。”
<br/>　　“看见怕什么，老东西正抱着那小丫头亲热呢，其他人就算看见了，我也不怕他们！”
<br/>　　“呸，坏人，就喜欢你男人的样子，跟个土匪一样。”
<br/>　　“呵，土匪算什么，有机会我也杀他一个俩的。”
<br/>　　咚！
<br/>　　房门被一脚踹开，握剑的汉子道：“不许动，蛤蟆寨做事……”
<br/>　　咦！
<br/>　　进门四个人的眼睛都愣住了，目光在那妇人露在外面的肩头流转。
<br/>　　“体面，注意体面。”程大雷轻咳一声，用剑押着二人脑袋：“说说，你们两个人是什么人？”
<br/>　　“好汉爷饶命，我叫王朳朳。”
<br/>　　“混蛋，竟敢占我便宜，怎么和我一个毛病。”
<br/>　　“好汉，我真叫王朳朳，王员外是我父亲。”
<br/>　　“还真叫这破名字。”程大雷又问那妇人：“你是什么人？”
<br/>　　“我是王员外的三姨太……”妇人怯生生答应。
<br/>　　咦！
<br/>　　程大雷再次睁大眼睛，这大家族里这么乱么。你不能因为自己叫王朳，怀恨在心就让自己老子当王八呀。
<br/>　　四人面面相觑，这明显是闯错门了。本来奔着王员外来的，却闯到小王员外的屋子，他还正给自己老子戴绿帽子。
<br/>　　“大当家，现在怎么办？”
<br/>　　“绑了！”
<br/>　　四人迅速动手，用棉被将两个人光秃秃包在一起，吊在房门口。程大雷兴致上来，还各在二人脸上画了两只小蛤蟆。
<br/>　　“蛤蟆寨做事，王家作恶多端，欺霸乡里，该有此报！”
<br/>　　四人扯着嗓子大喊，又在屋里放了一把火。王家大乱，等人都跑过来时，就看到被绑在门口的两人。
<br/>　　等王员外到了，王家如何大乱，且先不说。
<br/>　　此时，程大雷四人已经跳墙走了，又奔向下一家。
<br/>　　一夜之间，程大雷血洗黑石城七大家族，其实也说不上血洗，无非是大闹一番，留下一只黑蛤蟆后扬长而去。
<br/>　　如此，搅得满城风雨，人心惶惶，城内守军加入到追捕中来。
<br/>　　程大雷搞得满城大乱，本想趁乱混出城去。
<br/>　　然后，没混出去。
<br/>　　真要混出城外，六人目标太大，谁也不能踏雪无痕，循着脚印就能找到六人。如此，反而是城里更安全些。
<br/>　　天亮时分，六人又被逼回义庄。
<br/>　　嘟，完成任务，奖励抽奖次数1 。
<br/>　　是否现在抽奖？
<br/>　　No。
<br/>　　这个时候，程大雷收到完成任务提示。他又查看了一下自己的恐惧值，目前恐惧值在四万多，距离十万目标还差不少。不过，恐惧值一直在增长，这都是昨夜大闹黑石城留下的余波，相信增长到十万也用不了多长时间。
<br/>　　如此说来，程大雷这次行动的目的已经达成，该回到蛤蟆寨，等着恐惧值达到十万后来个十一连抽。
<br/>　　可如今的问题是……他们出不了城了。
<br/>　　问题还不止这些，在大闹贾家之后，所有人都以为程大雷出了城，所以也只是城门增加了守军，别让程大雷再过来捣乱便是。
<br/>　　可昨夜程大雷这么一闹，谁还不知道程大雷还在城里，拉网式的搜索肯定会展开。
<br/>　　躲在这义庄，也并非长久之计。
<br/>　　“军师，你再出去转转，看城里还有什么躲人的地方，狡兔三窟，咱们也不能一直躲在这里。”
<br/>　　“其他人也别长吁短叹，终结一下咱们这次行动的经验，从失败中学习经验么。”
<br/>　　如此说，只是给大家找些事情做。其实程大雷心里没觉得有什么要终结的，无非就是倒霉而已。
<br/>　　徐神机是下午回来，除了给大家带了饭，还给大家带回来一个人。
<br/>　　罗子开。
<br/>　　程大雷推测得不错，城内果然开始针对他们的搜捕。徐神机在街上偶遇罗子开，对方答应让程大雷到他们家藏身。
<br/>　　“你……信得过么？”程大雷心里有些犯疑。
<br/>　　“程当家上次替我出了口恶气，我以性命作保，绝不会出卖程当家。我家里大，没有其他人。”
<br/>　　程大雷倒是信不过罗子开，关键是信不过读书人，读书人脑袋要是一抽风，会做出什么事来谁也猜不透。
<br/>　　不过如今也没有办法，只好暂且在罗子开家安身。他家果然够大，虽然都是断壁残垣，但足够六个人藏身。
<br/>　　而且除罗子开外，家里就只有一个老娘，这下程大雷才稍微放些心，罗子开就算要出卖程大雷，难道会让老娘落在山贼手里。
<br/>　　大家暂且在罗家藏身，等着风声过去，再想办法混出城去。
<br/>　　日子其实还过得去，只是有件事一直搁在程大雷心里，让他耿耿于怀。
<br/>　　这件事情就是……究竟谁他妈的偷了我的马！
<br/>　　这一夜，程大雷窝在屋里酣睡，忽然从梦中惊醒，额头森森冷汗。
<br/>　　“出事了！”
<br/>　　
<br/>(本章完)
