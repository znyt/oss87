　　回到苍穹神宫的苏莫，便又清闲了下来。
<br/>　　每日除了修炼还是修炼，日子枯燥而无味，好在有夕儿的陪伴，倒也倍感温馨。
<br/>　　约莫过了半个月之后，苍穹世界无数大势力的宗主、门中等人物，纷纷来到了苍穹神宫。
<br/>　　这一日，苏莫在苍穹神殿中，召见了所有人。
<br/>　　东海龙王、北海女皇、究楚北、战空等超级势力的掌权者，也全部来了。
<br/>　　望着端坐在帝座之上的苏莫，所有人都是心中忐忑不已。
<br/>　　数年没有现身的苏莫，真不知道强大到了什么地步！
<br/>　　而且，现在苏莫召集他们，不会又是要资源吧？
<br/>　　众人真是怕了，他们个个宗门内都是穷的叮铛响，苏莫若是再要资源，他们真的是要哭了！
<br/>　　“苏宫主，不知你招我等前来，有何要事？”东海龙王率先开口问道，他倒是不担心苏莫索要资源，因为这几年来，苏莫从来没有向他东海蛟龙族索要过半点资源。
<br/>　　他猜测，定然是因为大劫的事，苏莫有事情宣布。
<br/>　　毕竟，试炼者之劫，已经迫在眉睫了！
<br/>　　苏莫眸光如电，扫视大殿中的数万强者，朗声道：“各位，大劫将至，你们有何看法？”
<br/>　　众人听闻苏莫之言，心中稍稍松了口气，原来不是索要资源，那就好说了！
<br/>　　“苏宫主，我东海蛟龙一族，唯苍穹神宫马首是瞻。”东海龙王率先表态，因为龙腾的关系，他蛟龙一族早已和苍穹神宫绑在了一起，自然不会有其他想法。
<br/>　　“我战神殿唯苍穹神宫马首是瞻！”
<br/>　　“我北海冰宫唯苍穹神宫马首是瞻！”
<br/>　　“我金云门唯苍穹神宫马首是瞻！”
<br/>　　“……”
<br/>　　众人见东海龙王表态之后，纷纷表态，同样表示愿意听从苏莫和苍穹神宫的安排。
<br/>　　没办法！现在苏莫实力无敌，众人不表态也不行啊！
<br/>　　苏莫见此，微微点头，道：“很好，你们回去以后，立刻着手，所有势力之人，全部搬进苍穹神宫。”
<br/>　　他让众人搬来苍穹神宫，是要在大劫之时，将所有人都带走。
<br/>　　反正他有空间宝器，方圆五百里的空间，就算是百亿人，也勉强能装的下。
<br/>　　众人闻言一怔，均是心中有些惊疑。
<br/>　　“苏宫主，你这是做何打算？”北海女王疑声问道。
<br/>　　这么多大势力，门下弟子、长老加起来人数之多，何止十亿，估计有百亿之多。
<br/>　　让这么多人都来苍穹神宫，所谓何意？
<br/>　　所有人都是凝望着苏莫，好奇不已，大劫将至，不是应该暂时遣散宗门吗？
<br/>　　“对抗试炼者，杀出苍穹世界！”苏莫眸光如电，朗声说道。
<br/>　　“什么？”
<br/>　　此言一出，大殿中顿时一片哗然，除了东海龙王之外，所有人均是一脸愕然之色。
<br/>　　对抗试炼者？杀出苍穹世界？
<br/>　　这可能吗？
<br/>　　众人心中掀起惊涛海浪，不是众人没想过苏莫要对抗试炼者，而是无数万年来，试炼者不可战胜的神话早已深入人心。
<br/>　　面对大劫，所有人的想法，均是逃避、躲藏，尽量争取能活下来。
<br/>　　所以，他们根本没想到，苏莫如此的大张旗鼓，居然是要对抗试炼者。
<br/>　　“苏……苏宫主，万万不可啊！强硬对抗试炼者，只有死路一条！”一名大家族的家主，脸色有些苍白，苦口婆心的劝告。
<br/>　　在此人看来，对抗试炼者，简直就是自杀，而且还会赔上他全族的性命。
<br/>　　“苏宫主，试炼者个个战力超凡，不可力敌！”
<br/>　　“苏宫主，就算我们侥幸胜了，也会死伤掺重，而且根本没有逃出去的希望！”
<br/>　　“还请苏宫主三思而后行！”
<br/>　　“……”
<br/>　　随即众人纷纷开口，劝说苏莫，几乎没有人认为他能成功。
<br/>　　公然对抗试炼者，开什么玩笑？
<br/>　　这根本就不可能成功！
<br/>　　退一万步来说，就算成功了，巫族发现试炼之人全部死光了，那还得了？定然派大能强者降临苍穹世界。
<br/>　　到那个时候，谁还有活路？
<br/>　　苏莫见此心中暗叹，这些人都是如此的没有锐气吗？连站起来反抗的勇气都没有？
<br/>　　也罢！
<br/>　　他暗暗的摇了摇头，他也不指望这些人出什么力，更不会强求这些人！
<br/>　　既然他们不愿意一搏，想永远的留在苍穹世界，他自然不会阻拦。
<br/>　　“愿意与我共进退之人留下，其他人十息之内离开苍穹神宫！”苏莫扫视众人，朗声喝道。
<br/>　　霎时间，大殿中便寂静了下来，变得落针可闻。
<br/>　　很多强者都是面带惊疑之色，众人面面相觑，不知如何是好。
<br/>　　苏莫会如此好说话？
<br/>　　就这样放他们离开？
<br/>　　一时间，根本没有人敢离开，谁都怕苏莫出手。
<br/>　　直到过去了七个呼吸左右的时间，一名武皇境七重的强者忍不住了，他猛一咬牙，身形一闪，急速的飞离了苍穹神宫。
<br/>　　直到对方飞出了千余里，苏莫也没有出手阻拦。
<br/>　　众人见此，再也不犹豫了，纷纷急速的飞出了大殿，离开了苍穹神宫。
<br/>　　几乎不到两个呼吸的时间，数万名强者便走得只剩下了十几人。
<br/>　　这十几人中，有东海龙王、战乱之父战空、究楚北、北海女皇等等。
<br/>　　十几人之中，除了少数几人是武皇境九重修为之外，其他的全部是准帝强者，其中还有两位妖族准帝。
<br/>　　“苏宫主，我蛟龙一族，愿意陪你一搏！”东海龙王沉声说道。
<br/>　　虽然他也知道，能逃出去的希望极为渺茫，但总归要尽力一试。
<br/>　　只有试了才有一丝机会，不然的话他只能被困死在苍穹世界。
<br/>　　“苏宫主，我战神殿也不会退缩！”
<br/>　　“我太玄宗同样如此！”
<br/>　　“奴家也愿意一试！
<br/>　　“……”
<br/>　　剩下十几名强者纷纷开口，面色坚定，均是愿意一试。
<br/>　　这倒不是众人相信苏莫的实力，而是他们太想离开了。
<br/>　　剩下的这十几人，个个都是苍穹世界的巅峰人物，他们留在苍穹世界，基本上没有活路，就算躲过了试炼者的屠杀，最终也是要化为一杯黄图。
<br/>　　所以，他们才会决定和苏莫联手，奋力一搏，博出一丝希望。
<br/>　　苏莫面上露出一丝笑意，微微点了点头，虽然这些人不能给予他多少帮助，但能看到众人视死如归的决心，他还是很满意的。
<br/>　　不过，众人也不是没有顾忌，随后便纷纷表示，不会倾尽全宗或者全族之力，只会带领一部分强者前来。
<br/>　　毕竟，此事危险性太大，搞不好便会全部被灭，各自门下或族中那些低阶的后辈武者，不宜范险。
<br/>　　对此，苏莫没有异议，因为他也没有把握能带众人离开。
<br/>　　若是无法离开的话，带的人越多，死的人也就越多。
<br/>　　最后，苏莫和众人商议了片刻，便吩咐几人立刻返回，各自回去安排。
<br/>　　而苏莫，便在苍穹神宫中，静等试炼者降临之日。
<br/>(本章完)
