　　苏莫转身向广场外大步走去，他现在脑中有很多感悟，需要仔细梭理一番。
<br/>　　他在苍穹世界，哪里有今日这种听前辈强者授课的机会！现在整个人都开阔了。
<br/>　　因为只有苏莫一人离开广场，木璃一眼便看到了苏莫，她不禁柳眉微微皱了起来。
<br/>　　她都来到此地准备授课了，苏莫居然还离开，明显是不给她面子。
<br/>　　不过，她也没有为难苏莫，毕竟听不听课是每一名弟子的自由，他没有权利要求弟子必须听她讲课。
<br/>　　而且，在她的眼中，苏莫连火元圣师都拒绝了，根本就是个不可理喻的人。
<br/>　　“大家不必客气！”木璃轻挥袖袍，示意众人免礼。
<br/>　　不过，木璃没有怪罪苏莫，但却有人找苏莫的麻烦了。
<br/>　　“站住！”
<br/>　　只听一声爆喝陡然响起，震动了整个广场，广场前方的一名紫衣青年转头望向苏莫，冷喝道：“谁让你走的？回来！”
<br/>　　苏莫闻言眉头一皱，停下了脚步，转头向对方看了过去。
<br/>　　这是一名相貌英俊的青年，五官英挺，肤色白皙，穿着华贵的紫衣，仿佛一位翩翩公子一般。
<br/>　　苏莫目光和紫衣青年对视，眸中闪过凌厉之色。
<br/>　　此人很强，苏莫看不透对方的修为，但他能感觉到对方最少都有武帝境五重、乃至六重的境界。
<br/>　　“我让你回来，你没听到吗？”紫衣青年再次开口，冷冷的喝道，其头颅微扬，语气强势霸道，蕴含不容拒绝之意。
<br/>　　广场上其他的弟子，不少人看向苏莫的目光，都留露出戏虐之色。
<br/>　　这个小子，得罪了不改得罪的人，今天怕是要遭难了！
<br/>　　苏莫闻言眉头一皱，眼眸微眯了下来，他心中杀意凛然，但也忌惮对方的实力。
<br/>　　“太史轩，你想做什么？”
<br/>　　这时，木璃开口了，其冷视着紫衣青年，声音冰冷无比。
<br/>　　紫衣青年闻言，转头望向木璃，脸色立刻恢复如常，轻笑道：“木璃导师，此人居然不给你面子，我实在看不下去，要教教他怎么做人！”
<br/>　　“你不要闹事了，让他走吧！”木璃摇了摇头道。
<br/>　　她的心中非常无奈，这个太史轩，仗着自己的身份背景，在神武学府中目中无人，无法无天。
<br/>　　更主要的是，对方对她纠缠不休，表现的异常殷勤。
<br/>　　“木璃导师，我可没有闹事！”
<br/>　　紫衣青年太史轩摇了摇头，继续道：“你来此授课，是我们南苑的荣幸，此人若是不来也就罢了，来了之后居然要走，简直就是对你的侮辱！”
<br/>　　言到此处，太史轩转头凝视苏莫，冷声道：“侮辱木璃导师，便是侮辱我太史轩，不教训一番他不长记性。”
<br/>　　太史轩的目光，如同冰冷的刀锋，不断的切割在苏莫的身上，凌厉万分。
<br/>　　“那你也要有那个本事才行！”苏莫淡淡的道，他虽然没有丝毫把握战胜对方，但也不想忍气吞声，弱了气势。
<br/>　　他不想惹事，但也不怕事！
<br/>　　“呦呵！你还挺硬气！”
<br/>　　太史轩见苏莫居然敢顶嘴，顿时双眉一挑，心中来了兴趣。
<br/>　　在这南苑之中，敢和他对抗的人可是不多，这个面生的弟子，居然敢顶撞他，着实胆子不小。
<br/>　　踏！踏！踏！
<br/>　　随即，太史轩抬步向苏莫走了过去，他的身上玄力涌动，仿佛要强势出手一般。
<br/>　　“我倒要看看，你有什么能耐，如此的嚣张！”太史轩眸中精光四溢，戏虐的说道。
<br/>　　苏莫见此，立刻警惕了起来，体内五枚神丹同样催动了起来，随时准备出手。
<br/>　　“住手！”木璃俏脸冰寒，顿时一声娇喝，一股强大的尊者威压弥漫全场。
<br/>　　浩荡的威压恐怖无比，如同高山大岳，又如同瀚海奔腾。
<br/>　　霎时之间，所有人均是身形一沉，仿佛大地引力再次暴涨万倍，让得他们顿时被束缚的不能动弹。
<br/>　　木璃也是怒了，让太史轩给她气怒了。
<br/>　　对方的霸道，让她心中非常的不喜，若是换了其他人，他早已出手了。
<br/>　　随即，木璃冷声告诫道：“太史轩，他是火元圣师看重的人，你莫要胡来！”
<br/>　　她抬出了火元圣师的名头，好吓住对方。
<br/>　　太史轩的身份太大，即便是她也不能将其怎么样，只能用火元圣师的名头，镇住对方了，免得再生出什么事端。
<br/>　　至于苏莫，的确是火元圣师看重的人，木璃自然不能让他有什么事。
<br/>　　“火元圣师？”太史已经被强大的尊者威压压制，立刻停下了脚步，闻言面色一怔。
<br/>　　随即，太史轩面带惊疑的打量着苏莫，目露恍然道：“难怪你这么面生，难道你就是那个拒绝成为火元圣师弟子的人！”
<br/>　　“居然是他！”
<br/>　　“我还当是什么人物？原来是这小子！”
<br/>　　“他的炼器天赋难道真的很厉害吗？”
<br/>　　“真是脑残啊！”
<br/>　　广场上其他的弟子闻言，亦是纷纷恍然，个个面带惊奇的打量着苏莫。
<br/>　　火元圣师要教导苏莫炼器之术一事，不知怎么流传了开来，居然变成了火元圣师要收苏莫为徒！
<br/>　　而苏莫的拒绝，更是让所有人大跌眼镜。
<br/>　　火元圣师是什么人物，圣级炼器圣师，身份在整个古灵星河都是巅峰般的存在。
<br/>　　能有勇气拒绝火元圣师的人，这在众人看来，不是超凡人物就是脑残。
<br/>　　但在众人眼中，苏莫很普通，并无什么不凡之处。
<br/>　　苏莫目光看向木璃，眸中闪过一丝歉意和谢意，他也没想到自己的做法，会产生侮辱木璃之意。
<br/>　　不过，对方还如此帮助他，倒是让他心中有些歉意了。
<br/>　　“你走吧！”木璃收起了尊者威压，对苏莫清冷的说道。
<br/>　　苏莫点了点头，向对方一抱拳，又瞥了太史轩一眼，便转身大步离开。
<br/>　　太史轩没有再刁难苏莫，不过，他望着苏莫离去的背影，却是冷喝道：“下次若你再敢如此，我让你后悔来到神武学府！”
<br/>　　太史轩的声音，充满了无边的狂傲，在苏莫的耳边回荡，让得他不由得捏紧了双拳。
<br/>　　此人真是欺人太甚！
<br/>　　不过，现在形势比人强，苏莫只能先暂时忍耐了。
<br/>　　很快，苏莫便远离了广场，消失在了众人的视线之中。
