　　嗖嗖嗖！！
<br/>　　一瞬间，所有人都下意识的暴退，并且全身源力涌动，护体源力滚滚而出。
<br/>　　众人之中，当属太苍和凤晴两人，最为强大。
<br/>　　太苍的身上，金光耀眼，恐怖的锋芒如同火山爆发。
<br/>　　而凤晴的本体，是一只冰凤，体外立刻出现了厚厚的坚冰，企图挡住司空炎的攻击。
<br/>　　不过，众人虽然都想防御，但是，毕竟有些晚了，全部是仓促之下，根本组织不出太厉害的防御。
<br/>　　轰隆隆！
<br/>　　一阵惊天爆响，司空炎的两道拳茫，如同两轮烈日，击在了人群之中。
<br/>　　爆炸开的火焰拳劲，火光四溅，让得整个通道之中，变得炙热无比，仿佛化为了一片火海。
<br/>　　啊啊啊！！
<br/>　　一瞬间，惊呼声、惨叫声、不绝于耳。
<br/>　　血肉四射，人影横飞，一片狼藉。
<br/>　　只是一瞬间，一百余人之中，足有一大半人被轰击的倒飞而出，口喷鲜血，纷纷撞击在通道尽头的墙壁之上。
<br/>　　有二十余人，被轰击的直接肉身崩溃，只有元神得以逃脱。
<br/>　　当然，也有十余人，当场神魂俱灭，陨落于此。
<br/>　　太苍和凤晴两人，虽然实力强大，但是也同样被轰击的飞了出去。
<br/>　　只不过，有众人分担火焰拳劲，他们两人并未受什么伤势。
<br/>　　轰隆隆！！
<br/>　　滚滚火浪席卷，那些肉身崩溃之人，瞬间便消失不见，被传送了出去。
<br/>　　一百一十余人，转眼间便只剩下了八十余人。
<br/>　　一众人，几乎是被打懵了，上来就惨败，折损了接近四分之一的战力。
<br/>　　“宝物都得到了吗？”苏莫看向司空炎，在众人脱离梦境之时，他也第一时间脱离了梦境。
<br/>　　“都得到了，一件没留！”司空炎咧嘴一笑，显得颇为得意。
<br/>　　苏莫闻言，微微点头，随即上前两部，凝视众人，道：“我不想为难你们，你们可以自行出去。”
<br/>　　众人闻言，面面相觑，很多人的脸色，极度难看。
<br/>　　“佩服！”
<br/>　　“告辞了！”
<br/>　　“在下心服口服！”
<br/>　　有十余个受伤较重的人，向苏莫和司空炎抱拳，随即大步而行，走心出口处。
<br/>　　他们的伤势不轻，也见识到了苏莫和司空炎的不凡，所以即便心中不甘，也只能无奈的放弃了。
<br/>　　“你们……！”巫铨见此，面色铁青，这些家伙居然如此懦弱，这样就认输了。
<br/>　　不过，此时此刻，巫铨更在乎的是太苍和凤晴的态度。
<br/>　　“诸位，这两人抢夺了我们的宝物，甚至是使用的兵器，我们决不能就此认输！”太苍的面上，现出滔天的怒火，对面两人实在太可恶了，抢夺宝物也就算了，连他的兵器都抢夺了。
<br/>　　众人闻言沉默，他们心中也很愤怒，但是，却是没有了自信。
<br/>　　百余人联手攻击，都伤不到对方，更遑论现在。
<br/>　　凤晴同样沉默不语，她知道，眼前的两人，是非常可怕的对手。
<br/>　　“冥顽不灵，你们若是自找苦吃，别怪我不客气了！”司空炎朗声说道。
<br/>　　“猖狂，若不是他有天梦元神，就凭你能击退我们这么多人？”太苍指了指苏莫，冷声向司空炎说道。
<br/>　　“吆喝，看来你很不服气啊？”司空炎大笑了起来，这个家伙真是有胆量。
<br/>　　“看来你很有自信，你若是能击败我和凤晴，我们便甘愿认输！”
<br/>　　太苍声音低沉，继续道：“若是你输了，就将我们的宝物，全部还给我们如何？”
<br/>　　太苍心中清楚，有苏莫在，他们基本上是必输无疑了，想抢会宝物可能性太小。
<br/>　　毕竟，一个防御无敌的人，他们现在根本不可能击败了。
<br/>　　所以，他用了一个策略，激将司空炎。
<br/>　　凤晴闻言，顿时眼眸一亮，若是对方答应下来，她们就有翻盘的机会。
<br/>　　“哈哈，激将我吗？我还真吃这一套！”司空炎闻言大笑出声，他丝毫无惧。
<br/>　　“交给你了！”苏莫向司空炎淡淡的说了一句，便转身向回走去，走到了出口之处站定。
<br/>　　他对于司空炎的实力，自然是丝毫都不怀疑，司空炎能解决的事情，他懒得亲自出手。
<br/>　　“你同意了？”太苍眸光一闪，眸中闪过一道犀利的精光。
<br/>　　“当然！”司空炎点了点头。
<br/>　　太苍闻言，顿时心中暗喜，然后向凤晴看了一眼。
<br/>　　凤晴点了点头，这是最后的机会了，她和太苍两人，只要能胜了，就能翻盘。
<br/>　　除非对方返回，不过，对于万古天骄级别的天才而言，都是极度自负的，还不至于出尔反尔。
<br/>　　巫铨等其他人，亦是心中大动，如此的话，他们还有机会，全看太苍和凤晴的了。
<br/>　　随即，众人都不在出声，凤晴和太苍两人，面色凝重的望着司空炎，身上强大的气息涌动。
<br/>　　太苍的手中，金色的源力凝聚，凝结成了一柄金色的战刀，熠熠生辉。
<br/>　　啾！
<br/>　　就在此刻，一声嘹亮的凤鸣声，如同利箭破空，洞穿人的耳膜。
<br/>　　只见凤晴的身躯，瞬间化为了一只五丈大小的冰封，急速向司空炎冲了过去。
<br/>　　冰封美轮美奂，如同是冰晶雕琢，充满了异样的美感。
<br/>　　不过，冰封的气息极为恐怖，身上散发的冰寒之气，足以冰冻人的神魂。
<br/>　　冰凤的速度极快，如同一道闪电，划破长空，向司空炎冲了过去。
<br/>　　而太苍也没有闲着，他身形一动，跟在冰凤之后，向司空炎冲了过去。
<br/>　　“冰火相克，我倒要看看，你这只冰凤有何不凡！”
<br/>　　司空炎轻笑一声，他的身上，陡然升起了炽烈的火焰，几乎是一瞬之间，他整个人，便化为了一个直径三丈大小的火球。
<br/>　　火球的火焰，不是火红色，而是紫红色，仿佛是一轮紫色的太阳。
<br/>　　嗖！
<br/>　　紫色的火球一往无前，迎着冰凤便冲了过去，几乎是瞬息之间，便互相临近在了一起。
<br/>　　唰！
<br/>　　只见绝美的冰凤，口中陡然吐出一股浩大的白色气流，向火球之上狂涌而去。
<br/>　　白色的气流，是一股极寒之气，乃是冰凤体内的冰属性精华，乍一出现，便冰冻了虚空，让得火球都瞬间变成了一个冰球。
<br/>　　“败吧！”
<br/>　　一声暴喝响起，太苍身形一闪，冲向了冰球，整个人如神灵降世，双手握刀怒劈而下，狠狠的劈向冰球。
