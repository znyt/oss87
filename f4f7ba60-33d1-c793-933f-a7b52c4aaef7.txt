<br/>　　但这条骨龙，对吞噬技能却没有任何反应。
<br/>　　这样的话，也就只有一个解释了，它还没死。
<br/>　　如果他没有猜错，那枚尸丹绝对被下了什么手脚，如果刘哔按骨龙吩咐的做了，后果谁都无法预料。
<br/>　　天上或许真的会掉馅饼，但除了馅饼之外，随着一起掉下来的，还有陷阱。
<br/>　　只是，有一个地方刘哔还是想不明白。
<br/>　　这条骨龙已经足够强大，但为什么还要费尽周折搞这些事情？
<br/>　　还是说……是有什么特殊的限制，让它无法对自己出手？
<br/>　　关于这些，他并不清楚。
<br/>　　但现在要考虑的，是如何处理尸丹和骨龙尸体的问题。

<br/>　　昏沉沉的醒来，白小飞的身体很是酸痛，他依稀记得，自己是在和一个变异动物对抗的时候，昏迷过去了。

<br/>　　最终，他还是没能舍弃那枚尸丹以及骨龙尸体，毕竟这两样东西，都可以说是极为珍贵的宝物。
