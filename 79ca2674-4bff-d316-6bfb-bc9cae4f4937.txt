    直到深夜一点，陈兴才猫着腰站起来，朝正在发呆的两个人招手，示意行动开始。两人早已等不及了，迅速整理了一下装备，就猫着腰，跟在陈兴的身后，朝黑暗中的营地移动。<br/>    <br/>    大约二十分钟后，他们来到了营地的外围。负责守夜的佣兵的都在打瞌睡，似乎刚经历了一场大战，疲惫不堪。有的靠在车旁睡着了，有的靠在山壁上睡着了。只有两个还算精神的，但都心不在焉，不时望向溶洞的洞口，然后闭着眼睛，似乎在幻想着什么。<br/>    <br/>    陈兴朝卡西做了个“左右包抄”的手势，然后又朝白人佣兵做了个“原地等候”的手势。他之所以这样安排，是因为卡西比较灵活，白人佣兵迟钝一些，而且块头大，不利于隐蔽。两人同时点头，陈兴立即绕到一边，靠着沙地车的遮掩，逐步逼近目标。<br/>    <br/>    数十秒后，陈兴和卡西分别就位。随着陈兴一个割喉的手势，两人同时扑上去，捂着卫兵的嘴巴，一刀割喉。<br/>    <br/>    “唔，唔！”<br/>    <br/>    卫兵挣扎了几下，喉咙喷出大量鲜血，瘫软下去。陈兴又分别指了指三名正在瞌睡的卫兵。三人一刻也没停留，迅速包抄过去，割喉放血。<br/>    <br/>    不一会儿，负责守夜的佣兵全被处理掉了。三人端着突击步枪，悄声无息地来到溶洞洞口。陈兴贴在洞旁石壁上，竖起食指放在嘴前，做出“噤声”手势，然后侧过脑袋，轻轻地掀开布帘的一角，望了进去。<br/>    <br/>    里面火光通明，白花花的一片。尽管陈兴三世为人，自诩见多识广，但这样的场面，却还是第一次见。他原本以为，地球上的那些岛国片，都只是存在于视频中，全部是假的。却没想到，这里正上演着一出真实的戏码。<br/>    <br/>    他很难用语言去形容他现在所看到的。那情形，就像一群游客正在公园里围观一只母猩猩，然后不断喂香蕉给它吃。母猩猩应接不暇，已经吃得肚子又圆又胀，快要撑不下去了。可大伙还是一个劲地喂它，逼它吃下去。而更可怕的是，后面还有大批的游客排着队，等着上来喂食……<br/>    <br/>    陈兴扫视了一圈，大概记住了里面的人的位置，然后缩回来，指着布帘，朝卡西和白人佣兵做了个“观察”的手势。<br/>    <br/>    卡西掀开一点，观察了小一会儿，缩回来，深表同情地摇了摇头。白人佣兵深深地吸了口气，从卡西的位置看进去，顿时握紧双拳，整个人都在发抖。<br/>    <br/>    陈兴拍了拍白人佣兵的肩膀，示意对方冷静。<br/>    <br/>    大约几秒钟后，白人佣兵恢复了平静。陈兴端起突击步枪，手掌朝洞内切了一下。其他两人立即会意，端起枪，做出突击的准备。<br/>    <br/>    陈兴伸出一根手指，然后两根手指，当第三根手指伸出来的时候，掀开布帘，钻了进去。<br/>    <br/>    三人鱼贯而入，没有一丝犹豫，朝着人多的地方，抬枪就射。<br/>    <br/>    “哒哒哒！”“哒哒哒！”“哒哒哒！”<br/>    <br/>    三支突击步枪同时开始开火，子弹长驱直入，射进毫无防备的人群中。<br/>    <br/>    霎时间，血肉横飞，哀嚎遍地，惨叫声响彻了溶洞。<br/>    <br/>    里面的人大部分都是底层佣兵，没有防护技能，鲜有的几个身上冒出蓝光的，也被陈兴蓄能击杀。<br/>    <br/>    鲜血瞬间染红了地面，血腥味浓郁得化不开。炽热的子弹飞旋而入，在赤条条的身体上不断开出血洞，大量的鲜血喷涌而出。<br/>    <br/>    一具具尸体倒在血泊中，场面混乱无比。上一刻还在享受极乐的佣兵们，此刻却像一窝仓皇逃窜的地鼠，互相挤压着、踩踏着，恨不得找个缝隙钻进去，根本没有想过要反抗。别说装备了，他们连衣服都没穿，武器全部放在外面的车上。<br/>    <br/>    而另一边，陈兴等三人都是老手，即便在这种情况下，依然保持着点射，最大化攻击效果。而且他们之间很有默契，轮番射击，陈兴率先打完子弹，重新上弹，然后是卡西，再然后是白人佣兵。<br/>    <br/>    而就在他们刚冲进来的时候，位于场中间的阿乔木先是呆了几秒，随即反应过来，直接卧倒在地上，避免被自己人误伤。<br/>    <br/>    在求生本能的驱使下，里面的人挤在两侧的角落里，让屠杀变得更加方便。<br/>    <br/>    数轮射击过后，已经没有人能站起来了，倒在地上挣扎着，抽搐着……<br/>    <br/>    白人佣兵正想上前，却被陈兴拦了一下，示意他继续补枪。<br/>    <br/>    “哒哒哒！”“哒哒哒！”“哒哒哒！”<br/>    <br/>    三支突击步枪无情地点射着，没过多久，场内没有一个会动的目标了。尸体横七竖八，互相堆叠着，宛如地狱。<br/>    <br/>    枪声停止后，过了一小会儿，尸体堆中爬出一条浑身是血的人影，眼中透着巨大的恨意，仿佛从地狱归来的罗刹。<br/>    <br/>    她左右环视，从血浆中捞出一把匕首，扑在旁白的尸体上，双手握着，一下接一下地扎下去。<br/>    <br/>    “死！”“死！”“死！”“死！”“死！”“死！”“死！”<br/>    <br/>    她疯狂地呐喊着，嘶声力竭，宣泄着无穷无尽的恨意。就连在场最理智的陈兴，也禁不住有些动容。很难想象，在这十多个小时里，她究竟遭遇了怎么样的折磨。<br/>    <br/>    三人都没有上前，默默地看着，任由阿乔木发泄心中的恨意。这是最好的选择，也是最好的尊重。<br/>    <br/>    然而，就在三人有些松懈的时候，尸体堆中突然扑出一条粗壮肥大的身影，一把夺下阿乔木的匕首，并扣住她的脖子，将匕首架在上面。<br/>    <br/>    白人佣兵马上认出来了，这个“血人”正是这群黒鹫佣兵的头领。<br/>    <br/>    三人同时端起枪，却投鼠忌器，不敢射击。<br/>    <br/>    “让开，不然我杀了她！”佣兵头领沉喝道，两只眼睛睁得铜铃大，眼底的眼白和满脸的血污形成了鲜明的对比。<br/>    <br/>    “离远点！”佣兵头领押着阿乔木，步步逼近。三人对视了一眼，朝洞外退却。<br/>    <br/>    佣兵头领盯着三人，一步步挪出洞外，再挪到武装蟋蟀的旁边，朝车里看了看，确定钥匙在上面后，猛地一下推开阿乔木，窜上驾驶位，钥匙一拧，踩死油门，朝外面冲去。<br/>    <br/>    <br/>
