　　苏莫的实力，震动了迷宫城之外的所有人，毫无意外的成为了所有人关注的焦点。
<br/>　　不过，对此苏莫并不知情，当然，他就算知情也不担心。
<br/>　　毕竟，只要不暴露身份，他的实力越强，对于他的龙族，反而好处越大。
<br/>　　苏莫没有去追踪冰儿，对方逃了也就逃了，他继续跟着出口移动。
<br/>　　接下来的时间之中，不时的有人来到出口之处，有人单独前来，也有两人或者三人结队而来。
<br/>　　不过，这些来到出口处的人，结局都是一样，全部被苏莫抢夺一空，然后被从出口处丢了出去。
<br/>　　没有人是苏莫的对手，基本上都是一招之下就被擒拿，让得迷宫城之外的人，纷纷无语。
<br/>　　现在的情况，就是一个人，要抢夺所有人的宝物。
<br/>　　这让得很多有奴仆在迷宫城的龙族之人，纷纷面色凝重，心中有些郁闷。
<br/>　　龙城的脸色，阴沉如水，这不仅是龙腾的话应验了，还因为，巫铨正在飞快的接近出口，即将和苏莫相遇了。
<br/>　　嗖！
<br/>　　通道之中，巫铨在飞速的疾驰，一步数十丈。
<br/>　　这一次在迷宫城之中，他只得到了一件宝物，略有些失望。
<br/>　　不过，他并没有什么不甘，这么多奴仆之中，能得到一件宝物，也算是不错了。
<br/>　　他现在，要立刻离开这里，毕竟，现在剩下的人，都是高手，继续留下来风险很大。
<br/>　　唰！
<br/>　　来到一处岔路口，巫铨转头，便看到了通道尽头的出口，同时，他也看到了出口前伫立的苏莫。
<br/>　　“是你？”巫铨眼眸一凝，眸中闪过一道璀璨的光芒。
<br/>　　出口之前，苏莫亦是看到了巫铨，嘴角勾起一丝笑容，道：“想离开？留下身上的宝物，你就可以离开？”
<br/>　　望着巫铨，苏莫心中微微有些疑惑，此人不是跟着司空炎进入迷宫城的吗，居然没有被淘汰？
<br/>　　难道此人的实力，如此之强，能在司空炎的手中逃脱。
<br/>　　“你更狂妄！”巫铨微眯着眼眸，上下打量着苏莫，此人堵住出口，这做法不可谓不狂妄。
<br/>　　不过，巫铨没有急着靠近，他知晓苏莫的实力绝对不会弱。
<br/>　　毕竟，他与司空炎交过手，司空炎都那般强大，苏莫绝对不可能弱。
<br/>　　苏莫沉默不语，对方不敢靠近，他也不会去追击对方。
<br/>　　巫铨沉吟了片刻，手掌一动，掌中源力形成了一条一丈长的灵蛇，快速的向苏莫飞了过去。
<br/>　　他想要以此，来试探一番苏莫的实力。
<br/>　　不过，面对这种攻击，苏莫云淡风轻，只是一挥衣袖，灵蛇便被击溃。
<br/>　　“不比那家伙弱！”巫铨的脸色，凝重无比，他绝望的发现，苏莫的实力，不比司空炎弱。
<br/>　　毫不犹豫，巫铨身形急退，从侧面的一条通道离去。
<br/>　　他要远离苏莫，等到有其它人到来，再联手出击。
<br/>　　对此，苏莫并未阻拦，少倾，出口移动到远处，他跟着出口离去。
<br/>　　巫铨没有离开，就在出口附近转悠，等待其他人的到来。
<br/>　　每等到一人，巫铨便向来人说明情况，约莫半刻钟左右，他便纠集了包括冰儿在内的九人。
<br/>　　“巫铨，那人真这么强大？”
<br/>　　有人问道，他感觉巫铨有些夸张了，但是，由于他和巫铨算是比较熟悉，倒也没有拒绝一起行动。
<br/>　　他们这些人，都是天才人物，都得到了宝物，彼此之间，都还算克制，没有互相出手。
<br/>　　“此人绝对强大，万不可大意！”巫铨郑重的说道。
<br/>　　“之前我和他交过手，不是他一招之敌！”冰儿提醒道。
<br/>　　“这么强？”
<br/>　　众人闻言震惊，既然如此的话，他们只能联合出手了。
<br/>　　“差不多了，凭我们十人的实力，应该足以击败他了！”巫铨环视众人，沉声说道。
<br/>　　目前，包括他在内，已经有了十人，应该足以击败那个龙腾的奴仆。
<br/>　　“走！”
<br/>　　其中一人，朗喝一声，率先行动了起来。
<br/>　　随即，众人纷纷跟随，向着出口的位置而去。
<br/>　　这一幕，迷宫城之外的人，全部看在眼里，十名天才奴仆联手，这一次能击败那位屠霄吗？
<br/>　　众人心中很是期待结果，若是十人胜了，这在情理之中，若是十人败了，那屠霄当真的不可思议了。
<br/>　　出口之处，苏莫背负着双手，眼眸看向通道的尽头。
<br/>　　他的耳中，已经听到了阵阵的脚步声，有一群人正飞速而来。
<br/>　　呼吸之间，道道人影出现在了他的视线之中，有巫铨还有冰儿，共计十人。
<br/>　　“交出你们身上的宝物，你们便可以出去！”苏莫面色古井无破，淡淡的说道。
<br/>　　“小子，你未免太猖狂了吧，我们有十人，你认为你还有胜算吗？”巫铨嗤笑一声，冷冷的说道。
<br/>　　不过，他心中却是惊疑不定，因为苏莫太镇定了，镇定的有些不正常。
<br/>　　“交出宝物，是你们唯一的出路！”苏莫面色不变。
<br/>　　“和他费什么话，联手将他轰杀！”有人大声说道，立刻得到了所有人的同意。
<br/>　　随即，一行十人，大步向苏莫靠近。
<br/>　　“动手！”
<br/>　　当距离苏莫还有数百丈之时，巫铨怒喝一声，率先出手了，一拳轰击而出。
<br/>　　轰轰轰！！
<br/>　　一瞬间，其它九人，全部都出手了。
<br/>　　刀光闪耀，剑气纵横，寒气肆虐，十道强大的攻击，在宽敞的通道之中，以绝杀之势，向苏莫袭杀而去。
<br/>　　如此一幕，让迷宫城之外的众人，纷纷倒吸一口冷气。
<br/>　　如此强大的攻击，就算是万古天骄，也很难抵挡的住吧！
<br/>　　面对十人的联手攻击，苏莫面色不变，手中剑光一闪，长剑出现在了手中。
<br/>　　此剑，自然不是终极神剑，只是普通的神剑而已。
<br/>　　唰！
<br/>　　苏莫动了，人剑合一，直接迎着十人的攻击，暴冲而去，整个人如同一道疾驰的闪电。
<br/>　　虽然他不动如山，也能挡住众人的攻击，但是，在不暴露所有实力的情况之下，很难将众人擒住。
<br/>　　所以，他整个人向十人冲了过去。
<br/>　　血脉之力和五行源力，赋予了他无敌的防御之力，他势如破竹，如利箭一般，洞穿了所有攻击，直接杀向十人。
<br/>　　这些人，在苏莫的眼中，就是土鸡瓦狗，实力根本就不在一个层次。
<br/>　　“什么？”
<br/>　　“这怎么可能？”
<br/>　　“快撤！”
<br/>　　怒吼之声不绝于耳，十人简直不敢相信，苏莫居然视他们的攻击如无物。
<br/>　　眼见苏莫已经冲到身前，十人疯狂暴退，巫铨和冰儿有心理准备，反应最快，急速逃窜。
