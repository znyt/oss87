        果不其然，季思妍今夜助人为乐，一路狂飙车技，就连警车都冒出来为她保驾护航，救助孕妇前往医院生产的举动上了新闻头条。
<br/>
<br/>        落落大方、机智果断、心地善良、人美心美……
<br/>
<br/>        一夜间，所有赞美的词语，在她季思妍的身上都觉得不够用。
<br/>
<br/>        只是，美中不足的是，季思妍没想到那个孕妇会生的那么快，在她的豪车里就直接把孩子给生了下来。
<br/>
<br/>        弄得她那豪车的后座上全是血水，又脏又恶心。
<br/>
<br/>        季思妍虽然心里嫌弃，但是不得不强忍着，装出一副非常担忧那孕妇生命安危的样子。
<br/>
<br/>        好在她让那个产科男医生一路陪在那孕妇身边，保住了那个妇人和宝宝的生命安全。
<br/>
<br/>        这一步，季思妍是铤而走险，就为了博得傅夜沉对自己的好感。
<br/>
<br/>        季董事长听完女儿的叙述后，目瞪口呆得一句话也说不上来。
<br/>
<br/>        他真的没想到，女儿的这个计划居然完美到无懈可击。
<br/>
<br/>        在来之前，他家那位“正宫娘娘”就有交代，一定要让他们的女儿思妍和傅家的大少爷傅夜沉攀上联姻的关系。
<br/>
<br/>        傅家地位在滨江城里几乎是处于金字塔的顶端，他们傅家高高在上，压根就不屑与其它集团公司的儿女用联姻的方式来巩固自己的实力。
<br/>
<br/>        所以，季思妍的母亲就告知季思妍，既然不能唐突地提联姻之事，那么就得先学会如何抓住那个男人的心。
<br/>
<br/>        一个美丽优雅，又不失可爱纯真，心地善良又乐于助人，家庭富裕又高学历，愿意努力工作学习的女人，怎会不让一个男人动心呢？
<br/>
<br/>        这样几近完美的女人，是许多男人梦寐以求的吧！
<br/>
<br/>        就那么两三个小时的时间，季思妍把自己的优点，在傅夜沉面前展现得淋淋尽致。
<br/>
<br/>        季董事长却突然唉声叹气：“可惜啊！傅夜沉那小子似乎真如传闻的一样不近女色，所以他只把你安排在他的子公司里，还让他弟弟带你！真不知道他是怎么想的？”
<br/>
<br/>        季思妍听着父亲那惆怅、抱怨的话，红唇微扬，似笑非笑，心思难测。
<br/>
<br/>        想必，她今晚那光荣的事迹，傅昼景一定在电视机前看到了吧！
<br/>
<br/>        她不信，阿景会对她如此绝情。
<br/>
<br/>        阿景心里一定还爱着她，她这么优秀，他不爱她，还能爱谁？
<br/>
<br/>        季思妍心里笃定，阿景只是在生她的气，才在表面上拒绝她，正因为如此，他越是冷落她，越是代表他在逃避心里对她的爱慕。
<br/>
<br/>        母亲让她搞定傅夜沉，但她却想要同时搞定傅家的这兄弟两。
<br/>
<br/>        傅夜沉和傅昼景，这两个男人的心，她季思妍都想要。
<br/>
<br/>        傅家的两个儿子都爱她，拜倒在她的石榴裙下，光想想，她就觉得自己好像站在了金字塔的顶端，成为了受人瞩目的女王。
<br/>
<br/>        是啊！
<br/>
<br/>        在国外的这些年，季思妍俘获了不少男人的心，什么样的男人都有，偏偏又让他们对她爱而不得。
<br/>
<br/>        她就站在那儿当他们心目中的女神，让他们心里痒到，只要她一句话，他们就会乖乖听她吩咐的地步。
<br/>
<br/>        电视机里，哪怕只是手机拍摄的画面，季思妍在镜头下的面容就像一个当红女明星一样光彩照人。
<br/>
<br/>        傅昼景手里拿着遥控器，看到季思妍出镜时，不由地皱了下眉头。
<br/>
<br/>        当他刚想把电视给关了去睡觉时，却被童筱颖连忙阻止，并夺走了他手里的遥控器。
<br/>
<br/>        童筱颖看着电视机荧幕上的季思妍，由衷地感慨、夸赞道：“这个女孩长得真的很漂亮，而且心肠又这么好！她救了一个女人和一个小宝宝啊！而且，她的车，看起来好像很贵的样子。她都不嫌弃，还一直在那问母子是否平安。”
<br/>
<br/>        “我去睡了，你早点睡。”傅昼景眉头紧锁，背过身去，直接从沙发上起身离开。
<br/>
<br/>        当童筱颖下意识地侧过脸去看他的时候，只看到他那抹修长的背影，疾步走进了房门内。
<br/>
<br/>        此时此刻，傅昼景的心情是复杂的。
<br/>
<br/>        这都五年过去了，思妍还是那么的心地善良，她一直没变……
<br/>
<br/>        那么她白天对他说的还爱着他的事情……
<br/>
<br/>        也还是真的吗？
<br/>
<br/>        不，都这个时候了，他还去在乎她是否一直还爱着他做什么？
<br/>
<br/>        傅昼景躺在床上辗转难眠。
<br/>
<br/>        童筱颖看完晚间新闻后，就把电视给关了。
<br/>
<br/>        她特意走到傅昼景的房门口，跟他说了一声“晚安”。
<br/>
<br/>        直到听到童筱颖的问候声，傅昼景的思绪才从初恋季思妍的身上拉回来。
<br/>
<br/>        “晚安。”傅昼景温和地应了声。
<br/>
<br/>        童筱颖甜甜地笑了笑，带着一丝娇羞，进了隔壁的房间。
<br/>
<br/>        两个人虽然是分房睡，但是两张房门都是敞开的。
<br/>
<br/>        傅昼景未关房门，童筱颖索性也不关房门。
<br/>
<br/>        她觉得两人的距离，仅仅只是隔着一堵墙而已。
<br/>
<br/>        甚至，在夜深人静的时候，她光听着他那幽微的呼吸声，就觉得自己很幸福。
<br/>
<br/>        而傅昼景思绪慌乱，侧身躺在床上，单手枕着脑袋，一双点漆的眸子在昏暗的室内晃动着微弱的眸光。
<br/>
<br/>        他不得不承认的是，今晚在电视上看到季思妍的善举之后，他平静的心田里忽然泛起了一波涟漪。
<br/>
<br/>        然而，这波涟漪却恰逢其时地再次出现在了他的面前。

<br/>
<br/>        当季思妍扎着清爽的卷马尾辫，梳着秀丽的斜刘海，妆容精致，唇红齿白，穿着一袭黑白西装包臀裙，肉色丝袜，脚踏黑色高跟鞋，明艳动人地出现在办公室门口的时候，傅昼景无意间地抬眸一瞥，看得身心一怔。

<br/>        一个介于稚气少女与成熟女人之间的独特风韵，季思妍把自己的美，向在场的男人展现得淋淋尽致。

<br/>        就连傅昼景的助理赵斌，在看到季思妍后，水杯里的水被倒满了，他都未察觉。
<br/>
<br/>        直到开水烫到他的手，他的目光和心绪才从季思妍的身上拉回来。
<br/>
<br/>        赵斌被开水烫到手的动静，同时也拉回了傅昼景的神魂，余光里，只见赵斌慌忙抽了数张纸巾，擦拭着洒落在茶几上的开水。
