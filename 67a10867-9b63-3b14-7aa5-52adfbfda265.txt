　　照片上的自己，和芮冷玉都穿着汉服，并肩坐在一间模拟古代风格的房间里，对着镜头开心地笑着。
<br/>　　说好了空界那件事办完，就结婚的，连日期都快定好了……没想到，结局，却是那么的残酷。
<br/>　　“我总感觉，我们的婚事会有意外，好像会出什么事似的。”
<br/>　　芮冷玉当初说的这句话，自动浮出脑海。
<br/>　　叶少阳还记得自己当时安慰她，如今想来，这真的是现实的写照。
<br/>　　一语成谶。只有一切成为定局，再回想过去的时候，才会有这么恍如隔世的感觉。
<br/>　　不光是芮冷玉，还有小九，瓜瓜，雨晴，静茹……明明之前还在一起，如今却天各一方，不，连天各一方都算不上，而是两个世界……
<br/>　　巨大的痛苦攫取了叶少阳的心，他躺在床上，感觉自己难受的快要死了。
<br/>　　在床上挺尸了好几个小时，叶少阳最终还是平静了下来，内心更加坚定了自己的想法：自己一定要回到属于自己的世界！去就回冷玉，救回小九！
<br/>　　那个世界，还有很多自己该做的事，等着自己去做！
<br/>　　公元二〇一六年，石城，叶少阳的家里。
<br/>　　叶少阳家的房子很大，这个时候却坐满了人：谢雨晴、周静茹、叶小萌、张小蕊……还有所有捉鬼联盟的成员，他们都被道风从山河社稷图中找了出来。
<br/>　　风之谷的道风，杨宫梓，小马。
<br/>　　阴司的徐文长。
<br/>　　所有人都沉默无语，周静茹在低声啜泣。谢雨晴没哭，但是双眼红肿，显然之前也没少哭。
<br/>　　房间里死气沉沉，充斥着一种悲伤的气氛。
<br/>　　“没想到军师是个叛徒！”小马气的一拳砸在桌子上，但他忘了自己现在是鬼，这一拳砸空了，从桌面直接穿透下去。
<br/>　　瓜瓜道：“军师不可能是叛徒，这一点我好不怀疑。”
<br/>　　“那为什么他把小叶子带走了？”小马反驳，“为什么捉鬼联盟只有他不见了？如果真的不是阴谋，为什么事前不跟大家说清楚？”
<br/>　　林三生放出徐福，然后徐福用山海印收走了叶少阳一干人，这是道风亲眼所见，说出来之后，立刻引起了大伙对林三生的怀疑。
<br/>　　李琳琳怒道：“三生不可能是叛徒，你怎么可以怀疑他，他提前没有说，肯定是有自己的苦衷！不然你说他为什么要这么做？”
<br/>　　小马怒哼一声道：“军师有自己的打算，我跟你们说，你们还不信，他一定是让徐福送他去明朝，找他的婉儿去了，那个凤兮也是，一定是去找他的情郎了，别忘了，他的情郎就是因为徐福才失踪的！”
<br/>　　“那他们也不可能出卖少阳，别说三生不会，凤兮姑娘也不会的！”李琳琳坚持说道。
<br/>　　雪琪说道：“军师肯定是为了这件事跟徐福结成的同盟，不过要说出卖少阳，我觉得不可能。”
<br/>　　小白道：“我也相信军师和凤兮，我们捉鬼联盟，不可能出叛徒！”
<br/>　　小马摆手道：“行了行了，我也就是这么一说，我说气话，你们别跟我叫真！”
<br/>　　谢雨晴吼道：“你们吵够了没有，这些都不重要好吗，我现在就想知道，少阳去哪了啊，谁能告诉我！他什么时候能回来！”
<br/>　　大家默然无语。
<br/>　　经过之前的讨论，现在能够确定的是，徐福一定是用山海印将叶少阳带去了别的空间，至于是哪个时代，就没有人知道了，至于他什么时候能回来……更是没人知道。
<br/>　　杨宫梓道：“我觉得，我们现在应该确定一下，究竟失踪了几个人。”
<br/>　　小青说道：“除了老大，还有宝爷，军师，凤兮，还有广宗天师，对了，吴嘉伟那小子也被吸进去了……别的还能有谁？”
<br/>　　一直沉默的道风说道：“我最担心的是，有一个人也跟着去了。”
<br/>　　杨宫梓道：“李浩然？”
<br/>　　道风颔首默认。“李浩然失踪了。极有可能是在徐福师作法的时候靠近，被一起吸了进去。我很担心，他会对少阳不利。”
<br/>　　众人面面相觑。
<br/>　　老郭皱眉说道：“李浩然对小师弟，好像没什么深仇大恨吧？”
<br/>　　道风道：“我跟少阳一起灭了悬空观在人间的山门，逼死他师父，如果换成是他对我茅山这么做，你猜我会怎么样？”
<br/>　　众人默然。
<br/>　　杨宫梓忧心忡忡地说道：“少阳不是李浩然的对手……”
<br/>　　这是不容置疑的事实，李浩然虽然在空界声明不显，但这次空界一战，李浩然所表现出的实力，堪称恐怖，甚至还要压过黎山老母一头。
<br/>　　杨宫梓道：“就我们见过的空界强者来说，李浩然应该是第一强者了吧。”
<br/>　　道风道：“不一定能压过摇光仙子，但也在伯仲之间。”
<br/>　　摇光仙子在空界之战中大放异彩，只有道风知道，她并没有全力发挥，倒不是有所保留，而是她一直在顾全大局，照应着整个战场，并没有真的跟人殊死缠斗。
<br/>　　大伙听了这话，心里更加紧张。周静茹趴在谢雨晴身上哭了起来。
<br/>　　“我觉得，李浩然想杀少阳，没这么容易！”老郭喃喃说道，“我知道李浩然很强，少阳不是对手，但是你们想想，少阳哪一次遇到的对手，不是比他强大的多，修罗鬼母、化蛇、女魃……就算没打赢，至少也都全身而退了。我相信少阳，就算遇到李浩然，也一定跑的掉。”
<br/>　　小白撅着嘴巴说道：“可是以前老大有我们大家帮忙，现在他可是只有一个人了。”
<br/>　　小青说道：“也不是一个人，不是还有宝爷和吴嘉伟他们，总之，我也相信老大能够安全回来。”
<br/>　　大家互相鼓励和安慰，逐渐坚定了信念，心里头好受多来。
<br/>　　美华说道：“那我们现在能做什么？”
<br/>　　一句话，问得大家哑口无言。
<br/>　　杨宫梓面对徐文长道：“徐公有什么见解？”
<br/>　　徐文长长叹一口气，道：“我还能有什么好说的，我这次来，也是带给你们一个消息，阴阳司全体成员，戴罪立功，前往空界驱逐尸族。”
<br/>　　小青说道：“不是我说，阴司这是想让我们当炮灰吗？”
<br/>(本章完)
