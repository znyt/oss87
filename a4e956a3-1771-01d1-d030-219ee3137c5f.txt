　　“苏莫胜！”
<br/>　　褐袍长老的声音响起，宣告着苏莫的胜利。
<br/>　　苏莫轻笑一声，随即转身飞回了看台，显得干净利落，潇洒无比。
<br/>　　不少人暗暗唏嘘，苏莫连败两大妖孽，这是真的在武帝境无敌了吗？
<br/>　　武帝境八重修为，拥有这等战斗力，整个明虚星河，好像都没有吧！
<br/>　　妖孽，这是一个十足的妖孽！
<br/>　　真不知晓道宗，是在哪里招揽的这么逆天的弟子！
<br/>　　阴泽长老暗暗点头，苏莫如此天赋，将来势必要被吸收进太阴族之中，不过，还要看其日后的表现，以及对太阴族的忠诚度。
<br/>　　“悠悠，这人真是厉害，是真正的妖孽！”灵萱望着苏莫的背影，眸中露出惊叹之色。
<br/>　　“的确是妖孽！”云悠悠眸中同样露出惊叹之色，就算是太阴族内族之中，天赋战力能达到这个地步的人，都是屈指可数，堪称真正的妖孽。
<br/>　　战败的扬天成，脸色涨成了猪肝色，只感觉如芒在背，灰头土脸的返回了看台。
<br/>　　苏莫返回看台之后，立刻得到了真枯半圣的赞许。
<br/>　　真枯半圣老怀大慰，苏莫出尽了风头，他们道宗也跟着颜面大涨。
<br/>　　冯光长老更是面色激动，仿佛比自己胜了还要开心。
<br/>　　有人欢喜有人忧，其它六大势力的长老、天才，均是面色凝重不已，出现了苏莫这样的妖孽，会武第一的位置就悬了。
<br/>　　“剑豪，有没有把握？”
<br/>　　御剑谷阵营，一名长老向身边的卓剑豪问道，虽然现在对方和苏莫还没有对上，但想要得到第一，最后无可避免的会对上。
<br/>　　“五成把握吧？”卓剑豪叹了口气，他心中也没有绝对的把握，不过，考虑到自己强大无敌的本命剑阵，他也还算有些底气。
<br/>　　“五成把握！”长老点了点头，五成把握的话，那还有很大的机会。
<br/>　　另一边，黑炎族的炎澜天，闪灵族的灵枫，雷云禽族的雷云皓，三大妖孽人物，同样是脸色凝重。
<br/>　　苏莫的强大，给予了他们巨大的压力，让他们心中沉重异常。
<br/>　　不过，他们虽然面色沉重，但也没有气馁，不亲自战斗一场，谁胜谁负还尚未可知。
<br/>　　会武继续，挑战继续。
<br/>　　苏莫坐在看台上，静静的观看比斗，他能感觉到无数的目光，纷纷杂杂的落在自己的身上。
<br/>　　忽然，苏莫神情一动，感受到了一道熟悉的目光，抬头完全，正是云悠悠在望着他。
<br/>　　苏莫微微一笑，向对方点了点头。
<br/>　　云悠悠见此，抿了抿红/.唇，便也轻点螓首，随后便转过了目光。
<br/>　　苏莫见此，目露沉吟之色，自己今日的表现，算是引起了青璇的注意，以后也好接触对方了。
<br/>　　时间流逝，战斗无止境，一场接着一场。
<br/>　　约莫又过去了一日之后，第三轮的战斗终于结束了。
<br/>　　三轮一过，只要是连败三场的人，全部被淘汰出局。
<br/>　　道宗弟子，除了苏莫和吕罡、林皇、唐古三大天才之外，居然被淘汰了九成之多，只剩下了二十余人。
<br/>　　其它势力，哪一方都比道宗的人多，均是有五、六十人左右。
<br/>　　所有被淘汰出局的人，全部离开了南方看台，分别落座到了东西两方的看台上，以做区分。
<br/>　　接下来的比斗，规则不变，但若是败了，败绩会再行叠加。
<br/>　　比如，之前败了两场，还没有遭到淘汰的人，只要再败一场，就会被淘汰。
<br/>　　所以，接下来的比斗，每一轮都会有人被淘汰。
<br/>　　“第四轮开始！”
<br/>　　褐袍长老的声音响彻而起，其话音一落，便见一位青衣声音，闪电般的落在了战台上。
<br/>　　众人见此，神情顿时一凝，因为这第一个上台的人，居然又是苏莫。
<br/>　　众人不由的好奇了起来，这苏莫迫不及待的上场，又是要挑战谁？难道还要挑战其它几大势力的妖孽？
<br/>　　果不其然，很快苏莫便给出了答案。
<br/>　　“炎澜天，就你了！”苏莫目光直视黑炎族的炎澜天，朗声说道。
<br/>　　众人闻言一惊，纷纷心中唏嘘，这苏莫每一次挑战的对象，均是妖孽天才啊！
<br/>　　先是击败了幽狼，又挑战扬天成，现在居然又要挑战炎澜天了。
<br/>　　这是要把六大势力的天才，全部都挑战一遍的节奏啊！
<br/>　　“挑战我？”炎澜天眉头一皱，他没想到苏莫这么快就要挑战他。
<br/>　　“不错，不仅是你，其他几位妖孽也会轮着来！”苏莫点了点头说道。
<br/>　　“是吗？”炎澜天眉头一皱，面色略显难看，苏莫的这种姿态，仿佛完全不将他们几大妖孽放在眼里。
<br/>　　“不知所谓！”
<br/>　　“真是自大啊！”
<br/>　　“苏莫，你真是太狂妄了！”
<br/>　　卓剑豪等几位妖孽，听闻苏莫之言，全部怒喝出声。
<br/>　　因为苏莫的姿态，虽然不显得嚣张，但那无敌的自信，以及眼眸之中的淡然，无不表明了其对于他们的藐视。
<br/>　　苏莫，完全不将他们放在眼里！
<br/>　　所有观战之人亦是讶然，这个苏莫是真的霸气啊，居然直接扬言挑战所有势力的妖孽！

<br/>　　苏莫闻言冷冷一笑，随即轻轻的点了点头，道：“既然你们认为我狂妄无知，那我就狂妄无知了，现在，我要挑战你们四人，一起挑战！”
