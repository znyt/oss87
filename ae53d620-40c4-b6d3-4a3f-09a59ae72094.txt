　　　　“喂，怎么是你”
<br/>　　刚带上耳机，突然一阵好闻的幽香从后面袭来，接着眼前多了一位穿着牛仔短裤，露着两条雪白长腿的俏美少女。
<br/>　　只见少女含着略微激动的笑，一屁股坐在了涂小安旁边的空位子上。
<br/>　　涂小安顺眼一看，顿时头大，冤家路窄呀。
<br/>　　“喂，小哥哥，你不会不认识我了吧”
<br/>　　李清照跟对方打招呼，见对方好像没什么反应。
<br/>　　涂小安无奈皱了皱眉，：“我们不过一面之缘，说不上认识”
<br/>　　这丫头怎么阴魂不散啊，自己坐大巴车去温城，都能在一辆车上撞见她。
<br/>　　之前做个任务，这丫头出现，蛇王节逛庙会，这丫头也出现，现在敢情好了，自己好不容易进城了，还能遇见她。
<br/>　　这是说明这丫头跟自己缘分不浅？
<br/>　　是孽缘吧？
<br/>　　“嘻嘻，一面之缘也是认识呀”李清照自来熟，眼睛弯成月牙儿。
<br/>　　之前蛇王节，他就对这个清秀而透着邪魅的少年印象深刻，尤其是斗蛇，他拿出来的迷你小蛇，简直让她爱的不行。
<br/>　　李清照发誓，哪怕让她拿出全部身价，她都想买对方手中的那条蛇。
<br/>　　目前为止，李清照虽然一直做着蛇类手机户外直播，给观众看各式各样的毒蛇。
<br/>　　但她自己却连一条蛇宠物都没有。
<br/>　　按照这丫头的话，我要的皮皮蛇，一定要非常非常喜欢的，绝不能将就。
<br/>　　她差不多是已按照男友的严格要求找的。
<br/>　　这不，到现在，她自己连一条皮皮蛇都没有。
<br/>　　自从李清照见过涂小安的迷你小蛇，她就惦记上了，可恨之后就再也没遇见涂小安。
<br/>　　苍天有眼，她大学暑假结束，准备去温城上学，坐大巴车，被她给撞见了。
<br/>　　这是蛇王在保佑她啊。
<br/>　　“小哥哥，你也去温城吗”
<br/>　　李清照开始套近乎。
<br/>　　“嗯”
<br/>　　“那你是去上学，还是去游玩”
<br/>　　涂小安微微一楞，上学这个词汇对他还真的有点陌生，其实按照他这个年龄，也是上大学的。
<br/>　　只是涂小安在上完高中，就一心扑在了捕蛇事业上去。
<br/>　　当初他认为，既然自己决心当一位出色的捕蛇人，那么读大学好像完全没有意义，简直在浪费时间。
<br/>　　之后他就不幸身亡，沦为家蛇。
<br/>　　李清照忽闪忽闪眼眸的等待他回答，涂小安只能无奈道：“不知道”
<br/>　　这是老实话，他就想去温城闯荡一下，具体可一点计划没有。
<br/>　　“小哥哥，你能将上回那条可爱的小蛇蛇再拿出来给我看看嘛”
<br/>　　李清照跟涂小安一阵没话找话，最后憋不住的开口。
<br/>　　这才是她真实的目的。
<br/>　　“你就不怕被蛇咬吗，之前斗蛇，你应该见过那蛇的恐怖”涂小安淡淡的开口。
<br/>　　一条三米的眼镜王蛇，被迷你小蛇来了一个透心凉，变成干尸，那一幕李清照可是历历在目。
<br/>　　“不怕，我就想看看而已”
<br/>　　对于李清照这种爱蛇如命的人来说，什么事情都能豁的出去。
<br/>　　涂小安话题一变：“你是去温城上学吗”
<br/>　　“对啊，我上的可是温城最好的大学，里面美女多多哦”李清照冲着涂小安挤眉弄眼的套近乎。
<br/>　　就差说，你把小蛇拿出来，我给你介绍我们学校的美女。
<br/>　　涂小安思忖一下：“到了温城，你带我去你学校看看，我就答应你”
<br/>　　没上过大学，起码看看大学的面貌。
<br/>　　反正涂小安到温城也没落脚的地方。
<br/>　　说不定到了大学，被里面的气氛感染，他想上个学也不一定。
<br/>　　李清照眼眸一亮：“一言为定”
<br/>　　“快点，将我可爱的小蛇蛇拿出来”
<br/>　　李清照都迫不及待了，典型的爱蛇如命，跟她清纯的形象完全背道而驰。
<br/>　　她这种花季年龄的美少女，看到蛇的正常反应是吓的尖叫，脸色发白才对。
<br/>　　“不就在你的大腿上了”
<br/>　　李清照今天穿着牛仔短裤，露着两条雪白长腿，此时，雪白的大腿上，有一条红白相间的小蛇趴在上面，微微的蠕动。
<br/>　　“哇，好清凉”
<br/>　　李清照看到心仪已久的小蛇趴在自己的大腿上，不但不怕，还赞叹了一声。
<br/>　　她的大腿一阵冰凉传来，非常非常的舒服。
<br/>　　“好可爱，好卡哇伊，爱死你了”
<br/>　　李清照一点都不怕被咬，双手将太阴蛇给捧了起来，爱不释手。
<br/>　　太阴蛇灵动的蛇眸泛着智慧的光，直接蛇尾直立，如筷子一样的立在李清照的手掌心中。
<br/>　　惹的李清照时不时的发出花痴般的尖叫。
<br/>　　可她时不时的大呼小叫，惹的车上一些人不满，但很多大老爷们看到她青春无敌的颜值，也就没去计较什么。
<br/>　　美女在任何时刻都是拥有特权的。
<br/>　　尤其是这种泛着满满青春气息的美少女。
<br/>　　“小哥哥，这是条什么品种的蛇呀”李清照一双眼眸烙印在太阴蛇的身上，然后发问。
<br/>　　涂小安翻了翻白眼：“太阴蛇”
<br/>　　“太阴蛇？”李清照想了想，貌似从没听过，只是眼神满是贪婪的说：“小哥哥，能不能商量一下，这条蛇你转卖给我怎么样”
<br/>　　“你放心，我会对它好的”李清照信誓旦旦的补偿了一句。
<br/>　　涂小安一头黑线，干脆懒得回答他。
<br/>　　“小哥哥，别这样呀，我们商量一下呀，价格随你开”
<br/>　　涂小安微微看了她一眼，这丫头的语气很大呀，看来还是一位小富婆。
<br/>　　李清照还真有这个底气，她现在可是人气主播，一个月下来，起码能赚几十万，而且这还是她很谦虚的说法。
<br/>　　如今，别看她还是一位大学生，但可是有几百万的身价。
<br/>　　有钱人。
<br/>　　按照蛇类品种来说，珍稀蛇种市场价百万以上，李清照决定拼了，为了自己心爱的小蛇蛇，她日子不过了。

<br/>　　“你买不起的”涂小安抬头看了她一眼。
<br/>　　李清照急了：“你先出价啊，你都没出价，怎么知道我买不起”
<br/>　　本来准备坐大巴车，涂小安是为了沿途看看风景，听听歌，过一过普通人的惬意生活，可现在全被这个丫头搅和了。
<br/>　　她是不到黄河心不死，不撞南墙不回头。
<br/>　　涂小安冲着她，伸出了一根手指。
<br/>　　李清照眨巴眨巴眼眸，这一根手指，应该不会是一万或十万，因为她已经知道，小蛇蛇是珍稀蛇种。
<br/>　　那么这一根手指，应该就是一百万。
<br/>　　旋即她的嘴角上扬美丽的弧度，顿时笑靥如花，花枝招展了起来。
