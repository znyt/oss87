
<br/>　　<br/>　　    年轻人都是躁动的，茅小草和王惊蛰自然也不例外，虽然他们一个口口声声说不感兴趣，一个说小女子要淡定，但在子时之前本来应该就早睡的两人还是钻出了家门，只不过两人都略微乔装打扮了下，避免让人给认出来。<br/>　　<br/>　　    良水城中的人都睡了，可良水河畔还在人头攒动，几条渔船从上游开始往下游划动，岸边也有人在来回的漫步着。<br/>　　<br/>　　    中元鬼门开，这扇门并不是一直都开着的，前后差不多从十一点到凌晨一点左右，只有这段时间是开启的。<br/>　　<br/>　　    也就是说，想要进入阴曹地府的人，需要在这两个小时之内，办好自己该办的事，然后赶在关门之前出来，否则你恐怕就得要一直留在阴间了，除非你能挺到下一次鬼节时鬼门再开然后出来，不然别无他法，而且寻常人是很难在阴间呆太久的。<br/>　　<br/>　　    阳间充满着阳气，阴间则到处都是阴气，寻常人沾上一些的话就受不住了，时间稍微久一点被阴气侵入体内就得会病上一场，如果长期身处阴曹地府恐怕谁都没办法习惯得了，一年的时间是万万挺不过去的。<br/>　　<br/>　　    就像阴差来阳间带人魂魄回阴间，牛头马面，黑白无常基本是立马带人就赶紧往回走，而不会停留过长的时间。<br/>　　<br/>　　    “我就说嘛，这个时候总归得会碰见点熟人，果不其然啊·······”王惊蛰和茅小草并肩走在河岸上，一艘渔船从他俩身旁顺着水流朝下方划过。<br/>　　<br/>　　    渔船上站着几个人，齐峥坤，常子敬和胡胡，三人都没有注意到他，站在渔船上交头接耳，眼睛瞄着前方。<br/>　　<br/>　　    “有资格打探到良水河连着阴曹地府的，有胆量敢过来的，也无非就是这些人了，比如正一教，龙虎山”茅小草随即就看见另外一艘渔船上，船头站着一根筋的秋成子，在他身旁还有一男一女两人。<br/>　　<br/>　　    那一对男女似乎地位都颇高，秋成子和他们讲话的时候始终都半低着脑袋没有抬头，那女人还算挺有礼貌，差不多对方每说几句话就含笑回应一句，至于那个男的，则一直都鼻孔朝上，眼神斜着飘了出去，并且非常风骚的是他居然还穿了一身白色的西服，在漆黑的夜里显得十分的扎眼。<br/>　　<br/>　　    “那女人好看么······”茅小草看了几眼之后就收回目光，然后转头就看见王惊蛰的眼睛也盯着那边呢。<br/>　　<br/>　　    “我这个人脸盲，说实话，根本分不清谁漂亮谁不漂亮”<br/>　　<br/>　　    茅小草瞬间无语了：“这话听着怎么有点耳熟呢？”<br/>　　<br/>　　    此时，时间正好已经到了午夜子时。<br/>　　<br/>　　    原本七月份里应该很炎热的夜晚，突然之间温度陡然下降了好几度，一股冷风徐徐吹来，渔船和岸边的人都不禁神情一紧。<br/>　　<br/>　　    秋成子那艘渔船上的几人拿出沾着牛眼泪的柳叶擦在了眼睛上，然后目光四处搜寻着。<br/>　　<br/>　　    “子敬，快些算出来”胡胡皱眉说道。<br/>　　<br/>　　    “好，等下就行了”常子敬应了一声，翻手拿出几枚铜钱卜了一卦。<br/>　　<br/>　　    这时候，突然之间有一道亡魂不知从何处飘了出来，眼珠子耷拉在眼眶下，七窍都渗着血迹，面目狰狞的荡在水面上。<br/>　　<br/>　　    秋成子那艘船上的人，顿时全都齐刷刷的望了过去，那厉鬼看了一眼后“嗷”的一声掉头就走，似乎知道自己碰上这伙人没有什么好处。<br/>　　<br/>　　    秋成子轻声说道：“没有看清是从哪出来的，但应该离我们不远了”<br/>　　<br/>　　    “不急，我们没看到，别人一样也没找到，玄门那帮人还在算来算去的，也没找到我们前面去，咱们别落后就行了，实在发现不了就干脆等着好了，前人栽树我们乘凉也可以”秋成子旁边，那女人忽然开口了，声音跟人一样，动静好听人也漂亮。<br/>　　<br/>　　    “小瑜，过会如果通往阴间的鬼门出现了，你就不要跟着过去，在船上等着就行了，女子本来阴气就重，你进了阴间的话比较麻饭”姜冰啸柔声说道。<br/>　　<br/>　　    陈小瑜摇头说道：“你不用担心我，我心里有数的”<br/>　　<br/>　　    姜冰啸顿时不吭声了，但却转头跟秋成子说道：“一会进入阴间，你全力护着点小瑜，我这边不用你管”<br/>　　<br/>　　    秋成子连忙点头说道：“明白了”<br/>　　<br/>　　    正在卜卦的常子敬忽然伸手斜着指向一个方向说道：“西南，坤二十二！”<br/>　　<br/>　　    “呢就说往哪划船就行了，说点呃能懂的可以么？”船家有点懵的问道。<br/>　　<br/>　　    常子敬干咳了一声说道：“往西南方向划过去，大概二十二米左右，然后船头朝向三十五度角”<br/>　　<br/>　　    船家顺着常子敬的指示调转船舵，渔船划出去后，离他们不远的几条船全都闻风而动的跟了过去，几分钟后当船到了地方时，就看见几道孤魂野鬼正蜂涌着冒了出来。<br/>　　<br/>　　    除了划船的船家看不见这些东西外，附近的人全都看到了，就知道鬼门差不多应该是在这一带了。<br/>　　<br/>　　    常子敬指的方向，已经是河岸对面了，那里有一处高耸的峭壁，上面长满了杂草和灌木还有荆棘，几乎布满了整片岩壁上。<br/>　　<br/>　　    在离地不到四五米的地方，有个黑黝黝的洞口，此时正往出飘荡着几条孤魂野鬼。<br/>　　<br/>　　    在阴曹地府，并不是所有的亡魂都被阴差所管辖的，总有一些漏网之鱼游离在外，他们就跟人间犯了罪的通缉犯一样，也在躲着黑白无常和牛头马面，这时候关了一年的鬼门突然打开，有知道这处鬼门的孤魂野鬼就全都从这里跑了出来，回到阳间他们至少不用担心在被阴差捉拿了，可能面对的不过是那些喊着除魔卫道的佛道两派罢了。<br/>　　<br/>　　    渔船划去的同时，岸边也有人再往那边走，茅小草和王惊蛰就上了一条横跨在良水河上的木桥。<br/>　　<br/>　　    “你老实讲，咱俩是只看热闹，还是也要跟着进去？”王惊蛰叹了口气问道。<br/>　　<br/>　　    “外面哪有什么热闹啊，热闹都在里面呢，当然进去了”茅小草说道。<br/>　　<br/>　　    王惊蛰心里略微一颤，看着身前小草姑娘的背影，忽然想到那天开车离开长安，两人真是凑巧落脚到了良水城么？
