　　第95章
<br/>　　黑石城一座义庄。
<br/>　　寒冬腊月的天气，很少有行人在街上走。一裹着皮袍的老者从远处过来，头上戴着厚重的狗皮帽子，他行到义庄前，左右看看，迅速拐了进去。
<br/>　　走进义庄后才把帽子摘下，拍拍身上的土，露出脸上猥琐的山羊胡。
<br/>　　“大当家，你的计策可真高呐。他们昨夜在城外搜了大半宿，毛都没找到一根，让他们想破脑袋都想不到，咱们又返回城里来啦。”
<br/>　　“还高呢！”程大雷靠在棺材板上：“陷进去啦，现在咱们还出得了城嘛！”
<br/>　　“这……城里城外都加了守军，几个城墙豁子也有人把守。”徐神机。
<br/>　　“别吹啦，你敢吹我都不敢接，丢人丢大发啦。出来抢劫，马却被人偷了，这件事要传回山寨，咱们的脸往哪儿搁，让山寨的鸡鸭鹅怎么看我，今后在山贼界怎么混！”
<br/>　　“这件事和鸡鸭鹅有什么关系？”徐神机。
<br/>　　“情绪需要。”程大雷圈起袖子，身子往墙上靠了靠。
<br/>　　这座义庄年久失修，停着十几口无主的棺材。西北风嗖嗖的往里面灌，天寒地冻，秦蛮、林少羽、刘悲、张肥、徐神机各自散坐在地上，脸上的表情也很落魄。
<br/>　　程大雷闭着眼睛检查系统界面，昨天劫了贾家，任务进度已经推进到3|10，自己也收割了一波恐惧值，目前拥有的恐惧值一万有余，距离十万目标还有很大差距。
<br/>　　“大当家，你估计会是谁偷了我们的马，有没有可能是高飞豹或者小白狼干的？”徐神机此时道。
<br/>　　“不是没有这个可能呐。”程大雷睁开眼睛，莫看他也和小白狼、高飞豹一起赌色子喝酒，可山贼之间的交情……世上有没有这种东西都不好说。真如果二人背后摆程大雷一道，也不是没有可能。
<br/>　　不管如今并无法确定，说不得是别人顺手牵羊。
<br/>　　“不管是谁，要让我逮住了，我和他不死不休！”程大雷恶狠狠道。
<br/>　　“不过几匹马而已，也不值得如此生气，平常心平常心。”刘悲看破红尘的声音响起。
<br/>　　“斧头，我的斧头！”程大雷大呼小叫：“我的斧头也被偷走啦！”
<br/>　　不仅程大雷的兵器，秦蛮和林少羽的兵器都被人顺走了。除此之外，还包括这次行动带来的干粮和酒，都被人偷得干干净净。如今程大雷几人已经一夜水米没打牙了。
<br/>　　“究竟是谁偷了老子的斧头！”程大雷咬牙切齿，脸上青筋暴起。
<br/>　　……
<br/>　　离城三十里，一座土丘。
<br/>　　十几个大汉散落在地上，身上裹着厚厚的风衣，中间一堆篝火，腊肉烤得正香，滋滋冒着油花。
<br/>　　几人坐在一起，饮酒吃肉，时不时响起洪亮的笑声。
<br/>　　“这酒可是真够劲儿，比咱们的马奶酒强多了。”
<br/>　　“要说这伙山贼可真够猛地，几号人就敢进城大闹。”
<br/>　　“不过他们绝对想不到，马被咱们顺走了，说不得他心里怎么恨我们呢。”
<br/>　　“哈哈哈哈！”
<br/>　　洪亮的笑声响起，树顶的雪粉扑索索落下。
<br/>　　“小王子，这斧头不错，以后就让我用吧！”
<br/>　　“野德，你小子胡说什么，这斧头是我先看上的。”
<br/>　　“赤虎，凭你也想跟我争！”
<br/>　　两个大汉从地上站起来，为一柄斧头怒气冲冲，当场在雪地上动起手来。扯开胸口的皮衣，露出强壮的胸膛，嘴里呼哧呼哧冒着热气。
<br/>　　周围一帮人叫好起哄，呼尔勒喝酒吃肉，笑看着这一幕，却也不阻拦。
<br/>　　“好啦，好啦。”等二人打累了，呼尔勒才挥挥手让他们散开：“你们想要这斧头，这斧头一百多斤重，你们二人使得动嘛？”
<br/>　　名叫赤虎和野德的两个大汉面面相觑，二人虽然都有力气，可真扛着一百斤重的兵器上阵，却也谁都不行。
<br/>　　呼尔勒哈哈大笑，从雪堆中站起来，一只手便将那大斧提了起来。挥了两下，感觉得心应手，看准身边一棵大树，一斧头便砍了上去。
<br/>　　咔嚓！
<br/>　　碗口粗细的树干应声而倒，呼啦啦雪粉扬起。
<br/>　　“好斧，好斧！以后这斧头就归我用了。”呼尔勒道：“该给它起个名字，就叫它……”
<br/>　　北风袭来，将地上的积雪卷起，铺天盖地而来。呼尔勒持斧站在旷野中，眼如饿狼，意如猛虎。
<br/>　　“就叫它风暴战斧吧！”
<br/>　　呼尔勒，戎族北蛮部王子。戎族不是一个国家，而是有上百个大小不一的部落组成，互相之间势力错综复杂。北蛮部算是极北地区最强大的部落之一，当然，北蛮王也不会只有一个儿子，几百个不好说，几十个却是有的。
<br/>　　可呼尔勒却是北蛮王最喜欢的小王子，他虽年轻，在部落中却极有威信。部落中的人常常用一句话来赞扬他。
<br/>　　『一头即将咆哮四方的寒地苍狼』
<br/>　　今日得到趁手的兵器，呼尔勒十分开心，他看了看野德和赤虎二人，道：“你们两个也别不开心，不是还有两杆枪么，以后就由你们用。”
<br/>　　“好！”二人齐声大笑。
<br/>　　“小王子，那我们呢……”
<br/>　　“赤虎比我差多了，枪凭什么给他！”
<br/>　　“小子，你不服咱们来打一场。”
<br/>　　“吵什么吵！”呼尔勒伸手一指：“那里是一个花花世界，美酒、茶叶、兵器、丝绸……娇滴滴能掐出水来的女人，有本事能抢过来都是你们的。”
<br/>　　呼尔勒伸手所指，是黑石城方向。一伙人往那个方向看过去，眼中露出炽热的目光。
<br/>　　“小王子，咱们什么时候动手！”
<br/>　　“是啊，兄弟们可都已等不及了，早就想去城里快活快活！”
<br/>　　“只等小王子一句话，我们兄弟杀进城去，抢人，抢粮，抢钱。”
<br/>　　“不用着急，凭咱们几个人可闹不了一座黑石城。”
<br/>　　“守军那些酒囊饭桶，咱们兄弟可不怕他。”
<br/>　　“哈，你们不用着急，部落的一百勇士正在路上，不时就能到。”呼尔勒道：“咱们还要先混进城中，等咱们的人到了，里应外合打开城门，好好在黑石城抢一番。”
<br/>　　
<br/>(本章完)
