        这暗恋的滋味并不好受，结果，还来了个女人给她添堵。
<br/>
<br/>        落以柔吸了吸鼻子，目光无意间落在了傅夜沉的那件挂在落地衣架的西装外套上。
<br/>
<br/>        灵机一动，她疾步离开了总裁办，不一会儿，再回到这里的时候，她将手里的东西悄悄地塞到了傅夜沉的那件西装外套的内兜里。
<br/>
<br/>        离开傅氏集团大厦后，童筱颖还去帮夏凌焰赶了一下午的设计稿，但在正常的下班时间点，她便早早地回到了傅夜沉的私宅里，和刘婶一起准备晚餐。
<br/>
<br/>        “今天连医生也在家，菜里不要加胡椒粉。”刘婶嘱咐道。
<br/>
<br/>        童筱颖点头应了声“好”，将锅里的菜盛入了盘子里：“连医生还住在这里？”
<br/>
<br/>        她住进来的这几天，好像没见着连华生。
<br/>
<br/>        刘婶笑着说道：“现在住得不多了，不过也会隔三差五地来住一下，陪陪大少爷。”
<br/>
<br/>        “他俩感情真好。”童筱颖感叹，再想想她自己，自从跟着傅昼景去了澳洲后，她就跟以前的那些所认识的朋友们失去了联系。
<br/>
<br/>        在澳洲的那三年，她的世界越来越小，小到只能容下她和小卫。
<br/>
<br/>        童筱颖将菜端上餐桌的时候，连华生正慵懒地从楼上下来。
<br/>
<br/>        他点漆般的黑眸，向她投来冷淡的目光。
<br/>
<br/>        童筱颖微笑着问候：“连医生，好久不见。”
<br/>
<br/>        连华生瞥了童筱颖一眼，淡淡地说道：“可否借一步说话？”
<br/>
<br/>        “嗯！”童筱颖点了点头，解开身上的围裙，随手搭在了椅背上。
<br/>
<br/>        两人一前一后，去了私宅的后花园。
<br/>
<br/>        连华生忽然转过身去，直接开门见山地说道：“你还是离开阿沉吧？你不适合阿沉。”
<br/>
<br/>        关于她童筱颖跟傅夜沉领了结婚证的这事，他一回来，就被刘婶拉着说个不停。
<br/>
<br/>        刘婶倒是高兴不已地把事情的来龙去脉叙述得一清二楚，但是连华生听着却很不高兴。
<br/>
<br/>        作为傅夜沉的好兄弟，这么大的事情，傅夜沉居然对他只字不提。
<br/>
<br/>        若是三年前，他或许还会支持童筱颖和傅夜沉在一起。
<br/>
<br/>        但是，现在，他觉得，童筱颖已经配不上傅夜沉了。
<br/>
<br/>        倘若他没记错的话，童筱颖和傅昼景已经有了孩子了吧！
<br/>
<br/>        就算没有孩子，童筱颖跟过傅昼景，现在又来跟傅夜沉，连华生怎么看她童筱颖，都看着不太顺眼，总觉得这女人身心都不干净。
<br/>
<br/>        连华生知道，傅夜沉为了她童筱颖，在感情上堕落了三年之久，她这次跟傅昼景离婚，傅夜沉一定会失去理智的强娶她。
<br/>
<br/>        娶一个心里还藏着其他男人的女人放身边，对傅夜沉来说没什么，但在连华生看来却是一道能给傅夜沉致命的伤害。
<br/>
<br/>        “我给你看样东西。”见童筱颖不吭声，连华生从裤兜里掏出手机，点开了手机里的相册，递给了童筱颖。
<br/>
<br/>        童筱颖怔愣地接过手机，垂眸看着屏幕上的相片。
<br/>
<br/>        一张相片是在傅夜沉的总裁办里，落以柔扑在了傅夜沉的身上，两人四目相视，面对面的距离近到鼻尖看起来像是碰在了一起。
<br/>
<br/>        还有一张相片是在医院里，傅夜沉躺在病床上，一只手搂着半个身子匍匐在他怀里的落以柔。
<br/>
<br/>        照片所显示的日期，是在他和她结婚之前。
<br/>
<br/>        童筱颖看完后，从容不迫地将手机还给了连华生。
<br/>
<br/>        连华生接过自己的手机收好后，接着问道：“这下，你能明白我的意思了吧？”
<br/>
<br/>        “你偷拍的？”童筱颖抬眸看着连华生，淡定地反问。
<br/>
<br/>        “很明显，是。”连华生勾了勾唇。
<br/>
<br/>        “他之前的那个未婚妻突然悔婚，也跟相片里的这个女孩有关？”童筱颖又问道。
<br/>
<br/>        连华生愣了一下，默不吭声。
<br/>
<br/>        他沉默，就代表默认了吧！
<br/>
<br/>        童筱颖却冷冷一笑：“谢谢你，给了我一个答案。”
<br/>
<br/>        原本，她还想不通，傅夜沉和落以柔身上那共同的薄荷香气是怎么来的，现在，她总算是明白了。
<br/>
<br/>        今天，她一定是疯了，才会向他打开那扇心灵的窗户。
<br/>
<br/>        才被她打开的那扇窗，在此刻，她又毫不犹豫地给关上了。
<br/>
<br/>        童筱颖啊童筱颖，你什么时候变得这么物质呢？
<br/>
<br/>        一个毛绒公仔、一束蓝玫瑰、一张卡片、一辆价值上亿的跑车、一个吻、还有他当众抱着她宣誓她总裁夫人的特权……
<br/>
<br/>        他今天对她的百般宠溺与甜言蜜语，还有“青蛙与天鹅”，原来都只不过是他的逢场作戏罢了！
<br/>
<br/>        或许，苏漫雪离开他后，这个落以柔又闯入了他的生活中，与他紧紧地交织在了一起。
