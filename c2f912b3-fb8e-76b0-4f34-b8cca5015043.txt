<br/>　　人王宫之人，均是有些紧张，不清楚姬月松能否战胜天尘山。
<br/>　　毕竟，天尘山成名已久，名声比姬月松要响亮的多。

<br/>　　平台之上。
<br/>　　姬月松看自己一击无功，并没有丝毫的意外，天尘山的强大，他心中有数。
<br/>　　适才一击，只是试探而已。
<br/>　　“天尘山，拿出你的真实战力吧！”
<br/>　　姬月松朗声大喝，随即，他的身上，迸射出夺目的金色光芒。

<br/>　　“是灭世金乌元神！”
<br/>　　“姬月松的血脉之力，也是无比的强大，听说是烈日圣体！”
<br/>　　“姬月松修至阳之力，配合上灭世金乌元神和烈日圣体，可谓是无比的契合，这也造就了他强大的战斗力！”

<br/>　　灭世金乌元神，在太古武魂榜，排名第十五名。
<br/>　　当然，这不是最主要的，最主要的是，姬月松还拥有烈日圣体。

<br/>　　两者同在一人之身，可见其威力之大。
<br/>　　而且，听说姬月松靠着灭世金乌和烈日圣体，将自己的肉身凝练成一种特殊的至阳战体。
<br/>　　骆妤、褚彦、沁儿三人，倒是没有什么意外，他们早就知道姬月松的底细。
<br/>　　“月松哥才是真正的强大，那段惊天还远远不如！”褚彦阴沉着脸，向骆妤和沁儿说道。
<br/>　　沁儿和骆妤，均是并未出声，她们心中清楚，褚彦只不过是嘴上不承认而已。
<br/>　　段惊天的实力，虽然不太可能比的上姬月松，但是，越级战斗能力，或者说天赋要大大超越了姬月松。
<br/>　　毕竟，段惊天才一品高级的修为。
<br/>　　“灭世金乌！”苏莫低声自喃，这姬月松的元神，果然不简单。
<br/>　　要知道，灭世金乌武魂，太古武魂榜排名第十五，只比万化神剑武魂低一名。
<br/>　　不过，虽然低了一名，但是灭世金乌元神，配合上烈阳圣体和至阳源力，威力会非常的大。
<br/>　　只见姬月松再次出手了，双掌之间，凝聚出一个巨大的光团，如同是一个金色的小太阳。
<br/>　　小太阳之中，散发出毁天灭世的气息，骇人无比。
<br/>　　“败吧！”
<br/>　　姬月松怒喝一声，双掌一推，金色的小太阳便如同一颗炮弹一般，向天尘山袭杀而去。
<br/>　　金色的小太阳，速度快到极致，焚化一切，瞬间便临近了天尘山。
<br/>　　不过，天尘山面不改色，甚至眸中却是露出了一丝讥讽之色。
<br/>　　“想击败我你还不够资格！”
<br/>　　天尘山冷喝一声，体内陡然传出了阵阵龙吟之声。
<br/>　　霎时之间，他的周身群龙乱舞，共有九条神龙闪现而出。
<br/>　　这九条神龙，颜色各不相同，有金龙、有黑龙、还有血龙等等。
<br/>　　九条庞大的神龙，散发出滔天的凶煞之气，更有吞天噬地之威。
<br/>　　轰隆隆！
<br/>　　九条神龙离体，铺天盖地的向金色的太阳扑去，瞬息之间，就将金色的太阳撕扯成了碎片，并且吞食一空。
<br/>　　“什么？”姬月松见此，顿时面色一变，他知晓天尘山拥有九龙弑天元神，但是并没有想到会如此的强大。
<br/>　　轰隆隆！！
<br/>　　九条神龙，吞食了金色的小太阳之后，飞速的向姬月松扑了过去，气势如虹，仿佛也要将姬月松吞食一般。
<br/>　　“去！”
<br/>　　姬月松身形飞退，身上笼罩的巨大金乌，同样离体而出，向九条神龙飞去。
<br/>　　他灭世金乌的排名，还在九龙弑天之上，自然不惧。
<br/>　　巨大的灭世金乌，双翼一扇，如同烈日划过长空，裹挟焚天灭世的气息。
<br/>　　九条神龙气息更加的狂暴，不仅凶煞无比，更是蕴含吞天噬地的气息。
<br/>　　“这股气息？”苏莫眉头一皱，他发现，九条神龙的身上，居然蕴含吞噬的气息。
<br/>　　只不过，这种吞噬的气息，与他的吞噬元神的吞噬气息，并不相同。
<br/>　　轰隆隆！！
<br/>　　一阵今天动地的爆响，猛烈的炸开，灭世金乌和九条神龙，展开了厮杀。
<br/>　　这是巅峰的对决，不仅是元神的比拼，更是巅峰战力的体现。
<br/>　　因为，元神和武者体内的圣丹、气血等等均是彼此相连，元神也能爆发出武者巅峰的战斗力。
<br/>　　轰轰轰！
<br/>　　虚空崩塌，劲力炸泻，灭世金乌和九条神龙，疯狂的交锋，如同凶猛的巨兽。
<br/>　　不过，寡不敌众，灭世金乌虽然强大，在九条神龙的围攻之下，很快便节节败退。
<br/>　　随着金乌落入下风，姬月松的脸色，则是越来越苍白。
<br/>　　九条神龙，如同洪荒猛兽，虽然每一条都略弱与金乌，但是联合起来，简直凶猛无比。
<br/>　　嘎！
<br/>　　少倾之后，一声刺耳的尖叫声响起，在九条神龙的攻击之下，金乌受创，身躯幻灭不定，仿佛是要崩溃一般。
<br/>　　噗！
<br/>　　一口鲜血，从姬月松的口中喷出，其脸色瞬间惨白。
<br/>　　嗖！
<br/>　　金乌舍弃了九条神龙，闪电般的回归了姬月松体内。

<br/>　　他认输了，不得不认输。
<br/>　　众人震撼，姬月松这么强的实力，依旧败了，此次第一名，估计飞天尘山莫属了，其他人虽然也强大，但是，很难与天尘山抗衡。

<br/>　　“无敌！百岁之内无敌！”

<br/>　　嗖！
<br/>　　苏莫身形一闪，在天尘山离开的同时，他飞落在了平台之上。
