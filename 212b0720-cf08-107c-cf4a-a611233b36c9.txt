<br/>				第二天，陈默走出修炼室后，手中拿着两个自己做的玉石药瓶。作为一个雕刻师来说，制作的药瓶很漂亮，还用木塞塞着，整体上雕刻花草等图像，让人看到后就感觉非常精致。<br/>　　<br/>　　一个晚上的劳作，陈默炼制了很多药，不过不是丹药，基本上都是散，比如止血散、金石散等等，都是一些应急药物。这些因为不是成丹，所以失败率反而没有那么高，倒是让他炼制了不少。<br/>　　<br/>　　这些药炼制好以后都收到了乾坤珠内。此时手里拿着的玉石瓶，一个是吴老的药，一个是何老的药。<br/>　　<br/>　　吴老的药加入了一些疏通血栓的中药材，而何老的药主要是清肺利咽，降压等中药，所有药物中的中药都是出自乾坤珠内，药效可以说是杠杠的。<br/>　　<br/>　　炼制成散以后，一点点的溪水混合后揉成药丸的。每个药瓶中也就十颗药丸，但是光是揉搓这些药丸，都让陈默感觉有些耗费时间，他已经在络上订购了一个手工制药丸机械，等到了以后，直接用机械制作药丸，绝对的方便。<br/>　　<br/>　　陈默走到院子里的时候，已经看到吴老和何老都起来，并且在院子里活动着身体，两个人都会太极，所以打的还挺好看的。主要是陈默不会太极，看到两个老人打太极，没有丝毫的内力，只能用挺好的来总结了。<br/>　　<br/>　　昨天一说定以后，何老和吴老就没有走，直接就住到了这里，至于外面的医护人员和保镖之类的，他也没有看到是怎么处理的，反正最后都消失了，不过陈默知道这些人肯定在村子里住下了，至于住在哪里，就和他没有啥关系。<br/>　　<br/>　　为了安排两个老人和两个陪护人员的伙食，陈默专门让自家的小舅过来做饭。小舅学过厨师，做些平常的饭菜还是没有问题的。<br/>　　<br/>　　小舅过来后，他就交代了一下，一定要用真材实料，能用自己地里产的就用自己的，没有再去买。<br/>　　<br/>　　从昨天晚上开始，两个老头就已经吃上了，而且有地里出产的青菜，所以特别受到欢迎，他们还没有吃过这么好的青菜，味道是在太棒了。<br/>　　<br/>　　至于看病的钱和住宿的钱，昨天晚上就打到了陈默的账户上，而且吴老一下子就打了两个月的钱，一副不差钱的样子，也让陈默重新理解了有钱人的生活，真的不在钱多钱少，而是能不能满足他们的需求。只要满足了需求，那么钱就不算什么。<br/>　　<br/>　　昨天晚上，两个老头分别睡觉后，就感觉一躺下就进入了睡眠。这一觉沉的，让保镖和医护人员都害怕了，一个晚上查看了好几次，也造成快到早上的时候，一夜没有睡觉的保镖和医护人员迅速的轮替。没有办法，一个晚上的煎熬，特别的困乏。<br/>　　<br/>　　两个老头可是一觉就睡到了大天亮，起床后，就感觉身体的轻松和精神焕发，简直和以往的情况有着质的区别。<br/>　　<br/>　　“房费没有白花啊！”都是一阵感叹，本来昨天还想着房费这么贵，虽然嘴上不说，但是心里还是颇有微词的，这么好的一个小伙子，怎么就钻进钱眼里去了呢。好在今天早上起来后，感觉还真是没有白花钱，顿时身心舒爽万分，一点点的芥蒂也就烟消云散了。<br/>　　<br/>　　这里这么好的情况，再体验个几天，要真是身体有很大的变化，也要推荐给自己的一些老伙计，让他们也来这里住几天，好好修养一下身体。<br/>　　<br/>　　院子里有十米宽的砖石区域，大概有个四十米的长度，一直通道院门口，所以两人在住宿的房间门口练习太极拳，非常的宽敞。今天早上替换后的一个医护一个保镖站在门口，看着两个老头在锻炼。<br/>　　<br/>　　陈默看了看，也就收回了眼睛，两个老头都在锻炼，也不好打断，所以就直接坐在院子里的躺椅上，先等等再说。<br/>　　<br/>　　昨天一晚上，又是绘制符箓，又是炼制药散，还有修炼等等，可以说时间上安排的满满的，直到陈默坐在这里的时候，才算是休憩片刻。深深地吸了一口气，感觉非常的舒服。院子里自从布置好聚灵阵以后，就形成了一个小环境，在这个环境中生长的植物，绝地是一天一个样子。<br/>　　<br/>　　等到快半个小时后，两个老头都缓缓收功。陈默就上前，拿出两个小药片，分别递给两个老头。药瓶虽然都都用玉石雕刻的，但是上面的花纹却不一样，另外瓶塞上还有粘贴的姓氏，也不会让人搞混了。<br/>　　<br/>　　“何老、吴老，这是你们二位的药，每隔一天，在清晨的时候，空腹温水吞服一颗药丸。这是一个月的量，不能少也不能多吃。”陈默交代道。<br/>　　<br/>　　“这瓶子不错，看着雕工，绝对是大师级的作品！”何老接过玉瓶后，就被上面的雕刻所吸引了。虽然玉瓶的玉石属于杂玉，但是上面的雕工和制作手法，都让这个小玉瓶脱胎换骨。<br/>　　<br/>　　“真不错啊，很好！”吴老也是赞叹不已。两个老头自从退休以后，就开始热心收藏，本身地位就高，见过不少的珍品，这眼睛也就样成了火眼金睛，但是今天这个小小的玉石瓶，却让两个人眼前一亮。<br/>　　<br/>　　“陈小子，你这个玉石瓶是从哪里得来的？”何老问道。<br/>　　<br/>　　“自己做的！”陈默回答道。<br/>　　<br/>　　“这是你自己做的？”何老的声音陡然直接扩大不少，里面的惊讶和震惊，只有他自己明白。<br/>　　<br/>　　要知道收藏这么多年，自己对于玉石的喜爱已经到了一定的高度，凡是上手后的玉石雕工，只要一眼扫过，就能知道出处和年限。但是这次陈默却给了自己一次震惊，没有想到这个家伙的雕刻技艺，已经可以说等同于大师级别了，心中对这个年轻人又有了新的认识。<br/>　　<br/>　　“有些药丸，用玉石瓶子装着能保证药性，所以就随手雕刻了几个。”<br/>　　<br/>　　装比的最高境界，就是无形的装。此刻陈默的表情在何老看来，就已经达到了无形装的境界。当然何老和吴老两人对于一些新鲜词汇不是很了解，但是装比倒是听过的，此刻，陈默就在装比，还是无形的装比。<br/>　　<br/>　　无语了，何老和吴老只能将玉瓶收好，转身就走，如果再和陈默待的时间长一点，害怕心脏受不了。<br/>　　<br/>　　陈默也是很无语，真没有礼貌啊，自己忙活了一个晚上，都不知道感谢的，真是两个可恶的小老头！<br/>　　<br/>　　其实陈默的雕刻技艺，虽然没有练习多久，但是自从成为修士以后，就在时刻练习着不说，由于修士的神识，还有修真以后对身体、手指、力量等等的细微控制，经过练习一段时间后，绝对可以说成是大师级的人物。<br/>　　<br/>　　要知道，阵法的阵基可不是那么好雕刻的，而且混合着真元的时候，随着刻刀的笔锋随走，真元也点点滴滴的融入其中，一旦有一点差别或者真元波动等，那么阵基就会整个都毁了，这对雕刻技艺的要求就非常的高了。<br/>　　<br/>　　在修真界里，每个修士都是有专攻方向的，要不就是专业绘制符箓，要不就是专一制作阵法，要不就是专一炼制丹药等等，甚至大部分修士都是专一的修炼，不去涉及其他一切的杂学。不过陈默的师父可以说是修炼天才，而且对于杂学也非常的喜爱，所以就在传功玉符中录入了很多自己的杂学，另外他对于阵法的研究可以所是独树一帜，传功玉符中对于阵法的知识就特别多。<br/>　　<br/>　　由于陈默在蓝海星上，可能大概只有他一个修真者，所以一切都要靠自己，那么杂学就不免涉及的多了。而且陈默也不知道光是靠着灵气荒漠似的蓝海星，自己能够修炼到何种程度，所以有所涉及，也是能够在修炼的时候能够帮助自己。<br/>　　<br/>　　将药物给了两个老头以后，又去找大海和陈金贵，交代了几下。今天他要去西市，因为药材市场的黄老给自己电话，说手头已经有很多的药材种子和植株了，让他去趟西市。<br/>　　<br/>　　另外就是地里的青菜基本再过个两三天就能上市了，他要拿些青菜去西市找沈婷婷的小叔，让他检测一下青菜，也好确定要货信息。<br/>　　<br/>　　一大早，陈金贵就在青菜地里，给陈默准备好了老大一包青菜，上面还淋上水，能够起到保湿的作用。<br/>　　<br/>　　陈默这才想起来，自己恐怕还需要去买一辆保鲜车，要不然蔬菜运到市区后绝对是蔫的。<br/>　　<br/>　　不过想想，先暂时不卖，等过段时间看看再说。今天他就去找沈婷婷的小叔沈博文，看他哪里有没有保鲜车，运输他们自己来好了，自己只负责买菜就好。<br/>　　<br/>　　如果感觉还可以的话，请给个推荐和收藏一下，顺便投个月票，谢谢了！<br/>　　<br/>　　7<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
