<br/>        能来这种地方的人，都是有钱人！

<br/>

<br/>        炽热的气息，将她包围。

<br/>        这个不明意义的动作，还未令诺筱颖缓过神来，唇瓣再次被人覆上。

<br/>        放大的瞳孔里，是一张英俊的脸，五官精致立体。

<br/>        对方忘情忘我地吸吮着她的唇瓣，就在他想更进一步的时候，诺筱颖用力推开了对方，却再一次被对方搂入了怀中。

<br/>        她真的没想到，他的力气会这么大，一只手撑伞，还能一只手紧紧地扣住她。
<br/>
<br/>        “你……”
<br/>
<br/>        看清楚对方是谁后，诺筱颖欲言而止，脸上露出惊愕的表情。
<br/>
<br/>        傅夜沉勾唇一笑，抬手摸了摸诺筱颖的头顶：“上车。”
<br/>
<br/>        “你、你是……”诺筱颖有些支支吾吾，不确定自己是否认对了人。
<br/>
<br/>        这个男人一袭军装，身材高大却不威猛，但霸气十足。
<br/>
<br/>        傅夜沉是连拉带拽，将诺筱颖塞进了自己的越野车。

<br/>        他们的车刚走，后面便有一辆醒目的玫瑰红私家车驶进了别墅区。

<br/>        因为傅夜沉的特殊身份，傅夜沉并没有固定的手机号码，苏漫雪也只能听从刘管家的传话，来花园别墅区等这位傅大少爷。
<br/>

<br/>        苏漫雪不得不开车往别墅区这边赶，当她赶到别墅区后准备下车时，又犹豫了。
<br/>
<br/>        正式跟傅大少爷见面，她突然莫名其妙地感到紧张起来。

<br/>        万一……
