<br/>　　“今日你的手段再多，也无济于事，被吞天秘术所吞噬，化为我的力量吧！”
<br/>　　轰隆！
<br/>　　吞天秘术的漩涡，再次转动，恐怖的力量拉扯着楚风眠，要将楚风眠给吞噬其中。

<br/>　　宫殿之内大量的陈家弟子，都是高声呼唤着，让陈少主的脸庞都是露出了几分得意之色。
<br/>　　他的目光看向楚风眠，露出几分玩弄之色道。
<br/>　　“小子，不用在挣扎了，你今日……”
<br/>　　这陈少主的话还没有说完，他的瞳孔突然间的放大开来。
<br/>　　只见一道剑光，突然间的从天而降。

<br/>　　楚风眠运转九玄剑术，施展出了这九玄剑术的最新一招，也就是第四式，天机。

<br/>　　那一次还是楚风眠第一次施展，所以无比生疏，在跟小叶帝的一战后，楚风眠又潜心领悟了九玄剑术的精髓。

<br/>　　陈少主见到楚风眠斩出的一剑不可置信的大喊道。
<br/>　　陈家，也是阎罗星域的大家族之一，他当然知道仙术，但是这仙术，可是就连古帝都难以掌控的。
