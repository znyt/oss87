夏夕绾站在回廊里探着小脑袋东张西望，就是没有看到他的人影。
<br/>
<br/>        这时蓝蓝的天空突然下起了一场稀里哗啦的春雨，竟然下雨了。
<br/>
<br/>        他去哪儿了？
<br/>
<br/>        夏夕绾气的在原地跺脚。
<br/>
<br/>        这时前方突然多了一道身影，夏夕绾抬眸，陆寒霆从前方的雨幕里奔跑而来，他身上穿着婶儿们手工制作的黑色衬衫黑长裤，并不是商务精英那种冷贵质感的衣料，衣料很是柔软，没有束进长裤里，而是休闲的垂落，衬的他整个人多了几分英俊的少年感。
<br/>
<br/>        夏夕绾纤柔娉婷的站在台阶上望着他，望着他从雨里一路奔跑到了她的面前，“陆太太，你找我？”
<br/>
<br/>        夏夕绾还生气呢，并不想理他，但是他额前碎软的刘海已经被这一场春雨给打湿，沾着水雾的男人真是格外的年轻俊俏，让人心软，“你去哪里了？”
<br/>
<br/>        这时夏夕绾看见他一只手藏在身后，好像藏了什么东西，她有些好奇，“你手里是什么呀，送我的么？”
<br/>
<br/>        说着她就摊开了自己柔白的小手心，娇蛮的跟他要，“快给我，我看看是什么东西，再决定要不要原谅我。”
<br/>
<br/>        陆寒霆将东西递给她，“陆太太，送给你。”
<br/>
<br/>        他手里是一大束的红色玫瑰。
<br/>
<br/>        红色玫瑰刚刚绽放，娇艳欲滴，花瓣上面还挂着这一场春雨的露珠，芬芳扑鼻。
<br/>
<br/>        夏夕绾双眼一亮，迅速接过玫瑰花抱在怀里，她垂着纤长的羽捷用手指数了一下，一朵，两朵，三朵…十一朵。
<br/>
<br/>        正好十一朵。
<br/>
<br/>        十一朵红色玫瑰，代表一生一世。
<br/>
<br/>        夏夕绾悄然的勾起了红唇，她低头嗅了一下红玫瑰的花香，心里的欢喜都快溢出来了，这是他第一次送她玫瑰花。
<br/>
<br/>        陆寒霆看着女孩，“喜欢么？”
<br/>
<br/>        夏夕绾点头，“恩，喜欢啊，看在玫瑰花的份上，我就原谅你一次了。”
<br/>
<br/>        陆寒霆将薄唇勾出一道柔软而缱绻的弧线，上去亲她的额头。
<br/>
<br/>        但是还没有亲到，远处就传来了婶儿撕心裂肺的尖叫，“啊，我的玫瑰花呢，哪个杀千刀的将我的玫瑰花给偷了！”
<br/>
<br/>        陆寒霆的吻顿住了，夏夕绾仰起绝丽的小脸望着他，“是你…偷的？”
<br/>
<br/>        “好像…是的。”
<br/>
<br/>        婶儿捶胸顿足要哭了，“这可是移植过来的玫瑰，花中之后，好贵的，今天刚绽放，现在都没了，十一朵都给摘了，竟然一朵都不留给我，这个杀千刀的好狠的心啊！”
<br/>
<br/>        夏夕绾看了看手里的十一朵红玫瑰，又抬头看了看男人。
<br/>
<br/>        陆寒霆一张俊脸很罕见的露出了尴尬的神色，“怎么办？”
<br/>
<br/>        还能怎么办，跑啊，再不跑婶儿就来了！
<br/>
<br/>        夏夕绾拉住了他的大手，拽着他就跑。
<br/>
<br/>        春雨已经停了，这山村的空气格外的新鲜舒爽，夏夕绾拉着他一路跑，跑到了金色的沙滩上才气喘吁吁的停下来，“陆先生，谢谢你送的花，这也太刺激了，我都跟你当了一次小偷了。”
<br/>
<br/>        陆寒霆也是第一次做这么尴尬的事情被小偷，他的人生除了那些灰暗的记忆，其他都可以当做教科书的，出身豪门且贵为一代商界骄子的他还没有这么狼狈过。
