　　芮冷玉怔怔地看着他，说道：“这里是零界？”
<br/>　　“没错。没有人可以来这里找到你。除了尸族溃败，被人追杀到这里来，不过这不可能。”
<br/>　　“溃败？”芮冷玉很聪明，也很敏感，察觉到了他言语中的问题。
<br/>　　后卿把外面的形势跟她讲了一遍，芮冷玉听完，惊得说不出话来。
<br/>　　“你们要占领整个青冥界？”
<br/>　　“不错，我们在这个苦寒之地蛰伏了太久，如今世无东皇太一，也无东岳大帝，我们为什么还要等下去？”
<br/>　　说到这里，后卿脸上也是露出了得意的笑容，“我要让这战火烧遍整个空界，让空界亿万生灵臣服在我脚下！我要带领尸族走向荣耀，迎接天劫！”
<br/>　　芮冷玉冷笑，对他的豪情不屑一顾。“原来你们是害怕东皇太一和东岳大帝，才龟缩在这里这么久吗？这样的战争，算不算胜之不武？”
<br/>　　后卿摇摇头，“我们也失去了我们的首领。若有他在，就算是那两位还在，我们也能一战。”
<br/>　　“你们的首领？”芮冷玉皱起眉头。
<br/>　　后卿缓缓吐出两个字：“将臣。”
<br/>　　芮冷玉倒吸冷气，喃喃道：“真的有将臣存在……”
<br/>　　“当然有，当初这三位开辟青冥界，阐教，截教想要奴役我们尸族，三方大战，最终三位一同殒命，阐教与截教占据空界，我尸族却被封印在零界这苦寒之地……”
<br/>　　后卿握紧了拳头，“尸族永不为奴。属于我们的荣耀，必将在我的手中拿回来！”
<br/>　　芮冷玉不理会他的豪言壮语，只是被他说的事情震惊得说不出话来，不由问道：“这三位都死了？”
<br/>　　“因为是开辟和分割青冥界的一战，我们叫做鸿蒙之战，那时候黎山老母这些人还没有飞升空界。大战的过程，除了我们尸族还有传承，已经没人知道了。
<br/>　　鸿蒙一战，本来阐教和截教联手，我们是必败之局，但后来两教内讧，给了我们喘息的机会。最终东皇太一身死，魂魄虚化，附着在东皇钟上，余下九尾天狐一脉，东岳大帝下落不知，我真主将臣也只存尸身……”
<br/>　　芮冷玉震惊地听着这段秘辛往事，她实在没有想到，青冥界的历史真相，居然是这样……
<br/>　　猛然间想到什么，芮冷玉问道：“你为什么跟我说这些？”
<br/>　　后卿耸了耸肩道：“你太闷了，这里没人跟你说话，我陪你聊一聊解闷。”
<br/>　　芮冷玉看着他，叹了口气，说道：“这一切跟我没有关系，我只想知道，既然你说是为了救我，为什么要把我关在这里？”
<br/>　　“为了保护你。”
<br/>　　“只是为了保护我？”芮冷玉不买账。“我记得尸族不是你一个人说了算吧？女魃，还有后卿呢？他们都同意你把我这个麻烦留在你们尸族？”
<br/>　　后卿犹豫了一下，说道：“你是转世鬼童，他们希望你能跟尸族合作，拉拢叶少阳和他的门人，还有风之谷。”
<br/>　　果然是这样……
<br/>　　芮冷玉摇了摇头，非常坚定地说道：“死了这条心。”
<br/>　　“我刚说的是第一点，第二，无极鬼王机关算尽，花费巨大的经历将你送到人间，绝不希望你就这样被杀死，有你在这里，假如战局逆转，对我们不利，我们我可以跟鬼王谈合作，用你作为条件。”
<br/>　　“你们……”芮冷玉震怒，却不知道该说什么，整理了一下情绪，冷冷说道：“我不会让少阳答应你们的！”
<br/>　　“你什么都不用做，有你在，叶少阳绝不敢跟真正跟跟尸族作对，而你……我希望你能把他忘了。你是转世鬼童，他是人间天师，你们，不可能在一起了。”
<br/>　　不可能在一起……我是转世鬼童……
<br/>　　芮冷玉眼泪再度落下来，摇了摇头，咬牙说道：“我不是转世鬼童，我相信你的话，但我相信，一定是某个地方弄错了，我不是鬼童我不相信！”
<br/>　　“我给你个证据，你还记得当初的王曼思吗？”
<br/>　　王曼思？“你是说，女魃的那个分身？”
<br/>　　“是她，当初她在灭亡之前，在你身上打了一掌，她的本意是让你变成僵尸，给叶少阳最大的报复，看他一个法师，能把你怎么办。结果……你中了一掌，但是并没有死，尸气在你体内没有蔓延就自动化解了。常理来说，这是绝无可能的事。这说明，你压根就不是正常人类。”
<br/>　　芮冷玉表情呆滞了。
<br/>　　她想到了那件事情，在那之后，叶少阳等人还专门检查了自己的身体，也是感到匪夷所思，猛然间，她想到了更多……
<br/>　　道风一次次说自己试图杀死少阳，自己一直觉得很委屈，现在想来……难道自己真的试图杀死过他？在那个时候……自己是觉醒状态？
<br/>　　这一切，都是自己中了王曼思那一掌之后发生的，难道那个时候自己已经觉醒了，只是觉醒得不够彻底，失败之后，会立刻恢复正常，并且没有任何的记忆。
<br/>　　可是自己潜意识中……或者说，那个真正的身为鬼童的自己，为什么要杀死少阳？
<br/>　　这个问题，芮冷玉不得而知，但是后卿所说的事实，是可以确定了的，自己，就是转世鬼童。
<br/>　　芮冷玉双手捂着自己的脸，再度失声痛哭了起来。在这一刻，让她感到最悲伤的不是自己的身份，而是……如后卿所说，自己是鬼童，少阳是天师，这种巨大的身份差异，让他们根本不能在一起了。
<br/>　　更不要说，现在全天下都在追杀自己，少阳如果跟自己在一起……以他的性格，势必会保护自己，结果他所有的荣耀都会离他远去，从最高位跌落，成为阴阳两界的罪人。
<br/>　　芮冷玉猛然间想到，也许，在冥冥之中，这一切是由无极鬼王或什么人在掌控着，自己存在的目的，就是去坑少阳，让这个承载着人间法术界未来的希望之星陨落……
<br/>　　一切都结束了。
<br/>　　想到自己跟少阳差一点结婚，想到那几乎成真的家……芮冷玉惨笑着摇摇头，自己当初就有种到手的幸福会被人破坏的感觉，原来应在这里。
