　　听闻古天意的话，苏莫面色凝重了起来。
<br/>　　但是，他并未说话，因为既然对方来找他了，必然是有所意图。
<br/>　　见苏莫沉默不语，古天意凝声道：“据我所知，你和火元圣师的关系，也并没有多么的牢靠，他若是不帮你，你必死无疑！”
<br/>　　“的确，不知古兄有什么办法？”苏莫点了点头，沉声问道，他知道，对方必定是有备而来。
<br/>　　“苏兄，不瞒你说，你的天赋在整个神武学府，都是最巅峰的存在，堪比各大派系的圣子人物，比之神子估计都不差多少！”
<br/>　　言及此处，古天意略微停顿了一瞬，面色郑重的继续道：“在下也是对你的天赋见才心喜，你若是愿意加入我皇祖圣朝派系，此事在下为你一力扛下！”
<br/>　　“哦？”苏莫闻言眸光一闪，对方的来意，没有出乎他的预料。
<br/>　　不过，此人真是不简单，不直接拉拢他，也不许诺任何好处。
<br/>　　首先点明他目前的处境，然后发出邀请。
<br/>　　这个时候，他无法拒绝，也不能拒绝。
<br/>　　而且，加入了对方的派系，不仅得不到任何好处，还要感谢对方！
<br/>　　当然，好处无所谓，苏莫要的是有人撑腰，确保安全。
<br/>　　虽然此人心机深沉，但的确是双赢的局面。
<br/>　　苏莫面上露出沉吟之色，虽然已经决定加入了对方的派系，但他也没有立刻答应。
<br/>　　古天意见此，继续道：“苏兄啊！你有所不知，虽然你天赋绝世，但现在其它派系，却是望而却步了！”
<br/>　　“哦？他们忌惮太史世家了？”苏莫问道。
<br/>　　“嗯！”
<br/>　　古天意微微颌首，道：“若是你不杀这么多人，他们都会来拉拢你，但现在，太史世家派系怒发冲冠，哪个派系若是敢将你拉拢，就要正面面对太史世家的疯狂报复！”
<br/>　　“那你不怕吗？”苏莫目光直视对方，其它派系都忌惮了，对方就不忌惮？
<br/>　　或者是说，对方愿意为了他，不惜和天使世家派系正面抗衡！
<br/>　　“怕！”古天意重重的点了点头，但话锋一转，郑重的道：“但是，为了能得到你这种天才，就算与太史世家派系开战，又有何妨？”
<br/>　　古天意豪气冲天，大有为了苏莫，不惜一切的姿态。
<br/>　　“既然古兄如此看重，那在下岂有推脱之理！”
<br/>　　苏莫微微点了点头，对方的诚意已经完全表露了出来，他自然不会拒绝。
<br/>　　不过，他也暗暗警惕了起来，因为就连他，听了对方的话语之后，心中都忍不住生出一丝感动之意。
<br/>　　这要是换了一些情商低的人，估计都要向对方顶礼拜谢了。
<br/>　　此人，不菲吹灰之力拉拢了他，还要让他心生感激，这手段虽然简单，但却是非常不凡。
<br/>　　“哈哈！既然如此，那从今以后，苏兄就是我皇祖圣朝派系之人了！”古天意的脸上露出了笑容。
<br/>　　他比其它派系的圣子，更希望拉拢住苏莫这种天才，因为他与其他派系的圣子不同。
<br/>　　其它派系的圣子，不过是为了完成各自派系背后势力的任务，而拉拢天才。
<br/>　　而他，却是为了自己，为自己培养班底，为将来谋的大业做准备。
<br/>　　“嗯，多谢古兄了！”苏莫向对方抱拳道谢。
<br/>　　“苏兄不必多礼，事不宜迟，我现在就去想办法，尽力化解你与太史世家派系的恩怨！”古天意站了起来。
<br/>　　“劳烦古兄了！”苏莫道。
<br/>　　“告辞！”古天意没有与苏莫过多寒嘘，便告辞离去。
<br/>　　望着对方离去的背影，苏莫眸光闪烁，他估计事情并不会像对方说得那般严重。
<br/>　　十三个派系，十三大势力，相互合作、相互竞争，相互依存，怎么可能开战？
<br/>　　或许解决此事，对于对方来说，并不是什么困难之事。
<br/>　　随即，苏莫便不再多想，安心修炼了起来，他相信对方既然敢拉拢他，就一定会解决此事。
<br/>　　还是自身的实力最为重要，他现在资源无数，要加紧修炼的进度了。
<br/>　　在苏莫修炼的时间里，他加入皇祖圣朝派系一事，迅速传遍了整个神武学府。
<br/>　　学府中顿时一片哗然，如此关头，苏莫加入皇祖圣朝派系，这是代表了什么？
<br/>　　这代表，皇祖圣朝派系，要给予苏莫庇护。
<br/>　　这让太史世家派系上下震怒，闹的一片沸腾。
<br/>　　但是后来，古天意和太史爵见了一面，两人相谈了一个时辰。
<br/>　　不知古天意付出了什么代价，其离开之后，太史爵宣布，不再追究苏莫残杀七八十名麾下弟子之事。
<br/>　　这一场巨大的风波，就这样渐渐平息了下去。
<br/>　　当然，所有人都知道，这只是表面上平息了，太史世家派系不太可能真的放过苏莫。
<br/>　　时光飞逝，眨眼间又过去了半个月。
<br/>　　半个月后，苏莫结束了修炼，他的修为根基已经相当扎实了，可以尝试冲击武帝境五重了。
<br/>　　之所以能这么快稳固根基，是因为从天冥星回来的路上，那几个月的时间，他也没有懈怠，赶路的时候都在修炼。
<br/>　　对于太史世家派系一事，他也从聂乾和韩盈口中得知了，这是最好的结果。
<br/>　　不过，太史世家虽然不追究此事了，但苏莫却是不会就这么算了，以后若是有机会的话，他必杀尤天韩。
<br/>　　几日前，一名皇祖圣朝派系的弟子，前来邀请他去古天意的‘天意峰’居住。
<br/>　　对此，苏莫婉言谢绝了，他住在火元峰很好。
<br/>　　结束修炼之后，苏莫用大石封住了自己的洞府，随即进入了玉镯空间中，准备冲击境界瓶颈。
<br/>　　身形出现在玉镯空间中，苏莫立刻便察觉到了巫娴等三名巫族之人。
<br/>　　随即，他面露思索之色，这三人貌似已经没有任何用处了，可以杀了。
<br/>　　不过，想到日后还可能与巫族交手，他压下了心中的杀机，先留着三人吧！说不定日后还有用处。
<br/>　　在玉镯空间中的宫殿内盘膝而坐，苏莫取出了几枚强大的神丹，开始冲击武帝境五重境界。
<br/>　　吞噬战魂释放而出，神丹直接被吞噬战魂吸了进去。
<br/>　　霎时之间，浑厚无比的玄力，直接炸开，冲进了苏莫的体内。
