　　元始站在昆仑山玉虚宫，遥遥望到人族部落中的一个小孩，这个小孩便是未来的人族三皇之一的地皇神农，亦称炎帝。
<br/>　　小孩抬头望天，渴望像小鸟一般，飞翔天际，自由自在。小孩低头望地，看到的不是大地的宽广，生命的多姿，而是看到的无尽病魔在折磨这人类，无尽的灾难，降临人间。
<br/>　　小孩虽然只有三四岁，但却十分懂事。这一刻，小孩下定决心，将来一定要学到本身，帮助人族脱离苦难，于病魔之中拯救人类。
<br/>　　若是其他人知道小孩内心的想法，定会嘲笑他，笑他不自量力。但他们又何曾想过，没有尝试，又怎么知道不会成功呢？
<br/>　　这一日，一个道人从天边飞过，他看到了小孩，更是看到了他的内心想法，他顿时觉得有趣，便飞了下来，落到这个小孩面前。
<br/>　　小孩看着天上飞下来的道人，面上不但没有露出害怕的情绪，反而有些好奇的看着这道人，开口问道：
<br/>　　“你是仙人吗？”
<br/>　　在小孩的心中，他十分羡慕那些会飞的仙人，听说那些人能够飞天遁地，神通广大，他是多么希望自己有那样的本事，那样，他的族人们便不会再遭受苦难了。
<br/>　　那道人听了小孩的话，哑然一笑，道：“你就当我是吧。”
<br/>　　“听说仙人都是神通广大，不死不灭，是那样吗？”
<br/>　　“的确如此。”
<br/>　　“那你能收我为徒吗？”小孩认真道。他都没有去管这人是好是坏，他虽然小，但很聪明，他能够感觉到，眼前这人不是坏人。
<br/>　　“你为什么要拜师呢？”道人沉默了一会儿，开口问道。
<br/>　　“若是我学会了本事，我就能够为我们部落里的族人治病了，部落中一旦有病魔降临，就会死去好多好多人，我不想看着他们死去。你能教我救人的仙术吗？”
<br/>　　说到这里，小孩眼中流露出无比坚定的神色，同时望向道人的眼神中，充满了渴望，那是对生命的渴望。
<br/>　　看到小孩那纯洁担忧充满坚定的神色，道人的内心颤了颤，曾几何时，我也是一个这样的小孩啊，可是，我如今有了实力，但却又好像忘了自己的初衷。他沉默片刻，回了一个字：
<br/>　　“好。”
<br/>　　“多谢老师。”小孩大喜，拜谢道。
<br/>　　道人亦微笑着点了点头。
<br/>　　这一切，元始都看在眼中，这个道人便是太上的徒弟玄都，小孩便是未来的炎帝。
<br/>　　元始收回目光，不再去管其他，而是将重心放在自己最近研究的一项工程上。元始准备创世。
<br/>　　创世！
<br/>　　不错，就是创世。修士一到金仙，便能开辟一些小空间，大罗更是能够开辟洞天福地等空间，准圣能够开辟小世界，圣人更是能够开辟中千世界，当然，更高级的圣人更是能够开辟大千世界。
<br/>　　元始因为修炼功法的特殊性，如今以他混元七重天的修为，再加上《玄黄不灭》第九重的功效，他体内已经出现了三百六十五方大世界，更有无数细胞之中，进化成了一个个中世界、小世界。当然，这种大世界只是初生的大世界，不仅在宽广度上，还有法则的完善度等，都还比较弱小，比不上盘古所开劈的这种完美大千世界，但就算如此，也非常恐怖了。
<br/>　　试想，一个人全身上下全部是世界，他的力量该有多强大？如今元始若是全力攻击洪荒，他都可以轻易将整个洪荒打成碎片。
<br/>　　当然，元始也不会真的无聊的去将洪荒打碎，先不说鸿钧不允许他那样做，就算他允许，元始也不会那么做。毕竟，洪荒天道意志不会让他这么做，天道可是堪比一位天道境强者，若是天道不顾一切想要灭了他，那他也只有逃跑的份。
<br/>　　如今元始想要创造的世界并不是如同盘古这样的世界，也不是如同女娲那般在混沌之中开辟一方世界，依附于洪荒世界。他所要开创的世界，是一方特殊的世界。怎么个特殊法，因为这方世界是介于真实与虚幻之间的精神世界。
<br/>　　他要开创的世界有点像准提圣人的梦幻空间，但又比他那种世界更真实。准提的世界全是梦幻法则造成了，里面全是虚假的，还没有达到真实的地步。而元始要开创的，便是一方介于真实与虚假之间的世界。这比完全虚幻或者完全真实的世界更加困难。
<br/>　　简单来说，那是一方能够让人精神进入的世界。让人能够感同身受，在此方世界经历的一切，都像是真实的。
<br/>　　我们身处的世界，是物质世界，但元始所要创造的世界，便是精神上的世界。
<br/>　　任何一种生灵，其精神都是最神秘的，更胜于他的肉身。若是能够开创出一方精神世界，让人的精神就进入其中，都宛若经历一番再生，与真身在此没有任何区别。
<br/>　　而且，精神在其中磨砺，回归现实后，更是会将所有领悟都带给肉身，这便是元始所要创造世界的神秘所在。
<br/>　　当然，最为关键的是，在这方世界之中，不必担忧死亡，纵然战死在其中，最多也只是损失一些精神力而已，不会令本体死亡。
<br/>　　当然，要除了一些特殊的手段，因为有些手段能够通过精神，攻击到他的肉体。如因果法则，它能够直接连同精神与灵魂一同灭亡，还有命运法则、诅咒法则等神秘法则会牵连到本体。
<br/>　　若是别人，要开创这样的世界，可能有些困难，但元始却不一样，他有着别人没有的优势。
<br/>　　他身为混沌魔神之时，便有掌握梦幻大道法则，之后更是精通灵魂法则，灵魂乃是万物之灵，最为神秘，有了灵魂法则与梦幻法则的结合，那就更加完美了。当然，最重要的是他有领悟了三大本源法则。混沌物质，太初能量，鸿蒙灵魂，三大本源的结合，对于开创任何世界，都是非常容易的。
<br/>　　若是真的能够创出这样的精神世界，洪荒便会进入一个全新的时代。
<br/>　　当然，元始这样做并不是没有目的，天下没有免费的午餐，元始又怎会辛辛苦苦创造一个这样的世界来便宜别人？他从来都不是那么无私的人。
<br/>　　元始所做的一切都是为了规则。
<br/>　　若是世界开创成功，无尽种族，浩瀚文明，各种道法，都会在这里碰撞出智慧的火花，而元始身为精神世界的掌控者，便会获得其中的一切，到那时，各种道法规则奥妙，都会被精神世界吸收，之后演化出来，元始到时便可以通过这些规则演化，感悟这其中的智慧，淬炼自己的大道。
<br/>　　他的元始大道便是要吸收众多万千大道来进化，而一个人感悟，终究是有限的，洪荒如此多人，那意味着无尽的智慧，无数的想法，若是元始能够利用起来，到时便相当于无尽种族在他的精神之中，演化各种道法，他若是能够吸收，道行定会大增。
<br/>　　更为重要的是，每一个进入精神世界中的人，都会或多或少的留下一部分精神力，那些精神力，便会被精神世界吸收，增强精神世界的强度。而且进入世界内的众生，更是能够产生一种精神念力，或者信仰之力，有了那些力量，千万年不断积累，纯化之后，便可以吸收，更是能够节省元始许多时间。
<br/>　　“嗯，这样的世界，一定要尽快创造出来。”元始在自语，他想到其中的好处，非常心动。不再多想，他很快便沉浸到参悟世界规则的美妙之中去了。
