　　从水里捞上船的大食人很快交待了李二的位置和眼下的情况，李佑在知道了一切之后，一颗提着的心终于是放回了肚子里，摆摆手命人将那几个分别打捞上来的大食人重新丢回大海。
<br/>　　“怎么样，这下放心了吧？”李震看着脸色已经恢复正常的李佑，拍了拍他的肩膀，李佑则是翻着白眼将他的手打到一边，没好气的说道：“别套近乎，我跟你很好么？”
<br/>　　面对恼羞成怒的李佑，李震无奈的耸了耸肩：“怎么样，接下来我们怎么办？是直接杀过去，还是等消息。”
<br/>　　“还等什么消息，这里打的这么热闹，估计父皇那边早就已经看到了。”李佑看着刚刚大食人所指的方向，毫不犹豫的说道：“今天晚上修整一个，留下人警戒，明日一早发起进攻。”
<br/>　　“好，那就这么办！”李震点了点头，摆手示意传命兵把命令传达下去。
<br/>　　刚刚他们这边打的火光冲天，大半夜的估计离着几十里都能看到，所以李佑说老头子已经知道他们来了这一点李震毫不怀疑。
<br/>　　现在差的就是那些大食人应该也知道他们来了，只是不知道他们会不会趁着夜色跑来偷袭，又或者会不会趁着夜色逃走。
<br/>　　不过一晚的时间估计那些大食人就算是跑也跑不了多远，如果他们足够聪明的话就一定会固守，又或者学老头子，找个地方登岛，避免与北海舰队碰面。
<br/>　　……
<br/>　　而事实证明，李震的判断并没有错，阿里·伊本在看到海面上那可以映红天边的火光时，立刻意识到了情况不对，随后在召集人手商量了很长时间之后，他们得出的结论同样是登岛，躲避大唐的援兵。
<br/>　　大唐的远程武器实在是太厉害了，在海上与他们交战，那种武器打过来一死那就是一船人，远不如在岛上作战，至少被那种武器打了不会一下子死那么多人。
<br/>　　有了这样的决定之后，阿里·伊本再次作出了一个错误的决定，全员登岛，争取在大唐的援兵赶到之前将李世民抓住，这样就可以用他来当成筹码。
<br/>　　可是阿里·伊本并不知道，李世民早就已经算计到了他的下一步计划，当看到天边火起的那一刻，他就知道，这一次大食人根本别无选择。
<br/>　　所以他命令秦琼和尉迟恭整军，借着夜色，整只队伍退入海岛的更深处，将营地扎到了岛上唯一一座高山的另一侧，等待着大食人的进攻。
<br/>　　当然，李世民选择退入海岛深处也有他自己的考量，援军虽然到了，但他并不知道领军的是谁，退入岛的深处也是为了防止援军突然反水，与那些大食人合兵一处。
<br/>　　做好了一切准备之后，李二伫立于山巅之上，任由海风吹拂着他的衣襟，目光盯着远处的海面，如一只盯住了猎物的雄鹰，随时准备发动致命一击。
<br/>　　秦琼、尉迟恭就站在李二的身后，如两尊石雕，但身上不断逸散的杀气却压的四周的野草全部伏低。
<br/>　　东面天空渐渐泛白，李二也终于开口：“怎么样？都准备好了么？”
<br/>　　“陛下，已经准备好了，随时可以发起攻击。”秦琼说道。
<br/>　　“那就准备吧，很快那些大食人就会攻上来，到时候好好给他们一点颜色看看！”
<br/>　　“是！臣等遵旨！”得了命令之后，秦琼、尉迟恭重重一抱拳退了下去。
<br/>　　……
<br/>　　太阳一点点的升起，很快便将金色的光辉铺满整个海面。
<br/>　　大食人的船队分批次的靠近岸边，无数大食人从船上涌下来，在海边列阵。
<br/>　　时间紧迫，容不得阿里·伊本过多的等待，待集合起一万余人之后，他便命那些土著打先锋，自己带着一万余人压阵，向着此前唐军的防线扑了上去。
<br/>　　这是一场碰硬碰的战斗，不是你死就是我活，大食人的身后就是大唐的援兵，如果不能在短时间里将李二拿下，等待他的绝对没有什么好果子。
<br/>　　唐军的防线没有人，一个人都没有，阿里对此并不意外，只稍稍犹豫了一下之后便再次下令向前。
<br/>　　无名岛并不大，大唐的太上皇在没有船的情况下根本不可能离开，所以只要追下去就一定能够遇到。
<br/>　　又向前推进了大概有两里左右，先头部队已经到了岛上唯一一座高山的山脚下，同时阿里·伊本也看到了半山腰处，两员骑在马上的唐军将领。
<br/>　　……
<br/>　　没有任何的交流，甚至没有任何犹豫，双方见面的时候各自下达了进攻的命令。
<br/>　　万余唐军在秦琼和尉迟恭的带领下，直接从半山腰扑了下来，向着正在往山上冲的土著碾压过去。
<br/>　　是的，这的的确确就是碾压，前几天还攻的有声有势的土著在唐军的攻击下连反击的机会都没有，只一个碰撞就变成了一地的尸体。
<br/>　　疯了，真特么疯了！
<br/>　　一直在采用远距离攻击的唐军突然间改变了作战方式，抛开那种可以隔着几百步就能打死人的武器，挥舞着横刀冲下山的时候，阿里·伊本几乎忘记了应该如何指挥战斗。
<br/>　　为了应付那种看不见的武器，阿里将一万余人的队伍排成了散兵线，这是他从土著攻击唐军防线的方式中总结的阵型，在他看来完全可以应付一段时间，只等后面的部队上来之后就可以发起致命一击。
<br/>　　结果唐军根本不按套路出牌，这一次竟然没有使用远距离攻击的武器，反而是一鼓作气的冲杀了下来，跟他玩儿起了近距离作战。
<br/>　　看着唐军那密集的阵型，就算是傻子都知道散兵线根本无法阻止他们的进攻，而且那两个骑在马上的将军简直不是人，面对如雨的箭矢竟然完全不知道躲闪，就那么一路杀了下来，一杆长枪，一条钢鞭瞬间替他们身后的那些士兵开出两条血路。
<br/>　　而那些唐军士兵也不是什么弱不禁风的家伙，右手横刀，左手短弩，近战配合远攻，不消片刻已经将战果扩大的数倍，万余大食人眨眼间便被打的毫无还手之力，防线隐隐有崩溃的趋势。
