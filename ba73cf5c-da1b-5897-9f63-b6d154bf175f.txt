        因为，忘记了他们的过去，也就代表着她把傅昼景也一并给忘了，还有他曾经做的那些伤害过她的事情。
<br/>
<br/>        不记得那些糟糕的事情，真好……
<br/>
<br/>        虽然，她也忘记了对他的爱。
<br/>
<br/>        这一切，傅夜沉决定一个人默默地承担。
<br/>
<br/>        童筱颖又继续自顾自地吃饭，突然又停了下来，看向傅夜沉，微笑着补充了一句：“还好我记得你的脸。”
<br/>
<br/>        忘记了所有的人，也包括忘记了自己，却仍旧记得他的脸，记得她对他的昵称，还记得对他脸红心跳的感觉。
<br/>
<br/>        而这些缘由，原来只因他是她的老公。
<br/>
<br/>        对于自己莫名其妙地就多了一个老公这种事情，童筱颖没有感到很排斥，但也没有很快就接受了这个事实。
<br/>
<br/>        “我会尽快适应你是我的老公这件事情。”顿了顿后，童筱颖又接着说道。
<br/>
<br/>        傅夜沉听着她的这句话，不知道是喜还是忧，不过，他发现，筱颖已经不需要再戴助听器了。
<br/>
<br/>        看样子，她的听力，在实验室里也被他们治好了。
<br/>
<br/>        就好像他和她之间，一切都可以重新来过一样。
<br/>
<br/>        “我会好好珍惜你。”傅夜沉故作淡定地嘴角微扬。
<br/>
<br/>        她说她要去环游世界，第二天，他便帮她把一切出国手续全都办妥，就连行李都全部替她准备妥当，她随时都可以买机票出发。
<br/>
<br/>        童筱颖还真就后顾无忧地拖着行李箱，买了机票一个人出国玩去了。
<br/>
<br/>        对于这件事情，连华生得知后，一脸惊悚地看着傅夜沉：“我还以为，你会带着她去寻回你们过去的回忆。结果，你就这么让她一个人出国去玩了？”
<br/>
<br/>        “是。”傅夜沉坐在办公桌前，一边签署着文件，一边应着连华生的话。
<br/>
<br/>        他和夏阳枭之间的约定，连华生知道后，也不好说什么，但是对于童筱颖的事情，他就大有意见了。
<br/>
<br/>        连华生汗颜地瞥着傅夜沉，忍不住对他泼冷水：“你就不怕她去了国外，在旅途中邂逅其他男人并爱上了对方，然后回来跟你提离婚？”
<br/>
<br/>        “怕。”傅夜沉握着钢笔的手停顿了一下，然后拿起另一本文件又继续签了起来。
<br/>
<br/>        “怕的话，那你还这样放宽心了让她去？”连华生难以理解傅夜沉的这种行为。
<br/>
<br/>        傅夜沉却只笑不语。
<br/>
<br/>        以前，他把筱颖当金丝雀，霸道地将她圈禁在自己为她准备的金屋子里。现在，她不再是那只金丝雀了。
<br/>
<br/>        至少，那天她在机场跟他道别的时候，眼神里对他没有一丝的恋恋不舍，就连一个拥抱，她都没有想过要给他。
<br/>
<br/>        她只是微笑着对他挥手告别。
<br/>
<br/>        他说，要她记得给他打电话报平安，但是至今都已经过去一个星期了，她一个电话都未打给他。
<br/>
<br/>        不过，他知道，她很安全，因为他派人在暗中保护着她的安危，知道她玩得很开心。
<br/>
<br/>        现在的筱颖，活得潇洒自在，过着她想要过的日子，难道不好吗？
<br/>
<br/>        经历过那些事情后，是他想开了。
<br/>
<br/>        连华生见傅夜沉不理会自己，于是又没话找话，存心打击他傅夜沉：“我觉得吧！你要不干脆就放她走好了，反正她不记得你了，也对你没有爱意了。而且，最主要的是，她不孕不育。没法给你生继承人。”
<br/>
<br/>        “你说什么？”傅夜沉倏然间抬眸，黑眸冷冽地瞪向连华生。
<br/>
<br/>        连华生不痛不痒地接着说道：“本来我以为她死了，这件事情就没必要跟你提了。其实半年前，你出事后，童筱颖有很积极地想要怀上你的孩子。但是，我陪她去医院做检查，才从她那些检查结果上得知，她不孕不育。我想帮她，才带她去我爸的实验室，让我爸手下的医生替她查原因，看看还有没有法子治疗。结果，她就被绑架了。不过，我猜的是，童筱颖当时一心赴死，确实是为了保护你，但还有可能也与她知道自己无法生育的这件事有关。她想你彻底忘了她，重新找个女人过日子。而且，我看她现在似乎一点都不在乎她以前的事情，对你也没有任何的依恋，反倒是想着她自己今后要怎么开心就怎么过便好。事到如今，你何不如她所愿？”
<br/>
<br/>        “你觉得，除了她，我还能接受其他女人吗？”傅夜沉若无其事地反问。
<br/>
<br/>        连华生一脸诧异，又忍不住滔滔不绝起来：“难道，你想要傅家断后？傅氏集团的未来，你该不会是想交给傅昼景的儿子吧？你爷爷不会答应的。再说了，你若是无法接受其他女人，那你可以做试管婴儿，然后找个女人代孕便是。”
