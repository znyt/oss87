        &lt;h3&gt;第三卷第一百七十八章  闫芈的悲哀&lt;/h3&gt;
<br/>
<br/>        闫欢有很多话要问闫芈，但是还是得耐下性子，毕竟闫芈才回到家，过新年的时候，秦少卿也没有回来，就是给闫芈打了电话，问候了大家。
<br/>
<br/>        闫芈虽然极力隐藏着，但是闫欢还是发现了闫芈有心事。
<br/>
<br/>        这天晚上，闫欢等着小皓儿睡着了，闫欢和闫芈聊开了。
<br/>
<br/>        “你这生个孩子，都几年不回家呢。”闫欢看着睡熟的小胖墩说。
<br/>
<br/>        “秦家有不成文的规矩，说孩子三岁之前不让回外婆家，我就只能等了。”闫芈有些无奈的和姐姐解释。
<br/>
<br/>        “秦家对你好吧？”闫欢不问秦少卿对闫芈好不好，而是问秦家对闫芈好不好。
<br/>
<br/>        “怎么说呢，也还行吧，之前没有住在一起，是没有多少矛盾，住在一起了事情就多了，可能每个家庭都一样，需要磨合吧。”闫芈眼神有些飘忽。
<br/>
<br/>        “你们是什么时候搬到一起的啊？”闫欢没有嫁人，也不知道俩个家庭怎么磨合。
<br/>
<br/>        “孩子一岁多吧。”闫芈想着时间，大概就是陆玉到甸城的那一段时间吧。
<br/>
<br/>        “磨合了这么多年还没有磨合好？”闫欢知道有故事了。
<br/>
<br/>        “可能生活习惯和代沟的问题。”
<br/>
<br/>        “你现在的工作和你的专业匹配吗？”
<br/>
<br/>        “不，我学的是经济管理学。”
<br/>
<br/>        “芈儿，你实话告诉我，你和秦少卿的关系怎么样？”
<br/>
<br/>        “姐。”闫芈心里很苦，但是她不想告诉姐姐。
<br/>
<br/>        “芈儿，我们是姐妹，你离得那么远，身边一个亲人都没有，如果心里有委屈，不说出来，会憋坏的。”
<br/>
<br/>        “我和秦少卿之前的关系很好，只是后来，后来........”闫芈不知道怎么说。
<br/>
<br/>        “后来怎么样了？”闫欢问。
<br/>
<br/>        “后来来了一个叫陆玉的，他们说他们几个从小关系就很好，大概就是陆玉来了之后吧，我们就搬回去了，少卿经常陪着陆玉工作，经常见不到人。”闫芈说。
<br/>
<br/>        “你是怀疑秦少卿和陆玉有问题？”闫欢听到陆玉心里一惊，世界真是太小了。
<br/>
<br/>        “不是怀疑，而是我亲耳听见陆玉说她怀孕了。”闫芈低着头，手用力的绞着衣角掩盖着她内心的疼。
<br/>
<br/>        “陆玉到甸城多久了？”
<br/>
<br/>        “快三年了。”
<br/>
<br/>        “你认识一个叫谷强的人吗？”闫欢终于是问出来了。
<br/>
<br/>        “认识，他很能干，这几年伙同秦少卿和陆玉在准备做信息化，现在项目资金都申请下来了，准备明年起动。”闫芈不知道闫欢怎么会认识谷强。
<br/>
<br/>        “你是说陆玉和谷强他们一起搞信息化？”闫欢接着问。
<br/>
<br/>        “是的，听说谷老大让他放下所有的事情，专门成立了一个新的公司，这几年主攻信息化。这次审批的项目资金非常大，资金是陆玉帮忙申请的。”
<br/>
<br/>        “呵，要是资金不大，估计也不会让谷强放下所有的事情来做这件事情吧。”闫欢心里在想，谷强的大事情是不是没有做了？
<br/>
<br/>        能让谷强放下手中的事情，这个信息化该有多大啊？
<br/>
<br/>        “陆玉要在甸城呆很久吗？”闫欢又问闫芈，闫芈觉得很奇怪，姐姐问的话都是跳脱性的，似乎没有逻辑可言。
<br/>
<br/>        “之前是经常出差，现在是正式调任，已经快两年了。”闫芈如实回答。
<br/>
<br/>        “你在甸城干的工作是不是也很重要？”闫欢接着说。
<br/>
<br/>        “还好，不过现在的我就是一傀儡，都是别人说了算。”闫芈还是老总的位置，不过她已经是被架空的老总。
<br/>
<br/>        “芈儿，我想到甸城开连锁超市你觉得怎么样？”闫欢问闫芈。
<br/>
<br/>        “甸城的超市也有，不过都是小型的。你想开什么样的超市？”
<br/>
<br/>        “明天带你去我的商场转一圈，你看看可不可以？如果可以的话我过去给搭个伴。”闫欢看着妹妹那无助的样子有些心疼。
<br/>
<br/>        秦少卿劈腿陆玉，渣男渣女在一起，闫芈性子柔弱，只能是被欺负，但是陆玉如果知道她会去甸城，肯定会为难她的。
<br/>
<br/>        不过正因为陆玉会为难她，她才能在甸城扎下根来。
<br/>
<br/>        谷强能让陈子昂过去，肯定是在谋划着一个大事情，还有陈子昂和谷强很是想象，他们不可能不认识彼此。
<br/>
<br/>        能耗费两三年的时间来做信息化，肯定不是一件小事情，谷强，你是在拖延时间吗？你是在寻求帮助吗？
<br/>
<br/>        “姐，我在甸城势单力薄，我怕我帮不了你什么。”闫芈知道自己目前的处境，她还想着怎样和秦少卿摊牌。
<br/>
<br/>        后期她想再去国外，她联系了之前的导师，导师可以帮助她安排一份工作，想着自己放弃了国外优厚的待遇和秦少卿在一起，如今落得如此下场，闫芈心里充满了悔恨，爱情真的不能当饭吃。
<br/>
<br/>        “呵呵，放心吧，姐不需要你帮助我什么，我是过去投资的，没有人能拒绝金钱的魅力。”闫欢笑着说。
<br/>
<br/>        “芈儿，你打算和秦少卿怎么办？”
<br/>
<br/>        “我后期想去国外发展。都已经这样了还是离婚吧。”
<br/>
<br/>        “能离得了吗？”
<br/>
<br/>        “怎么不能呢，他们俩个都在一起了。”
<br/>
<br/>        “芈儿，你说的那个陆玉，是京城陆家的女儿，心狠手辣，放，荡不已，她的老公是张家的张函，虽然陆玉作风不好，但是一直都是遮遮掩掩的，你如果想要和秦少卿离婚，你只能拿到秦少卿出轨的证据，没有证据一切枉谈。所以，芈儿，压下你心底的悲伤，找到他们的证据。”闫欢虽然没有读多少书，但是生活的苦难和磨练已经让她坚不可摧。
<br/>
<br/>        “姐？”
<br/>
<br/>        “当然，如果你愿意过那样的生活也可以，装作什么都不知道。像陈子悦一样过一种同在屋檐下两个陌生人的生活。”闫欢和妹妹很多年不见，也不知道她是怎么个想法。
<br/>
<br/>        “我可以装作什么都没有发生，但是我做不到再和秦少卿在一起，我感觉很恶心。”他们曾经那样深深的爱着彼此，当陆玉说出她怀孕的时候，闫芈差点都吐出来了。
<br/>
<br/>        “人，要想别人爱，首先要学会爱自己，尊重自己，对于爱情我没有太多的发言权，因为我没有谈过恋爱，但是我觉得，爱情是双方面的事情，是平等的，不是一方需要苟延残喘的迁就。一方如果变心，当断就断，没有必要纠缠不清。”闫欢看着闫欢郑重的说。
<br/>
<br/>        “姐，我知道了。”闫芈是受过高等教育的，智商很高，但是情商有点欠缺，要不然怎么会相信秦少卿和陆玉真的是少年朋友关系？
<br/>
<br/>        “过几天你先回去，要装作什么事情都没有发生，等拿到他们的证据再说！”闫欢再次叮咛。
<br/>
<br/>        “另外，你说的那个信息化项目，会过去一个人，这个人和陆玉有关联，甸城也许会上演一场好戏。”
<br/>
<br/>        陈子昂有孩子的事情，有可能张函会知道，张函上一次说他一直爱着陈子昂，如果陈子昂去了甸城，张函说不定也会赶过去。
<br/>
<br/>        陆玉将她送到甸村那个地方肯定是和那边的人联手，要不然凭着陆玉那么小的年龄是没有那样的本事，陆家和秦家一定有什么不可告人的秘密。
<br/>
<br/>        还有一个重要的问题，张函怎么会知道她在甸村？
<br/>
<br/>        闫欢有着缜密的思维清晰的头脑，她要利用一切可利用的机会，帮助谷强翻身，是帮助她复仇，隐忍了这么多年，她也应该出现了！
<br/>
<br/>        那些黑暗的日子像蛆一样腐蚀着她的心智也让她像阳光一样散发着光芒！
