　　　　“嗯！”
<br/>　　阳鼎天闻言点头，随即他双手掐诀，一个个印诀飞快的结出，打入了阵法各处。
<br/>　　霎时之间，庞大的火炼八荒大阵嗡嗡震动了起来，随即内部燃烧起了熊熊的烈焰。
<br/>　　整个天海城，仿佛是被一个巨大的火球包裹在了其内，阵法内的温度立刻急速攀升，疯狂的暴涨。
<br/>　　“怎么办？”
<br/>　　“完了！我们都完了！”
<br/>　　“真是疯子啊！我们做鬼也不会放过你们！”
<br/>　　天海城之中，一片嘈杂，无数的武者脸色惨白，面无人色，彻底的慌了神。
<br/>　　面对死亡，无论是低级的武者，还是一些武王境、武皇境的武者，均是没有什么不同。
<br/>　　“苏宫主，我们怎么办？”海明江同样面现焦急之色，若是逃不出去，他们真的要陨落于此了。
<br/>　　“能怎么办？先集合众人之力，试试能不能破开阵法吧！”
<br/>　　苏莫叹道，天海城有近亿武者，虽然因为天涯海阁总部的毁灭，天海城变得清冷了许多，人口锐减，但也有数千万之多。
<br/>　　这么多人联手，破开阵法不是没有可能。
<br/>　　言罢，苏莫身形冲天而起，为飞上了高空中。
<br/>　　“诸位，我们不能坐以待毙，若想活命，只有联合起来，击破阵法，方可有一线生机！”苏莫眸光如电，扫视四方，朗声高喝。
<br/>　　他的声音化为滚混浪涛，传遍了天海城每一个角落，让整座城微微一静。
<br/>　　“不错，我们这么多人一起联手，说不定能破开大阵！”
<br/>　　“苏莫，我们听你的！”
<br/>　　“大家一切联手，我们必定能逃出去！”
<br/>　　“……”
<br/>　　苏莫话音刚落，立刻有无数的武者响应，爆发出震天的呼声。
<br/>　　众人都知道这是唯一的办法，只有这样才有脱身的希望，当然，最主要的是，有苏莫和海明江两人在，众人心中也有一些底气。
<br/>　　“那还等什么？大家一起出手！”
<br/>　　苏莫见此，立刻大喝一声，随即身上玄力鼓动，手中帝皇剑爆发出万丈剑光，一剑狠狠的劈向阵法气罩。
<br/>　　海明江同样立刻出手，没有任何废话，手中长矛疯狂的刺向阵法气罩。
<br/>　　“出手！”
<br/>　　“出手！”
<br/>　　“杀！”
<br/>　　霎时间，怒喝声四起，天海城无数的武者都出手了。
<br/>　　各式各样的攻击，五颜六色，千奇百怪，组成遮天蔽日的攻击洪流，如同长江大河一般，向阵法气罩攻击而去。
<br/>　　一起出手的人，足有数千万人，数千万道攻击，震撼人的心灵。
<br/>　　磅礴的威势席卷天地，强大到难以想象的地步。
<br/>　　所有人均是爆发出了最强的实力，为了活命，每个人都没有什么保留！
<br/>　　这些攻击之强大，足以瞬间将一名准帝强者灭杀成渣。
<br/>　　“呵呵！！”
<br/>　　面带这么多人的攻击，阳鼎天等人却是没有丝毫的担心，反而笑了起来，显然并不怕阵法被破。
<br/>　　“诸位，我们一起加持阵法！”阳鼎天对二魔老等人说道。
<br/>　　随即，二魔老、大魔老和殷棕几名准帝强者，纷纷出手，将玄力注入阵法之内。
<br/>　　不仅是五大强者出手，几人麾下的那些武皇强者全部都出手了。
<br/>　　阴阳神宫的武皇强者，还有黄泉魔宗的一些强者，以及殷家和北堂世家的一些强者，全部都出手了。
<br/>　　这些黄泉魔宗和殷家、北堂家的强者，数量虽然不多，但也有近千人。
<br/>　　这些强者就是诛灭天涯海阁总部，以及帝玄宫的强者，个个实力强大，高阶武皇都有不少。
<br/>　　五大准帝强者，和众多武皇强者，虽然大多数修炼的不是火属性玄力，无法加强阵法的威力，但他们的力量却是能最大程度的稳固阵法，让得阵法不会被破开。
<br/>　　轰隆隆！！！
<br/>　　阵法空间之内，浩大恐怖的攻击洪流，碾碎空间，狠狠的冲击在了阵法气罩之上。
<br/>　　一阵惊天动地的巨响，在阵法气罩上爆开，巨大的威力让得阵法气罩疯狂震动，严重的扭曲了起来。
<br/>　　啊啊啊！！
<br/>　　一声声惨叫声响起，猛烈的攻击不仅没有轰开阵法，反而让得阵法上无数的火焰迸射而开，将不少修为低的武者灼烧而亡。
<br/>　　其它人则是吓的立刻远离了阵法气罩。
<br/>　　这阵法气罩上的火焰，极为恐怖，就算是武皇境的武者沾染上，也是非死即伤。
<br/>　　“好强的阵法！”海明江满脸震惊之色，集合了数千万人之力，居然都轰不开阵法。
<br/>　　“是他们的加持之力太强了！”苏莫沉声道，对方有这么多强者加持阵法，才能让阵法保持不破。
<br/>　　而且，他们虽然有数千万人，但大多都是低级武者，武皇境武者都很少。
<br/>　　而对方的阵营，一名准帝就能抵的上无数的武者。
<br/>　　就在这个时候，异变再起，只见巨大的火炼八荒阵法，居然缓缓的收缩了起来。
<br/>　　大阵收缩的速度，虽然不算快，但也不慢，几乎是一个呼吸的时间就能收缩数十米。
<br/>　　阵法气罩所过之处，一切都被焚化成了虚无。
<br/>　　而且，阵法上的火焰之力，越发的强大，让得整个阵法之内的空间中，温度暴涨了百倍，而且还在继续增长之中。
<br/>　　“怎么办？我们怎么办？”
<br/>　　“完了！我们破不开阵法！我们都要完了！”
<br/>　　“我不想死！我不想死！”
<br/>　　“……”
<br/>　　无数的人哭丧着脸，如丧考妣，不断的呼喊。
<br/>　　“大家不要停，继续攻击！”
<br/>　　苏莫放声大喝，他手中东洲不停，一道道剑气疯狂的斩击在阵法气罩上。
<br/>　　不过，阵法在持续的收缩，变得越来越厚实，即便苏莫动用不朽奥义，也没有多大用处了。

<br/>　　没有人想死，整个天海城之中，无数的武者和苏莫一起，疯狂的攻击了起来。

<br/>　　而这个这个时候，阵法已经缩小了十分之一，阵法内火焰滔天，温度已经达到了骇人的地步。
<br/>　　啊！啊！啊！！
<br/>　　不断的有惨叫声响起，炽烈的高温，根本不是那些低级武者能够承受的。
