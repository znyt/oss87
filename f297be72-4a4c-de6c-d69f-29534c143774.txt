<br/>　　殷家参与了灭帝玄宫，罪无可赦，要怪只能怪殷家选择了一条死路。
<br/>　　啊啊啊啊！！

<br/>　　一个个殷家族人俱是身体瞬间干瘪了下去，瞬间变成了干尸。
<br/>　　扑通！扑通！扑通！
<br/>　　密密麻麻的人影从空中跌落，犹如是下雨一般，闷响声不断。
<br/>　　嘭！嘭！嘭！
<br/>　　更有无数的武者，身体直接爆炸，炸成了一团团血雾，连尸骨都不存在了，凄惨至极。

<br/>　　就算是一些武皇境一、两重的武者，都抵挡不住吞噬的力量。

<br/>　　“这是怎么回事？”
<br/>　　“救命啊！大长老救我！”
<br/>　　“不！！！”
<br/>　　无数惊恐的嚎叫声此起彼伏，震撼人的心灵，整个天殷城变成了修罗地狱。

<br/>　　但下一刻，他便看到了百万武者齐齐陨落的场景。
<br/>　　那干尸一般不断坠.落的人，那如妖艳花朵一般不断炸开的血雾，让他这名准帝强者，都不禁心神剧震。
<br/>　　“这……这是什么力量？”
<br/>　　海明江瞪大了眼眸，满脸不可思议之色，他自然也能感受到这股吞噬之力，这力量对于他来说，虽然不强，但着实不弱。
<br/>　　目光看向苏莫头顶之上悬浮的武魂，海明江震惊不已，这是什么武魂？居然这么恐怖？
