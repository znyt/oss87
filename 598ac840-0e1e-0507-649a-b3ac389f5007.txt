　　院落内，一座精美的凉亭之中，端坐着两名青年。
<br/>　　这两名青年之中，其中一人正是太史轩，而另外一人，是一名身藏青色长袍的青年。
<br/>　　这名青年个子不高，面貌较为普通，气质也远不如太史轩，但身上的气息深邃无比，仿若无尽的深渊，深不可测。
<br/>　　而聂乾和韩盈两人，此时正站在凉亭之前，脸色难看异常
<br/>　　苏莫刚一走进院落，太史轩和藏青色长袍青年，以及聂乾和韩盈，便将目光落在了他的身上。
<br/>　　“尤师兄、太史师兄，苏莫来了！”
<br/>　　蓝衣青年将苏莫带至凉亭之前，向藏青色长袍青年和太史轩抱拳。
<br/>　　太史轩挥了挥手，蓝衣青年便转身离去。
<br/>　　苏莫目光先是打量了凉亭中的太史轩两人一眼，随即看向聂乾和韩盈，沉声问道：“你们两没事吧？”
<br/>　　“苏兄，我们没事！”聂乾和韩盈摇了摇头，强自笑道。
<br/>　　不过，苏莫确实从两人的眼眸中，看出了深深的担忧。
<br/>　　“苏莫，我来为你介绍，这位是尤天韩大哥！”太史轩微微一笑，指向藏青色长袍的青年，向苏莫介绍道。
<br/>　　苏莫瞥了藏青色长袍青年一眼，微微点头，便将目光直视太史轩，开门见山的问道：“太史轩，你到底想怎样？”
<br/>　　“苏莫，你无需激动！”
<br/>　　太史轩摇了摇头，继续道：“我已经调查清楚了，我太史世家外族的廖云剑，上次和你们三人一起去了魔云海，然后便再也没有回来。”
<br/>　　“而廖云剑的兵器，在你的手中，苏莫，你没有什么话要说的吗？”
<br/>　　太史轩面上笑意连连，嘴角噙着若有若无的戏虐之色。
<br/>　　上次去拉拢苏莫之时，他并不清楚廖云剑是否死了，说被苏莫杀了，只是他自己的猜测而已。
<br/>　　但现在，他已经清楚了，廖云剑是死于苏莫和聂钱、韩盈三人之手。
<br/>　　“廖云剑的确是我杀的，但他想残害我们三人，死有余辜！”
<br/>　　苏莫冷漠的说道，他并没有狡辩，因为他知道狡辩也无用。
<br/>　　估计对方，已经将此事查清楚了。
<br/>　　而且，无论他是否狡辩，凭对方太史世家派系的实力，若是想杀他，根本不需要证据。
<br/>　　“不！廖云剑不是你杀的，是你们三人联手杀得！”太史轩闻言，立刻摇了摇头。
<br/>　　苏莫眉头微皱，对方这是什么意思？将他们三人全部拉下水？
<br/>　　转头看向聂乾和韩盈，苏莫目露询问之色。
<br/>　　聂乾见此，立刻开口道：“苏兄，你也不要为我们揽责任，廖云剑是我们三人所杀！”
<br/>　　“不错，有什么事，我们三人共同承担！”韩盈亦是如是说道。
<br/>　　苏莫闻言恍然，看来聂乾和韩盈，是想和他共同抗下此事，不想让他一人担责。
<br/>　　这不禁让苏莫有些感叹，这两人倒是有骨气、有胆色，但这样却会让他们都陷入其中。
<br/>　　“苏莫，如果我将此事禀告学府执法堂，你知道有什么后果吗？”太史轩笑着说道。
<br/>　　“什么后果？”苏莫眯着眼问道。
<br/>　　“你们三人都会死！”太史轩眸中寒光闪动，继续道：“只要我太史世家一脉，向执法堂稍稍施压，你们三人必死无疑！”
<br/>　　略一沉吟，太史轩继续道：“当然，听说你和火元圣师有些关系，但就算你能勉强保命，他们两人依旧要死！”
<br/>　　苏莫闻言，脸色彻底黑了下来，威胁！赤.裸.裸的威胁！
<br/>　　他的猜测果然不错，这一次对方就是专门威胁他的。
<br/>　　聂乾和韩盈的脸色，则是苍白了起来，面对太史世家一脉，他们没有任何反抗之力。
<br/>　　太史世家派系，在学府中势力极大，只是外府弟子中，加入太史世家派系的人，便不下六七千人。
<br/>　　若是再加上内府的弟子，都有接近万人了。
<br/>　　而他们两人，因为天赋一般，并没有加入任何派系。
<br/>　　“当然了，苏莫，我们也不想闹到如此地步，大家和平相处更好！”
<br/>　　太史轩摸了摸下巴，满脸笑意的继续道：“只要你加入我们太史世家一脉，我们便不追究此事了，廖云剑的死，就算是个意外吧！”
<br/>　　苏莫眼帘低垂，心中的愤怒简直无以复加，他恨不得一巴掌呼死对方。
<br/>　　不过，他并没有冲动，先不说他是不是太史轩的对手，其身边的那位尤天韩，绝对是更强大的人物。
<br/>　　这时，一直未开口的尤天韩开口了，他目光直视苏莫，沉声道：“苏莫，我也听说了你的天赋，的确是难得的天才人物，连我们太史世家一脉的圣子，都对你很是看重。”
<br/>　　“现在，摆在你面前的，只有两条路，一是加入我们太史世家一脉，大家共同进步，二是我们将你三人，送进执法堂！”
<br/>　　“怎样抉择，你自己看着办吧！”
<br/>　　尤天韩徐徐说完，便端起面前石桌上的一杯茗茶，小酌了起来。
<br/>　　苏莫心中怒火升腾，杀机涌动，这两人真是欺人太甚。
<br/>　　转头看了聂乾和韩盈一眼，见两人面色苍白，同样在望着他，苏莫顿时犯难了。
<br/>　　他倒是无惧对方，但若是不同意，怕是聂乾和韩盈两人，真的要死了。
<br/>　　这两人虽然和他相处的时间不长，但心思简单，为人重情重义，他已经将两人当朋友看待了。
<br/>　　若是害死两人，这不是他想要看到了。
<br/>　　“苏兄……我们……！”聂乾和韩盈两人吞吞吐吐，一副欲言又止的模样。
<br/>　　他们想当然不想死，但也不能要求苏莫为了他们，而向太史轩和尤天韩屈服。
<br/>　　苏莫心中沉思，少倾之后，他抬头看向太史轩和尤天韩，沉声道：“两位，给我几天时间考虑一番如何？”
<br/>　　他没有立刻答应，而是准备拖延几天，再想办法应对。
<br/>　　“当然可以，给你三天时间，三日之后，若是你不同意，那别怪我们无情了！”尤天韩说道。
<br/>　　苏莫闻言点了点头，随即招呼聂乾和韩盈一声，三人便一起转身离开了院落。
<br/>　　“短短几日，居然突破到了武帝境界！果然不凡！”待苏莫三人离去之后，太史轩低声自喃，他自然看出了苏莫的修为，已经突破到了武帝之境。
<br/>　　“的确不凡，他的玄力异常的浑厚，不下于武帝境三重武者！”尤天韩点头道。
<br/>　　随即，尤天韩略一沉吟，问道：“太史轩，我们如此威胁于他，就算他加入了我们太史世家一脉，估计也不会真心效忠吧！”
<br/>　　“这无妨！”
<br/>　　太史轩闻言摆了摆手，道：“天韩大哥，只要他加入了我们太史世家一脉，以后，自然有的是办法让他效忠！”
<br/>　　太史轩极为自信，太史世家是何等强大，苏莫日后入了太史世家，为了前途和实力，不效忠都不行。
