        &lt;h3&gt;第三卷第二百零三章  巧合&lt;/h3&gt;
<br/>
<br/>        凌辰拜访了刘诚之的父亲，也见到了宁雅的父亲，两位老人家对凌辰还比较赏识，尤其是刘诚之父亲神助攻的将凌辰和宁雅的关系透露给宁雅的爸爸，宁雅的爸爸才知道这些年他都没有关注宁雅的生活。
<br/>
<br/>        细细想了这么多年，宁雅也不是一无是处，工作上兢兢业业，不亚于现在的这个儿子，尤其是比宁玥更懂事理，本来自己家里有公司女儿还去别人公司上班，这样说出去好像有点丢人。
<br/>
<br/>        今天那个姓凌的小伙子还不错，如果宁雅真的也很喜欢的话不如撮合他们在一起，虽然家世没有宁家的强，看来小伙子很上进，男人只要上进也是很有前途的。
<br/>
<br/>        看看时间不早了就告辞刘父回家，并且邀请第二天让刘诚之带着董亚子和凌辰去他家里，这些年轻后生们要多和他们打交道，将自己的儿女们介绍给他们，以后都是不可多得的助力。
<br/>
<br/>        凌辰心里暗暗欢喜，宁雅的父亲邀请他们去他家里，虽然是一种客气的邀请，但是凌辰是一定要去的，这样就能离宁雅又近了一步。
<br/>
<br/>        刘诚之原本也是想要父亲帮帮凌辰的，这种顺水推舟的事情也就求之不得。
<br/>
<br/>        刘诚之对他爸爸说：“爸爸，我有事情想麻烦您。”
<br/>
<br/>        刘爸爸：“凌辰的事情？”
<br/>
<br/>        刘诚之：“什么都瞒不住你。”
<br/>
<br/>        刘爸爸：“我给你宁伯伯今天就提一下，明天你们可以去你宁伯伯家，我再打个电话给他，让他安排宁雅和凌辰见一面吧。”
<br/>
<br/>        刘诚之：“有爸爸出面，宁伯伯肯定会考虑他们的关系的。”
<br/>
<br/>        刘爸爸：“你宁伯伯摇摆不定，亚子和凌辰来的刚好是时候，也许对他会有好处。”
<br/>
<br/>        刘诚之：“你是怕他和那一家走的太近？”
<br/>
<br/>        刘爸爸：“他们原本也走得不远，只是我和你宁伯伯关系也很好，不忍心看着他走下坡路。”
<br/>
<br/>        刘诚之：“爸爸，你是在担心什么？”
<br/>
<br/>        刘爸爸：“今年落家和吴家的联姻会影响整个局势，一切怕是要等到过一段时间才能看清楚啊。”
<br/>
<br/>        刘诚之：“我上班之后会去甸城，厂家要求今年配合三一五运动，做好打击仿冒伪劣产品，我估计会在甸城呆一段时间。”
<br/>
<br/>        刘爸爸：“这几年据我的观察，甸城虽然说表面上一片和谐，估计不会那么简单，你过去需要小心点。”
<br/>
<br/>        刘诚之笑着说：“你看像我这样青葱的少年，谁会将我当回事？”
<br/>
<br/>        刘爸爸：“确实没有人将你当回事，不过你已经不是青葱少年了。好像亚子很喜欢你？”
<br/>
<br/>        刘诚之：“今天请你给别人当红娘，我的我自己搞定，你就不用操心我了。”
<br/>
<br/>        .......
<br/>
<br/>        刘诚之带着凌辰和董亚子去拜访宁家，而宁雅也带着陈子昂回到了她家。
<br/>
<br/>        新年宁雅没有回家，这个家对她来说，能记得她的也只有她的父亲，回来的时候家里人还是吃了一惊。
<br/>
<br/>        “小雅，这过年你也不回家，你出去干嘛了？”阿姨见到宁雅进门就质问宁雅，连身后的陈子昂也不打招呼，在她的影响中，一个过气了的女儿能有什么好的朋友，家里坐着的这几个才是人中翘楚。
<br/>
<br/>        在客厅的沙发上坐着刘诚之和凌辰及董亚子，宁玥和宁雅的父亲。
<br/>
<br/>        宁雅对凌辰到他家里有些吃惊，他怎么来家里了？但是看到刘诚之有些释然，都是生意场上的人，刘诚之带着凌辰来他家里拜访也无可厚非。
<br/>
<br/>        “我去韩国了，给爸爸说过了啊。爸、凌辰、诚之，你们来了。”宁雅并不知道董亚子。
<br/>
<br/>        “陈总。”宁雅的父亲见了陈子昂，站起来相迎，刘诚之和凌辰都站起来了，留下阿姨和宁玥有点发呆。
<br/>
<br/>        “叔叔好，早就说来拜访您，一直不得空，今天随着小雅一起过来，唐突了。”陈子昂微笑着和大家一一打招呼。
<br/>
<br/>        “子昂，你先坐，我回房间放东西。”宁雅没有想到今天家里这么热闹。
<br/>
<br/>        宁雅很久没有回来，到了自己的房间，看着房间冷冷清清的似乎很久都没有人打扫的样子心里有些难过，她走了，房间都没有人打扫，如果不是有意为之，家里的保姆怎么可能不打扫卫生呢？
<br/>
<br/>        不过时间也不允许她伤感，陈子昂还在客厅坐着呢。
<br/>
<br/>        凌辰见宁雅回来了，那张冰坨子的脸上像如沐春风般温柔起来，整个人一下就感觉热情洋溢起来了。
<br/>
<br/>        “陈总年轻有为，短短几年已经是你们行业上耀眼的一颗星了啊，真是后生可畏，你看看我们家的孩子，年龄和你差不多。”
<br/>
<br/>        “宁叔叔就叫我子昂好了，我比小雅大几岁呢。”陈子昂礼貌的笑着，确实她比宁雅要大了几岁，而且今天是随宁雅一起来的，不算是商业上的应酬，纯粹就是一个朋友之间的来往。
<br/>
<br/>        当陈子昂随宁雅进门的时候她就了解到了宁雅在家里尴尬的处境，所以她就低调的和宁雅的父亲应酬着。
<br/>
<br/>        不过她不了解刘诚之和凌辰还有董亚之之间的关系，感觉微微有些尴尬，等宁雅从楼上下来了，宁雅的父亲就给他们做着介绍：
<br/>
<br/>        “这个是山西凌辰，这个是刘诚之，这个是贵州的董亚子，也是诚之的女朋友。”宁雅的父亲为陈子昂做着介绍，同时也是给宁雅介绍。
<br/>
<br/>        陈子昂是认识凌辰和刘诚之的，就是不认识董亚子。
<br/>
<br/>        “陈总，没想到今天能见到你呢，你可是我的偶像，曾经被我们老师拿来做教材的。”董亚子心直口快。
<br/>
<br/>        “是吗？就我这样还能做教材啊？”陈子昂几年前就做过论文，也曾经被有关的金融系拿来做教材，这个她自己也很清楚。
<br/>
<br/>        凌辰的眼神一直都没有离开过宁雅：“雅儿，你好像瘦了些？”
<br/>
<br/>        凌辰轻声的喊着宁雅，称呼亲密，让宁雅微微一愣。
<br/>
<br/>        “没有瘦。”宁雅礼貌的笑着，他们是同学关系了呢。
<br/>
<br/>        “诚之都带着女朋友，你怎么没有打女朋友？”
<br/>
<br/>        “凌辰哥哥的女朋友不是你吗？”董亚子脱口而出，她可是耳朵里听出来厚厚茧子。
<br/>
<br/>        “是啊，凌辰哥哥的女朋友是你啊。”刘诚之也笑着说。
<br/>
<br/>        “我......”宁雅刚想开口被她父亲打断掉：“小雅啊，都谈了男朋友不给我们说，这样不对啊。”宁雅的爸爸笑着说。
<br/>
<br/>        “凌辰哥哥，雅姐是你的女朋友？那我？爸！你不是.....”
<br/>
<br/>        “兰乔，今天很开心，安排厨师好好做几个菜，我和这些年轻人一起喝几杯。”宁爸爸吩咐他的老婆兰乔，留下几个年轻人在家里吃顿饭，家宴不同于寻常商业的应酬，这样更能促进彼此的感情交流。
<br/>
<br/>        至于宁玥说的话，可以暂时忽略。
