<br/>    紫霞仙子看向自己手中的丹药，狠狠道：“宗师现在都是在为我争取时间，我绝对不能拖累了宗师，要尽快突破到超脱！”
<br/>    
<br/>        道完，紫霞仙子已经是回到了自己临时开辟出来的洞府之中，在一块青石上坐下，盘膝开始炼化丹药。
<br/>    
<br/>        陈明行走在漆黑的暗流通道之中，这里，相当于是一条暗河，通向外界的九幽寒泉。
<br/>    
<br/>        在前方，有着一颗颗夜明珠闪耀，点缀在洞壁之上，解千愁一身黑袍，盘膝坐在一张桌案之前，桌案之前，放置着两个白瓷杯，还有着一壶酒，在解千愁的右手处，一柄长刀插在他的右手处。
<br/>    
<br/>        解千愁独自饮下一杯酒，看向陈明，朝着陈明冷眼看去，“想必这位便是最近突破的新君吧，魔宗十六宗主解千愁，请君对饮。”
<br/>    
<br/>        陈明也丝毫不惧，走到桌案前盘膝坐下，水流在两位大君的威势之下散开，靠近不了两人分毫，卓青瑶也是在两人十步开外，背着荒古巨剑默默站立。
<br/>    
<br/>        陈明看了一眼解千愁，这一次却是看到了更多，并不像之前一般有很多问号，是因为自己的实力提升了，能够看到的更多了吗？
<br/>    
<br/>        在解千愁的身上，分明有着两个光环：
<br/>    
<br/>        
<br/>    
<br/>        介绍：君者，威临天下！
<br/>    
<br/>        效果：对超脱之下的存在拥有部分压制之力。
<br/>    
<br/>        
<br/>    
<br/>        介绍：秉承死忠之节，岂因生死而惧之？
<br/>    
<br/>        效果：开启之后不畏生死，进入狂暴状态，攻击提升20%，防御降低30%。
<br/>    
<br/>        看来能够成大君的都不是一般人物，都是有着光环的存在，这个，倒是也有点像打鸡血状态，当一个人怒气满满的时候，力量会大很多，但是对于身体的负荷也会增大。
<br/>    
<br/>        至于这个光环，陈明估计是超脱都有的一个基本光环，不然，危险指数只差五百的情况下，大君是怎么一招秒了悟道榜上的天才的？
<br/>    
<br/>        陈明有着，倒是丝毫不惧这个君威。
<br/>    
<br/>        解千愁倒下一杯酒放在陈明的面前，淡淡道：“阁下修行到大君的修为，想必也是数百年的苦修，数百年的苦修，数百年的风霜寂寞，就要在此毁于一旦，阁下甘心吗？”
<br/>    
<br/>        陈明端起酒，一口饮尽，然后笑道：“哪里花了数百年，只是花了三年左右而已啦！”
<br/>    
<br/>        解千愁的手微微一颤，不是，你这人会不会聊天？这要怎么接啊？我本来还想说你修行数百年不易，这么死了多可惜，不如投入我魔宗，成为一个宗主，岂不是两全其美？
<br/>    
<br/>        “额，嗯，我听魔宗中的弟子说，阁下也是修行过魔功的，想必对我魔宗并无偏见，既然愿意修行我魔宗功法，想必阁下也不是很在意门户之见，不如加入我魔宗，我魔宗大宗主胸怀天下，海纳百川，必定能够给出一席魔宗宗主之位！如此，也好过在此，化作一捧黄土！”
<br/>    
<br/>        解千愁的声音很大，似乎是想要让洞口之后的人王们也听到。只要陈明犹豫片刻，那么就可以乱其军心，军士不相信将领，我倒要看看你这一仗怎么打？
<br/>    
<br/>        陈明可谓是丝毫犹豫都没有，我投靠你魔宗了，我上哪儿去找那么多冤大头买丹药？“放屁，我堂堂仙门正道，怎么可能与你魔宗同流合污，浩浩天下，朗朗乾坤，我虽然只是一个大君，我或许会死，但是，死了我一个正道，还有千千万万个正道！这天下的正道，你杀之不尽！我陈明，虽然身单力薄，也必要还这天下一个乾坤浩荡！”
<br/>    
<br/>        在后方的人王们本来心中还有着担心，毕竟陈明只是刚刚突破到大君，而对面的那一位，可是魔宗大宗主的师弟，更是君临已久，新君没有大君之间的战斗经验，很有可能不是对手。解千愁更是魔宗大宗主的师弟，他的话还是很有分量的，一时之间,他们也是有些担心陈明投靠了魔宗。
<br/>    
<br/>        然后，他们便在不到一息的时间之后听到了陈明的话，“宗师，当真是我正道楷模！”
<br/>    
<br/>        “宗师，一身正气，何惧魑魅魍魉！”
<br/>    
<br/>        “有宗师这般仙门正道，我道不孤！”
<br/>    
<br/>        解千愁的面色一寒，听门下弟子说，这个家伙行事倒是多有魔宗风范，却是一个正道死忠？还真是怪胎！自己一计不成，反倒是替陈明刷了一波声望，“你真的想要找死不成？”
<br/>    
<br/>        陈明抬手道：“来，我们试试看！”
<br/>    
<br/>        解千愁握住刀柄，然后道：“这一柄刀，是师兄亲手交到我手里的，它的名字叫做斩草，神兵榜排名三十一，就像是你背后这些废物，在我面前就如同枯草一般弱小，既然你要寻死，我便成全了你！”
<br/>    
<br/>        锵！
<br/>    
<br/>        长刀出鞘，刀面如水，波光粼粼，神兵之威凛然降临，魔气荡漾，万道天竹轻摇，一缕青风出现，一时之间，漫天竹叶飞舞，与魔气交织在一处。
<br/>    
<br/>        解千愁看向陈明手中的万道天竹，轻蔑地笑笑，“不过是还没有彻底成型的仙器雏形而已，也敢与斩草争锋？”
<br/>    
<br/>        单臂一提，斩草锋鸣不止，那一抹刀光，如同能够割开天地，朝着陈明劈去，陈明抬手，万道天竹横挡，格住了斩草。
<br/>    
<br/>        斩草落在万道天竹之上，并没有留下半点痕迹，解千愁露出一抹欣赏的神色，“当真是好一柄仙器雏形，只是仙器雏形，竟然能够和我的斩草争锋！”
<br/>    
<br/>        陈明向来不是喜欢说废话的人，作为一个大boss，要有着杀人不说话的基本素养，在陈明的身后，八十八颗星辰凝聚而出，穆然间已经携着风雷之声飞向了解千愁。
<br/>    
<br/>        解千愁眼中一急，抬腿便是一脚踢飞了桌案，后退了三里。
<br/>    
<br/>        这星辰只是凝聚出来他就能感受到其中的威势，绝对能够伤到自己，对面那个带着面具的家伙，什么时候悄悄念得口诀？是面具挡住了，他是无声念出来的，还真是行事有我魔宗的风范啊！
<br/>    
<br/>        此时，一颗星辰已经到了解千愁的面前，解千愁当空而起，一刀斜撩，只是一刀，便是已经将身前的这一颗星辰直接劈成了两半！
