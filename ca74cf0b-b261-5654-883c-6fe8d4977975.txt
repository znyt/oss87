　　慕寒扪心自问，假如把自己换成叶少阳，会怎么做。
<br/>　　别说自己想不到这样的计策，就算想得到，怕是也不敢冒这个险吧，多半会是让大家掩护自己突围……
<br/>　　“包子用神识控制着穷奇，如果杀了穷奇，他的神识也会受到反噬，后果不堪设想，因此我才放走了穷奇。”叶少阳望着包子，却对大家说道：“秋莹不会白死，我一定会为她报仇。”
<br/>　　“妈的，小叶子，干脆咱们杀上太阴山吧，连带无极鬼王也一起灭了！”小马愤然骂道。
<br/>　　“好。”叶少阳出人意料地答应下来。
<br/>　　“叶天师，了不起！”冲和子走过来跟他打招呼，神色十分激动，之前，当叶少阳他们被困在重重包围之中，所有人都认为他们必死无疑了，冲和子自然也这么想，结果叶少阳出奇谋将所有人带回，让冲和子内心震撼，也对叶少阳充满了景仰之情。
<br/>　　“诸位都劳累过度，先随我去草堂休息吧，别的事稍后再说。”
<br/>　　徐文长也冲叶少阳点头。
<br/>　　叶少阳本来也想了解有关海眼的更多，于是带领大家一起，跟在冲和子后面，走到了高台的另一面。
<br/>　　这是他们第一次来到岛屿的另一面，丛林之中，到处弥漫着雾气，叶少阳敏锐地感知到雾气中充斥着浓郁的灵气，对任何生灵来说，都是绝佳的修炼场所，这一点与岛屿之外灵力贫瘠的整个南海形成了鲜明的对比。
<br/>　　树林里，有很多草庐，是用树皮和茅草搭建的，上圆下方，外形有点像是蒙古包，放眼看去多的数不清，都散落在丛林中。
<br/>　　冲和子叫来几个弟子，分别带叶少阳他们去不同的蒙古包休息。
<br/>　　“我们要三四个就好了，三四个人挤一个，这样安全一些。”
<br/>　　冲和子笑道：“海眼都被激活了，还有什么危险的。”
<br/>　　叶少阳淡淡地瞟了一眼跟在他们身后不远处的慕寒等人，道：“那可不一定，还是小心点好。”
<br/>　　最后冲和子给他们找了几个连成一片的草庐，捉鬼联盟两人一间，进去休息。说是休息，他们也不用睡觉，主要就是吐纳调息，他们从叶少阳到下，所有人都受了伤。尤其是四宝，一直昏迷不醒，叶少阳之前检查过了，他元神有损，需要养息恢复，于是把他弄到自己的草庐里，先自己调息了两个周天，大体恢复了，再帮四宝养息元神。
<br/>　　很快四宝也醒了，半晌才缓过神，瓜瓜跟他讲了整件事的经过，四宝得知秋莹之死，也潸然泪下，半晌不做声。
<br/>　　叶少阳走到外面的草地上，吹了一声口哨，不一会工夫，大家全都来了，看他们的样子，基本上都恢复了，只是因为秋莹的死，一个个情绪低落。
<br/>　　“四宝，你之前在高台上，靠一己之力守住高台好几分钟，你是怎么做到的。”林三生想起当时的情况，问道。
<br/>　　“我在轩辕山的时候，鸠摩祖师教了我完整的诸天观想术，又给我启发，把诸天观想术与罗汉金身融合在一起，我现在可以观想出任何一尊神佛，都有金身罗汉加持，如同实质一般的存在。”
<br/>　　说到这个，四宝语气中带着一种骄傲，“鸠摩祖师给我讲解了须弥天，我现在能观想出须弥天，其中装满诸天神佛，只要中了我的观想术，就会进入须弥天，在漫天神佛面前，自己内心的弱点也会最大程度放大，用游戏术语来说，这叫弱点攻击。所以我才要少阳给我C位，没让你们失望吧。要没我之前守着高地，泉水早就被打爆了！”
<br/>　　叶少阳看他眉飞色舞的样子，打压了他几句，不过内心还是为他感到高兴。
<br/>　　林三生回想着当时的情景，结合四宝之前所说的，问他：“照你这么说，你这诸天观想术，跟包子的通灵梦有些类似？”
<br/>　　四宝想了一下说道：“那不一样，我这是法术，耗费的是法力，而且也不会制造什么迷惑人的幻境，进入幻境的人会清楚知道是幻境。”
<br/>　　两人继续讨论有关法术的学术问题，吴嘉伟也在一旁仔细听着。
<br/>　　叶少阳找到杨宫梓，看着她说道：“今天多亏你赶来帮忙。”
<br/>　　“你说这个，就是把我当外人了，再说我也没帮上什么忙。”
<br/>　　叶少阳知道她是在谦虚，看上去她跟小马没帮什么忙，但多了他们两个强者，情况当然不一样，少了一两个人，之前对穷奇的围攻也未必能造成杀伤，量变累计产生产生质变。
<br/>　　“你们风之谷现在怎么样，我之前听说，道风让你们全都躲起来了？”
<br/>　　杨宫梓抱怨道：“你那师兄是个甩手掌柜，他自己躲在山河社稷图里修炼，什么事都交给我们，现在太阴山大军横扫鬼域，风之谷损失惨重，好在核心还在。”
<br/>　　“损失惨重？”叶少阳皱眉，“道风不是让你们躲起来吗？”
<br/>　　“你说的轻松，几十万大军，往哪里躲，大部分都走失了，只剩下一些精英还在。”
<br/>　　小马抱怨道：“孤的白云城啊，也陷落了。”
<br/>　　杨宫梓也叹道：“招兵买马这么久，一下子又打回原形了。”
<br/>　　叶少阳等人大为震惊，问她怎么会这样，杨宫梓解释道：“道风创建风之谷的目的，是建立在太阴山先侵略的人家的基础上，他从来没想过保护阴司，假如太阴山对人间作战，阴司就会展开总攻，在鬼域拖住太阴山，风之谷则侧面侵袭，寻找一剑封喉的机会。
<br/>　　谁也没想到太阴山会主动出击，跟阴司硬碰硬，连酆都大帝也没有料到，所有诸侯势力都被冲散，风之谷也是没办法。”
<br/>　　“原来是这样，那真是可惜了。”叶少阳叹道。
<br/>　　杨宫梓淡淡一笑，道：“还有一支精锐在的，那是道风的心腹部队，而且四大灵兽还在，这就还好。”
<br/>　　叶少阳心中一动，道：“什么四大灵兽，是白狐他们吗？”
<br/>　　“朱雀白狐，青牛玄武，这四人可是道风的命根子，所有人都在保护着他们。”
