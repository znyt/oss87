　　涂小安重新回到副院长办公室密谈了一会儿，二个小时后，他从古学院中光明正大的走了出来。
<br/>　　青枭也被涂小安收入了系统之中。
<br/>　　近些天，他没准备回古学院的打算了，有不少事情要做。
<br/>　　刚来到市中心的街道上，涂小安就接了一个电话。
<br/>　　“喂，来我店喝牛汤”
<br/>　　电话另一头，传来一道含妖含俏的女子声音，很动听。
<br/>　　涂小安直接道：“不去”
<br/>　　“为什么”
<br/>　　“我不想去你店里，看到你那张丑脸”
<br/>　　“咯咯...那你是喜欢看我没“化妆”的脸吗”电话那张传来花枝招展的声音。
<br/>　　涂小安：“呃...算是吧”
<br/>　　简单说了几句，挂掉电话，涂小安想了想，还是决定去古街面馆一趟，说起来牛妖人可是他干掉的。
<br/>　　白白送给了落子烟，不去喝几口牛汤，好像有点太便宜那个变态女人。
<br/>　　出来的时候，就已经快夜幕降临，吃晚餐的饭点到了，反正免费。
<br/>　　刚准备朝着古街走去，涂小安的身子微微的一顿，一下子蹙眉起来。
<br/>　　有人跟踪他。
<br/>　　而且看样子一出古学院，就跟踪了上来。
<br/>　　有意思，看来想清闲会都不行。
<br/>　　涂小安换了一个方向，就好像在逛街似的，慢慢的朝着一片小区走去，二十分钟后，他来到了一栋独立的豪华别墅。
<br/>　　未进去之前，他一个转身，浅笑吟吟道：“既然都跟来了，就到我的新家坐坐吧”
<br/>　　一道挺拔的身影出现，看了看他：“那就客随主便”
<br/>　　两人一起走近了豪华的别墅，大院内，涂小安看着跟踪自己的人，他不是别人，正是刚死了弟弟，超等天才班的大神学员罗子杰。
<br/>　　果然，这个人没那么好糊弄，怀疑自己了。
<br/>　　“你有什么话就问吧，学弟我尽量知无不言”涂小安笑呵呵的道。
<br/>　　出了古学院，他无所顾忌。
<br/>　　罗子杰其实并不想惹这位最强新生，因为白君然败在这个人的手上，足够说明一切。
<br/>　　但他现在没的选择，弟弟惨死，他要知道真相。
<br/>　　阴郁着一张俊逸的脸，罗子杰没闲情雅致先大量这豪华别墅，直接问：“我弟弟到底是怎么死的”
<br/>　　“被蛇妖所杀”涂小安不带想的回。
<br/>　　“你别告诉我，就是那条巨蟒，我知道不是，那条巨蟒根本不是什么三阶妖灵级别”
<br/>　　涂小安点了点头：“确实不是那条巨蟒所杀”
<br/>　　“真正的蛇妖在哪”罗子杰眼中一片寒冷，他就知道如此。
<br/>　　“别急，我给你看样东西吧”涂小安意念转动，他的眼前空间波动起来，凭空的出现了一头通体银白，威风凛凛的巨狼。
<br/>　　罗子杰看到巨狼，诧异的脱口而出：“银月巨狼”
<br/>　　这是他弟弟的战宠，他在熟悉不过。
<br/>　　“银狼拜见主人”银月巨狼冲着涂小安尊敬的叫了一声，非常乖巧。
<br/>　　“主人？”
<br/>　　自己弟弟的战宠，居然管他叫主人，罗子杰简直五雷轰顶了一般，突然想明白了什么，愤怒道：“原来我弟弟是你杀的，你杀了他，夺了他的战宠”
<br/>　　“既然如此，你为什么说我弟弟是蛇妖所杀”
<br/>　　“是蛇妖所杀没错，而且蛇妖就远在天边近在眼前”
<br/>　　“....”罗子杰一楞，一时不懂对方说的什么意思。
<br/>　　涂小安见对方既然兄弟情深，没理由不让这位哥哥知道弟弟死的真相。
<br/>　　他做事还是很厚道的。
<br/>　　他嘴角微勾，身上悄无声息的散发出了一股恐怖的蛇类气息。
<br/>　　这种气息一出现，蓦地，罗子杰怔了一下，短促而痉挛地呼了一口气，一双眼睛慢慢的睁大。
<br/>　　几个呼吸后，罗子杰的一张俊逸的脸立刻扭曲了起来，眼神凌厉，杀意如狂风骤射。
<br/>　　“你这个最强新生既然是化形的蛇妖”
<br/>　　这对他简直就是平地惊雷，将罗子杰炸的懵逼了，不可思议，太不可思议了，饶是罗子杰想了很多种可能，但独独没想到这点，只能说他脑洞还不够大。
<br/>　　副院长对着整个古学院的人说谎，居然为了隐藏这真正的蛇妖。
<br/>　　他老人家为什么要这样做，罗子杰想不明白，不过，现在他也没心情想那些。
<br/>　　杀死弟弟的凶手就在眼中，他体内的元力爆涌出来，怒喝：“谢谢你这条蛇妖如此坦诚，我会尽量让你死的舒服一点”
<br/>　　“你比白君然如何”涂小安忽然问。
<br/>　　罗子杰不假思索道：“伯仲之间，难分高下”
<br/>　　“既然如此，那白君然都败在了我的手中，你觉得你是来给你弟弟报仇，还是来送死的”
<br/>　　此话一出，罗子烟闻言哈哈大笑，眼中血红：“弟弟之仇，不共戴天，蛇妖，今天，我必杀你，为我弟弟报仇”
<br/>　　到是一位好大哥，涂小安耸了耸肩膀。
<br/>　　“今天我打了一天了，不想在动手”涂小安看着他：“况且，你也不值得我动手”
<br/>　　罗子杰元力爆涌，周围掀起阵阵狂风：“蛇妖，你不要太过自大，你纵然是三阶妖灵，我罗子杰也要把你碎尸万段”
<br/>　　“好，那就让我先看看你的实力吧”
<br/>　　“牛妖王何在”
<br/>　　别墅二楼处，传来声如洪钟的声音：“老牛来也”
<br/>　　轰...
<br/>　　一个浑身横肉的大胖子跟千斤顶一样的从二楼坠下，地面震了三震。
<br/>　　“打败我的战宠，我就任由你处置”涂小安看着愤怒的罗子杰开口。
<br/>　　“战宠？”
<br/>　　罗子杰看着这个从天而降的大胖子，他身上的妖气可谓滔天，这很显然是一只化形的三阶妖灵。
<br/>　　“啧啧...人类小娃子，敢来杀我老牛的主人，你怕是活到头了”牛妖王眼中满是轻蔑，看他就跟看一个死人一样。
<br/>　　一个还未达到三阶生灵的人类修武者，也敢来到这里，谁给他的勇气。
<br/>　　“学长，你可要加油我，我这战宠力气很大，别被他打死”涂小安坐到了银月巨狼的背上，朝着一边走去。
<br/>　　收服的三阶妖灵，总算要派上用场。
<br/>　　什么事情都要自己做，还挺累的。
<br/>　　牛妖王的强大气场，让罗子杰这位天才班的大神学员背后冷汗涔涔。
<br/>　　在学院中，他威风八面，可出了学院，就是弱肉强食了。
<br/>　　在三阶妖灵面前，他好像还有点不够资格。
