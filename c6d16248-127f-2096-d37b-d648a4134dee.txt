　　听到屋内清冷的声音，慕清霄微微一笑，声音清朗道：“故人来访，小倩姑娘不打算请我进去吗？”
<br/>　　话语刚落，屋内陷入宁静，随后就响起一阵脚步声。
<br/>　　“吱吱……”
<br/>　　当房门打开，映入眼帘的是一名身着雪白薄衫的身影。
<br/>　　肌肤如玉，黑丝如瀑悬挂及腰，绝美容颜惹人怜惜。
<br/>　　聂小倩怔怔的望着门外的身影，秋水眸子中闪过一抹温柔。
<br/>　　慕清霄打量了她一眼，美女真是怎样都看不腻，更何况聂小倩身上本就具有一股大家闺秀的气质。
<br/>　　“这两年，过的可好？”
<br/>　　聂小倩微微颔首，将门打开让他进入闺房。
<br/>　　能让慕清霄进入自己的闺房，可见聂小倩在心底里已经接受了他，毕竟两者已经有过夫妻之实。
<br/>　　“托郎君的福，若非郎君，小倩依旧是孤魂野鬼，连个安身地都没有，现在与姐妹们谋生，日子虽平淡，倒也不错。”
<br/>　　跟在她的身后，一缕幽香就钻入鼻中，让慕清霄觉得有些痒痒。
<br/>　　慕清霄打量了一眼闺房，微笑道：“原本我还有些担心，看来你的修炼没有落下，金丹一重已经有些自保的能力。”
<br/>　　“嗯。”
<br/>　　当初你离开后，我和小蝶四处飘泊，天下之大可终究没有能让我们安身的地方。
<br/>　　毕竟，我们是孤魂野鬼。
<br/>　　后来，我与小蝶重返兰若寺，取出了自己的骨灰盒，发现有些姐妹并未离开，于是就将她们聚集了起来。
<br/>　　随后就回到京城，开设了似水年华。
<br/>　　京城中很危险，除魔师也有许多，好在小倩修炼更进一步，他们不敢随意招惹。
<br/>　　姐妹们也未重操旧业，虽说平日里会吸收一些男子的阳气修炼，但并没有触及底线，才足以在京城安身立命。
<br/>　　话语从嘴中说出来很简单，但谁也不知道她们经历了多少坎坷。
<br/>　　聂小倩转过身来，美眸凝视着他，粉/唇轻启：“郎君回来，准备何时离去？”
<br/>　　“很快就要离开，回来只是看一眼，顺便给你带了些东西。”说罢，慕清霄取出一枚纳戒递了过去。
<br/>　　看了眼纳戒中的物品，聂小倩瞳孔一缩，惊呼道：“好浓郁的阴气，究极……”
<br/>　　“玉瓶中的都是鬼丹，乃是天地间最精纯的阴气，相比阳气，鬼丹更利于你修炼，这批鬼丹足以让你修炼上百年。”
<br/>　　说着，慕清霄伸出手，温柔的抚摸着她的脸颊，肌肤有一些冰凉。
<br/>　　“罗刹鬼莲据说是强大的鬼修凝聚而成，对魂体修炼事半功倍，这些东西切勿保密，毕竟防人之心不可无。”
<br/>　　抬起玉手抓住慕清霄的温暖的手掌，聂小倩心中划过无尽的暖流，美眸中柔情似水。
<br/>　　她是聪明女子，当然知道慕清霄说回来看一看，其实就是专门给她送来了这些东西。
<br/>　　心中最后一丝抵触也在顷刻间消散，能有这样一个在乎自己的男人，聂小倩觉得就算为他魂飞魄散也是值得。
<br/>　　如同高冷的猫咪，放下心中的所有警惕，聂小倩身躯渐渐向慕清霄靠拢。
<br/>　　望着靠在怀中的佳人，慕清霄嘴角牵起一抹弧度，轻轻挑起她雪白的下巴……
<br/>　　如此近距离的望着慕清霄俊逸如画的面容，聂小倩一时间不由的有些痴了，等到回过神来，后者的面容已经近在咫尺。
<br/>　　顿时，绝美的容颜涌上两团绯红，羞涩无比。
<br/>　　慕清霄轻轻搂住她的纤腰，在聂小倩娇/羞扭捏的摸样下，温柔的吻了下去。
<br/>　　顿时，一种酥/软冰凉的感觉就回荡在心间。
<br/>　　聂小倩早已经不是懵懂少女，自从第一次被慕清霄夺走后，在这两年间日思夜想。
<br/>　　终于，寂寞在这一刻按耐不住，极力的迎合起慕清霄。
<br/>　　慕清霄直接将聂小倩轻轻抱起，拥/吻的同时，缓步向床榻上移去，同时一层无形的结界将卧室笼罩。
<br/>　　不多时，卧室中就响起一阵令人遐想的婉转低吟，春光无限。
<br/>　　整整一个时辰后，云收雨歇。
<br/>　　聂小倩趴在慕清霄宽广的胸膛上，脸颊透着绯红，美眸痴痴，满足无比。
<br/>　　回过神来，聂小倩如猫咪一样缩在慕清霄怀中，清冷的语气中携着无尽的温柔。
<br/>　　“夫君，何时回来？”
<br/>　　“何必在这里等，不如随我离开，以后我们永远都会在一起。”
<br/>　　搂住她的香肩，让她舒适的躺在自己怀中，慕清霄柔声说道。
<br/>　　聂小倩不愿现在跟他离开是有原因的，因为她的父母尚在人世，并且有一个十多岁的妹妹。
<br/>　　他们乃是京城的大户人家，虽算不得高官，但在朝廷中也有一点地位。
<br/>　　按照聂小倩的意思就是，她想在这里守护自己的亲人，直到父母去世才会离开。
<br/>　　“你不用担心我，有空回来看看，毕竟是我这个做女儿的不孝在先。”
<br/>　　闻言，慕清霄并不打算强求，沉吟片刻道：“等有空就回来，好好修炼，记得管束好酒楼中的女子，免得牵连了你。”
<br/>　　“嗯，我知道。”
<br/>　　柔声回了一句，聂小倩就躺在慕清霄怀中香甜的睡去。
<br/>　　待聂小倩沉沉睡去，慕清霄替她捋了捋发丝，星眸中浮现原子图案，在她身上留下一道极强幻术。
<br/>　　若是有强者对她不利，在聂小倩不敌的情况下，幻术会直接将敌方抹杀。
<br/>　　随着灵魂力量的增强，慕清霄的瞳力也就越强，永恒万花筒早已经超越了火影位面的限制。
<br/>　　布下幻术，慕清霄俯下身子在她玉额上轻轻一吻，穿戴起衣服，随后转身离开卧室。
<br/>　　打开卧室门的一瞬间，门外就响起一声痛哼。
<br/>　　来到门外，只见小蝶蹲在地上捂着额头，大眼睛里泪水打转，慕清霄有些无语，这小丫头竟喜欢听墙角。
<br/>　　抬起臻首，看到慕清霄后，小蝶露出一个傻乎乎的笑容。
<br/>　　“慕公子，你要离开了吗？”
<br/>　　微微点头，慕清霄取出一个空间玉瓶，其中有百颗鬼丹，直接递给了小蝶。
<br/>　　“好好照顾小倩，跟在她身边算是你的福气。”
<br/>　　说罢，慕清霄就转身离开，只留下小蝶一人拿着玉瓶发出黄鹂一般的笑声。
