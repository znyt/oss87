        恍然间，落以柔才意识到一个问题。
<br/>
<br/>        网络曝光的苏漫雪与傅夜沉的亲密照，照片里的女主角其实压根就不是苏漫雪，而是她童筱颖！
<br/>
<br/>        那么至于李忆莎为什么会突然单方面的提出悔婚，想必一定也跟这个童筱颖有关！！
<br/>
<br/>        童筱颖这个女人太可怕了不是吗？她一定是耍了什么手段，才让大哥上当的！！！
<br/>
<br/>        在落以柔看来，比起苏漫雪和李忆莎，童筱颖这个女人更加令她觉得心机深重。
<br/>
<br/>        她家大哥怎么就那么傻，居然愿意牺牲自己的幸福，去给这个心机婊当复仇工具！
<br/>
<br/>        落以柔感慨不已，替傅夜沉感到万分不值。
<br/>
<br/>        同时，她的心情也一落千丈。
<br/>
<br/>        尽管她心情不好，但是她仍旧会把自己的这一天的工作给认认真真地做好。
<br/>
<br/>        只是最近，落以柔觉得自己在傅夜沉面前总是有种患得患失的错觉。
<br/>
<br/>        童筱颖离开傅氏集团大厦后，本想去买高铁票回云城，但是肚子仍旧痛得有些难受，她又不得不先放弃这个念头，先去医院看医生。
<br/>
<br/>        医院里弥漫着消毒水的气味……
<br/>
<br/>        童筱颖拿着各种检查化验单走出了急诊室。
<br/>
<br/>        医生说，她只是吃得太饱，又加之少了胃后面的那一颗肾，且受到外力重压，造成肠胃轻微位移，导致胃痉挛以及缺肾的那块地方空着痛而已。
<br/>
<br/>        所以，归根究底……
<br/>
<br/>        她以后还是不能吃太饱，不能做剧烈运动……
<br/>
<br/>        和之前在澳洲给她看诊的那个医生一样，都是这样建议她日后所需注意的事项。
<br/>
<br/>        童筱颖捂着肚子，坐在了急诊室门外走廊的休息椅上，忽然间回想起了以前在澳洲与小卫生活在一起的那些日子。
<br/>
<br/>        好像，每次她不舒服，都是自己先去看的医生，小卫事后才来医院里找她，然后接她一起回家。
<br/>
<br/>        这次，不会再有小卫来接她回家了。
<br/>
<br/>        直到一双锃亮的黑皮鞋，走入了她的视野中。
<br/>
<br/>        往上是极具垂质感的西裤，剪裁合体，衬得两腿格外修长。
<br/>
<br/>        “小卫！”童筱颖欣喜地抬起头来。
<br/>
<br/>        对上了一双如潭水般深沉，眸光却流溢着焦虑的眼睛。
<br/>
<br/>        干净利落的短发略微凌乱，深邃立体的五官，容貌英俊，倒是与她的小卫神似。
<br/>
<br/>        可他不是她的小卫。
<br/>
<br/>        而是……
<br/>
<br/>        “大哥。”童筱颖干笑着改了口。
<br/>
<br/>        傅夜沉垂着深邃的黑眸，看到童筱颖手中拿着的单子，二话不说地夺到自己手中，一张一张仔仔细细地翻看了起来，直到他看到医生的诊断书后才松了口气。
<br/>
<br/>        但下一个瞬间，他刚舒展的眉心，又蹙了起来，整个人有些微微晃神。
<br/>
<br/>        “你为什么只有一个肾？”他说这句话的语气，令她无法用语言去形容。
<br/>
<br/>        似乎是心疼又似乎是好奇又似乎是……
<br/>
<br/>        总而言之，是她说不出来的意境。
<br/>
<br/>        “还有一个……给了小卫。”童筱颖不敢直视傅夜沉的眼睛。
<br/>
<br/>        这一刻，他的心被刺痛之时，才明白，她对阿景的爱，到底有多深了……
<br/>
<br/>        傅夜沉的眼睛里布满了血丝，心痛地看着童筱颖，明明他很气恼咽下肚后又满是怜惜。
<br/>
<br/>        这是他爱的女孩，为什么会被折磨成这样？
<br/>
<br/>        她把肾给了阿景，把爱给了阿景，把一切都给了阿景，哪怕最后被阿景抛弃了，她心心念念地还是阿景。
<br/>
<br/>        傅夜沉什么话也没说，只是蹲了蹲身，一手环住她瘦削的肩，一手从她腿窝穿过，搂着她匀称的双腿，将人打横抱了起来。
<br/>
<br/>        在身体悬空的那一瞬，童筱颖下意识地抱住了傅夜沉的脖子，近距离的接触，她又闻到了那股淡淡的薄荷香气，和那个女秘书落以柔身上的香气一模一样。
<br/>
<br/>        他到底不是她喜欢的人，所以，在他身上闻到其他女人的气味时，她一点醋意也没有，不光如此，她也不打算过问他这香气的来源。
<br/>
<br/>        潜意识里，有个声音在警告她，不要再爱上任何一个男人。
<br/>
<br/>        至少没有爱，就没有痛……
<br/>
<br/>        到时候若是要离婚，至少她会活得比现在洒脱。
<br/>
<br/>        车窗外，即将下山的太阳，小得就像一只烧红的铁球，穿梭在树影间，余晖染红了城市的高楼大厦。
<br/>
<br/>        坐在豪车的后座上，傅夜沉总是把驾驶座后这个最安全的位置，让给她童筱颖坐着。
<br/>
<br/>        或许，这不叫“让”，而是这个位置于他而言，就是属于她的。
<br/>
<br/>        童筱颖双手抱臂，蜷缩着身子，坐在车椅上一言不发。
<br/>
<br/>        傅夜沉一直在接电话，语言中英结合，精简流畅，声音透着磁性，又不失威严。
