　　众人都被苏莫的所为震惊了。
<br/>　　事实上，这种做法以前不是没有人想过，但是谁敢破坏规则？
<br/>　　这第二关的规则，就是看谁撑的时间长，谁能最后留在平台上，谁就能过关。
<br/>　　所以，就算是偶尔有这种想法的人，也不会去破坏这种规则，只会老老实实的抵挡圣雨。
<br/>　　但现在，苏莫做了，以雷霆之势击飞所有人，顺利的过关了，这让所有人下巴碎了一地。
<br/>　　尤其是和苏莫有仇的巫人王、尤天韩等人，眼珠子差点没瞪出来。
<br/>　　“可恶！”
<br/>　　巫人王见苏莫轻松过关，心中比吃了苍蝇还要恶心，不过，他转念一想，苏莫顺利过关也是好事，到了第三关对方就没得逃了。
<br/>　　“过关越快，死的也越快！”尤天韩心中冷哼。
<br/>　　不过，对于此事众人来不及过多考虑，因为圣雨已经降临了，众人急忙出手抵挡。
<br/>　　铺天盖地的圣雨，仿佛一柄柄灵剑，落在了一座又一座平台上。
<br/>　　啊！啊！啊！！
<br/>　　这一瞬间，惨叫声不断，由于受到了苏莫的影响，导致很多人分神，躲避剑雨不及，当场就有不少人比剑雨击杀。
<br/>　　这是真正的死亡，被剑雨击杀不仅被淘汰出局，也彻底陨落了。
<br/>　　更多的人被苏莫影像，导致身受重伤。
<br/>　　“可恶！”
<br/>　　一轮剑雨过后，不少受伤的人对苏莫怒目而视，若不是苏莫他们怎能受伤。
<br/>　　伤势较轻的人还好，伤势较重的人，只能主动飞出平台了，免得挡不住下一轮圣雨而陨命。
<br/>　　而此刻，通过了第二关考验的苏莫，面色平淡。
<br/>　　能一巴掌将所有人击下平台，一是他有吞噬战魂之助，二十那些人都猝不及防，完全不知道他会出手。
<br/>　　此时，他脚下的平台已经消失，他身形刚想向真圣碑飞近，但一股无形的力量阻隔在他的前方，让他无法前进。
<br/>　　苏莫心中无奈，或许要等到其他平台的考验结束，或者是第三关结束，才能靠近真圣碑。
<br/>　　现在，苏莫心中已经有了逃命的想法了，那就是擒拿巫族的几名顶尖天才，以此为要挟。
<br/>　　当然，这种办法逃生的可能性极低，武圣强者可不是那么好威胁的，但总算是有一丝希望。
<br/>　　“都给我滚吧！”
<br/>　　就在此刻，一声暴喝声突然响起，如同雷霆炸响，震动四野。
<br/>　　只见其中一座平台上的巨摩，突然暴起，如水缸般大小的巨拳抡起，暴轰而出，向平台上的其人出手了。
<br/>　　啊！啊！啊！！
<br/>　　硕大的拳茫闪过，惨叫声不断，有十几人猝不及防，顿时被巨摩一拳轰的吐血飞退。
<br/>　　“杀！”
<br/>　　“给我下去！”
<br/>　　“你们都下去吧！”
<br/>　　几乎同一时间，数百个平台上都彻底大乱了起来，下一轮圣雨攻击还没到来，这些平台上的人已经开始大打出手了。
<br/>　　姜神夜、飞长云、黑锋、巫殇、甚至是巫人王等，全部都出手了。
<br/>　　既然将其他人打下平台就能过关，那些强大的超级天才人物，自然不想被动的再比拼谁能撑的时间长，而是纷纷向同平台的人出手了。
<br/>　　只要将其他人打出去，他们也能想苏莫一样过关。
<br/>　　霎时之间，几乎有大半的平台之上，纷纷爆发出了惊天动地的大混战。
<br/>　　轰！轰！轰隆隆！！
<br/>　　爆响不断，巨响震天，气浪汹涌，各式各样的攻击，在各大平台上交锋。
<br/>　　“呃~~~！”
<br/>　　苏莫见此也是有些无语，这些人都不想正常的通过第二关了，居然都开始效仿他了。
<br/>　　只不过，他是出其不意的攻击，再加上有吞噬战魂之助，才能轻易得手。
<br/>　　现在所有人都知道了这个方法，都有防备了，就算是巨摩那种人物，也不可能轻易打败同平台的所有人。
<br/>　　……
<br/>　　外界。
<br/>　　此时已经人山人海，铺天盖地都是人影，因为第一关没有通过圣门的人，都被传送了出来。
<br/>　　声浪冲天，嘈杂的声音此起彼伏，很多被淘汰的人，均是抱怨不已。
<br/>　　但是，突兀的，现场立刻寂静了下来，所有人都望着宗门长辈或者族中长辈前方显化出的影像。
<br/>　　众人看到真圣界内，苏莫突然出手，将所在平台的其他人全部击飞，顿时目瞪口呆。
<br/>　　“我去，苏莫过关了！”
<br/>　　“他将其他人都打出去了，只留他自己，自动算过关了！”
<br/>　　“我的天，居然还带着这样的！”
<br/>　　“这太卑鄙了吧！”
<br/>　　少倾之后，众人一片哗然，与真圣界内那些天才一般无二，全部都是几乎被惊掉了下巴。
<br/>　　那些各大势力带队的强者，亦是面色怔怔，满脸无语。
<br/>　　“不知所谓！”
<br/>　　大祭司见此，脸皮抽了抽，面色阴沉了下来，苏莫能一巴掌将同平台的所有人打下平台，这个实力在武帝境之中，可是非同凡响。
<br/>　　虽然苏莫属于偷袭，攻击出其不意，但以武帝境六重的修为，一招击飞大量武帝境八、九重的天才，这种实力依旧很逆天。
<br/>　　“这小子有意思！”逍遥剑圣微微一笑，对于苏莫的这种做法，感到非常有趣。
<br/>　　另一边，吴晋、唐婉等神武学府的导师，对于苏莫的所为，也是一脸无语，哭笑不得。
<br/>　　“这小子不仅天赋逆天，做事也是奇葩！”众导师心中暗想。
<br/>　　就连一直冷若冰霜，不苟言笑的木璃导师，面上都是忍不住露出了一丝笑容。
<br/>　　不过，想到巫族之人对于苏莫的杀机，她脸上的笑容很快便消失不见，眸中露出了一丝担忧。
<br/>　　嗡嗡嗡~~
<br/>　　此时，不远处的空间一阵波动，和苏莫同平台的两百多人被传送了出来。
<br/>　　众人乍一出来，均是有些懵圈。
<br/>　　但很快，他们想到自己被苏莫突然打出平台，惨遭淘汰，个个都怒火冲天。
<br/>　　“苏莫，老子和你没完！”巨煌一声咆哮，震动天地，仿佛要将心中的憋屈发泄出来。
<br/>　　他怒啊！他恨啊！
<br/>　　这种憋屈几乎要让他疯狂了！
<br/>　　他本来自信满满，能在第二关撑到最后，从而顺利过关。
<br/>　　但苏莫突然出手偷袭，让他连承受第二关圣雨攻击的机会都没有，就被淘汰了，他怎能不怒？
<br/>　　飞华等其他两百多人，不比巨煌好多少，个个几乎被气的吐血，几乎骂娘的心思都有了。
<br/>　　这尼玛太憋屈了，等苏莫出来，绝对不能轻饶。
<br/>　　不过很快，他们想到巫族对于苏莫的杀机，心中的怒气才稍稍平复下去。
<br/>　　苏莫这个混蛋，肯定走不出真圣界了。
