    
这天晚上的酒局让范二哥和王惊蛰的关系给拉近了一步，推心置腹莫逆之交倒也谈不上，但友谊的小船肯定已经浪起来了。<br/>　　<br/>　　他俩之间的关系就是，一个有意结交，另外一个并不抗拒，可距离却一定会有的，以后也会照此延续下去，赊刀人古来都有训诫，君不登庙堂，若即若离即可！<br/>　　<br/>　　一晃几天过去，时间进入到十二月下旬眼看着就是另一个年头了。<br/>　　<br/>　　这阵子，东四环那块地正在紧锣密鼓的操持着，关于布局差不多是已经要到了尾声。<br/>　　<br/>　　十二月二十一日，夜。<br/>　　<br/>　　那片荒地中间被油毡布给围拢了起来，围的密不通风，胡胡，韩观山和王子文一脸严肃的站在里面。<br/>　　<br/>　　油毡布围拢的中间，被搭建起了一个台子，左右两侧竖立起了两根高有四五米的石柱，两根柱子从上到下都刻着一条长龙，栩栩如生，但是这两条长龙的眼睛却是闭着的，俗称龙眼未点睛。<br/>　　<br/>　　长龙柱后面的台子上是个供桌，下方摆着九只公鸡，猪牛羊牲口也同样各九只，供桌上面是个硕大的香炉，此时炉中还未点香，炉子两旁摆放着龙九子，囚牛，睚眦，嘲风，蒲牢，狻猊，霸下，负屃，螭吻的玉雕。<br/>　　<br/>　　玄门中命字脉四人站在台前神情肃穆。<br/>　　<br/>　　这是古时传说中的祭龙台，据传古人如发现龙有潜宗时可以此台祭拜，遇山河中有龙也以此台祭拜，望云层中有龙腾云驾雾也以此来祭拜。<br/>　　<br/>　　总得来说，这祭龙台就是尊重龙来用的，华夏五千年上下的历史，对于龙这一莫名其妙的生物来讲，始终都是抱着憧憬，尊崇，礼敬等各种朝拜的心思的。<br/>　　<br/>　　上到各朝王孙贵族和皇帝，下到黎民百姓贩夫走卒都是如此！<br/>　　<br/>　　这祭龙台就是由此而来的，而在风水一术中，此台的用处则是勾连龙气的，避免引得龙气翻滚，搅动龙脉，不然就有可能是坏了根基的。<br/>　　<br/>　　今晚，月黑风高，残月如钩（这两个词出现的频率，等同于主角的名字？）。<br/>　　<br/>　　命字脉一老人点上三柱长香，忽然“噗通”一下跪在地上，手举长香说道：“敬，四方天地，鬼神，敬，万物上苍……龙行踏绛气，天半语相闻，混沌疑初判，洪荒若始分”<br/>　　<br/>　　“敬上苍有龙，单于浪惊喜”<br/>　　<br/>　　“八龙三虎森如也，万古千秋”<br/>　　<br/>　　这老人祭语说完，忽然挥手说道：“祭，龙台……”<br/>　　<br/>　　“噗嗤！”忽然间龙台前的牲口全都在同一瞬间被抹了脖子，鲜血顺流入柱洒向地面，从古至今的祭奠都是以活物来祭的，用牲口还好一点，以前几百年用童男童女的都不少见。<br/>　　<br/>　　一篇祭文过后再加上活祭，这天边的云彩忽[八一中文网
