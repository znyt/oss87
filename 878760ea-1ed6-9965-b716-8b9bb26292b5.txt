　　蝾螈……多半就是蝾螈了。一种据说是上古时期的生物，种类有很多，大多数都生活在地下洞穴里，在民间，这是一种神秘的生物，在法术界却是一种有名有姓的灵物。
<br/>　　法术界各大门派的典籍里，都有一些关于灵兽的介绍，其中就有这个蝾螈。在场几人都是成名的法师，自然都看过这些资料，虽然版本不同，但是没什么差别，至少都会介绍出这种灵兽的特征。据说蝾螈其实也没灭绝，只是一般都生活在山体中的地下洞穴里，数量也极少，法术界里，差不多有几百年都没听说过有人见过蝾螈的事情了，于是当这个怪物就出现在面前的时候，没有人想到它就是蝾螈。
<br/>　　不过在叶少阳的提醒之下，大伙回想了一下蝾螈的特征，还真的完全符合典籍上的介绍……
<br/>　　“真的是蝾螈……”妙心凝视着面前这个庞然大物，吸着气说道：“我们家族的手札上，提到过用蝾螈来守墓的事情，据说汉武帝的墓地就是……不过我家祖祖辈辈也没人见到过蝾螈，真没想到……传说竟然是真的。”
<br/>　　毛小方一拍脑门，说道：“可不是，井研上那两个虫字旁的字，就是‘蝾螈’啊！”
<br/>　　众人都是一惊，在脑海中想了一下，还的确有点像是蝾螈两个字。
<br/>　　小心蝾螈……
<br/>　　妙心深吸了一口气，望着面前这头不知道活了多少年的蝾螈，郎朗说道：“不管它是什么，既然对我们有威胁，那就只能解决了他，诸位，一起使出全力吧！”
<br/>　　毛小方和叶少阳也立刻加入了战斗。
<br/>　　看着蝾螈在他们手下挣扎的样子，叶少阳心中也是没底，说道：“大家小心点，这个怎么说也是灵兽，肃然靠的是被动修炼，毕竟也不知道活了多少年了。”
<br/>　　蝾螈这种灵兽，跟犼、烛九阴，还有叶少阳自己对付过的化蛇，虽然同属上古灵兽，但是实力完全不在一个水平线上：蝾螈是灵兽不错，生而为灵，这一点跟鲛人一样，不过灵智跟最普通的动物没什么区别，不会主动修炼，只是因为长期生活在山体内部的洞穴里，又浸淫在山泉之中，山精水气吸收了不少，长期下来，也积累了不弱的修为。
<br/>　　这种生灵，最大的特点其实还是块头大，这么一个庞然大物，就算是普通生物，也不是人类容易对付的，更不要说它还是一头灵兽。因此，叶少阳相信，想要对付蝾螈，并不是那么简单的事。
<br/>　　听了叶少阳的话，曹雨兴哼了一声说道：“不要长他人志气了，哦对，对你来说，这蝾螈的确是个难对付的主，不过对几位师兄来说，也算不了什么吧，再说它都被困住了，还能翻出浪花不成？”
<br/>　　他的话说完，陈晓宇和卢晓清一起转头朝叶少阳看过来，虽然没说什么，但眼神和嘴角的微笑，已经对叶少阳展开了嘲讽，嘲讽他胆小。
<br/>　　不过下一秒钟，妙心的一句话，就让他们这些胆大的感觉自己被打脸了：“小心，他好像要降命了！”
<br/>　　大伙一惊，定睛看去，果然，在这些人的合力进攻之下、本来已经被牢牢困住的蝾螈，身体突然又剧烈地抖动了起来，本来已经趴在地上的四肢，又在尝试着一点点站起来，口中发出好像暴风雨降临之前的那种哼哼声，两只眼睛也爆红起来，流出了绿色的粘液。
<br/>　　大伙顿时感到了来自结界下面的压力，逐渐增加，很快就到了压不住的地步。
<br/>　　“再撑一会，它也耗不起的！”叶少阳一边作法，一边咬着牙关说道。在这几个人里面，他的实力是最差的，因此他作法借助了阴阳镜的灵力，至少在操控结界方面，不比他们差多少。
<br/>　　他感觉自己还能撑一会，而被困在结界之中的蝾螈，却是用自己庞大的身躯在抵抗着结界的束缚和绞杀，身上已经完全皮开肉绽，血流不止，在地上都形成了一片池塘似的血泊，大家都看得出，它绝对撑不了太久了。
<br/>　　然而，他们还是低估了这头上古灵兽的实力。
<br/>　　就像游戏里那些BoSS在红血的时候，都会暴走一样、这头几乎被消耗得只剩下半条命的蝾螈，总算也是爆发出了一股强大的力量，更加猛烈地冲击着结界。
<br/>　　就在叶少阳说完那番话不到两分钟的时间，一声惨叫，在墓室里响彻起来。
<br/>　　是曹雨兴，他是这几个人之中实力最差的人（有法器加持的叶少阳除外），第一个顶不住压力，被结界的反噬之力震飞了出去，一头撞在墓道坚硬的岩石上，昏死过去。
<br/>　　结界之所以能维持下去，靠的就是平衡，曹雨兴被震飞出去之后，结界便是被打开了一个缺口……蝾螈浑身一机灵，操控自己的修为，对着这个缺口撞了过去。
<br/>　　妙心等人试图调整站位，补上这个缺口，但终究还是慢了一步：蝾螈将自身修为汇聚在一个点上，通过这个缺口，将整个结界在一瞬间撞的粉碎。
<br/>　　妙心等人都被强大的反噬之力震飞了出去。
<br/>　　不过在这一刻到来之前，他们凭着强者的敏锐，已经意识到了这个结果，虽然慢了半拍，但也算是提早做了准备，并没有像曹雨兴一样被直接撞在墓道的墙上，而是临空作法，踉跄落地，总算没有受伤。
<br/>　　总算脱困了……蝾螈大吼一声，完全陷入了狂暴，朝着离它最近的梧桐飞了过去。
<br/>　　梧桐虽然没料到它会突然自己，但是以她的实力，本来是可以躲开这一击的，但是在蝾螈脱困的一瞬间，她的五宝金莲也被弹飞起来，朝墓室的一角飞去。
<br/>　　五宝金莲是她的本命法器，当然不能丢了，梧桐自己落地之后，第一时间就是去捡五宝金莲，虽然她脚步很快，但就是这一个瞬间的延误，让蝾螈抓住了机会，飞扑过去。等到梧桐转身应对的时候，已经躲避不开，只好本能地往外推出双手，结了一个无相劫印。
<br/>(本章完)
