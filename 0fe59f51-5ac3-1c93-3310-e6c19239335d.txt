　　周凯见此，就道：“林峰，我先走了，老龙口的位置我知道，这几天我安排下我的事情，然后就去老龙口找你。”
<br/>　　说着他就离开了。
<br/>　　留下陆勇等几人。
<br/>　　“刚才走的是铠王吧，可惜了，之前邀请他加入我军部，不怎么同意，没想到，却被林峰兄弟折服了，还真是羡煞我也。”
<br/>　　陆勇瞥了眼周凯的背影无奈道。
<br/>　　这年头，好将难求。
<br/>　　林峰也很无奈，他压根都没去邀请人家，倒不是看不上，而是在他看来，这种实力的家伙，肯定有所归属了，结果谁知道，人格魅力把人家征服了，站在那里寂寞如雪的吃着辣条，人家就主动求收留，这让林峰怎么办。
<br/>　　“林峰，给你介绍一下，这位就是从八峡发电站那边过来的快速作战部队，这位是张旅长，张浪。”
<br/>　　陆勇道。
<br/>　　张浪啪的一下给林峰敬礼。
<br/>　　“见过首长。”
<br/>　　在军衔上，林峰是将军，他一个旅长，喊林峰首长没毛病。在实力上，张浪是九阶，这么喊，也没问题。
<br/>　　“林峰兄弟，张旅长的加入，对于我们曙光基地来说，就是一针强心剂，大大的弥补了我们火炮缺少的问题，尤其是防控火炮，这几天一直在布置，日后，曙光基地的防空火炮网要比之前强大最起码一倍！！”
<br/>　　陆勇小道。
<br/>　　张浪点了点头，“我们八峡发电站作为国家战略重地，防空力量自然是不可忽视的。不过可惜，我们若是第一时间就来到曙光基地和陆将军回合，那么也许就不会损失那么多兄弟了。”
<br/>　　“结果到头来，发电站被那个怪物破坏了，还死了那么多兄弟，竹篮打水一场空。”
<br/>　　张浪摇头道。
<br/>　　“张旅长，你当时的作战指令没有任何问题，一旦出现战争，第一时间保护住这些国家战略重地是我们军人的职责。”
<br/>　　陆勇拍了拍张浪的肩膀郑重道。
<br/>　　“在这一点上，我不如你，对了，张旅长，还是说说你的那个东西吧。我想林峰小兄弟肯定会喜欢的。”
<br/>　　张浪也是个汉子，不墨迹，立即拿出了一个东西。
<br/>　　看到此物，林峰目光微微眯起。
<br/>　　“这个是？”
<br/>　　“这是我们斩杀了那个盘踞在发电站内的十阶怪物后在他体内得到的，名为剧毒之液，乃是个三星的灵枪，射出来的不是子弹，而是一种压缩了毒液的弹药，腐蚀性非常强大，腐蚀范围也很广，威力很强，而且不需要换弹夹，唯一缺点就是射速比较慢，大概四秒钟可以射一发毒液弹。”
<br/>　　张浪道。
<br/>　　“说也奇怪，这一切信息就是自动出现在我脑海之中的。”
<br/>　　果然。
<br/>　　林峰目光一闪，这么看来，他的猜测完全正确，只有十阶以上的怪物才会有灵器的伴随。
<br/>　　林峰看着这把剧毒之液，这是一把手枪一样的武器。
<br/>　　“小兄弟似乎知道灵器？”
<br/>　　陆勇随时注意林峰的表情，毕竟他当时知道这东西时候很震惊的，但林峰表情上没有任何震惊反而欣喜，这让陆勇猜测，莫非林峰早知道了不成？
<br/>　　“嗯，实不相瞒，末世后没多久我也得到了灵器，就是我脚上的这双鞋子，名为追风靴，也是个三星灵器。可以加快我的速度。”
<br/>　　林峰道。
<br/>　　“追风靴？”
<br/>　　张浪和陆勇都朝林峰脚上看去，果然这一看，还真的和寻常鞋子不一样，他们之前还真没有发现。
<br/>　　“另外，我有个推测，这个灵器，只会出现在十阶以上的怪物或者丧尸身上。”
<br/>　　林峰说出了他的判断。
<br/>　　“哦？”
<br/>　　陆勇和张浪都是为之一动。
<br/>　　林峰说的话，那么基本八九不离十了。他们都很信服林峰。
<br/>　　“不过，至于这灵器从何而来，又为何只有十阶的怪物才可以拥有，这就不得而知了，不过可以肯定的事这些武器明显比我们现在的武器要强大很多。”
<br/>　　“林峰小兄弟言之有理！！”
<br/>　　陆勇也颇为赞同。
<br/>　　“对了林峰小兄弟，我们计划，五日后，对巨树之森发动进攻！！你意下如何！！”
<br/>　　林峰闻言点了点头，“没有任何问题。”
<br/>　　“好！到时候，希望林峰小兄弟能够助我们一臂之力！！”
<br/>　　陆勇道。
<br/>　　“自然，战门这群人杀了我不少手下的手下，这血海深仇，我不会坐视不理。”
<br/>　　说完后，陆勇和张浪离去了，为这几天做好一切战前动员。
<br/>　　林峰在原地沉思了一下，然后也离开了曙光基地回到据点。
<br/>　　他把五天后开战的事情，仅仅告诉了苏云等他信得过几个，毕竟他现在老龙口也是人多口杂，且不说是否有奸细，就是这大嘴巴一传十十传百，泄露军机，那可不是什么好玩。
<br/>　　一个先发制人，和被动还击，在战略意义上差距还是很大的。
<br/>　　到了下午，周凯也回来了，林峰第一时间就给大家介绍了一下新成员。对于周凯，林峰很重视，对方的实力且不谈，最主要的是，对方人品很不错，有实力，人品还很好，这种人才最为难得，林峰也很重视。
<br/>　　毕竟现在很多实力强的人都花天酒地，一天几个嫩模换着来，乐不思蜀，酒池肉林，一旦需要他发挥职责的时候，躲得比谁都快，极其怕死，能力越大责任越大这种话，对于他们来说就是屁话，这种人在曙光基地内不在少数。
<br/>　　“周凯，你以后就加入李广的安保部队吧。”
<br/>　　林峰把周凯安排的清清楚楚。
<br/>　　“是。”
<br/>　　周凯点了点头没有任何意见。反正他是跟着林峰杀怪物的，林峰叫他做什么他就做什么，只要别让他去炊事班就可以。
