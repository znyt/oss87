
<br/>　　<br/>　　    “来，喝汤……”余生婆洋溢着慈祥的笑脸给他盛了一勺菜汤，善良的就好像家里的老奶奶在溺爱着自己的孙子，但王惊蛰知道这笑容的背后，搞不好就藏了一把刀子。<br/>　　<br/>　　    “汤就不喝了，撑得慌”王惊蛰往嘴里塞了最后一块馍馍，擦了下手上的油渍，挺认真的看着她说道：“我想多活几年，越多越好，没有上限的那一种”<br/>　　<br/>　　    余生婆幽幽的说道：“可是这借命，大部分人就只是借了三年五载不等啊，就比如那虬髯客到最后也是没借成，不得不熬了一千多年”<br/>　　<br/>　　    “那是他们，我和大部分人不同，主要就体现在都不是一路人上了，婆婆，龙生九子还各有不同呢，人也分三六九等，他们办不到的事，我未必办不到，你也可以说我贪，但我真的真的想多活那几年，越多越好的那种”<br/>　　<br/>　　    王惊蛰在说这句话的时候，语言中充斥着难以掩饰的情感，这两句话说的完全没有一丁点演戏的成分，真真是发自内心的那一种。<br/>　　<br/>　　    因为这些年来，为了自己三十岁的那道天诅，王仙芝和王冬至疲于奔命了很多年，他们两个几乎放弃了所有关于自己的生活，而想方设法的要帮他迈过那个坎，在王冬至的这个年纪，凭着她的脸蛋和身材，最应该享受的是名媛的生活，至于王仙芝则最好找个老伴然后来个完美的夕阳红，但是他们两个迄今为止却把所有的时间全都用在了王惊蛰的身上，到最后却不一定能不能见到啥效果，用王惊蛰曾经讲过的一句话来讲就是，他们大好的时间都喂了狗了。<br/>　　<br/>　　    有些话没用嘴说，其实憋在心里才最痛苦。<br/>　　<br/>　　    “可是这借命真的不一定能借到多少呢，哪怕是……”余生婆顿了顿，忽然伸出手指了指脚下说道：“哪怕是阴曹地府中判官手里的生死薄被勾画了一笔也是没一定的，毕竟天道昭昭，说不定还瞒不过去的，我说是三十年，可万一没准就只是十年八年的呢？”<br/>　　<br/>　　    王惊蛰笑了：“这是丑话说在前头了”<br/>　　<br/>　　    “当然要说清楚了，交易就是交易”余生婆皱眉说道。<br/>　　<br/>　　    “十年八年也比三年五载多了不少，挺难能可贵的了”王惊蛰搓了搓手，说道：“婆婆，汤也喝了不少了，来点干货吧”<br/>　　<br/>　　    “生死薄说白了也不过就是几页特别点的纸，只是在判官的手里能写出人的生死阳寿才难得的，除了在阴曹地府外，还有个地方也能拿到这页纸，只要一页就够了……”<br/>　　<br/>　　    道门中曾有一传言，世间有天地人三书，天书封神榜，地书山海经，还有就是人书生死薄，其中前面两个几乎已经成了传说从来都没有人见到过了，不过生死薄却实实在在的存在于阴曹地府中，记载着阳间所有人畜的生死时间，这生死薄上还能查到人的是非功过，往上可以追溯到最高的九十九世，一旦被定上了死期就再也无法更改了。<br/>　　<br/>　　    不过这死期可不是从人的一出生就定下的，最开始的时候在生死薄上是模糊不清的，然后会根据人这一生的功过来浮动，直到确定你这辈子已经无法在改变，最后才会逐渐清晰，然后定了你的生死。<br/>　　<br/>　　    余生婆说道：“这八字借命术法大成就差了最后一道……将人的生辰八字写在那页纸上”<br/>　　<br/>　　    余生婆话到这里就停住了，再没有往下接着说了，言多必失，话在往下说可能秘密就不是秘密了，王惊蛰也没有在顺着她的话往下问，而是说道：“那一页纸除了在阴曹地府，哪里还能在找到？”<br/>　　<br/>　　    “八拐里坡”余生婆张嘴，就说了四个字，一个地名。<br/>　　<br/>　　    王惊蛰则是忽然转过头，看着墓崖村的西边，倒吸了一口气后，点头说道：“难怪墓崖村会在这里呢，原来如此了……”<br/>　　<br/>　　    八拐里坡是个地名，不过却不被常人所知。<br/>　　<br/>　　    八里地，拐八道弯，这是在字面上的解释，但除了明面上的解释，八拐里坡还是个禁地，是生人勿近的地方，就在墓崖村的西面，在澜沧江边上。<br/>　　<br/>　　    古往今来只有通风水走阴阳里的人可能会有人听说过八拐里坡，但听说归听说却绝对没人敢去，因为去了的话，是完全有可能就出不来的，八拐里坡中充满了各路牛鬼蛇神。<br/>　　<br/>　　    饭桌旁，忽然寂静下来，只闻呼吸声而无人说话。<br/>　　<br/>　　    李红拂放下手中的汤碗看着王惊蛰，余生婆则慢吞吞的喝着汤，只有眼神里有一点悸动和渴望。<br/>　　<br/>　　    良久之后，王惊蛰挠了挠鼻子，扣着眼角说道：“这个地方啊，还真是绝了……”<br/>　　<br/>　　    “要去的话，从村西面走，翻过一座山后，下面就是澜沧江了，江边有摆渡人你可以坐着竹筏过去”余生婆见他俩都吃完了，扔下一句话后就起身收拾着碗筷，拿到了厨房里。<br/>　　<br/>　　    王惊蛰放下手，扣着指甲缝里的眼屎：“好事多磨，我就说么，幸福来的太快太突然了也未必是什么好事，背后搞不好就是心酸，现在妥了，没准三五年的阳寿摸不到，先把自己给搭进去了”<br/>　　<br/>　　    李红拂笑了：“你也可以选择不去的”<br/>　　<br/>　　    “唰”王惊蛰抬手点了点她，说道：“阅历还是浅，男人心摸不透”<br/>　　<br/>　　    李红拂白了他一眼，说道：“都是男人想方设法来摸我的心，我摸你们这帮男人干嘛？”<br/>　　<br/>　　    王惊蛰提了下褂子，站起身子就从藤床上拿起了自己的破包，说道：“唉，富贵险中求啊……这个点正好出门，你先在这等着我吧”<br/>　　<br/>　　    李红拂顿时瞠目结舌的跟了出去，扯着嗓子喊道：“不是，你这猫一出狗一出的，来的也太快了吧，说走就走啊？”<br/>　　<br/>　　    王惊蛰的反应有点出乎她的意料了，这人的想法来的太风风火火了一些，怎么着不得仔细琢磨一晚上吧。<br/>　　<br/>　　    王惊蛰头也不回的摆手说道：“嗯，今天大姨夫来了，心境有点不太平和，现在不走还更待何时啊，我怕在等一晚上，自己都说服不了自己，该变卦了”
