　　“给我吞！”厉喝一声，苏莫立刻将吞噬武魂催动到了极致，强大的吞噬之力，顿时将所有的凶灵全部笼罩住了。
<br/>　　从进来开始，苏莫便一直在观察这些凶灵，发现这些凶灵，全部是由魔念诞生而成，正是他吞噬武魂需要的养料，所以他并没有丝毫的担心。
<br/>　　嘎嘎嘎！！
<br/>　　遭到吞噬之力的笼罩，无数的凶灵顿时发出一阵惊恐的怪叫声。
<br/>　　而后，这些凶灵仿佛遇到了天敌一般，疯狂的挣扎了起来，想要向远处逃离。
<br/>　　不过，苏莫的吞噬武魂已经催动到了极致，吞噬之力强大无比，疯狂的吸扯着这些凶灵，让得众多凶灵止不住的向他飞了过去。
<br/>　　“这是……！”聂乾见此，立刻停止了出手，目瞪口呆的看着这一幕。
<br/>　　目光看向苏莫的武魂，聂乾神情怔怔，这是什么武魂？居然能吞噬凶灵？
<br/>　　甚至，在吞噬之力下，他隐隐感觉自己体内的战魂，都有一丝颤抖。
<br/>　　嘎嘎嘎！！
<br/>　　无数的凶灵惊叫，吞噬之力将一只只凶灵拉近了漆黑的漩涡之中。
<br/>　　这些凶灵的实力或许相当于武帝境的武者，但面对吞噬之力，它们只有被吞噬的份，这是天生的克制。
<br/>　　苏莫心中暗喜，他发现这些凶灵的吞噬效果，堪比战魂。
<br/>　　每一只凶灵被他吞噬，他都能感觉到一股强大的魂力，比之上次的途山镇，吞噬的那名途家家主的战魂，效果都丝毫不差。
<br/>　　一只只凶灵在惊叫之中，被吞噬武魂吞噬，仅仅是几个呼吸的时间，便有九成的凶灵被吞噬。
<br/>　　剩下的百余只凶灵，依旧在苦苦的挣扎，惊叫不断。
<br/>　　但是，任凭这些凶灵如何挣扎，也逃脱不了被吞噬的命运，片刻之后，所有凶灵便被吞噬一空，现场恢复了平静。
<br/>　　嗡嗡~~
<br/>　　就在此刻，苏莫的吞噬武魂一阵颤动，其上金色的光环金光大放。
<br/>　　不过，苏莫的反应很快，在第七道光环出现之前，便立刻收回了吞噬武魂。
<br/>　　而此刻，他的吞噬武魂，已经成功晋升到了天级七阶的地步。
<br/>　　嘶！
<br/>　　聂乾看到上千只堪比武帝强者的凶灵，居然片刻间便被吞噬一空，顿时倒吸了一口凉气，这让他震惊不已。
<br/>　　“聂兄，我们去宫殿！”苏莫向聂乾招呼一声，便闪身向远处的宫殿飞去。
<br/>　　“哦！”聂乾下意识的应了一声，便跟随着苏莫向宫殿飞去。
<br/>　　“苏兄，你的武魂是什么武魂？不会是太古武魂榜上的武魂吧？”飞行之中聂乾好奇的问道。
<br/>　　他也算是见多识广，但从未见过苏莫这种武魂，便猜测是否是太古武魂榜上的那种超级武魂。
<br/>　　“我也不清楚！”苏莫笑了笑，他也不是隐瞒对方，而是他对于自己的武魂都不太清楚。
<br/>　　因为他的武魂，并不是只有单纯的吞噬能力，还有升级和运营他人武魂的能力，甚至还有吞噬别人血脉的能力。
<br/>　　而且，他也不清楚太古武魂榜上，都有哪些特殊的武魂。
<br/>　　片刻之间，两人飞到了那座黑色的宫殿之前。
<br/>　　这座宫殿极大，方圆不下十里，通体用一座黑色的岩石建造而成。
<br/>　　宫殿只有一个入口，其内一片迷蒙，看不清状况！
<br/>　　“聂兄，那廖云剑拿我们当炮灰，等会若是有机会，我们灭了他！”伫立在宫殿的大门之前，苏莫沉声说道。
<br/>　　他没有急着进入宫殿中，毕竟廖云剑也跑不了。
<br/>　　聂乾闻言苦笑一声，无奈道：“苏兄，那廖云剑实力比我都强不少，我们哪里是他的对手，再说了，他背景很大，和太史世家有关系，我们不能动他！”
<br/>　　“太史世家？”苏莫闻言，想到了神武学府背后的东家之一，然后又想到了上次在广场上那嚣张无比的太史轩。
<br/>　　“我们进去吧！”苏莫摇了摇头，没有多聊此事，身形一闪，便掠进了黑色的宫殿中。
<br/>　　无论是谁，想让他死，都有被他反杀的觉悟！
<br/>　　这廖云剑如此阴毒，已经上了苏莫心中的必杀名单。
<br/>　　瞬息之间，苏莫和聂乾，便进入了宫殿之中。
<br/>　　……
<br/>　　进入宫殿之后，前方是一条宽广的通道，直通大殿深处。
<br/>　　苏莫向聂乾示意了一番，两人将身上的气息收敛到了极致，缓步前进。
<br/>　　“哈哈！这一次真是不虚此行啊！”
<br/>　　前方隐隐传来廖云剑畅快的笑声。
<br/>　　苏莫闻言，眸中杀机一闪，此人他必杀无疑。
<br/>　　“廖云剑，你骗我们，这里根本就没有什么宝物！你只是为了这具魔魂！”韩盈的娇喝之声响起。
<br/>　　“嘿嘿！没有你们的帮助，我怎么能来到这里呢！”廖云剑的声音再次响起。
<br/>　　苏莫和聂乾已经悄悄的来到了通道口，两人没有急着现身，灵识微动，便看清了通道之外的情况。
<br/>　　此刻，通道之外是一个无比辽阔的大殿，大殿的中央有一座巨大的祭坛。
<br/>　　祭坛高大数十丈，其上魔光闪耀，漂浮着一个巨大的水晶球，而在水晶球内，有一个挺拔的高大身影。
<br/>　　“这是……魔魂？”苏莫看清那水晶球内的身影，顿时神情一怔，随即立刻激动了起来。
<br/>　　这身影绝对是魔魂，而且是超级强大的魔魂，因为即便相隔极远，都能感觉到无与伦比的魔威。
<br/>　　魔魂属于魔甲人的战魂，虽然没有人族战魂那么千奇百怪，但亦是强大无比。
<br/>　　只不过，这战魂为何会在此地，还有没有意识？
<br/>　　若是没有意识的魔魂，应该早已消散了才对，而若是有意识，为何会在此地？
<br/>　　而廖云剑满脸喜意，他要这魔魂有何用？
<br/>　　此刻，在祭坛的下方，廖云剑和韩盈相对而立，前者笑容满面，后者脸色难看。
<br/>　　“韩姑娘，实不相瞒，在下一直想找一名双修伴侣，特别钟意与你，不知你是否愿意？”廖云剑眸光打量着韩盈丰腴的娇.躯，微笑着说道。
<br/>　　“做梦！”韩盈闻言，身形立刻后退，就想要离开大殿。
<br/>　　唰！
<br/>　　人影一闪，廖云剑却是立刻拦住了韩盈的去路。
<br/>　　“韩姑娘，今日.你若是不同意，怕是走不出这座大殿！”廖云剑微笑着说道。
<br/>　　韩盈俏脸铁青，她没想到对方居然是如此卑鄙的人，不仅骗他们，还想打她的主注意。
