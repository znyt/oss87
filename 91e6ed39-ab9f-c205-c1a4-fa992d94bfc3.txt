
<br/>　　<br/>　　    孔子在后来被人尊称为孔圣人的时候曾经说过，世间本无圣人，这都是被人吹嘘出来的。<br/>　　<br/>　　    孔德荣也用一种十分不要脸的方式证明了这句话，孔家的那些什么仁义道德一类的东西也是不存在的，是人都有私心和私情，在面对自私这种事情的时候，脸是完全可以不要的。<br/>　　<br/>　　    在发觉墨家巨子其实是一只身怀绝技的机器猫之后，孔德荣和王惊蛰就暂时退了出去，然后由孔家子弟采用车轮战的方式，一群人单挑了这位墨家巨子。<br/>　　<br/>　　    老人家无疑很悲愤，就像孔德荣说的那样，我他么就算你一身上下全是机关，你不也得有用完的那一刻么，不给你机会补充弹药，你拿啥来维持下去啊？<br/>　　<br/>　　    片刻之后，墨家巨子在接连展现了袖中飞刀，木匣弹珠，胸口藏剑等一系列的机关暗器后，他似乎渐渐的已经黔驴技穷了，机关闪现的间隔，时间也是越来越长了，明显是机器猫也有被掏空的时候了。<br/>　　<br/>　　    最关键的是，小草猜测的非常靠谱，这老人其实已经濒临油尽灯枯的时刻了，凉绣确实是为了给他续命才想要的那一页生死簿，而墨家巨子的阳寿其实已经剩下不到三年了！<br/>　　<br/>　　    这要是在他鼎盛时期，或者年轻个十来岁的话，局势也不会是这样了。<br/>　　<br/>　　    在一场车轮战下，体力和精气极具耗费着，七十来岁的墨家巨子的头上渐渐的冒出了汗水，而与此耗费的代价则是，孔德荣带来的孔家子弟，三人身死，多人重伤，还剩下两个眼看着也要支撑不住了，这要是换做在唐梁山上，或者巨子鼎盛的时期，可能这些人早就已经成为机关暗器下的亡魂了。<br/>　　<br/>　　    “手段脏是脏了点，但人不为己天诛地灭啊……”王惊蛰叹了口气，跟孔德荣说道：“咱俩也上吧？”<br/>　　<br/>　　    王惊蛰已经瞧出来了，这老人的头发一瞬间干枯粗糙了不少，眼窝下泛起了黝黑色，皮肤暗黄，这已是灯芯马上就要烧没了的征兆，再往后要不了多久，人就得被拖拉死了，就算是此时停手，这老人也活不过一月有余了。<br/>　　<br/>　　    王惊蛰敬他为一代巨子，也算不忍心看他就这么折在了孔家子弟的手中，说出去可就不太好听了，倒不如葬在自己手中，这个死法的话，也算是过得去的，不管咋说他也是传承了多少代的赊刀人啊。<br/>　　<br/>　　    孔德荣犹豫了下，明显还没有从墨家巨子的机关震慑中走出来，但他看到王惊蛰迈步过去之后，也只得硬着头皮跟了上来。<br/>　　<br/>　　    但谁曾想到的是，王惊蛰和孔德荣刚一过去，还没等出声的时候，墨家巨子忽然悲怜一笑，长笑了几声后就剧烈的咳嗽了起来，他摆手说道：“罢了，罢了，我认了……”<br/>　　<br/>　　    “唰”王惊蛰脚步一顿，没在往前走了。<br/>　　<br/>　　    墨家巨子说道：“到我这一代，墨家不缺战死的巨子，差不多有四五代的巨子都是死在围剿下的，但说来我可能是死的最憋屈的一个了，天可怜见啊……”<br/>　　<br/>　　    王惊蛰默然，孔德荣别过脑袋，脸色微红。<br/>　　<br/>　　    “王家的小孩儿”墨家巨子招呼了王惊蛰一声，然后转身回到树下的凉绣身旁，从她身上拿出那本写有连山和归藏之术的本子，说道：“这东西其实我从来都不曾感兴趣，只是绣儿自己的私心罢了，我觉得墨家只重自己的机关之术就可以了，知道的太多就是杂而不精了，所以我并不太在意这什么连山和归藏，如今我归还与你，还希望你能在我走后答应我一个要求”<br/>　　<br/>　　    王惊蛰皱了皱眉，他已经猜到这老人要提出什么要求了。<br/>　　<br/>　　    “放了这孩子一马如何？我这一条命够交代的了吧，虽然她死了墨家也不算是绝后，但总归我就这一个孙女，她若也死了我可就没脸在泉下见她早逝的父母了”墨家巨子拿着本子，边走边说道：“再说，她的精神都已经失常了，就算活着也没准都没有好的那一天了，得饶人处且饶人吧，行么？”<br/>　　<br/>　　    王惊蛰看着老态尽显，仿佛下一刻就要天人两隔的墨家巨子，忽然之间想到了离此不远，八拐里坡中的王风饶，同为爷爷辈的，对于自己的子孙后代，可真是煞费了苦心。<br/>　　<br/>　　    王惊蛰稍微一犹豫，就点头说道：“好，我答应你，不难为她……”<br/>　　<br/>　　    “哈哈，好，好，有你这话我也走的心安了，这东西就还给你吧”墨家巨子说完，也走到王惊蛰身前，抬手就把那本子朝他扔了过来。<br/>　　<br/>　　    王惊蛰伸手一接，刚接到手中的时候，注意力全都集中在这本子上了，屋里一直注视着墨家巨子的小草，在月光下忽然之间就看到了这老人脸上决绝的神情。<br/>　　<br/>　　    “惊蛰，他有诈！”<br/>　　<br/>　　    “咻”墨家巨子张嘴之时，从他口中一根一公分左右的钢针就急射而出，直奔着王惊蛰面门上的眉心处扎来。<br/>　　<br/>　　    墨家巨子浑身上下机关暗器不知凡几，举手投足间都可伤人于无形，但谁也没曾想到的是，这老人在说话间张口闭口之时还能防不胜防的突下杀手。<br/>　　<br/>　　    嘴里藏针，这得算是墨家巨子最压箱，也是最后的一个手段了，他想在临死之前，还把王惊蛰给拉下去。<br/>　　<br/>　　    小草那一声惊呼传来之际，王惊蛰手中正好接到了那本子，他听见那一声尖叫后，其实是完全不知道发生了什么事的，就是条件反射一样，抓住本子之后都没想着要放下手，就直接挡在了脑袋前面。<br/>　　<br/>　　    “噗嗤！”<br/>　　<br/>　　    王惊蛰先是感觉手上一凉，然后剧痛传来，紧接着眉心上就有了一种被针刺破的感觉。<br/>　　<br/>　　    那一枚钢针先是扎穿了他的手掌，然后穿透了本子，力道最后才卸下了一多半，针尖点在了他的额头上，要不是茅小草最后那一声，可能这根钢针就已经透进他的脑袋里了。<br/>　　<br/>　　    老而不死为贼者必然老奸巨猾！
