　　“龙族竟然也参与到此事中来，此言当真？”
<br/>　　众人大惊，龙族可不是现在的妖族，虽说龙族也落寞了，但也不是普通种族能比的，毕竟那可是远古时期称霸天地的种族，底蕴也非同小可。若真是龙族也参与到其中，那可不得了啊。
<br/>　　应龙皱了皱眉头，轻声说道：“龙族按理来说不会干这等事，想来只是这妖猴得到过龙族的传承或者与龙族有过关联，才会有如此本领。”
<br/>　　“呼——”
<br/>　　众人都松了一口气，也是，龙族又怎会做这等事情，他们那是自己吓唬自己了。
<br/>　　“那如今该如何是好？”
<br/>　　众人看向战场上，有些担忧道。
<br/>　　“放心，那妖猴已经落入了下风，再过一时半会儿，禹帝便可将其捉住。”玉玲珑淡淡道。
<br/>　　“嗯。”应龙淡淡点了点头。
<br/>　　果然，事情不出所料，妖猴与夏禹大战约半个时辰后，一个不慎，妖猴被夏禹用九州鼎收了，自此妖猴被镇压在九州鼎之中。
<br/>　　“拜见大王。”众人见夏禹走来，都恭敬行礼道。
<br/>　　“嗯，走，我等回去审问这妖猴，到底是何来历，竟敢在人族兴风作浪？”夏禹点了点头，便拂袖而去，后面众人连忙跟去。
<br/>　　……
<br/>　　都城之中，夏禹坐于上首，下方众臣立于两边，大殿中央站着一妖猴，正是之前被夏禹捉住的妖猴，只不过此刻他已经被封印了修为。
<br/>　　“妖猴，见到我王，还不跪下！”大殿上，群臣见妖猴被封了修为还如此放肆，明目张胆地打量着周边环境，便向他大声喝斥道。
<br/>　　“嘿嘿，俺连天都没跪过，又岂会跪拜一个人族的君王？”妖猴嘿嘿一声，不屑道，其言语之间，丝毫没有将众人放在眼里。
<br/>　　“放肆！”众人大怒，一武将上前，想要将其按倒在地。
<br/>　　“好了。”上首夏禹一挥手，示意那人退下，之后他看向妖猴，淡淡问道：
<br/>　　“妖猴，我且问你，你叫何名？”
<br/>　　“嘿嘿，俺行不更名坐不改姓，俺叫无支祁。”妖猴毫不畏惧，没有丝毫身为犯人的觉悟，直视夏禹，傲声说道。
<br/>　　“无支祁？”
<br/>　　众人一听此名，都没有听说过，显然，这是一个众人都不认识的妖。
<br/>　　“那好，我再问你，你为何要制造如此洪水，祸乱人间？是谁给你命令的？”说到此处，夏禹的语气一下子变重了，显然，他内心是愤怒的。
<br/>　　“没人命令得了俺，俺就是喜欢，不行吗？”无支祁诡异一笑道。
<br/>　　“好，看来你是要敬酒不吃吃罚酒了，真以为你不说，我们就没办法了。以你的罪孽，死千百次都不足惜！”夏禹气急反笑，接着大声道：
<br/>　　“来人，将无支祁拖出去斩首示众。”
<br/>　　“是。”
<br/>　　接着，便有人进来，将无支祁拖了出去，想要将他斩了。
<br/>　　大殿之内，大夏禹将目光看向右侧的一位老者，那人满头银色，一声白袍，却给人一种神秘莫测之感。
<br/>　　“可算到什么？”
<br/>　　大殿中众人都将目光投到这老者身上，这老者是王朝的占星师，得到过伏羲的传承，精通推演之术。
<br/>　　“回大王，老臣算到这妖猴的确名为无支祁，乃是豫南桐柏山中的花果山水帘洞的妖修，天生为神猴，后来不知从何学得一身本事，娶了东海龙女为妻，生有三子，却都是大魔头，之前他在淮水建立龙宫，那十万山精水怪大多都是他从东海之中带出来的，其余一些是他降服的，至于为何会弄出水患，老臣并没有算到。在我推算之时，好似有一股神秘的力量在掩饰着。也许是因为龙族，也许是他自己有什么秘密，也许是妖族想要给人族制造灾难。”老者躬身一礼，回答道。
<br/>　　“龙族？妖族？”夏禹眼睛一眯，放出危险的光芒，喃喃低语道。
<br/>　　正在这时，门外有人前来报告：“报告，大王，那妖猴杀不死啊！”那人脸上带着惊骇的表情，好似看到了某种匪夷所思的事情。
<br/>　　“什么杀不死，慌慌张张，成何体统！”一大臣大声责问道。
<br/>　　“是那妖猴杀不死，无论是用刀砍，还是水煮，火烧，都杀不死他。”那人快速说道。
<br/>　　“哦？竟有此事，他已经被封住了修为，竟然还杀不死？”
<br/>　　众人惊讶道，要知道无支祁不仅仅是境界被封了，他的肉体一样被封了，如今的他，就如一个凡人一般，竟然还杀不死。
<br/>　　“果然有几分道行。”夏禹说道，想了想，接着道：“既然如此，那么便用轩辕剑将他斩了吧。”
<br/>　　轩辕剑乃是人皇公孙轩辕的证道之剑，他证道后便去了火云洞，轩辕剑用不着，便留在了人族之中。
<br/>　　“是，大王。”
<br/>　　半刻钟之后，外面传来一声惨叫，之后便再无声息了，显然是那妖猴被杀了。
<br/>　　“应龙，你去龙族一趟，看看此事是否与他们有关？希望此事最好不是龙族所为，如若不然……”夏禹对着应龙说道，但其话中的意味，任谁都能明白。
<br/>　　“是。”应龙心中一冷，随即应了一声，便走了。
<br/>　　……
<br/>　　数日之后，应龙回来了，发现此事并不是龙族所为，而且大多数龙族都不知此事是无支祁所为。不过，那无支祁的三个儿子倒是与其中有点关系，都被应龙顺手杀了。无支祁的妻子，也被龙族赐死。
<br/>　　“嗯，如今看来，那么此事是与妖族有关了。”夏禹淡淡道。
<br/>　　“哼！妖族，真是妖心不死，上古时期放他们一马，此刻竟然还来我人族祸乱。来人，向外颁布法令，自今日起，我人族需以杀妖除魔为己任，若是遇到妖魔，尽管击杀！”
<br/>　　夏禹的语气之中，带着凌厉的杀伐之意，众人仿佛能够看到一股血腥之气扑面而来。
<br/>　　“昂！”
<br/>　　人族的气运金龙，发出一声高昂的声音，目露金光，震慑四方妖魔。混在人族都城中的妖魔，若是没有人族气运在身的，直接被震死一大片。也只有与人族气运相连的，如夏禹的妻子，或者其他在朝做官的妖族之人，才能免幸于难。
<br/>　　“不过，此事与龙族也有些关联，那十万水怪，大多是他龙族的，需警告龙王一番，若是他做不好龙王，那便换个龙来做。”夏禹目光冷厉，冷冷地说道。
<br/>　　此时的龙族龙王，并不是远古时期的龙祖四皇子敖玉，而是另外的龙族，至于敖玉，早已升任为太上皇了。
<br/>　　夏禹取出一根金棒，正是那无支祁的法宝，他将这金棒一番祭炼，加上自己的道则，之后将之扔进东海之中。
<br/>　　金棒随风而涨，变得巨大无比，仿佛一根金色天柱，从天而降，坠入东海之中，使得东海一阵晃动，与此同时，夏禹那冷冷的声音传了进来：
<br/>　　“东海最近波涛不平，会有动乱，我将此定海神铁送入东海，助你们定住波涛。”
<br/>　　威胁，赤裸裸的威胁，还定海神铁？定海，恐怕是警告龙族吧。
<br/>　　可是，知道了又能如何，要知道人族如今可不好惹，证道的就有好几位，龙族虽然也有高手，可是并不会蠢到与人族开战。就算知道这是人族的警告，他们也无可奈何，只能日后小心点。
<br/>　　至此，天下水患彻底平了。
<br/>　　夏禹的做法，洪荒众多大能都看在眼中，但却并没有多管，只能在心里感叹人族越来越强了，不可阻挡！
<br/>　　经此一事，人族威望在洪荒中更高了。
<br/>　　……
