　　第740章
<br/>　　了然和尚倒抽一口凉气，眼见程大雷冉冉升起，忍不住高呼一声：“真乃神人也。”
<br/>　　此时，山门被撞破，魏牧冲终于等不及，领着大军冲进清风寺。闯进山门之后，直扑后寺，就见院内空空荡荡，程大雷已经高高飞起。
<br/>　　了然和尚一激灵，忙道：“魏城主，你终于来了，那贼人折磨得我们好惨。”
<br/>　　众僧人看着这一幕，主持师傅果然是高僧，睁眼说瞎话的本事大家谁也比不上。
<br/>　　魏牧冲的注意力并不在了然和尚身上，依旧全神贯注于空中的程大雷。他心中想不明白，这程大雷是怎么飞起来的？
<br/>　　程大雷眼见地面上人越聚越多，忍不住哈哈大笑：“姓魏的，你留得住本当家么。”
<br/>　　魏牧冲气得三尸神暴跳，五灵豪气腾空：“姓程的，你卑鄙。”
<br/>　　“哈哈哈。”程大雷狂笑一阵，忽地收住笑声：“本当家乃是南极仙翁传艺，太白金星是我把兄弟，天蓬元帅是我师兄。哈哈哈，尔等凡夫俗子，见到真神还不磕头。”
<br/>　　地面上，众兵丁面面相觑，有人双膝颤抖，险些跪倒在地上。魏牧冲咬了咬牙，大手一挥道：“放箭。”
<br/>　　众兵丁被程大雷的把戏慑住，只有寥寥几十人放箭，由地面射向半空中，想要击中也并不容易。羽箭射到半空，便被风吹散。
<br/>　　看到这一幕，程大雷开怀大笑，双手一压，大声道：“蛤蟆大王，光明普照。”
<br/>　　“贼人休狂，吃我一矛。”
<br/>　　声音落地，有一人从兵丁中窜出，三两步便攀上屋顶，用尽全身力气，手中一杆短矛投了出去。
<br/>　　来势汹汹，程大雷一激灵，这短矛势头不小，自己想要阻拦，手中也没有能用的东西。直到此时，程大雷才发现自己露算了一节。身在空中也失去了借力的地方，只能给人当靶子。
<br/>　　长矛直直扎进竹篮，亏的是没有射中皮蓬，程大雷心中松了口气，那军官再想出手，程大雷已经渐渐升高，离开他的射程。
<br/>　　魏牧冲怒火攻心，双目沁血一般，盯着空中的程大雷，恶狠狠道：“追。”
<br/>　　虽然不知程大雷搞得怎样把戏，但多多少少知道他撑不了多久。以现在魏牧冲的愤怒，抓到程大雷后必然将他千刀万剐。
<br/>　　程大雷顺着风飘，穿过了整座汝南城。城中百姓都窥见了这一幕，有人站在街上抬头望着天空，有人扒着窗户将脑袋探在外面瞧。
<br/>　　就见一颗空中一颗流星相仿，熊熊火焰燃烧，也有人忍不住跪拜，口中高呼着『神仙保佑』之类的话。
<br/>　　魏牧冲带着士兵，在地上紧紧追赶，他今晚死活不肯放过程大雷。
<br/>　　而在空中，程大雷现在的心情并不算愉悦。他哪里想到，空中风这么吓人，夜风呼呼刮过，程大雷担心火炬被风吹灭，在里面上蹿下跳。
<br/>　　而程大雷也算错了风向，他本想着顺着往北飘，自己可以和徐神机几人汇合。但刮得是东风，程大雷一路往西走。
<br/>　　如果风继续吹，程大雷大概会被吹回凉州。
<br/>　　关键是上去容易，下去困难，假若火炬熄灭，在上百米的高空直直落下来，程大雷怕也要摔得粉身碎骨。
<br/>　　他开始熄灭火炬，渐渐往地面上飘落。四座火炬，程大雷灭了其中两座，渐渐往地面上坠落，眼看已经出了城，前面出现一片山林，一棵棵参天大树拦在前方。
<br/>　　程大雷提心吊胆，心脏险些从嗓子里蹦出来。人在空中，就算双脚立地了，智商也未必占领高地。
<br/>　　那黑牛也变得不安，发出声声呜呜响，像是也感觉到危险。
<br/>　　程大雷多多少少还有轻身功夫，能够随机应变，这黑牛摔在地上，便只有死路一条。
<br/>　　眼见地面上出现一条大河，程大雷不敢再耽搁，挥剑斩断了绳索，一人一牛直直向地面上坠去。
<br/>　　噗通噗通两声响，程大雷和黑牛分别落入水中，他扎了个猛子，再从水中探出脑袋时，忍不住哈哈大笑。
<br/>　　一人一牛先后上了岸，程大雷跨上黑牛，口中高呼一声：“憨货，走着。”
<br/>　　此时魏牧冲的大军还没杀过来，毕竟程大雷从空中走要快一些，而程大雷降落之后，他们也未必能找到程大雷。
<br/>　　大河下游，有一秋风渡。
<br/>　　徐神机几人等在渡口，在过去的这几天，刘发财已想办法搞到一艘船。从这里开始，可以由陆路改水路。
<br/>　　如今，现在已是与程大雷约好的最后时间，几人都在焦急等待。
<br/>　　“徐军师，你说大当家回不回得来？”刘发财远远眺望着汝南城。
<br/>　　“担心什么，大当家是神仙转世，自能逢凶化吉。大当家让我们等着，我们等着便是。”
<br/>　　刘发财没有徐神机这样的境界，心中也难免佩服徐神机，军师果然是军师呐，这眼界魄力自己比不了。
<br/>　　一共五人，徐神机老神在在，刘发财完全将他当做主心骨。剩下的鱼洞真是个木头人，万般不关心的表情。崔白玉与乔弄云在一起，崔白玉偶尔看一眼汝南城方向。
<br/>　　想要单人独骑从汝南城脱身谈何容易，亏得徐神机对程大雷这么有信心。但崔白玉并不敢有任何希望。
<br/>　　“假若他回不来，我们该怎么办？”
<br/>　　“怎么能回不来，肯定会回来的。”徐神机。
<br/>　　“那我们也该好好想想，如果他真无法回来，是该回凉州，还是继续去长安。”
<br/>　　“会回来的，会回来的。”徐神机本来不怎样担心，但被崔白玉说了几句，也有些六神无主。
<br/>　　崔白玉望着远处，心情隐隐有些焦躁。她忽地一楞，自己是担心他么，他若被杀死了，自己本该高兴才对。
<br/>　　转念一想，毕竟是同舟共济，他若出事，自己怕也没有好果子此。如此想，这担心就显得理所应当起来。
<br/>　　时间渐渐过去，几人表情都显得有些凝重。
<br/>　　就在此时，忽然听得咚咚声响，远处出现一团黑影，距离大家越来越近。
<br/>　　程大雷骑在黑牛背上，拎着匹夫剑，轰轰隆隆跑到诸人身边。
<br/>(本章完)
