        连星儿一直专注于自己手中的活，郎安就坐在她的身后，一双冷冽的黑眸，直勾勾地盯着她。
<br/>
<br/>        往日里，若是被郎安这么盯着，连星儿一定会觉得不寒而栗。
<br/>
<br/>        但今天，她太过专注自己的事情，以至于她完全把郎安还在这里陪她的事情给忘了。
<br/>
<br/>        到了凌晨十二点，明司寒收到了许多亲朋好友准点发过来的生日祝福短信。
<br/>
<br/>        但唯独没有连星儿的。
<br/>
<br/>        一个女生和一个男生，孤男寡女在外过夜，哪会有空理会他这么个外人？
<br/>
<br/>        明司寒看着这一条条的祝福短信，心里仍旧在期许着什么。
<br/>
<br/>        但是，他期许了一晚，一觉醒来，躺床上拿起手机继续翻看短信时，却仍旧没有收到连星儿的祝福信息。
<br/>
<br/>        难道，她不知道他今天过生日吗？
<br/>
<br/>        应该没理由不知道吧！
<br/>
<br/>        就算他不主动告诉她，她也看过他的资料吧！
<br/>
<br/>        就算她没去记他的生日，那他妈妈总归是是跟她妈妈提起过，那么她妈妈应该也告知过她吧！
<br/>
<br/>        种种迹象证明，连星儿好像压根就没把他的生日当回事，反而跟另一个男生在外彻夜不归。
<br/>
<br/>        明司寒心里苦闷着，有些郁郁寡欢。
<br/>
<br/>        此时，邢乐游和张子晋从食堂里买了早餐回来后，见明司寒已经睡醒了，连忙争先恐后地说道：
<br/>
<br/>        “寒哥，你今天真的是大发啊！”
<br/>
<br/>        “去去去，什么大发，是走桃花运了！”
<br/>
<br/>        邢乐游和张子晋一人一句，说得明司寒一头雾水。
<br/>
<br/>        而这个时候，他俩走进来时未关寝室门，所以门外的聒噪声全部传入了寝室里，走廊上更是有一些男人正站在护栏边往外眺望。
<br/>
<br/>        仲正浩跟着进屋后，笑容满面地吆喝道：“寒哥，真是绝了！楼下有个妹子，将九百九十九朵玫瑰摆成了‘贺思思爱明司寒’，那妹子还带着小提琴，似乎是要当众演奏的趋势。”
<br/>
<br/>        听到“贺思思”的名字，明司寒猛地一弹，直接从床上坐起身来。
<br/>
<br/>        “你说谁？”明司寒惊愕地质问道。
<br/>
<br/>        仲正浩咧嘴一笑：“贺思思爱明司寒！话说回来，贺思思是谁啊？寒哥，你这还藏妹子的啊！”
<br/>
<br/>        明司寒顿时眉头一皱，走廊外便响起了从扩音器里传出来的呼唤声：“明司寒，你给我听好了！我贺思思，喜欢你，不，是爱你。真的真的很爱你。祝你生日快乐！”
<br/>
<br/>        而楼下，连星儿提着礼品袋，和郎安一起，刚走到宿舍楼下的前坪里，就看到了这令她百般不是滋味的一幕。
<br/>
<br/>        那个女孩叫什么来着，如果不是她这样一边自我介绍，一边表白的话，她还真给忘了。
<br/>
<br/>        反过来，她倒是记得，第一次是在医院里与那女孩相遇，第二次是那女孩主动找上门来，想要联合她一起算计明司寒解除婚约结果被她拒绝了。
<br/>
<br/>        总而言之，这女孩是明司寒的爱慕者就对了。
<br/>
<br/>        而这一刻，贺思思拿着扩声器喊完后，周遭围观的男男女女顿时响起了一片热烈的掌声。
<br/>
<br/>        贺思思随后放下手中的扩声器，将小提琴架在了肩膀上，又优雅地拉起了生日祝福歌。
<br/>
<br/>        整个过程，大家就像是在看一场小提琴的独奏会。
<br/>
<br/>        人家一个女生，大庭广众之下，能做出这番举动，实在是令众多男生感动了。
<br/>
<br/>        仲正浩忍不住替楼下那女生说起了情：“寒哥，你就不出去看一眼吗？那个妹子长得又不比连星儿差！”
<br/>
<br/>        张子晋却否决仲正浩的提议：“人家寒哥心里只有连星儿，去看了也是白看。还是不要看的好！”
<br/>
<br/>        “喂！阿寒的心里什么时候只有连星儿呢？阿寒可没承认过他喜欢连星儿！”邢乐游插嘴道，说完后，还睐了一眼坐在床上盯着手机发呆的明司寒，问道，“阿寒，我说得没错吧？”
<br/>
<br/>        楼下，贺思思拉完一首生日祝福歌后，又拉起了表白情歌。
<br/>
<br/>        虽然明司寒没有出来，但是她知道，他一定听得到。
<br/>
<br/>        因为，她在小提琴的音箱处，装了扩声器。
<br/>
<br/>        整栋宿舍楼都能听到，所以他一定也能听得到。
<br/>
<br/>        大概是不好意思，所以就没出来吧！
<br/>
<br/>        贺思思暗自洋洋得意地这么认为。
<br/>
<br/>        当她的目光，在无意间落在人群中站着的连星儿的身上时，脸上的笑容在不经意间地渐渐收敛。
<br/>
<br/>        今天她要将连星儿比下去！
<br/>
<br/>        她要向全世界的人证明，她才是最爱明司寒最适合明司寒的那个女孩。
<br/>
<br/>        连星儿倒是一脸淡定地站在人群里看戏。
<br/>
<br/>        不，与其说是看戏，倒不如说是欣赏。
<br/>
<br/>        她冥思苦想还耗时做的生日礼物，似乎还真不如人家摆那么一圈玫瑰花拉几首曲子的效果。
