　　“啊！”
<br/>　　就在杨之凡等人惊疑不定之时，外面忽然传来一阵阵惨叫声。
<br/>　　继而，“哐当！”一声巨响，
<br/>　　龙盟武道社的大门，便被人从外面直接撞开，紧接着便看到，几名龙盟武道社的社员，从门外滚了进来。
<br/>　　再然后，又有一群人，从外面冲了进来，
<br/>　　“唐威！你竟然……没事？”
<br/>　　钱龙看到来人为首的唐威，顿时两眼一瞪，脸色为之大变。

<br/>　　“区区一点小伤而已，未必能伤的到我！钱龙，我们之间的事情，可还没完呢！”
<br/>　　那何坤山见唐威中气十足，也顿时瞳孔一缩，一脸的难以置信，
<br/>　　“不可能！你中了老夫一掌，怎么可能会没有事？”
<br/>　　何坤山越想越觉得邪门，直接身体一晃，便已经跨越十多米的距离，瞬间便冲到唐威的跟前！

<br/>　　众人顿时感觉到一股热浪迎面袭来，犹如飓风一般，吓得众人连连倒退，连唐威也不由得脸色大变。
<br/>　　这何坤山到底是化劲宗师，相比于这些学生，实在强大太多。
<br/>　　蹬蹬蹬！
<br/>　　唐威下意识的朝后方暴退而去，可惜他的修为，毕竟相差何坤山太多。
<br/>　　“小子！老夫倒要看看，你究竟有什么本事！”
<br/>　　何坤山大喝一声，抬起手掌便朝唐威狠狠拍落下来。
<br/>　　轰！
<br/>　　唐威顿时感觉到一股强大到令人心悸的威严，朝他铺天盖地压倒而来，犹如一座巨山砸落！
<br/>　　眼看对方一掌，他无论如何都无法躲过，唐威心头顿时一凉，
<br/>　　“完了！这才是化劲宗师真正的实力吗？我这下恐怕真的要死了！”
<br/>　　就在唐威心头一阵绝望之时，脑海中陡然响起一道熟悉的声音，
<br/>　　“别怕！打他！”
<br/>　　唐威顿时又惊又奇，
<br/>　　这声音，赫然正是叶尘！
<br/>　　只不过，叶尘的声音，就如同是他自己的念头，在他脑海中陡然响起，简直说不出的诡异神奇。
<br/>　　而就在叶尘的这道声音响起的同时，他又陡然察觉到，自己体内突然多出了一股难以形容的强悍力量！
<br/>　　“不管了！拼了！”
<br/>　　眼看那何坤山那足以碎石裂金的一掌，就要拍落下来，唐威也别无选择，当即双掌狠狠推了出去！
<br/>　　轰隆！
<br/>　　两人的手掌在空中，狠狠相撞在一处，周围空间顿时一阵晃动，整个大厅里掀起了一股热浪风暴，将附近的桌椅都震的粉碎。
<br/>　　而周围，无论是龙盟武道社的社员，还是唐门武道社的社员，全都被这恐怖场景吓得纷纷倒退。
<br/>　　等人众人稳定住身形，慌忙朝场中定睛一看，顿时全都傻眼了，尤其是龙盟武道社一方，更是个个目瞪口呆。
<br/>　　只见，
<br/>　　何坤山和唐威双掌相对之后，唐威竟然依旧纹丝不动的站在那里，而被众人视为无敌战神的何坤山，竟然倒飞了出去，直挺挺的砸落在地板之上，狂喷了一口鲜血，显然已经受伤不轻。
<br/>　　“不！这不可能！”
<br/>　　何坤山被唐威一掌打成重伤，顿时发出一道不甘的吼叫，连双眸也隐隐赤红。
<br/>　　而后方的钱龙和杨之凡，更是全看傻了，
<br/>　　要知道，之前的唐威，可是连何坤山一掌都接不住，两人的修为简直天差地远。
<br/>　　可是这才短短半天不到，唐威不但伤势痊愈，而且竟然如同换了一个人似得，居然一掌便击败了化劲宗师境界的何坤山！
<br/>　　这结果，实在令他们无法接受，但事实摆在眼前，却又由不得他们不信。
<br/>　　其实，岂止是他们，就连唐威自己看到眼前的一幕，同样是一脸懵逼，
<br/>　　要知道，这何坤山在整个华夏武道界，那也是响当当的人物。
<br/>　　哪怕是在他们川蜀唐门之中，恐怕也无一人是此人对手。
<br/>　　可是如今，他居然一掌将何坤山打到吐血？
<br/>　　“难道是因为……”
<br/>　　唐威惊愕过后，猛然想起方才脑海中，突然响起的那道诡异声音，于是慌忙扭头朝后方的叶尘望去。
<br/>　　却见叶尘一脸云淡风轻，只是冲他微微点了点头。
<br/>　　“尘哥……”
<br/>　　唐威实在按耐不住心头的好奇，正打算向叶尘好好询问一番。
<br/>　　然而就在这时，
<br/>　　方才被唐威一掌打飞的何坤山，猛然从地上跳了起来，双眼变得赤红，死死盯着唐威，头发根根倒立，脸色全是疯狂、狰狞之色，疯狂的咆哮起来，
<br/>　　“不！我不信！我不信我会输给你这个无名小辈！”
<br/>　　原来，何坤山无法接受失败，竟然气得发狂了！
<br/>　　“小子！我要宰了你！！”
<br/>　　愤怒之下的何坤山，已经完全丧失了理智，如同疯了一般，再次朝唐威扑了过来。

<br/>　　唐威虽然借助叶尘的力量，勉强击退了何坤山，毕竟战斗经验太少，而且面对的又是名震一方的武林名宿，一时间顿时被吓得连连倒退，竟然都不知道该如何抵抗。
<br/>　　“死吧！”
<br/>　　暴怒之下的何坤山，再也没有任何保留，气息竟然暴涨了将近一倍！
<br/>　　此消彼长之下，唐威顿时越发心惊胆寒，吓得双腿直打颤，险些有些立足不稳。
<br/>　　“唐威！”
<br/>　　后方的李虎和陈东看到这一幕，顿时急的惊呼一声。

<br/>　　轰隆！

<br/>　　“哼！”
<br/>　　叶尘冷哼一声，连眼皮都没有抬一下，只是随手一抓，
<br/>　　“啊！”
<br/>　　伴随着一道惨叫声响起，前一刻还犹如洪荒野兽一般的何坤山，竟然被叶尘一把擒住了脖子，直接拎了起来！
<br/>　　等到众人反应过来，看到那犹如死狗一般，被叶尘擒在手里的何坤山，顿时全部傻眼。
