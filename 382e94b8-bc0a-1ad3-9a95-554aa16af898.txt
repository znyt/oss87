<br/>				陈默心神一转，神识就来到了黑影处。<br/>　　<br/>　　这是一个微缩山脉，占地也就上百亩，最高也不过五、六百米。但是就是这么一个山峰，竟然上面一百多米的高度开始就有着皑皑白雪，空气的温度也散发着冰冷之感，越往上，温度也就越发的寒冷。<br/>　　<br/>　　这个地方，适合养殖一些寒冷地方生长的灵植。有些灵植生长的地方，终年冰雪，温度一高，灵植也就会死亡，所以乾坤珠内才有这么一个地方，而且随着空间不断的增加，这个山峰也会随之变大。<br/>　　<br/>　　根据乾坤珠内传来的信息，随着乾坤珠的禁制不断解开，内部的空间也会随着变大，各种各样的地形也就会随之出现。像是这种寒冷地带，在第二层出现，那么就应该还有热带，或者是比较严酷的沙漠带，火山带，还有沼泽等等各种各样的地形，可能一些蓝海星上没有的地形或者地带，也会在乾坤珠内出现。<br/>　　<br/>　　陈默也很期待，以后会出现的各种地形，这个乾坤珠内的一方世界也就可以变得生动形象起来，就像是真正的星球一样。<br/>　　<br/>　　而且乾坤珠的第二层，还有一个功能，那么就是有了空气的流动。这也是雪山的出现，造成空气的流动。<br/>　　<br/>　　等乾坤珠内更加的大，地形变得更多的时候，那么就不是空气流动了，而是风雨雷电自然气候都会形成。<br/>　　<br/>　　还有一个功能，比较实用，就是可以利用禁制，形成下雨，用来浇注乾坤珠内的花花草草。以前陈默浇花都是要用法决，将溪水吸取上来，利用法决释放云雨数，现在就不用了，只要用神识控制着禁制，就可以在任何想下雨的区域来场随心所欲的雨水。<br/>　　<br/>　　区域可控，雨水可控，容量可控，时间可控，可以说方便的很。<br/>　　<br/>　　陈默将乾坤珠内新增加的功能，一一实验了一番，顿时感觉比以前一层的时候，有了很多的方便功能，能够更好的利用乾坤珠种植一些药材。<br/>　　<br/>　　随心所欲的玩了好久，这才满意的收手。<br/>　　<br/>　　这次乾坤珠内还增加了养殖区域，这个区域和种植区域其实是隔离开的。主要是因为种植区域有很多灵植，对于灵兽的吸引可以说是绝对的致命，为了保护灵植，所以养殖区域是单独隔离开，这样也就省去了很多的麻烦。<br/>　　<br/>　　不过这点对于陈默来说没有任何的区别，因为他现在也没有什么灵兽可言。再说了，他自己在种植区域养鱼养虾的，还用蜜蜂什么的，都是不会对药材起到破坏作用，要是在这里养些牛羊什么的试试，绝对将种植的药材吃个七零八落的。<br/>　　<br/>　　心神一转，陈默就进入了养殖区域。但是一进入后，本来飘在空间的心神顿时就被一下趴在了地上，他感觉到一种威压，从前面传来，自己承受不住，只能从空中趴到地上。<br/>　　<br/>　　这是什么？怎么这么厉害的威压？努力地将自己的神识展望过去，只见一片青翠碧绿，一米多高的青草如同绿毯一样荡漾起伏，整个空间大概有个五百多亩地，两边分别是500亩的淡水湖和咸水湖。两边的湖水波光粼粼，在天空白雾散发的光芒下，闪耀着水光。<br/>　　<br/>　　其实这些都不重要，重要的是在青草陆地的中间，横卧着一头生物。远远看上去，就只能看到一个模糊的身影！这也让陈默大吃一惊，怎么这里有生物？<br/>　　<br/>　　神识在威压之下，努力的接近，想看清是什么生物。但是却离开自己不到几十米，就顿时感觉不继，被威压给缩了回来，陈默神识也一下子被踢出了养殖区。<br/>　　<br/>　　“轰！”脑海中的神识轰然作响，犹如一个大棒槌砸到了脑海中一样，让他眩晕了好久。<br/>　　<br/>　　赶紧心守合一，缓缓运转真元，好长一会，才将震荡的神识平静了下来。<br/>　　<br/>　　不过神识平静下来后，他却发出了：“咦？”的疑问声音。自己的神识似乎增长了一米多，达到了10米左右的长度，这可是在现实中，不是在乾坤珠内，神识可以操控自如，想延伸多远就延伸多远。<br/>　　<br/>　　实际上，陈默自从达到练气三层后，神识也就是9米过一点，不会达到10米以上，随着修炼的增加，功力的进阶，可能会慢慢增加一部分。<br/>　　<br/>　　但是却不应该如同这个时候，神识突然增加一部分。<br/>　　<br/>　　陈默回想了一下后，就发现了问题所在。那就是，刚才自己神识进入养殖区后，被威压压制后，自己在努力抗争的同时，想探明是什么的时候，神识也在这种状态下，有了部分的增长。<br/>　　<br/>　　顿时，他的眼睛发亮了，没有什么能比自己的功力增加，能够让自己快乐的了。竟然有这样的功效，那么自己是不是再接着实验一下？<br/>　　<br/>　　说道做到，陈默立刻心神沉浸到乾坤珠内，然后转瞬之间来到了养殖区。还是那样，一进入养殖区后，就被威压压制到地上。努力抗争，却感觉脑海中犹如针扎一样：“轰！”然一声，他的神识又被踢了出来。<br/>　　<br/>　　不过这次要比上次还要头痛，感觉就像是千万根细针在扎入脑海中一样，疼他的有些禁不住喊出声音来。<br/>　　<br/>　　赶紧喝了一滴灵液，然后缓缓运功，将真元在身体中运行了一边，疼痛才减轻了很多。刚才这种疼痛，陈默有些不寒而栗，怎么会这么痛呢？<br/>　　<br/>　　等稍稍换过劲来，试了试神识的距离，却发现神识增加的不多，没有刚才那一下子多处一米。只是增加了小半米。<br/>　　<br/>　　陈默再一次缓缓运转真元，彻底的将脑海中的疼痛去除后，发现自己练气三层的真元也有了不小的提升。<br/>　　<br/>　　没有想到两次养殖区域的探查，竟然就将神识和功力都增加了一些。有细细想了一遍后，陈默感觉第二次进入太莽撞了。<br/>　　<br/>　　自己的神识刚刚出来，又是增长了一些，稍稍恢复了一下，就着急的再次进入乾坤珠内，这实在是太莽撞了。<br/>　　<br/>　　神识如果受伤，那么损及的可是自己的灵魂。那么当灵魂受伤后，安可不好医治的！要不是乾坤珠已经认可了他，那么自己的神识还真有可能涣散，那绝对不是自己想要看到的。<br/>　　<br/>　　到底是不是自己猜测的这个样子呢，看来还需要修养一晚上，等明天进入乾坤珠的养殖区，就知道结果了。<br/>　　<br/>　　不过，养殖区有了一个生物，还有着这么大的威压，自己岂不是养殖区就白费了么？真的是，有些乐极生悲之感！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
