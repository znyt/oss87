　　面对魏仑和鲁东的道歉，苏莫面色淡淡，只是微微的点了点头。
<br/>　　对于这种墙头草，他懒得理会。
<br/>　　苏莫静静的等待执事归来，褚彦见苏莫的模样，也不再拿热脸去碰冷屁.股。
<br/>　　执事的速度很快，仅仅是片刻的功夫，便回来了。
<br/>　　“段惊天，这是奖励你的源石，你清点一番！”中年执事将一个特制的储物戒，交到了苏莫的手中。
<br/>　　苏莫接过储物戒，稍一检查，便点了点头。
<br/>　　储物戒中的源石，全部是五行源石，数量不知百万，他无需仔细的清点。
<br/>　　“多谢执事大人了！”苏莫向中年执事行礼，心中却是感慨，人王宫果然富裕。
<br/>　　拿出一百多万源石，居然如此的轻松，虽然只是下品源石，也足够阔气了。
<br/>　　“无需言谢，这是你应得的！”
<br/>　　中年执事微微一笑，随即道：“待会，我会将你闯到熔岩窟十八层执事，禀告上去，并通告整个人王殿！”
<br/>　　“啊！”
<br/>　　苏莫闻言，顿时心中一惊，忐忑的道：“执事大人，没必要这么郑重吧？”
<br/>　　苏莫有些郁闷了，这要是闹大了，对他极为不利。
<br/>　　毕竟，此事一大，他的身份怕是很难隐瞒了。
<br/>　　这还不是最主要的，最主要的是，此事若是传到天王宫和地王宫，风险太大了。
<br/>　　他目前，还完全没有实力，抗衡帝一魂和拓跋青云。
<br/>　　在没有明确得到人王宫的支持之时，他的身份绝对不能泄露。
<br/>　　“这是你的荣誉，而且，此事公告全宫，你就算不想拜火陵长老为师，也可以选择其他长老，估计没有哪个长老会拒绝！”中年执事道。
<br/>　　他这完全是为苏莫着想，毕竟，此等荣誉，能带来的好处，实在太多了。
<br/>　　“执事大人，您还是别这样了，我不喜出风头！”
<br/>　　苏莫急忙说道，言罢，他又道：“我会郑重考虑，要不要拜火陵长老为师的！”
<br/>　　“这样啊！”
<br/>　　中年执事闻言，略一沉吟，便点了点头，道：“好吧，那你自己考虑吧，或者，你想拜哪位长老为师，本执事都能为你引荐！”
<br/>　　“多谢执事大人！”苏莫心中大喜，急忙向对方抱拳行礼。
<br/>　　他的心中，亦是暗松了一口气。
<br/>　　“无需客气，以你的天赋而言，在人王宫前途无量！”执事笑着说道。
<br/>　　虽然他不清楚，苏莫到底具体天赋怎样，但是，能到第十八层，本身就是一种天赋。
<br/>　　以前到达第十八层的那十二人，已经有两位成为了神王境的长老，修为最低的，都是八品真神。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，道：“执事大人，没什么事情的话，弟子就先告退了！”
<br/>　　“去吧！”执事挥了挥手。
<br/>　　随后，苏莫便大步离去，用出口的传送阵，离开了熔岩窟。
<br/>　　“不骄不躁，孺子可教也！”
<br/>　　望着苏莫消失的身影，中年长老捋了捋胡须，面带赞赏之色。
<br/>　　如同厉害的天才，居然没有任何的傲气，平易近人，这是难能可贵的。
<br/>　　褚彦等人，目视着苏莫离去，均是心中感慨万千。
<br/>　　“褚彦，此人你还是不要和他交恶了！”
<br/>　　中年执事望向褚彦，沉声告诫道。
<br/>　　褚彦虽然身份不凡，但是，和一个绝世妖孽对抗，并不明智。
<br/>　　“多谢执事告诫，我心中有数！”褚彦阴沉着脸说道。
<br/>　　少倾，褚彦也离开了熔岩窟，带着满心的郁闷。
<br/>　　……
<br/>　　苏莫很快便回到了自己的阁楼，坐在阁楼之中，他有些头疼。
<br/>　　因为他知晓，今日之事，就算不公告全宫，也会渐渐传开的。
<br/>　　不过，他只要不主动去找那些神王境的长老，那些长老应该还不至于主动来收徒。
<br/>　　他的身份，迟早要公开的，但不是现在，时机还不到。
<br/>　　他要等到实力再强一大截，并且，和人王宫很多高层之间，有良好的关系，才能显露身份。
<br/>　　那样的话，人王宫支持他的几率，会大大的提升。
<br/>　　“还是先突破真神吧！”
<br/>　　深深的叹了口气，苏莫也没有查看那火陵长老的道统，便准备开始修炼了。
<br/>　　突破修为，才是第一要务。
<br/>　　阁楼有简单的防护阵法，他打开阵法之后，有进入了空间宝器之中，便开始了闭关。
<br/>　　取出大量的五行源石，他开始吸纳五行源气。
<br/>　　突破到真神，只要做到一点，那就是圣丹之中，玄力化为精纯的源力。
<br/>　　苏莫修炼的混沌之力，他只要让圣丹之中，混沌之力全部化为混沌源力便可。
<br/>　　这难度并不是太大，只是需要的消耗比较大。
<br/>　　苏莫屏弃心中所有的杂念，一心一意的修炼了起来。
<br/>　　当然，他在空间宝器之外，留有一缕意念，谁若是来到了他的阁楼，他第一时间就能知晓。
<br/>　　……
<br/>　　而在苏莫闭关之时，他闯到熔岩窟第十八层之事，也不胫而走，在人王宫之中传播。
<br/>　　很多人闻听此事，均是惊讶不已。
<br/>　　不过，对于‘段惊天’，很多人均是从未听闻，完全没有任何的了解。
<br/>　　一时之间，段惊天之名，在人王宫内外宫之中，都掀起了一股浪潮。
<br/>　　一座辽阔的院落之内，房间内的修炼室之中，容颜绝美的骆妤，有些无奈的看着眼前娇小灵秀的女子。
<br/>　　“沁儿，我在修炼呢！”骆妤有些郁闷，她在闭关，对方却是强行敲门。
<br/>　　“骆妤姐，你还不知道吧，上次那个段惊天，可是妖孽天才！”沁儿面带兴奋之色，眸中露着惊奇，大声的说道。
<br/>　　她倒不是为苏莫高兴，只是感觉很是神奇，自己随便遇到一个人，居然就是妖孽天才。
<br/>　　“段惊天？就是上次那人？他怎么了？”骆妤闻言皱眉，她一直在修炼，并不知道苏莫加入了人王宫。
<br/>　　“对啊，我引荐他进了宫中，你猜怎么着？那家伙居然闯到了熔岩窟第十八层！”
<br/>　　沁儿水灵灵的大眼睛极为闪亮，颇为兴奋的继续道：“真是想不到，他居然如此厉害，吓了我一跳！”
<br/>　　“闯到熔岩窟第十八层了？”骆妤闻言，亦是惊讶了起来。
<br/>　　那个段惊天，居然如此厉害？
<br/>　　“是啊！”沁儿郑重的点头，她心中充满了成就之感，她可是为人王宫，招揽了一位妖孽天才。
<br/>　　“此人又何不凡之处？”骆妤惊奇的问道。
<br/>　　“不清楚，据说有万化神剑元神，其它的就不知晓了！”沁儿摇了摇头道。
<br/>　　骆妤闻言，沉默了片刻，轻声道：“能闯到熔岩窟第十八层，的确是厉害，但是沁儿，这并能说明此人就是绝世妖孽！”
<br/>　　“为何？”沁儿问道。
<br/>　　“说不定此人，有某种厉害的抵御火焰之力的神器，所以才能进入第十八层！”骆妤沉声道。
<br/>　　这并不是她信口开河，虽然熔岩窟之中，有阵法之力，很多神器都无法抵挡，就算是躲在空间神器之中，都无法抵御住火焰之力。
<br/>　　但是，还是有极其特殊的神器，能起到防御火焰之力的效果，虽然可能效果不大。
<br/>　　“就算这样，段惊天也颇为不凡！”
<br/>　　沁儿不在意的摆了摆手，有些兴奋的说道：“我倒要看看，段惊天有多厉害，我们一起去找他吧？”
<br/>　　“这……好吧！”骆妤略一迟疑，便点了点头，毕竟，对方身份尊贵，她也不好推辞。
<br/>　　随即，骆妤出关，和沁儿一起，前去寻找苏莫。
