　　第1438章
<br/>　　宫殿之中，并没有会客大厅，一片空旷。
<br/>　　大殿终于，蒲团之上，苏莫盘膝而坐，他的对面真贤道圣亦是如此。
<br/>　　“不错，武帝境八重境界！”真贤道圣打量了苏莫一番，微微点头，得到了神灵液的洗礼，苏莫能在一年多的时间里，突破两重境界，在他的意料之中
<br/>　　旋即，真贤道圣微笑着问道：“苏莫，你怎么这么快就来了明虚星河，任务完成了？”
<br/>　　“呃~~！”
<br/>　　苏莫闻言有些尴尬，挠了挠头，问道：“真贤师兄，那任务能不做吗？”
<br/>　　“不做？”
<br/>　　真贤道圣闻言一怔，皱眉问道：“怎么了？你没有去做任务？”
<br/>　　“没有！”苏莫摇了摇头。
<br/>　　“苏莫，你为何不做任务？”
<br/>　　真贤道圣面色一沉，继续道：“本道刚从万界山回来，姜神夜的任务都已经完成了，得到了巨大的好处，现在已经突破到武尊境二重境界了！”
<br/>　　“什么？这么快？”苏莫闻言惊讶，姜神夜居然突破到了武尊境二重境界，这未免太快了吧！
<br/>　　要知道这可是武尊之境，每一重的跨度远非武帝境可比。
<br/>　　“当然，他上次经过神灵液洗礼之后，回去没有多久，便成功晋升武尊之境，现状有完成了任务，万界上有巨大的奖励，直接让他达到了武尊境二重境界！”真贤道圣说道。
<br/>　　“真贤师兄，我也是迫于无奈！”苏莫叹了口气，他不羡慕姜神夜，一切只是他自己的原因。
<br/>　　“说出你的无奈！”真贤道圣沉声道。
<br/>　　“真贤师兄，你也知道我在古灵星河的情况，巫族是我的的大敌，而我又无法长时间隐瞒身份，随时会被人认出来，所以只能离开古灵星河了！”苏莫说道。
<br/>　　真贤道圣闻言沉默不语，他知道苏莫和巫族的仇怨，上次在真圣界的分身，就是想要瞒天过海。
<br/>　　沉默了少倾之后，真贤道圣点了点头。道：“苏莫，这个任务本道可以帮你取消！”
<br/>　　“多谢真贤师兄！”苏莫闻言立刻向对方道谢。
<br/>　　“不过，作为师兄本道要提醒你的是，下不为例，不然你将没有资格再作为万界山的弟子，而万界山是不会让知晓其秘密的人活着！”真贤道圣郑重的说道。
<br/>　　他不希望苏莫走到那一步，而且，苏莫是他招揽进的万界山，一旦闹出什么事，他也有责任。
<br/>　　“明白！”苏莫心中暗凛。
<br/>　　“还有就是，你若想快速的成长起来，还需要努力完成万界山的任务，因为任务奖励，非同小可，是你在外界很难得到的！”
<br/>　　真贤道圣提醒道，言罢，他继续道：“不然，你将会被姜神夜远远的甩开，即便你的天赋要优于他！”
<br/>　　“嗯！”苏莫重重的点了点头，不过他并不认为姜神夜能甩开他，或许对方现在比他强，但他只要多抽时间修炼，不必任何人修炼的慢。
<br/>　　“你接下来准备去哪里？是留在道宗，还是自己自由自在？”真贤道圣问道。
<br/>　　“真贤师兄，我想留在道宗。”苏莫沉声说道，他来这里就是要加入道宗，怎么可能离开！
<br/>　　“好，我会被你安排，从今以后，你就是万界山的核心弟子！”真贤道圣道。
<br/>　　“多谢真贤师兄！”
<br/>　　苏莫向对方道谢，略一沉吟，问道：“真贤师兄，我加入了道宗，什么时候能有机会去太阴族？”
<br/>　　“你去太阴族做什么？”真贤道圣面带诧异的问道。
<br/>　　“呵呵！师兄，太阴族是明虚星河的霸主，武道圣地，谁不想去呢？”苏莫笑了笑，并没有告知对方真实原因。
<br/>　　关于青璇的事情，不能泄露，哪怕是真贤道圣，也不能让其知道。
<br/>　　“有机会，只要你能成为真传弟子，就有机会去太阴族！”真贤道圣说道。
<br/>　　“哦？如何能成为真传弟子？”苏莫闻言眼眸一亮，急忙问道。
<br/>　　“本宗真传弟子设有四十八个名额，你只要在不满五十岁之前，能打败其中一人，便能成为真传弟子！“真贤道圣说道。
<br/>　　“那些真传弟子，都是什么修为？”苏莫再次问道，年岁他不关心，关心的只是修为。
<br/>　　他如今才三十多岁，对于他这个修为的武者而言，简直是太年轻了。
<br/>　　“最低的也有武尊境二重，而且每一个都是战力强大之辈！”真贤道圣道。
<br/>　　“明白了！”苏莫点了点头，心中已经了然，不过，以他目前的战力，怕是很难战胜那些真传弟子。
<br/>　　“对了苏莫，以后在道宗之内，外人面前你不能再称呼本道师兄了！”真贤道圣提醒道。
<br/>　　“好！”苏莫应声，对于此事他也明白。
<br/>　　随后，真贤道圣又和苏莫闲聊了片刻，意念一动，探入下方山峰之上，招来了一名道宗的长老。
<br/>　　这名中年长老大腹便便，肥头大耳，偏偏还和真贤道圣一样，穿着一声道袍，显得不伦不类。
<br/>　　“见过真贤长老！”肥胖长老走进宫殿之中，向真贤道圣见礼，声音略显尖细。
<br/>　　“冯光，这是苏莫，从今以后就是本宗的核心弟子，你去安排一番！”真贤道圣指了指苏莫，向肥胖长老说道。
<br/>　　肥胖长老闻言，小眼睛打量了一番苏莫，心中暗暗好奇苏莫的身份。
<br/>　　能被真贤长老直接安排进宗内，这可是不简单啊。
<br/>　　“真贤长老放心，我现在立刻就去安排！”冯光立刻高声领命，对于真贤圣长老的吩咐，他一个普通的道宗长老，丝毫不敢违抗。
<br/>　　“嗯！”
<br/>　　真贤道圣微微颌首，随即对苏莫道：“苏莫，你跟他去吧！”
<br/>　　“嗯！”
<br/>　　苏莫点头，长身而起，然后便跟着冯光长老走出了宫殿。
<br/>　　“苏莫啊！你和真贤长老什么关系啊？”
<br/>　　远离了真贤道圣的宫殿之后，冯光满脸笑意，好奇的问道。
<br/>　　“没有什么关系！”苏莫摇了摇头。
<br/>　　“没什么关系？这不可能吧？”
<br/>　　冯光面上露出不信之色，肥胖的脸庞凑到了苏莫的面前，小声说道：“你告诉我，我绝对不告诉别人！”
<br/>　　苏莫见此，顿时心中无语，这人还是道宗的长老吗？居然如此没有节操！
<br/>　　就连一丝一毫长老的威严也没有，简直就像个好奇宝宝。
