        “我可没有乱说，是一种直觉告诉我的。”童筱颖接着说道。
<br/>
<br/>        季雅姗却紧抿着嘴唇，不再吭声。

<br/>        如果，婴宁不爱十爷。

<br/>        那么十爷这么多年来思念的是什么？
<br/>
<br/>        而她包容的又是什么？
<br/>
<br/>        季雅姗说不出这种感觉来。
<br/>
<br/>        简而言之，她苦苦守候的男人，如果思念着一个不爱他的女人，那么她的守候岂不是成了笑话？
<br/>
<br/>        童筱颖意识到自己不该妄自非议他人的感情，只好也跟着沉默了。
<br/>
<br/>        到了临海城的依山别苑后，季雅珊炒了两个十爷爱吃的菜，盛了碗米饭，放在了托盘上，一并递给童筱颖，示意她端去书房。
<br/>
<br/>        童筱颖点了下头，双手接过季雅珊递过来的托盘，在季雅珊的引领下，走进了书房。
<br/>
<br/>        书房里的光线很暗，厚重的窗帘将窗户遮得严严实实。
<br/>
<br/>        童筱颖四下看了看，没见到那个十爷，于是把饭菜放在了书桌上。
<br/>
<br/>        “阿十？”童筱颖轻轻地唤了声，这是季雅珊让她这么称呼十爷的。
<br/>
<br/>        她心里虽然觉得很是别扭，但嘴上叫出来时还挺顺口。
<br/>
<br/>        这房间里，她没见着十爷冷逸峰的人影，本打算去叫季雅珊的时候，不经意间地转身，便撞到了一个温厚而结实的胸膛上。
<br/>
<br/>        “婴宁，是你回来了，对吗？”男人的声音暗哑地从头上方传来。
<br/>
<br/>        童筱颖还未缓过神来，一双强劲有力的臂膀便将她的身子给圈住。
<br/>
<br/>        但这个怀抱仅仅只维持了几秒，她便被一股力道给推开。
<br/>
<br/>        “你怎么来呢？”刚刚那声温和暗哑的声音，突然间变得冷酷起来。
<br/>
<br/>        童筱颖仰头看向面前的男人，回答道：“大哥不希望你一蹶不振，所以让我来看看你。”
<br/>
<br/>        她没把雅珊供出来，是雅珊事先跟她交代的。
<br/>

<br/>
<br/>        “我觉得，你不要把好心当成驴肝肺！”童筱颖没好气地白了冷逸峰一眼，随后转身往窗户那边走去。
<br/>
<br/>        “嚯唰”了一声，她将窗帘挥向两边。
<br/>
<br/>        一道强光打进来，冷逸峰下意识地抬起手来挡在眼前。
<br/>
<br/>        童筱颖站在逆光之中，冷逸峰看不清她的样子，却看到了与婴宁一模一样的轮廓。
<br/>
<br/>        “……”冷逸峰沉默了。
<br/>
<br/>        童筱颖转过身来，看清楚冷逸峰的模样后，不禁微微皱起了眉头。
<br/>
<br/>        这男人也太邋遢了……
<br/>
<br/>        虽然身上穿着一身白色的休闲便服，但是头发以及长及耳蜗，唇上与下巴处满是青色的胡渣。
<br/>
<br/>        真不知道这个老男人有什么值得季雅珊去喜欢的？
<br/>
<br/>        “你是打算把自己饿死了，然后好跟婴宁去团聚吗？”童筱颖走到冷逸峰的跟前质问道。
<br/>
<br/>        冷逸峰往一旁的书桌上瞥了一眼，饭菜还冒着热气，扑鼻而来的香味，告知他，是雅珊为他准备的。
<br/>
<br/>        与其说是傅夜沉叫她童筱颖过来的，还不如说是季雅珊去“求”她过来的吧！

<br/>        冷逸峰心知肚明，但是没有把这件事情给戳破。
<br/>

<br/>        童筱颖趁机跟他谈起条件来：“你把饭吃了，我就答应你。”

<br/>        “好。”冷逸峰随之绕过书桌，坐在了书桌前的椅子上，拿起托盘上的筷子，端起饭碗，开始动口吃饭。

<br/>        童筱颖倒不觉得是她自己起到了作用，而是季雅珊的饭菜让这位十爷有了食欲。

<br/>        冷逸峰一边吃着碗里的饭菜，一双犀利的黑眸则直勾勾地盯着童筱颖的一举一动。
