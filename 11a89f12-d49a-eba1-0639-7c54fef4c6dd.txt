        要是换做别的男人，一定会质问甚至骂她为什么要这么作贱她自己？
<br/>
<br/>        可是，在他心里，他舍不得骂她。
<br/>
<br/>        他在问她的同时，也在质问自己，为什么就不能选择放弃她童筱颖？为何要如此执着于她童筱颖？
<br/>
<br/>        三年的时光没有让他放弃她，反而越爱越深……
<br/>
<br/>        一阵热流从腹部往上窜，喉咙里突然溢出一股铁锈的味道，傅夜沉抑制不住地捂嘴咳了一声。
<br/>
<br/>        他摊开手之际，手心里星星点点，全是血迹。
<br/>
<br/>        触目惊心！
<br/>
<br/>        童筱颖看到傅夜沉手心里的血，顿时惊慌失措地抓住他的手腕，失声关问：“大哥，你这是怎么呢？”
<br/>
<br/>        傅夜沉直勾勾地看着童筱颖，缓缓抬起手来，向她伸了过去，而她的容颜在视野里出现了重影，慢慢地越来越模糊。
<br/>
<br/>        他的手，还未触及到她的脸颊，眼前忽然一片黑暗……
<br/>
<br/>        童筱颖连忙上前抱住傅夜沉的上半身，随着他身体的重量，缓缓地跪坐在了地上。
<br/>
<br/>        “大哥，你醒醒？大哥！大哥！”她一声又一声地呼唤，他却软瘫在她怀中，像是失去了所有的知觉。
<br/>
<br/>        童筱颖连忙掏出自己的手机给刘婶打电话，并发了自己的位置，告知了傅夜沉的症状，让刘婶把连华生叫过来。
<br/>
<br/>        随后，她将傅夜沉弄进了屋，扶上了床，给傅夜沉做了一番身体检查后，开始在屋子里翻箱倒柜的找外婆生前开过的药单。
<br/>
<br/>        大哥刚刚捂着胸口，应该是心痛，还有吐血……
<br/>
<br/>        她在一堆药单里，找到了与傅夜沉症状相同的病例。
<br/>
<br/>        急火攻心、郁结难舒……
<br/>
<br/>        需消炎止血、益气强心、保肝护胃……
<br/>
<br/>        大哥这是心病所致？！
<br/>
<br/>        好端端，他怎么就患心病了啊！
<br/>
<br/>        童筱颖有点儿抓狂，但想到这里，她连忙起身去了自己前段时间打理好的药草园，照着药单上的名称开始采药。
<br/>
<br/>        连华生一定会带医疗队过来……
<br/>
<br/>        但是从滨江城赶过来，开车最快也要三个多小时。
<br/>
<br/>        不能就这么放任着大哥不管……
<br/>
<br/>        童筱颖采完药草去了厨房，将药草洗干净后，放入了药罐里开始炆药。
<br/>
<br/>        一个小时后……
<br/>
<br/>        她将一碗药端了过来，舀了一勺，吹了吹，自己试了下温度后，喂入了傅夜沉的口中，结果大部分药从他嘴角溢了出来。
<br/>
<br/>        童筱颖见状，于是自己端起药碗喝了一口，结果没含住给吞了下去。
<br/>
<br/>        好苦……
<br/>
<br/>        童筱颖皱着眉头，不得不再喝一口含在嘴中，然后俯下身去。
<br/>
<br/>        她一手托住他的后脑勺轻轻地往上抬，一手捏住他的下颚，迫使他张开嘴巴，然后缓缓地将药喂入他的嘴中。
<br/>
<br/>        一碗药喂完后，童筱颖又去拧了毛巾给傅夜沉擦脸。
<br/>
<br/>        忽然，屋外传来“嗡嗡嗡”的巨响。
<br/>
<br/>        童筱颖连忙起身跑出去查看情况。
<br/>
<br/>        只见半空中，悬浮着一架巨大的暗红色直升飞机，机舱的门打开后，连华生和两个穿着白马褂的男医生站在了升降梯上徐徐落地。
<br/>
<br/>        “你这个女人，给我离阿沉远点！”连华生解开安全绳后，疾步走到童筱颖的跟前，呵斥了一声后，带着两个医护人员往屋内走去。
<br/>
<br/>        童筱颖尾随其后，只见那两个医护人员将傅夜沉从床上扶起，其中一个男人将他背在了背上，另一个男人在跟在一旁护他周全。
<br/>
<br/>        “你给阿沉喂了什么？”连华生又看向童筱颖，指着一旁凳子上的药碗质问道。
<br/>
<br/>        童筱颖把那碗药里的药草名称，一字不漏地全报给了连华生听。
<br/>
<br/>        得知是一些消炎止血、保护内脏的药后，连华生想谩骂童筱颖的声音欲言又止。
<br/>
<br/>        他顿了顿后，瞪着眼睛看着童筱颖，没好气地说道：“这次就饶了你！”
<br/>
<br/>        “我也一起去……”去照顾大哥。
<br/>
<br/>        “你去做什么？我们这里已经没你什么事了！”连华生疾言厉色地打断了童筱颖的话。
<br/>
<br/>        随后，他和那两个医护人员，带着傅夜沉上了直升飞机。
<br/>
<br/>        童筱颖也跟着来到了屋外，仰头看着直升飞机缓缓升空，卷起一地的尘土和树叶，在半空中飞扬打转，然后急速地向着远方飞去。
<br/>
<br/>        虽然一个人被留了下来，但是童筱颖心里还是非常担心傅夜沉，却又不敢直接给傅夜沉打电话。
<br/>
<br/>        晚上，童筱颖坐立不安，终于按捺不住地给刘婶打了通电话，她只是想间接地从刘婶那儿得知傅夜沉是否安好。
<br/>
<br/>        “连医生说，大少爷还未醒过来，所以得留院观察几天。大少奶奶，你和大少爷不是好好的吗？怎么突然就吵架了呢？”刘婶感慨又惆怅着。
<br/>
<br/>        童筱颖坐在屋外的茅草亭里，举着手机贴在耳边，深沉地回答道：“我没跟他吵架，只是想跟他离婚……”
