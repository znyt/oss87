<br/>
<br/>        苏漫雪和童筱颖的背景，傅老太爷已经派人调查得一清二楚。

<br/>        好在，这两个女人的家世背景比较简单。只是童筱颖特殊的地方是，她的身世比较可怜。

<br/>        童筱颖惴惴不安地坐在豪车里，心里一直在想，傅昼景到底是什么身份。
<br/>
<br/>        真的仅仅只是一个普通的建筑设计师那么简单吗？

<br/>        “那个……林叔……我可不可以问您一个问题。”
<br/>

<br/>

<br/>
<br/>        林贵不禁反问：“二少爷没告诉过您吗？”
<br/>
<br/>        童筱颖摇了摇头。
<br/>
<br/>        林贵会意地笑了笑，又接着反问：“不知二少奶奶有没有听过傅氏集团？”
<br/>
<br/>        童筱颖点头。
<br/>
<br/>        林贵接着说：“傅老太爷是傅氏集团的董事长。”
<br/>
<br/>        “啊！”童筱颖惊怔了。
<br/>
<br/>        也就是说，傅昼景是傅氏集团董事长的孙子！
<br/>
<br/>        林贵只说到这，便没再继续说下去了。
<br/>
<br/>        他没告知童筱颖，傅昼景还是个私生子，在傅家非常没地位。
<br/>
<br/>        不仅仅只是没地位，而是有个时候，还得受限于傅老太爷和大少爷。

<br/>        所以，傅昼景是“孤独”的，有个家却等于没家一样。

<br/>        害她前不久刚从韩国回来，又急着再次跑去韩国做微整保养，为的就是要让那位傅家大少爷不发现自己的破绽。

<br/>        童筱颖和苏漫雪第一次进这样古老到只能在电视上才能见到的宅子。
<br/>
<br/>        傅家，家大业大，童筱颖像是参观古园一样，对冷宅惊赞不已。
<br/>
<br/>        这么一来的话，还有……她脖子上的这块玉坠！同样，来历不小吧！

<br/>        既然是祖传玉坠，那么十有八九也是个老古董了！

<br/>        苏漫雪这才老实回答道：“是……是我在韩国的一个拍卖会上拍下的。”

<br/>        “孩子，真的太谢谢你了。”傅老太爷突然语气深沉地说。

<br/>        一直存在苏漫雪心中的疑惑，终于迎刃而解，她一直不敢问刘管家，是怕自己会被穿帮。
