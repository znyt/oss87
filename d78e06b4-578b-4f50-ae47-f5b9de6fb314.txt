　　现在已经是贞观七年，转过年二月就是李承乾加元服，娶老婆的时候，这意味着贞观八年，他李某人就不再是一个少年，而是一个青年，一个成年人。
<br/>　　作为一个成年人，或者即将成年的成年人，李承乾认为现在到了在兄弟们面前亮亮肌肉的时候，打消他们心中一些不切实际的想法，让他们知道：现实世界到底有多么残酷！
<br/>　　但是很快，在李泰还没有认识到世界残酷之前，李承乾就领略到了，小屁孩儿李治骑着他的小四轮出现在他面前：“哥哥，你不守信用。”
<br/>　　“我了个去的！”看到李治，李承乾想起来，似乎在长孙皇后那里自己一时嘴贱答应过带这个小屁孩儿出去玩儿来着。
<br/>　　“哥哥，你是不是又在骗我？”可怜巴巴的李治抬着小脑袋，眼圈开始发红。
<br/>　　“雉奴，哥一直没有出去，不就是在等你么，对不对？”看到小屁孩儿要哭，李承乾连忙安慰。
<br/>　　“真的么？”李治擦了擦眼角，天真的问道。
<br/>　　“当然是真的，哥现在收拾一下，咱们马上就走。”在李治的头上轻轻拍了一下，然后牵着他的小手，李承乾带着满腹的无奈回去换衣服，准备出宫。
<br/>　　不过事实证明，李承乾想的有些多，李治这个小子实在是太屁了，带着他出去逛了一圈东市，吃了一顿迎宾楼之后，这小子就已经又累又困，还没等去下一个地方，就坐在车里睡着了。
<br/>　　“殿下，还去农场么？”无所事事的苏猛坐在豪华大马车的车辕上，回头看着马车里睡的香甜的李治问道。
<br/>　　“回去吧，今天就这样了。”李承乾苦笑着说道。
<br/>　　丫的，早知道小屁孩儿这么好糊弄，何必搞的这么麻烦，直接去西市逛一圈，连饭都能省下来。看来下次出来一定不能带马车，走着出来，估计连皇城都没出这能把李治这小家伙给累的爬回去。
<br/>　　李承乾一个人自娱自乐的想着，冷不防马车一个急停站了下来，前面一个陌生的声音说道：“站住！”
<br/>　　“你们是什么人，纵马拦路意欲何为？”好不容易控制住两匹马的苏猛拧着眉毛问道。
<br/>　　“我家公子看上你们的马车了，说个价儿吧。”拦路的看样子是个家将，不过傲然的样子让人以为他是某位大将军。
<br/>　　“哥哥，这是不是就是强盗啊？”被马车剧烈的摇晃震醒的李治刚好听到拦路之人的要求，还有有些迷糊的小脸转瞬间就变的神采奕奕，分明就是个不怕乱子大的。
<br/>　　“理论上说这个不叫强盗，但是性质应该差不多。”李承乾摊了摊手解释道。
<br/>　　一个小小的家将，还轮不到他一个太子出马，甚至就连李治也都没有要出面的意思，兄弟两个只是觉得这件事情挺好玩的。
<br/>　　踩纨绔，斗流氓，后世的经典桥段，李承乾以为自己这辈子应该都不会遇到，没想到这次竟然有人意外的撞到他枪口上了，这简直就是意外的惊喜。
<br/>　　毕竟李承乾时常会带着自己的豪华大马车穿街过市，长安城里有名的纨绔一般来说都认识他的马车，很少会上来找不自在。
<br/>　　“小子，你是说你要买这架马车？”苏猛也是第一次遇到这种事情，愣了半天才有些不确定的问道。
<br/>　　“怎么，某家说的不清楚么？这里有十两银子，马上拿着银子，带着你家主上……离开。”‘咣当’的一声，一块银饼子随着拦路之人的话音，被丢到了马车的车厢里面。
<br/>　　“若是某不卖呢？”苏猛捡起身边的银饼子，拿在手里掂量着。
<br/>　　车里的两位小爷就是长安城数得着的纨绔，老苏不觉得在这天子脚下还有谁是自己惹不起的。
<br/>　　况且车里两位到现在还没有出声，一边的夜魅也没有发话，分明是打算把戏继续看下去，索性苏猛也就不再着急，打起了戏弄一下那个所谓的‘公子’的主意。
<br/>　　“不卖？不卖回头就等着被家破人亡吧。”一个不阴不阳的声音在马车的一旁响了起来。
<br/>　　紧接着，两个身着锦衣，油头粉面的年轻人就骑着马从后面越过马车，停在了前面。
<br/>　　“家破人亡？就因为一架马车？”苏猛眼睛瞬间瞪大了一圈，嘴角抽了抽，回头与李承乾和李治对视了一眼。
<br/>　　这特么什么情况？要不要这么牛逼？这两个家伙从哪里冒出来的？张嘴就是家破，闭嘴就是人亡。
<br/>　　要知道，就算是大唐皇帝陛下，看好了谁家马车，人家不给，也不至于搞得人家家破人亡吧？
<br/>　　“不信你就试试，今天本公子话放在这里，只要你敢走，你就准备明年今日去你家主上坟头烧纸吧。”另一个油头粉面撇嘴说道，言词语甚是倨傲，似乎不按他说的去做，真的会有灭门惨祸发门一般。
<br/>　　“问问他们的来历，本王想看看，到底谁能砍了本王的脑袋。”小屁孩儿李治探出头，对车外自己带的护卫说道。
<br/>　　终归是个孩子，城府要比李承乾差上许多，一两句话的功夫已经被气的恨不能掏刀子捅人。
<br/>　　李治的护卫接到命令，不敢怠慢，催马上前高声问道：“尔等到底是谁，可敢报上名姓？”
<br/>　　“大丈夫行不更名，坐不改姓，某家武元庆，这是我弟弟武元爽，若是尔等心存不满，长安城里尽可去告，看看到底有没有人敢管这件事。”
<br/>　　“武元庆，武元爽？”李承乾愣了一下，没记错的话这两个家伙好像是武士彟家的两个小子。当然，他们两个的妹妹或许更有名一些，那就是历史记载的武曌武媚娘。
<br/>　　“哥哥，你认识？”李治脸上闪过一丝好奇，似脸问道。
<br/>　　“不认识，不过知道罢了。”李承乾皱了皱眉头，感觉这件事情有些不好处理。
<br/>　　毕竟他们两个是小武的哥哥，如果这事儿传到宫里，不管他们两个如此行事依仗是什么，自己这个太子似乎都会被他们两个牵连。
<br/>　　而且看着武家两兄弟现在的作派，似乎这种巧取豪夺的事情干的也不是一回，这样一来如果真的追究，将来引起的风波怕是小不了。
