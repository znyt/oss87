　　第458章
<br/>　　程大雷深呼吸，再一次深呼吸，压抑住心中肆虐的情绪。
<br/>　　山坡下，是重重包围自己的大军，而身边只有六百残兵。退无退路，进不可进，程大雷如今大概能体会林少羽之前的心情。
<br/>　　上天无路，入地无门，任你有天大力气，也像是被装进一个布口袋中，得到的感悟，只有施展不开四个字。
<br/>　　此情此景，除了放手一搏，似乎别无他法。
<br/>　　“银眸，将我盔甲取来。”程大雷压住心绪，用干净坚硬的声音说。
<br/>　　银眸捧过来一个黑布包裹，程大雷将义字旗扎在地上，将身上沾满鲜血的皮甲脱掉，换上了盔甲。
<br/>　　他身穿照月蛤蟆甲，身披黑色披风，脚蹬虎头战靴，座下是墨玉照月兽，重一百单八斤的鬼面斧拎在手中。
<br/>　　天色将明未明，尉迟离远远看着这一幕，道：“已经陷在绝境，他还想干嘛，来人，告诉他放下武器的话，我能给他一个痛快。”
<br/>　　“呔！”山坡下有人高声喊道：“逆贼程大雷，我家将军大人大量，你若俯首投降，可赏你一个全尸。”
<br/>　　程大雷眉头竖起，用手指比划了一下双方的距离，冲银眸道：“我要他的命，你能不能做到？”
<br/>　　银眸没有说话，当然她也不会说话。她立起来，由山坡之上往下看，最后搬起地上一块石头。
<br/>　　力从地起，劲随腰走，她单手便将西瓜大小的石头丢了出去，然后空中一个黑影不停扩大，咚地一声，将喊话之人的脑袋砸成稀巴烂。
<br/>　　三军皆惊，吃惊银眸的力气，也感慨程大雷的狠辣。
<br/>　　“找死！”尉迟离眉头拧成一团。
<br/>　　程大雷站在高处大喊：“这只是个开始，今日有一个算一个，烦杀我兄弟者，都要付出代价。”
<br/>　　说罢之后，无论下面再说什么，程大雷都不再搭理他们。据程大雷估计，帝国军正调兵遣将，摆好阵势，也收拾刚才大战造成的乱局。反正己方被困在此地，他们并不着急进攻。
<br/>　　这时间，就是程大雷喘息准备的时间。
<br/>　　现在程大雷唯一拥有的，就是六百残兵。徐神机又失踪了，莫说出谋划策，程大雷身边连个逗闷子的人都没有。一切都需程大雷自己决断，至于这六百残兵，都是伤痕累累，身体和精神都极其虚弱。
<br/>　　林少羽其实说得不错，这帮百战之兵，一旦恢复过来，就是一支锐不可挡的精兵。当然，他们也许再没有休养的机会。
<br/>　　程大雷从系统商店中兑换出一些青苓散，让伤重的人服下。
<br/>　　“大当家……”一名老卒接过青苓散的时候握住程大雷的手：“我们还能活得下去么？”
<br/>　　程大雷看着他的眼睛，感受到他虎口的老茧。这名老卒，从前可能是个农夫，铁匠，樵夫……但被正义教拉扯到这乱世中，成了战场上杀人的野兽。
<br/>　　程大雷轻叹口气，将他的手拿开，口中吐出两个字。
<br/>　　“不能。”
<br/>　　程大雷自然可以抛下他们一走了之，以他的本事，加上力大无穷的银眸，脱身应该不难。可如果能这么做，林少羽早如此做了。既然他们是林少羽交给自己的，自己便要对这六百条人命负责。
<br/>　　要么，带着他们突围，要么让他们一起拼死。
<br/>　　如林少羽所言，一个体面的死法。
<br/>　　程大雷心如电转，想着突围之策，看能从系统中借助什么。系统商店的货品，不过皮甲、火药、连弩，青苓散和战马。
<br/>　　奈何唯一有利用价值的火药和连弩也发挥不出太大作用，时间紧急，这六百人不懂得使用，无法将火药或连弩的价值发挥到最大。
<br/>　　唯一的希望，就是从点将台召唤几名武夫，靠他们的武勇冲杀出去。
<br/>　　程大雷决定只召唤顶级武夫，绝世等级强虽强，但不好控制。万一他们出现后，觉得你程大雷作恶多端，十恶不赦，要替天行道杀了你怎么办。
<br/>　　所以顶级最好，能打能抗，一百万恐惧值召唤一次，性价比也高。
<br/>　　想到这里，程大雷心中忽然一动，自己还有一样物事，似乎在此地可以发挥作用——诸神的祝福。
<br/>　　诸神的祝福每三天领取一次，有效期持续三天。每次随机或许一项属性加成，比如工匠之神的祝福，山寨内兵器铺出产的精品武器就多些。
<br/>　　关键是，这诸神的祝福你领就有，你没领就无法生效。程大雷之前准时按时领取，有些祝福作用大些，有些领了几乎等于没领。程大雷之前都是按时领取，但因为这些天面对天罗地网的围追堵截，一直疏忽了这件事。
<br/>　　只希望这次，能领取到有用的BUFF，比如战神的祝福，战斗力暴增，要是领取到爱神的祝福……几乎等于扯淡。
<br/>　　现在这种绝天绝地绝人的三绝之境，任何微不足道的力量能用上都要用上，这样才能在这三绝之境，寻到一丝几乎不可能的转机。
<br/>　　嘟，获得生命之神的祝福。
<br/>　　生命之神的祝福：体力恢复加成，生命恢复加成。
<br/>　　程大雷感觉似乎有一种莫名的力量涌入体内，顺着四肢百骸行走，本来筋疲力尽的身体，重新获得体力补充。不提这些起义军，程大雷一路行来，大战小战无数，本身也是累得够呛。
<br/>　　好在，借着生命之神的祝福，恢复了体力。程大雷还能提着斧头，再冲杀一阵。
<br/>　　回头看山坡上的困兵，脸上都有了些血色。他们不知这改变从何而来，目光呆滞的移动，最后落到程大雷身上。
<br/>　　这帮人本就是被林少羽忽悠进正义教的，林少羽在郭烦人的辅助下，还装神弄鬼显过几次神迹。今天听说程大雷是比林少羽更高妙的存在，在加上体力的恢复，忽然福至心灵，想起来程大雷。
<br/>　　六百残兵面向程大雷跪拜，同时高呼：“圣主大人……”
<br/>　　程大雷摆摆手：“快别扯淡了。”
<br/>　　这生命之神的祝福，最多恢复一半的体力，能打能杀一阵，但带不来太大的改变。
<br/>　　想要从此地突围，还是要想其他法子。
<br/>(本章完)
