　　真圣界内，苏莫伫立与高空之上，周身剑光闪耀，百柄灵剑绽放出夺目的光华。
<br/>　　下方青色平台上，巨摩面色凝重，全身气息涌动，做好了应战的准备。
<br/>　　“十方灵蛇剑阵！去！”
<br/>　　蓦地，苏莫一声轻喝，手指向巨摩凌空一点。
<br/>　　咻！
<br/>　　霎时之间，便有十柄灵剑飞射而出，洞穿长空，击向巨摩。
<br/>　　这十柄长剑首尾相连，彼此气机融为一体，仿若化为了一条游荡虚空的灵蛇，灵动十足。
<br/>　　这灵蛇威力极强，完全不比苏莫刚才巅峰一剑弱多少。
<br/>　　其一因为这是十柄本命灵剑组成，十柄灵剑气机相连，威力叠加。
<br/>　　其次，因为有苏莫剑之战魂的威力加持，威力再次暴增。
<br/>　　这十方灵蛇剑阵，便是苏莫最近才学会的剑阵，虽然才练习了两日的时间，但他已经能熟练操控了。
<br/>　　此剑阵以百柄本命灵剑为基，十柄灵剑为一组，共分十组，剑话灵蛇，威力极大。
<br/>　　咻！
<br/>　　灵蛇的速度奇快无比，快到肉眼难辨，瞬间洞穿层层虚空，直击巨摩的脑袋。
<br/>　　“滚！”
<br/>　　巨摩早有准备，暴喝一声，立刻出拳抵挡，一拳直接砸了出去。
<br/>　　轰！
<br/>　　一声爆响，灵蛇被砸飞了出去。
<br/>　　不过，灵蛇虽然被砸飞，十柄灵剑却并没有溃散，依旧彼此相连。
<br/>　　咻！
<br/>　　巨摩刚刚将灵蛇砸飞，又一条灵蛇裹挟惊天杀机，急速袭杀而来，直刺巨摩的胸膛。
<br/>　　轰！
<br/>　　巨摩再次出拳，立刻又将第二条灵蛇击飞。
<br/>　　然而，下一刻，只见苏莫手指连点，八条灵射四散而开，从前后上下左右几个方向，同时向巨摩展开了攻击。
<br/>　　“不好！”巨摩见此心中大惊，他虽然力量无敌，防御力也很强大，但还没有到能硬抗这灵蛇攻击的地步。
<br/>　　嗖！
<br/>　　不再坐以待毙，巨摩脚下一踏，身形冲天而起，直接迎着上方的袭来的两柄灵蛇冲去。
<br/>　　双拳齐出，上空的两条灵蛇被击飞，然而一瞬间的耽误，剩下的八条灵蛇，已经组成了合围之势。
<br/>　　“糟糕！”
<br/>　　巨摩面色大变，八条灵蛇的同时攻击，让他根本就来不及抵挡。
<br/>　　咻！咻！咻！！
<br/>　　八条灵蛇的速度快到极致，洞穿虚无，分别击向巨摩的胸膛、小腹、后背和后腰等部位。
<br/>　　吼！
<br/>　　眼见八条灵蛇瞬息而至，巨摩顿时狂吼一声，全身玄力涌动，凝聚成了一件玄力铠甲，笼罩全身。
<br/>　　与此同时，他手中光芒一闪，取出了一柄巨大的盾牌。
<br/>　　以盾牌挡住前方，他反手就是一拳，击向后方袭来的灵蛇。
<br/>　　轰！轰！轰！！
<br/>　　爆响声不断的响起，震耳欲聋，火光四溅。
<br/>　　啊！
<br/>　　一声痛呼声响起，一溜鲜血飚射长空。
<br/>　　巨摩挡住了四条灵蛇，扭转身体避开了两条灵蛇，但还是有两条灵蛇没有挡住，这两条灵蛇洞穿了他的玄力铠甲，分别洞穿了他的左肩和右腿。
<br/>　　咻咻咻！！
<br/>　　崩飞的灵蛇迅速返回，十条灵蛇顿时就封住了巨摩的上下前后和左右，完全将其包围了起来。
<br/>　　“巨摩，你败了！认输吧！”
<br/>　　苏莫伫立与高空之上，朗声说道，他没有再动手，只是控制着十条灵蛇将巨摩围困住了。
<br/>　　“这……！”
<br/>　　巨摩脸色有些发怔，身上鲜血潺潺而流，望着周身的十条灵蛇，他心中急思应对之策。
<br/>　　但是，这十条灵蛇的攻击力很强大，速度又奇快无比，他根本没有办法全部防住。
<br/>　　巨摩的脸色变换，他还想反击，但十条灵蛇将他围困，他稍有异动，便会给他雷霆一击。
<br/>　　众人见此，心中震撼不已。
<br/>　　败了！
<br/>　　战力无敌的巨摩居然败了！
<br/>　　苏莫，仅有武帝境六重的修为，也堪称在武帝境之内无敌了。
<br/>　　谁也没有想到，战力逆天的巨摩，居然败的如此之快。
<br/>　　真圣界之外雅雀无声，千万武者都沉默了。
<br/>　　苏莫，真圣榜第一，实至名归！
<br/>　　巨灵族之人脸色难看，大祭司和巫族之人的脸色同样难看，尤天韩的脸色则是已经有些扭曲了起来。
<br/>　　任云都、谢彬、木璃导师等熟悉苏莫的人，则是脸上露出了笑容。
<br/>　　尤其是木璃，心中即感慨又震惊，苏莫的本命灵剑，材料都是她所给予。
<br/>　　短短的时间内，苏莫不仅将本命灵剑都炼制出来了，还将百柄灵剑的剑阵熟练的施展了出来。
<br/>　　这种天赋，让她觉得不可思议！
<br/>　　“真是好苗子！”
<br/>　　逍遥剑圣眸光闪亮，心中暗暗点头，苏莫的十方灵蛇剑阵，虽然修炼的不怎么样，但操控的还算纯熟。
<br/>　　能以武帝境六重修为，熟练的操控十方灵蛇剑阵，已经破位不简单了。
<br/>　　真圣界内。
<br/>　　少倾之后，巨摩叹了口气，目光望向苏莫，沉声道：“你排在真圣榜第一，我服气！”
<br/>　　巨摩不准备再战了，他虽然还有底牌，但那是以命相博的手段，这种比试显然不值得。
<br/>　　虽然这样败了有些丢脸，但他从来不在乎别人的眼光，甚至心中隐隐有些兴奋。
<br/>　　自他踏入武道以来，同阶之中从未一败，这次终于有人能打败他了，还是修为低于他的人。
<br/>　　这样，能让他更有冲劲，更有目标。
<br/>　　别人都说他是战斗狂人，但他只是想找到打败他的人而已，因为站在高处，他已经没有了目标。
<br/>　　这一次真圣界之行，若是没有苏莫的话，他就会挑战姜神夜。
<br/>　　现在真的输了，巨摩心中并没有多少失落，反而心底生出了更强的信念。
<br/>　　听闻对方的话，苏莫点了点头，随即意念一动，十条灵蛇化为百柄灵剑，被他收了回来，进入了识海之中。
<br/>　　“苏莫，希望以后有机会，我们再战！”巨摩朗声说道，言罢便准备主动退出平台。
<br/>　　“你的战力够强，但手段太过单一了！”苏莫沉声道，对方的力量超级强大，但手段单一，弱点太多。
<br/>　　“的确！”巨摩闻言先是一怔，随即便轻轻的点了点头。
<br/>　　这一点他自然知晓，但在以前，他认为只要自己的力量足够强大，一力破万法，不需要太多的手段。
<br/>　　现在看来，这个想法却是错误的，他若是手段够多，说不定这次就不会败。
<br/>　　“告辞！”巨摩向苏莫拱了拱手，随即转身飞出了青色平台，而后身形消失不见。
<br/>　　随后，在巨摩的名字消失之后，苏莫便也退下了平台。
<br/>　　剩下的三名没有挑战的天才，继续挑战，三场战斗之后，彻底结束了第三关。
<br/>　　三关考验，最终只剩下了四十八人，真圣碑外的青光之上只留下了四十八个名字。
<br/>　　随后，巨大的青色平台缓缓消失了。
<br/>　　而真圣碑上，那青光所化的四十八个名字，则是缓缓移动，整齐的排列在了真圣碑上，渐渐的深入真圣碑，直至彻底定格，仿佛是刻印在了碑上一般。
<br/>　　“留名了！”剩下的四十多名天才见此，除了苏莫之外，几乎个个面露喜色。
