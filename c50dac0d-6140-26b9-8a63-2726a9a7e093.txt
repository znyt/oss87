　　听着这前奏，一些关注着叶帝的音乐人眼睛顿时一亮。
<br/>　　“又是一首好歌，这首歌曲的前奏很棒。”
<br/>　　“不愧是大魔王，每一首歌曲都会给人眼前一亮的感觉。”
<br/>　　“6666666666666，这首歌曲的前奏很欢快，想必这肯定是一首非常欢快的歌曲。”
<br/>　　一些坐在一起聊天喝酒的音乐人纷纷开口议论着，然而叶帝一开口，这些人却是瞬间被雷的目瞪口呆，口中的节水都喷出来了也不自知。
<br/>　　“有些事我都已忘记，
<br/>　　但我现在还记得，
<br/>　　在一个晚上，我的母亲问我，
<br/>　　今天怎么不开心。”
<br/>　　听着叶帝在那里喊出来的这段话，直播间里之前满怀期待的粉丝这个时候也都是差点喷出一口老血。
<br/>　　“卧槽啊，这尼玛什么鬼。”
<br/>　　“我怎么感觉我的三观尽毁，这还是歌曲么，大魔王你不要乱搞啊。”
<br/>　　“666666666666，汗毛炸立，惊为天人，此曲只应天上有，人间难得几乎闻。”
<br/>　　“炸你妹啊，这尼玛完全就是一个非主流的歌曲啊。”
<br/>　　“我的耳朵聋了，大魔王是不是已经江郎才尽了。”
<br/>　　“无语了真的，这一点节奏都没有啊，完全就是瞎喊么。”
<br/>　　“你们也不能这么为难大魔王，一周出一首歌，你们见过哪个人比大魔王还有效率的了，所以有些瑕疵也是在所难免的，我相信大魔王肯定会带来更好的歌曲的。”
<br/>　　“江郎才尽！！！”
<br/>　　听着叶帝这第一句歌词，直播间里的无数观众顿时纷纷开口议论了起来。
<br/>　　就是身后的蒋念他们几个听着叶帝唱出的这些玩意后，一个个也都是一脸的无语外加上几分担心，因为从弹幕上他们已经看到有人在喷叶帝了
<br/>　　虽然叶帝如今的人气很高，但是这也并不是所有人都喜欢他的啊，就比如日本或者其他国家的，他们都在找机会黑叶帝，只不过这个机会一直没有出现而已，叶帝表现的太过优秀了。
<br/>　　今天这首歌毫无疑问，是给了一个对方抨击叶帝的机会。
<br/>　　“摩擦，摩擦，
<br/>　　在这光滑的地上摩擦，
<br/>　　月光下我看到自己的身影，
<br/>　　有时很远有时很近，
<br/>　　感到一种力量驱使我的脚步，
<br/>　　有了滑板鞋，天黑都不怕。”
<br/>　　叶帝没有理会直播间里的那些闲言碎语，因为他这个时候已经唱嗨了，这首歌曲乍听之下可能会感觉非主流，但是他却是有着很神奇的魔力，越听越让人想继续听下去。
<br/>　　在地球上这个歌曲刚刚出现的时候也是被人喷，只不过呢，后来不还是成就了庞麦郎，他相信既然在地球上这首歌曲可以火，那么在这个世界上肯定也是可以的。
<br/>　　甚至唱到后来，叶帝这边还来了兴致，干脆把椅子往后一推，直接在地上跳了起来。
<br/>　　看到这一幕，直播间里的众人更加的无语了起来。
<br/>　　“咦？不对，大魔王跳的那个是什么舞，我怎么感觉大魔王明明在向前走，可是他却在后退呢。”
<br/>　　“我去，这个舞蹈加分了啊，虽然歌曲不怎么滴，但是这个舞蹈真的神了，给我的感觉就像是在月球上漫步一样。”
<br/>　　“没错，就是有种太空漫步的感觉，真的太帅了，卧槽，以前怎么没见过啊，不会又是大魔王自创的吧。”
<br/>　　“大魔王会唱歌也就得了，难道他现在又要去祸害这个舞蹈领域了么。”
<br/>　　看到叶帝的这个舞蹈，直播间里原本还有些失望的众人顿时来了兴致，一个个疯狂的在那边议论着。
<br/>　　而一些从事舞蹈职业的人，在看到叶帝的这个动作后，一个个却是都有些目瞪口呆，有些回不过神来，片刻之后才一脸癫狂的在直播间上刷起了弹幕。
<br/>　　叶帝跳的这个舞蹈正式迈克尔的命名的太空步，他的别名叫做月球漫步，在地球上有着无数的人在学习着这一种步伐，只不过这个世界上没有迈克尔这个人，所以太空步自然也就没有人能够研究出来了。
<br/>　　而在最近这半个多月的时间，叶帝主要就是在练习这个舞蹈，如今终于是有所收获，就是以他的天赋都需要花这么长时间，可想而知这个舞蹈是有多么的难。
<br/>　　只不过虽然有些人在夸叶帝，但是他的歌曲却是依旧让人诟病。
<br/>　　“一步两步，一步两步，
<br/>　　一步一步似爪牙，
<br/>　　是魔鬼的步伐，
<br/>　　是魔鬼的步伐，
<br/>　　是魔鬼的步伐。”
<br/>　　听着叶帝唱到这里，无数的观众更加的无语了起来，步伐确实像魔鬼的步伐，不过你这歌词什么鬼。
<br/>　　“大魔王这首歌的歌词真的是烂到家了，歌也难听，真的是江郎才尽了啊。”
<br/>　　“大魔王怎么能够创作出这样的歌曲啊，你如果真的江郎才尽就不要继续创作了啊，将这样一首烂歌拿出来，那不是找喷呢么。”
<br/>　　“2333333333333，笑尿了，这就是你们的大魔王？什么屁歌啊，一点都听不懂，这样的歌曲就算是街边的乞丐也能会唱吧。”
<br/>　　直播间里有的观众在夸赞着叶帝的舞蹈，有的观众则依旧是在那里抨击着他。
<br/>　　当叶帝这首歌曲唱完之后，这些弹幕依旧是没有停止下来。
<br/>　　“哈哈，这首歌曲唱的很开心，你们开心么？”结束后叶帝笑着开口说道。
<br/>　　听着他的话，那些一直担心他的粉丝顿时喷出一口老血。
<br/>　　我开心你妹啊，你开心我们倒是见到了，不过以前见你唱歌的时候也没这么开心过啊。
<br/>　　直播间里骂声、疑问声，什么声音都有，不过叶帝这边却是乐在其中。
<br/>　　“看的出来大家意见很大，那么今天的直播就到这里了，咱们下周见。”叶帝继续笑着开口说道，他这个时候已经是准备将小苹果还有最炫民族风这两首神曲拿出来了。
<br/>　　说不定在这个世界上也能够再现出老大妈广场舞的那个风潮，如果那样那可就真的是太有意思了，估计到时候骂自己的人可能就更多了吧，当然了这可能是年轻人骂自己。
<br/>　　不过如果能够给这些老年人带来快乐，自己被骂一骂又有什么大不了的呢。
