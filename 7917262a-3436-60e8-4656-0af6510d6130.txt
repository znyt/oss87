　　境界越高，修炼所需要的灵气便是越多。
<br/>　　在这武风云的洞府之中，楚风眠也只能是勉强修炼，但是效果微乎其微。
<br/>　　楚风眠必须还要得到更高等山脉中的洞府才行。
<br/>　　不过外门弟子，只有人榜前十的强者，才能拥有七等山脉的洞府，至于更高的，则必须是内门弟子，核心弟子才行。
<br/>　　除非是为门派，立下大功，才有可能直接赏赐下来。
<br/>　　“我现在的实力，已经达到瓶颈，倒是无妨，等到我实力突破，便是距离晋升内门不远了，倒时候成为内门弟子，再去夺取就好。”
<br/>　　楚风眠低声开口道，突然他想起来了那刚刚天际之上，交谈的二人，对着武风云开口询问道。
<br/>　　“这人榜前十中，有没有一个姓叶，一个姓云的人。”
<br/>　　从那二人的语气交谈看来，这二人必定是人榜前十的人，甚至可能是人榜前三的强者，不然不会言语中那般嚣张。
<br/>　　“少主说的莫非是叶广跟云斗二人？”
<br/>　　武风云听到楚风眠的话，恭敬的回答道。
<br/>　　“这二人，分别位列如今人榜榜首跟第二位，这二人，实力上便是早早有着晋升内门的实力，但是并没有选择晋升，一直是在外门中作威作福，论起实力，不比许多内门弟子要弱。”
<br/>　　“其中那叶云，更是据说有着大奇遇，是曾经武王后代，拥有武王血脉，实力不凡，据说足以跟许多巅峰圣者中的佼佼者抗衡，距离生死境已经不远。”
<br/>　　“居然有着如此实力？怪不得敢来算计我。”
<br/>　　听到武风云的话，楚风眠默念一声道。
<br/>　　“但是就算是真正拥有生死境的实力来，敢算计我，也是死路一条，等我真正踏入神力境，生死境武者，我也能灭杀。”
<br/>　　“必须要先攒够功勋值，兑换一次圣山修炼的机会，然后一举突破神力境，才是正事。”
<br/>　　楚风眠心中计算着，他在御风境巅峰，已经是停滞了许久，如今必须要寻找机会，突破神力境才行。
<br/>　　楚风眠感觉的到，只要是当他突破神力境，浑身上下的血脉力量，都会有着急剧的变化，对楚风眠有着无穷无尽的好处。
<br/>　　“你的功勋值又多少，全部交出来。”
<br/>　　楚风眠对着武风云开口道。
<br/>　　“是。”
<br/>　　武风云恭敬的地上身份牌，楚风眠将其中的功勋值全部转了过来，他身份牌中的功勋值，便是已经达到了一千两百点。
<br/>　　这武风云一个人，有着足足一千点功勋值。
<br/>　　“果然，身为外门中的风云人物，倒是有着不少功勋值。”
<br/>　　楚风眠看到这个数字，脸上终于是露出了几分笑容，不过就算是他现在得到了一千二百点，距离圣山修炼一次动辄上万点功勋值的消耗，也不过是九牛一毛。
<br/>　　楚风眠必须要去寻找一个机会，赚取大比的功勋值才行。
<br/>　　在北邙学院中，倒是一直有些一些任务，难度奇高，但是完成后，动辄都可以得到上万功勋值。
<br/>　　楚风眠便是打算要去直接接取这种任务，来一举得到大量功勋值。
<br/>　　八日时间。
<br/>　　楚风眠便是就在这洞府之中，缓缓修行，他并没有在任何的修炼，只是将身体状态，恢复到了最为完美的地步。
<br/>　　“时间到了。”
<br/>　　楚风眠低声说了自言自语一句，他的身形便是消失在了洞府之中。
<br/>　　广场上，如今已经是站满了人。
<br/>　　入门十日之后，这便是任务考核的日子，一眼望去，所有这一届新人中的外门弟子，都是集合在广场之上。
<br/>　　除了这些外门弟子之外，在广场上，还有着数十名紫衣弟子，这一袭紫衣，跟外门的蓝衣，截然不同。
<br/>　　这些紫衣弟子，可不是外门弟子，而是北邙学院中的内门弟子，每一尊的实力，都要超越寻常的巅峰圣者。
<br/>　　这些内门弟子，全部站在广场前方，气势如虹，人人都是傲气十足的一代天才。
<br/>　　“好可怕，这便是内门弟子的威势吗？”
<br/>　　“这些人的眼神，简直便是如同利剑一般，我们根本不敢跟他们对视。”
<br/>　　一声声议论，便是响起。
<br/>　　内门弟子，对这些外门弟子来说，便也是一种属于高高在上，只能仰望的存在。
<br/>　　楚风眠来到广场之上，四处望去，却是没有发现寒月漓的身影。
<br/>　　不止是寒月漓，燕古，牧野战等人，被那些长老带走，收为弟子的他们，也不在这考核之中。
<br/>　　这些人，都是直接晋升为内门弟子，更是长老弟子，自然不需要接受这等繁琐的考核。
<br/>　　“那小子便是楚风眠吗？这一届的新人王，据说是出了不少风头啊。”
<br/>　　一尊内门武者，看着楚风眠到来，心神一动，直接催动灵识，扫视着楚风眠，仿佛是要将楚风眠的一切都给看破一样。
<br/>　　不过楚风眠早有准备，他的底蕴，都全部隐藏了起来，表露出来的，只是表现出来的实力，就算是那些长老都看不穿他的隐藏。
<br/>　　这小小的一尊内门弟子，自然也是不可能看得穿的。
<br/>　　“出风头又如何？不懂隐忍，这样的人，迟早就死在别人的手中。”
<br/>　　又一尊内门弟子，扫视了楚风眠一眼，不屑的开口道。
<br/>　　“小小年纪，得到了几分奇遇，倒是觉得自己可以嚣张了，然而不过只是废物罢了，整个外门中，也就只有那叶广，云斗二人有些实力，剩下的人，全是废物。”
<br/>　　这一尊内门弟子的言语之中，便是嚣张到极点，直接便是将整个外门，不放在眼中。
<br/>　　不过他们这些内门弟子，都是超越外门的存在，随意一尊来到外门的，都是称王称霸的存在，自然不会在乎区区一些外门弟子。
<br/>　　“那倒也是，这小子充其量就是池塘中的鱼，在一个小池塘中嚣张，放在江河湖泊之中，算不得什么。”
<br/>　　又一尊内门弟子，开口道。
<br/>　　在这议论之际，一尊老者，突然从天而降，正是哪位苏凌长老。
<br/>　　看到苏凌长老出现，这些内门弟子也都停止了交谈，等待着苏凌长老开口。
