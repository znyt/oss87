        “我跟你一起去吧！毕竟，我现在是你的未婚夫。见见你朋友，也是应该的。”明司寒云淡风轻地说道。
<br/>
<br/>        连星儿顿时惊愕地抬起头来看着明司寒：“你刚不是说，我两的婚约不作数的吗？你还答应我，会解决这个问题。”
<br/>
<br/>        “在问题没解决之前，你都是我的未婚妻。所以，你最好不要背着我见其他男人！”明司寒翻脸比翻书还快，语气冰冷不说，脸色还很不好看。
<br/>
<br/>        “这个还给你。”连星儿向来吃软不吃硬，二话不说，就直接将手中的小礼盒往明司寒的怀里塞。
<br/>
<br/>        她收回手，转身就往前走。
<br/>
<br/>        当她离开酒店，走到路边打算拦出租车的时候，明司寒开着车，在她身旁停了下来。
<br/>
<br/>        车窗落下，明司寒坐在驾驶座上，说道：“上车吧！我送你过去。”
<br/>
<br/>        “……”连星儿视而不见地侧过身去。
<br/>
<br/>        明司寒接着说道：“你不上车，我两取消婚约的这事情就免谈了。”
<br/>
<br/>        下一秒，连星儿立即拉开了副驾驶座的车门钻了进去。
<br/>
<br/>        明司寒也只是苦笑，无言以对。
<br/>
<br/>        到了洛羽所在的酒店后，连星儿下车前，明司寒叮嘱道：“我就在附近随便逛逛，你玩好了给我打电话，我来接你，送你回家。”
<br/>
<br/>        “没关系，你不用管我。”连星儿微笑着拒绝。
<br/>
<br/>        明司寒冷着脸回答道：“你别忘了，我两现在是怎么出来的。”
<br/>
<br/>        “那我给你打电话也打不通，你应该把我给拉黑了吧？”连星儿说起这件事情时，才猛然想起朱小溪的那件事情，又反问道，“对了，你不是接受朱小溪的表白了吗？你俩好了，为什么同时把我给拉黑了啊？”
<br/>
<br/>        “朱小溪是谁？”明司寒不明所以然地看着连星儿。
<br/>
<br/>        连星儿眨巴着眼睛，心里怀疑明司寒这记性是不是被狗吃。
<br/>
<br/>        “就是那个在网咖里，你推了我，害得我泼了她一身冰激凌奶茶的女生。”
<br/>
<br/>        “……”明司寒这才想起那件事情，冷不丁地调侃道，“我发现，你交朋友真的很没眼光。不是利用你，就是陷害你。”
<br/>
<br/>        “你在胡说八道些什么啊？”连星儿忍不住对明司寒翻起白眼来。
<br/>
<br/>        “你那个朋友，那天在网咖门外，在你的奶茶里下了料。如果不是我推你，把那杯奶茶打翻，我想你现在也不会有机会在这儿跟我好好说话。还有那个……”提及洛羽，明司寒欲言又止。
<br/>
<br/>        连星儿此刻对洛羽一定是有特殊的感情，他若当着她的面，说洛羽的不是，她一定不会听，反而会对他很反感。
<br/>
<br/>        话题到这里，连星儿沉默了。
<br/>
<br/>        半晌后，连星儿靠在副驾驶座里，脸上露出一脸无奈：“我知道朱小溪和张蓉蓉不是真心想要跟我交朋友的，只是没想到，她们会在我的饮料里加料。”
<br/>
<br/>        “你从小到大，很缺朋友吗？”明司寒温和地关问。
<br/>
<br/>        连星儿望向车窗外，感慨万分：“她们跟我交朋友，都抱着不同的目的。我的第一个好朋友，跟我交朋友，居然是因为喜欢我哥哥。我哥哥拒绝她后，她便跟我也绝交了。后来，我转学到这个城市。班里的同学一个个都是尖子生，都在忙着学业，一个个生怕被我比下去，不敢跟我多说几句话。除了朱小溪和张蓉蓉。”
<br/>
<br/>        说着说着，连星儿发现自己的话题扯远了后，回过神来看向明司寒。
<br/>
<br/>        明司寒突然伸手搭在连星儿的头顶上，会心一笑：“你要不要跟我交朋友？我不会利用你、不会陷害你、不会欺骗你。”
<br/>
<br/>        “不对！你不是跟朱小溪……”
<br/>
<br/>        “我以为那封情书是你写的，看样子不是。”明司寒不傻，经过连星儿这么一说，他也大概明白了是怎么一回事了。
<br/>
<br/>        一定是那个朱小溪自己不敢表白，所以冒用了她连星儿的名字，还要连星儿这傻丫头替她来给他送信。
<br/>
<br/>        “我写的？”连星儿一脸懵。
<br/>
<br/>        明司寒会心一笑：“上面的署名是你的名字。”
<br/>
<br/>        连星儿顿时便明白了。
<br/>
<br/>        那个朱小溪又坑她。
<br/>
<br/>        算了，反正朱小溪已经将她拉黑了，她也懒得去跟她计较了。
<br/>
<br/>        看着连星儿一脸挫败的样子，明司寒揉了揉她的头顶，接着说道：“你不是要去见朋友吗？赶紧去吧！”
<br/>
<br/>        “哦，好。”连星儿讷讷地应了声。
<br/>
<br/>        明司寒摇了摇手机，咧嘴道：“我随时接听你的电话，你放心好了。能打通。”
<br/>
<br/>        连星儿这才喜上眉梢，脸上露出了灿烂的笑容。
<br/>
<br/>        明司寒忽然发现，其实连星儿的思想真的很简单。
