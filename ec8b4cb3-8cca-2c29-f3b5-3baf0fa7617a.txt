        连星儿再次点头，只觉自己的脸跟他的脸越靠越近，近到她能闻到他身上那淡淡的香草气息。
<br/>
<br/>        下一秒，明司寒脱口而出：“星儿，我喜欢你。”
<br/>
<br/>        就这瞬间，连星儿身心定格了。
<br/>
<br/>        “我觉得我真傻！从你说不想跟我解除婚约起，我就应该知道，你也喜欢上我了，对不对？”明司寒不假思索地问。
<br/>
<br/>        或许，这压根就不算是一个问题。
<br/>
<br/>        因为，他心里已经有了答案。
<br/>
<br/>        连星儿瞪大了眼睛，胸口似乎有只小鹿在不停地乱撞。
<br/>
<br/>        明司寒看着她震惊的样子，嘴角噙着幸福的笑意，微微偏头，直接吻上了她的唇。
<br/>
<br/>        她刚刚吃过棒棒糖，唇瓣上像抹了蜜糖般一样甜。
<br/>
<br/>        明司寒在她唇上食髓知味后，便不肯就此放过她了。
<br/>
<br/>        连星儿此刻脑海里是一片空白。
<br/>
<br/>        长这么大，平生第一次被一个男生这么热情的亲吻。
<br/>
<br/>        她一直瞪大着眼睛，余光里，她发现，明司寒的耳根子已经红了个通透。
<br/>
<br/>        不知被他吻了多久，当他放开她时，她尴尬地说了一句：“棒棒糖……掉了。”
<br/>
<br/>        明司寒下意识地看了一眼，那根彩虹棒棒糖，还真的掉到了车内的地毯上。
<br/>
<br/>        “没关系，待会我再去给你买。”
<br/>
<br/>        他一边说着，一边探身捡起那根脏了的棒棒糖，扔进了车内的垃圾桶中。
<br/>
<br/>        连星儿抿了抿唇，脸颊发烫，双手捏着衣角不停地绕圈圈。
<br/>
<br/>        明司寒见状，伸过手去拉起她的左手，爱怜地放在自己唇边，轻轻地吻了一下她的手背。
<br/>
<br/>        “宝贝儿，坐稳了，我要开车了。”
<br/>
<br/>        他哄着她，放下她的手后，便去拉了手刹。
<br/>
<br/>        车子驶动后，车内也跟着安静了下来。
<br/>
<br/>        连星儿双手捂着自己发烫的脸颊，看着车窗外，心情久久不能平静。
<br/>
<br/>        到了目的地之后，大家从越野车上将露营的帐篷包搬了下来，然后开始搭帐篷。
<br/>
<br/>        明明大家都在忙活，明司寒却让她连星儿单独坐在一旁看着就行了。
<br/>
<br/>        连星儿四下看了看，倒是觉得这里让她感到很熟悉，她之前似乎来过。
<br/>
<br/>        这里属于临海城的郊区了，前方是一片湖泊，他们扎营的地方是一片草地，而草地的后方则是一片深不见底的原始森林。
<br/>
<br/>        森林里是有防护网的，所以森林里的野生动物出不来，同时他们这群人类也进不去。
<br/>
<br/>        连星儿实在是坐不住了，走到明司寒跟前，眨巴着眼睛说道：“不如，我去捡些柴回来如何？”
<br/>
<br/>        “我已经安排人去捡了，你就乖乖坐在这里，看着我扎帐篷就行了。”明司寒微微一笑，抬起手来，宠溺地刮了刮连星儿鼻梁。
<br/>
<br/>        “可是我觉得好无聊啊！”连星儿瘪了瘪嘴，一脸不满地说道。
<br/>
<br/>        明司寒会心一笑，终于点头了：“那你去捡柴吧！不许跑远了。”
<br/>
<br/>        连星儿做了个“OK”的手势，便跑到顾千凝那儿，想要邀请顾千凝一起陪同自己去捡柴，结果顾千凝的任务是和其他几个女生一起准备晚餐。
<br/>
<br/>        无奈之下，连星儿只能一个人去了。
<br/>
<br/>        捡柴的人不止她一个，还有两男两女。
<br/>
<br/>        连星儿加入他们后，跟他们一起在森林的边缘上游走。
<br/>
<br/>        明司寒一边支帐篷的时候，还一边抬头望向了连星儿那边，见她还在自己的视野里捡干树枝，便放心的继续干着自己手中的活。
<br/>
<br/>        连星儿这里，不知不觉中，捡到了林子里去了，抬头时，看到了眼前这张绿色的防护网。
<br/>
<br/>        防护网那边，有参天大树，鸟叫虫鸣。
<br/>
<br/>        而就在这个时候，两片椭圆形的大叶子突然向两边打开。
<br/>
<br/>        草丛深处，有两只发着金光的兽眸。
<br/>
<br/>        “郎安？”连星儿下意识地唤了声。
<br/>
<br/>        耳畔忽然“嗖”地一声，一个黑影瞬间从草丛里窜到了她的面前。
<br/>
<br/>        连星儿瞪大了眸子之际，就隔着这张防护网，郎安出现在了她的眼前。
<br/>
<br/>        郎安的脸上有抓伤，身上的衣服也很破烂，碎发凌乱，眼神里透着一丝阴郁。
<br/>
<br/>        “郎安，你怎么在这儿？你没有回我外公的实验室吗？”连星儿关问道。
<br/>
<br/>        郎安只是安静地看着她，一句话也不说。
<br/>
<br/>        “你肚子饿不饿？你肚子了的话，我给你吃糖吧！”连星儿一手将捡到的柴抱在怀中，一手从自己的裙兜里掏出了一根棒棒糖，从防护栏的缝隙里伸了过去递给郎安。
<br/>
<br/>        这糖是刚刚在来的路上，明司寒重新买给她的。
<br/>
<br/>        郎安垂眸看了一眼连星儿的手，在抬眸之际，忽然说道：“你，跟我走。”
<br/>
<br/>        “什么？”连星儿怔愣。
