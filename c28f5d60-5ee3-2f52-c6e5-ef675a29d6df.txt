　　“好，天魔宗乃是曾经的魔道领袖，老夫要是得到了这天魔九宝，实力足以大成！”
<br/>　　天字第二号包厢之中，一尊老者猛然站了起来，一脸激动。
<br/>　　这老者的胸前，有着一道巨大无比的蜈蚣印记，看起来狰狞无比，一身浓厚魔意，几乎足以代表他的身份，绝对是一尊地魔门的大人物。
<br/>　　“地魔门中，有谁的胸口上，有着一个蜈蚣印记？”
<br/>　　楚风眠在一旁。悄然开口道。
<br/>　　“楚公子所说的，莫非是百蜈魔尊？”
<br/>　　听到楚风眠的话，李长老想了一下开口道。
<br/>　　“百蜈魔尊，可是地魔门的大人物之一，除了那些老古董中最强的几尊魔尊，怎么楚公子招惹上此人了？”
<br/>　　“那倒不是，只是这百蜈魔尊，如今也在拍卖会中，就在那第二号包厢之中。”
<br/>　　楚风眠点了点头道。
<br/>　　百蜈魔尊，地魔门的大人物，跟着二号包厢中的那老者，应该是一个人不错。
<br/>　　这百蜈魔尊的实力，隐约之间甚至还在不死魔尊之上，在地魔门中的地位绝不会低，甚至可能是如今地魔门实际上真正的大人物。
<br/>　　像是地魔门，北邙学院这样的宗门，并非是没有超越生死的强者，但是这些人，可都是活了数千年的老古董，平日里都在闭关。
<br/>　　像是百蜈魔尊这样的人，才是真正的中坚力量，掌控全局。
<br/>　　“什么，楚公子你居然可以看破这包厢的禁制？”
<br/>　　李长老脸上满是惊讶。
<br/>　　这天字包厢之中的禁制，可是要比起他们天商会的禁制，还要厉害无数倍。
<br/>　　就连李长老都无法看穿，楚风眠居然可以做到。
<br/>　　“有些特殊手段罢了。”
<br/>　　楚风眠随意笑了笑，也不解释。
<br/>　　让李长老心中，看向楚风眠，反倒是有着几分深不可测的感觉。
<br/>　　“这天魔九宝，起拍价，一百万灵石！”
<br/>　　灵尊真人，直接开口道。
<br/>　　“两百万！”
<br/>　　七号包厢之中，玉面龙率先开口。
<br/>　　“三百万！”
<br/>　　六号包厢中的声音，也是不断响起。
<br/>　　“四百万！五百万！一千万！”
<br/>　　价格节节攀升起来，足足是最后达到了七千万的层次，被那二号包厢中的百蜈魔尊买下。
<br/>　　这天魔九宝，虽然威力极强，但是却并不适合楚风眠，楚风眠有着祖龙至尊剑，丰饶碑，人皇罐，根本不需要其他灵器了。
<br/>　　尤其是这天魔九宝，乃是魔道之器，跟楚风眠功法相冲，楚风眠就算是得到，也没有任何意义。
<br/>　　“这最后一件拍品，相比各位也都为为此而来，老夫也不在这里卖关子了，请看！”
<br/>　　灵尊真人长啸一声，在他的手掌之中，一枚瓦片残片，浮现在了空中。
<br/>　　在这瓦片残片之外，蕴含有着无数禁制，显然是特意被设下，用来防备所用。
<br/>　　“这最后一件拍品，便是极道仙兵的残片，这极道仙兵的残片，其中的力量，各位可以感觉的出，跟天级灵器，是截然不同。”
<br/>　　灵尊真人缓缓的开口道。
<br/>　　“这虽然是一件残片，但是诸位若是谁可以得到的话，能够将其炼化，冥冥之中也会得到机遇，也许可以将所有残片收集到，到时候便是拥有者一件完整的极道仙兵。”
<br/>　　灵尊真人的话，令下方沸腾了起来。
<br/>　　极道仙兵。
<br/>　　在九域历史之中，都是至于在远古时代，上古时代中，才出现过的东西。
<br/>　　在如今这个时代，就算是万年之前，都没有存在过了。
<br/>　　谁要是可以得到一件极道仙兵，那也就意味着，近乎成为无敌的人物，谁还可能是他的敌手。
<br/>　　“不错，这真的是极道仙兵的残片，我可以感觉得到，其中的灵力，都不是寻常的灵力，而是仙元，仙力！跟天级灵器，都是云泥之别！”
<br/>　　三号包厢里的女子，看到这人皇罐残片的一刻，都是猛然站了起来，眼神中充满着贪婪。
<br/>　　“这残片，我必须要得到！”
<br/>　　“查查我们还有多少财富！今日就算是拼劲家财，这极道仙兵的碎片也要得到！”
<br/>　　“极道仙兵，谁可以得到，就是真正的无敌！必须要得到！”
<br/>　　“我们奉宗门命令，这极道仙兵的残片，必须要到手！”
<br/>　　这样的议论声，还有很多很多，几乎是在场所有势力，都是为了这极道仙兵的残片而来。
<br/>　　不管是任何一方势力，都绝不会放过这极道仙兵残片的。
<br/>　　在众人的议论之中，灵尊真人猛然开口道。
<br/>　　“极道仙兵的残片，起拍价，一千万！”
<br/>　　“一个亿！”
<br/>　　灵尊真人的话音刚落，十号包厢之中，古林便是猛然开口道，喊出了一个数字。
<br/>　　“什么？”
<br/>　　“直接喊出一个亿的价格？”
<br/>　　在场无数人都是惊呆了，一个亿的灵石，这极道仙兵残片，第一个开口，居然都是喊出了如此之高的价格。
<br/>　　“这不过就是一个极道仙兵的残片，又不是真正的极道仙兵，至于吗？”
<br/>　　一些围观武者，都有些嫉妒的说道。
<br/>　　他的话音未落，在天字包厢之中，又是连连喊出无数数字。
<br/>　　“我出两个亿！”
<br/>　　“两亿五千万！”
<br/>　　“三个亿！”
<br/>　　各个天字包厢之中，都开始了争夺，一声高过一声，这极道仙兵的残片价格，在疯狂飙升。
<br/>　　这简直让人，目瞪口呆。
<br/>　　“这，太疯狂了。”
<br/>　　李长老都是有些瞪大了眼睛，似乎也没有想到，这极道仙兵的残片，居然会价值如此之高。

<br/>　　这样一枚极道仙兵的残片，就算是七大宗门，只怕一个个都会想要得到。

<br/>　　一道声音，突然响起，正是三号包厢之中的那名女子。
