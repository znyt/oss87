　　被紫炎吞噬的毒龙立马打算用真气抵挡。
<br/>　　可是，迟了。
<br/>　　如此多的紫炎。
<br/>　　加之紫炎的温度以及破坏力太过骇人，在他反应过来前，已经将他烧成重伤。
<br/>　　远远看去，一条巨大的七纹黑蟒正在妖异紫芒中痛苦挣扎，不过几秒便停止了挣扎。
<br/>　　七纹黑蟒消失，只留下毒龙残破的尸体。
<br/>　　“散。”萧逸走到他的尸体前，轻喝了一声，紫炎才瞬间消去。
<br/>　　紫晶灵炎，在焚尽物体前，不会消失，除非用真气抵消掉。
<br/>　　至此，盘踞在破玄城外多年，臭名昭彰的毒龙山贼团首领，毒龙，死。
<br/>　　紫晶灵炎，乃是六级妖兽紫炎狮王的本命火焰，连一般的地元境武者都不敢硬抗，一旦被烧中，便会立刻抵消处理掉，不敢大意。
<br/>　　更别说地元境以下的武者了。
<br/>　　毒龙死于紫晶灵炎之下，只能说他运气太背，遇上萧逸这样的变态。
<br/>　　谁能想到，区区一个少年，一个先天武者，能够掌控如斯恐怖的火焰呢。
<br/>　　这时，周围的山贼见毒龙被杀，顿时慌张起来，并四处逃命。
<br/>　　“首...首领死了...其他先天武者也死了，我们快跑。”山贼们惊慌失措地逃命。
<br/>　　所谓树倒猢狲散，毒龙死，另外五个先天境也是死的死伤的伤，剩下一大群凡境和后天境，能有什么作为，不逃才怪。
<br/>　　“哼。”萧逸冷哼一声，数十道火刃打出。
<br/>　　嘭嘭嘭....
<br/>　　一阵阵爆炸声响起，百多个山贼被秒杀。
<br/>　　“阿...这个猎妖师好强，快逃...”火焰爆炸之中，山贼们仓惶嘶叫奔逃，在他们眼中，如今的萧逸简直是个恶魔。
<br/>　　原本在密林中的弓箭手，早就在之前的火海中死伤过半。
<br/>　　上千名山贼，如今能逃的，也不过两三百个。
<br/>　　萧逸本想追，但刚打算释放武技，便发现自己体内真气竟已消耗一空，只好打消念头。
<br/>　　“紫晶灵炎，厉害归厉害，可对真气的消耗同样恐怖。”萧逸暗暗皱眉。
<br/>　　如果靠普通火焰战斗，自己的真气足够战斗很久。
<br/>　　如果释放紫炎，顶多数十道紫炎就耗尽真气了。
<br/>　　半晌，萧逸眉头一松。紫炎这么厉害的手段，消耗再多真气也是值得的，且只要等到自己修为继续提升上去，待得真气多得能够肆意释放紫炎时，自己的战力必然强悍到一个恐怖的地步。
<br/>　　不多时，惊慌失措的山贼们已然狼狈地逃走。
<br/>　　其实，萧逸就算真气耗光，但单凭肉体力量，也能追杀他们。
<br/>　　只是，一群普通山贼罢了，若日后他们继续作恶，遇到其他猎妖师也是死路一条，萧逸便懒得追了。
<br/>　　这时，萧逸在毒龙的尸体上一阵摸索，从怀中摸出一个乾坤袋。
<br/>　　大致看了看，大多是银票和金条，少部分是天材地宝和一些杂物。
<br/>　　银票和金条加起来，大概有二十万两。
<br/>　　不愧是臭名昭彰的山贼团伙，身家果然丰厚。
<br/>　　至于天材地宝，都是些寻常三品药材，价值不大。
<br/>　　“嗯？”萧逸轻咦一声，乾坤袋内，竟有一本功法。
<br/>　　“清风剑罡，玄阶中级功法。”萧逸微微感到惊喜。
<br/>　　这不是武技，而是一本功法，记载着一套精妙的剑法。
<br/>　　这是一门以速度见长的剑法。
<br/>　　一旦练成，武者的剑如疾风般迅速，让敌人难以躲避；身法飘逸灵动，如清风般虚无，让敌人难以捉摸。
<br/>　　同时，还能凝聚出清风剑罡。
<br/>　　清风剑罡，锐利无匹。
<br/>　　之前毒龙便是用这套剑法伤了他，那打伤他肩膀的诡异剑气，就是清风剑罡。
<br/>　　“没想到毒龙还有一本玄阶中级的功法，难怪实力之强，让得破玄城内的猎妖师不是他的对手。”
<br/>　　“只不过，这本剑法对于我如今的实力提升，帮助不大，随便练练算了。”萧逸撇撇嘴，随手将功法装入自己的乾坤袋。
<br/>　　论速度，他有形意五绝中的豹形；论躲避，他有蛇形。
<br/>　　不需要这套剑法的飘逸灵动。
<br/>　　论攻击，无论是崩山斩还是紫炎都比清风剑罡的剑气强，更加不需要。
<br/>　　想罢，萧逸将毒龙以及其他山贼的武魂吸收，大致在这山贼窝逛了一圈，发现没其它东西之后，便离开了。
<br/>　　.....
<br/>　　萧逸一路往裂天剑派的方向赶路，哪怕路过大城也没有过多停留。
<br/>　　中途遇到有大型山贼团伙，便顺手清剿了，他可是接了不少清剿山贼的任务。
<br/>　　若遇到妖兽，则杀了取内丹和妖兽精血。
<br/>　　时间飞逝，一个月眨眼便过。
<br/>　　萧逸的日子便在这般一如既往的枯燥赶路中渡过。
<br/>　　算起来，他已经跨过了差不多二十座大城，距离裂天剑派越来越近了。
<br/>　　今日，他稍微停下了赶路的步伐，进入了最近的一座大城。
<br/>　　这里，有一座猎妖殿，他准备在此交任务。
<br/>　　北山郡共有十座猎妖殿，每隔十数城便有一座。
<br/>　　猎妖师可到任何一殿中交任务和接任务。
<br/>　　猎妖殿虽然分殿遍布，但它是一个势力，有着完整的体系。
<br/>　　进入猎妖殿，立马是喧嚣的声音传来。
<br/>　　如同其他猎妖殿一样，殿内不少猎妖师在办事，也有很多武者在此发布任务。
<br/>　　萧逸来到任务处，将令牌递出。
<br/>　　任务处的人员看了眼令牌，自语道，“哦？是破玄城分殿的任务吗？”
<br/>　　说着，又拿出任务手册和一些资料，认真对比起来。
<br/>　　半晌，他才点点头，道，“嗯，资料显示，这些山贼团伙的首领尽皆死了，各自的手下也死伤惨重纷纷溃散，可以算完成任务了。”
<br/>　　说罢，他再数了一下，道，“十五个四级任务全部完成，每个30点，总共450点。任务点已经给你记录好了，这是你的赏金。”
<br/>　　萧逸接回令牌和赏金。
<br/>　　“易霄先生，还有其他事吗？”猎妖殿人员问道。
<br/>　　萧逸摇摇头，再接了一大堆任务后便离开了。
<br/>　　离开前，他明显注意到殿内的其他猎妖师在窃窃私语，目光更是不时隐晦地看向他。
<br/>　　萧逸皱了皱眉头，冷冷地瞥了他们一眼，而后快步离开。
<br/>　　周围喧嚣的猎妖师，立马让开一条道来。
<br/>　　“嘶，好恐怖的杀气，被他看一眼，浑身打颤。”一中年猎妖师吸了口凉气，惊讶地看着萧逸离开的背影。
<br/>　　“这家伙怎么回事，虽然看不到样貌，但看起来，似乎只是个少年，怎么满身血腥味？”
<br/>　　旁边一些相熟的猎妖师说道，“你不知道刚才那是何人吗？”
<br/>　　那中年猎妖师摇摇头，面露疑惑。
<br/>　　旁边的猎妖师道，“你没有听说过吗？最近不知从哪冒出来一个疯子，专挑山贼匪窝下手。短短一月，已经有十五个大型山贼团被他剿灭。”
<br/>　　中年猎妖师惊道，“我最近都没出任务，快活去了。只是，短短一月，剿灭十五个大型山贼团？不可能吧？”
<br/>　　“什么不可能。”旁边的猎妖师瞪大眼睛说道，“知道我们城外两百里外的青鬼山贼团吗？”
<br/>　　那中年猎妖师道，“当然知道，那青鬼可是先天九重武者，手下还有三位先天八重，山贼过千，盘踞在城外，谁都不敢去招惹他们。”
<br/>　　“哼。”旁边猎妖师道，“就在昨日，他们被团灭了，包括首领青鬼在内，一个都没能逃掉，就是刚才那面具少年所为。”
<br/>　　“嘶。”中年猎妖师再次倒吸一口凉气。
<br/>　　猎妖师们平时忙于出任务，很少会在猎妖殿内闲聊。但这一聊起来，无论是否相熟，都一并过来凑热闹了。
<br/>　　“还有呢，我前些天在外斩杀妖兽，就见过那面具少年。啧啧，无论是落单的三级妖兽，还是成群的妖兽群，只要遇到他，通通被杀光，厉害得很。”
<br/>　　“就单单听到的传言，这附近十数城，他遇妖猎妖，遇山贼杀山贼，一路横扫。甚至有人认为，他可能是洞玄境武者呢。”
<br/>　　“年纪不大，却有如此实力，也不知从哪冒出来的天才武者，恐怕是某些超级家族中出来历练的天才子弟吧。”
<br/>　　“....”
<br/>　　周围不少猎妖师都面露惊讶和赞叹。
<br/>　　中年猎妖师听着旁人滔滔不绝的话语，打断道，“知道那面具少年叫什么名字吗？”
<br/>　　“易霄。”众猎妖师异口同声地说道。
<br/>　　.......
<br/>　　猎妖殿外，萧逸依稀能听到猎妖师们的对话，无奈地摇了摇头。
<br/>　　他没有多作逗留，往城外走去，继续赶路。
