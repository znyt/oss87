　　钦陵的到来让李承乾十分意外，他知道历史上有这么一号人物，不过这家伙一直没有出现，他便也没有去找。
<br/>　　结果没想到，在这个十分平淡的早晨，这货竟然自己送上门来，难道这还真是自己有什么王八之气？带着重重疑惑与不解，李承乾召见了这位传说中的人物。
<br/>　　可当钦陵出现在他面前时，李承乾立刻愣住了，半晌才喃喃说道：“东方亮？”
<br/>　　“对，我就是东方亮，也叫做葛尔·钦陵。”钦陵落落大方的站在书间的中间，与李承乾对视着，没有一丝恐惧。
<br/>　　当然，李承乾也没拍桌子眼睛，因为这完全没有必要。
<br/>　　“你为什么会回来？有什么事？”良久之后，等到宫女将茶水呈上来之后，李承乾对一直在摆弄电灯开关的钦陵问道。
<br/>　　钦陵抬头看着头顶上的灯泡，随着他手上的动作一明一暗的闪着，口中淡淡说道：“累了，不想再走了，所以就回来了。”
<br/>　　“这叫电灯，很快这东西就会装的满大唐都是。”李承乾意有所指的说道。
<br/>　　“好吧，我不玩了。”钦陵摊了摊手，坐正了正身体，然后对李承乾正色问道：“你想杀我么？”
<br/>　　“杀你干什么？你干什么了？”
<br/>　　“你不恨我？”
<br/>　　“恨，不过我认为你活着用处似乎更大一些，如果死了……我岂不是没了报仇的机会。”
<br/>　　李承乾与钦陵对视着，良久，两人一起发出莫名的笑声，随后那钦陵说道：“如果你知道我这段时间都干了些什么，应该就不会这样说了。”
<br/>　　“是么？那你就不想跟我说说？像你的敌人展示一下你的功绩？”李承乾吸溜了一口茶水，漫不经心的说道。
<br/>　　这再次让钦陵有一种拳头打在空处的感觉，沉默了看刻之后才说道：”我离开之前的事情你应该都知道了吧？“
<br/>　　”嗯。”李承乾点点头。
<br/>　　“那么你知道林邑国的叛乱也是我挑起的么？”钦陵看着李承乾问道。
<br/>　　“不是已经平息了么？还提它干什么？”李承乾不为所动的说道，毕竟这么大一个大唐，面对一个全国人口不足两万的属国发动的叛乱，实在提不起什么兴趣。
<br/>　　甚至如果不是老程因为把皇宫给炸了，很可以大唐派兵镇压都不会去做。
<br/>　　钦陵说着说着也觉得有些没有意思，仔细想了想这两年来自己鼓捣的那一点东西，好像真不怎么样，于是也就闭上了嘴巴不在言语。
<br/>　　现如今的大唐实在太大了，大到让人根本无法兴起挑战他的欲望，因为你不管怎么折腾都无法掀起太大的风浪。
<br/>　　而且这事情本身也不是那简单，钦陵记得自己在离开之前，大唐好像只是刚刚征服了西域，可现在他回来了，大唐已经征服了整个天竺，甚至就连倭国也被征服了一半。
<br/>　　一种无力的感觉在钦陵的心底蔓延，似乎不管他怎么折腾都无法阻止大唐前进的脚步，他的所作所为就好像一只挡在车轮前的蚂蚁，除了被冠以不自量力之外，再也不会有任何其他评价。
<br/>　　李承乾看着沉默下来的钦陵，也知道他是词穷了，于是便笑了笑说道：“其实你也不用气馁，在我看来你的能力并不在于这种小伎俩的使用，如果你能把自己能力用在正的一面，想必你现在的成就并不比松赞干布或你老爹差。”
<br/>　　“你是在安慰我么？”钦陵苦笑着问道。
<br/>　　“不，我只是在说一个事实，因为在你做这一切的时候你就在厌恶自己所做的一切，所以你根本就不会全力以赴，不知道我这样说对是不对。”
<br/>　　葛尔·钦陵可以说是一个心高气傲之辈，历料所载，此人乃枭雄，是初唐时候吐蕃除他老子禄东赞之外最杰出的政治家和军事家。
<br/>　　这样一个人，你让他去搞一些阴谋活动，说不好听些，那就是杀鸡用牛刀。
<br/>　　所以当钦陵一个人在外搞事情的时候，从他的内心来讲还是有些抵触的。
<br/>　　而且他并不是一个莽夫，也不会把事情做绝，必竟他老子还在大唐的治下为官，把事情做绝了，对他的家族来说也不是什么好事。
<br/>　　这样一来不管他做什么都会有些放不开手脚，失败也就再所难免。
<br/>　　又是长时间的沉默，直到茶水已经慢慢变冷，李承乾才打破了这份平静：“说说吧，你来找我想干什么？总不会是想要投案自首吧？”
<br/>　　“我想回高原看看，已经好多年没有回去了，来的时候我遇到一个老头儿，他说他的儿子在辽东，虽然他说想让儿子自己去为了未来打拼，但我看得出来那老头儿很想他儿子。”
<br/>　　“葛尔·钦陵竟然会多愁善感？这可是新鲜事儿，回头我一定让人记下来，写到历书里面。”李承乾看着钦陵的样子不由调侃了他一句，而后想了想喃喃低语道：“回去吧，回去看看，你家老头子年龄也不小了。”
<br/>　　“嗯。”钦陵点点头，随后又问道：“我能在军中某个差事么？”
<br/>　　“等你回来再说吧，你若是真的想，我可以安排你去辽东。”李承乾沉声说道。
<br/>　　钦陵点了点头，又摇了摇头啧了一声说道：“为什么你对辽东那么感兴趣？就我所知，那边似乎除了苦寒之外再也没有什么其它的东西了吧？”
<br/>　　“如果你有点耐心，再过十年你就会知道为什么。”
<br/>　　“现在为什么不能说？”钦陵十分好奇的问道。
<br/>　　“时机不到，天机不可泄露，若是现在说了，到时候就不灵了。”李承乾指了指头顶，故做神秘的说道。
<br/>　　“那就这样吧，半年之后我会回来，到时候便去辽东。”
<br/>　　“好，本宫祝你一路顺风！”李承乾看着钦陵离去的背影，淡淡说道。
<br/>　　辽东，倭国占领了那里，硬生生支持那个破烂的岛国在原大地上盘踞了八年，可见辽东北不是什么都没有，大量的矿产资源都埋藏在那片黑土地下面，无论如何大唐都要把那个现在还属于黑水部的地方抢过来。
