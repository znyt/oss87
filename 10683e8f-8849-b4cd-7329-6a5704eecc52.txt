　　“萧逸兄弟，这里有足足数百武者，那可全是分数阿。”林劲怂恿道。
<br/>　　“滚犊子。”萧逸低声骂道，“你行你上，没看到秦飞扬都受伤了吗，老子才不去。”
<br/>　　四人经过几天的相处，彼此都有了不错的交情，不时互相打趣。
<br/>　　“萧逸别听他们的。”柳嫣然道，“你们看，那数百武者为首的家伙，是长风公子，这趟浑水，我们还是别插手的好。”
<br/>　　四人藏身在一边，没有动作。
<br/>　　前方，数百武者将秦飞扬团团围住。
<br/>　　秦飞扬已经身上带伤，脚步一动，刚要凌空跃起，化作金翼雕飞离。
<br/>　　忽然，数十道大网凭空而现，逼得他不得不降落回地。
<br/>　　“哈哈哈哈。”为首的年轻男子得意地笑道，“秦飞扬，我既然早就知道你是金翼雕武魂，岂会不做好准备。”
<br/>　　“我手下的植物系武魂武者，还有铁链武魂的武者，足以让你上天无门。”
<br/>　　“如今你又被团团包围，可谓下地无路。今日，我倒要看看你还如何逃走。”
<br/>　　秦飞扬冷声道，“顾长风，没想到你竟亲自出手，还把你所有狗腿子都带来了。不过，若惹毛了老子，你也别想好过。”
<br/>　　“哈哈。”顾长风丝毫不惧，笑道，“秦飞扬，如今整个森林里，没几个独行武者，你算其中一人，且是最厉害的。一般人数较小的团伙都不是你的对手，这些天，你得了不少分数吧。”
<br/>　　“今日，只要你悉数交出分数，且承诺为我效命，我便饶了你。否则，你将成为这三百里森林内的一具尸体。”
<br/>　　顾长风，也就是长风公子，乃先天九重修为，北山榜排名15。
<br/>　　秦飞扬冷笑道，“顾长风，你敢杀我？你可知我是何人？”
<br/>　　“知道。”顾长风不屑道，“南林秦家的少家主嘛，呵呵，算什么东西。”
<br/>　　“你...”秦飞扬气急，“胆敢轻视我秦家，我要你死。”
<br/>　　说罢，秦飞扬率先发起了攻击。
<br/>　　两方的战斗，瞬间打响。
<br/>　　另一边，藏身的四人看得津津有味。
<br/>　　南林秦家，乃是北山郡的大势力之一。
<br/>　　家族中有一金翼雕之像，又有独门秘法，可让族中族人轻易觉醒金翼雕武魂。
<br/>　　故他们又称为金翼雕一族，处于北山郡的最南方。
<br/>　　金翼雕，是非常稀有的武魂之一，阶品为绿色巅峰。
<br/>　　除了是飞禽武魂外，还有一个特效，便是从觉醒之日起，每隔一年，便能让拥有者自动提升一重修为。
<br/>　　也就是说，十年时间，就算是不修炼，也能突破一个大境界。
<br/>　　效果一直持续到洞玄六重才会结束。
<br/>　　所以，秦家的族人，一般只要上了年纪，最弱的都是洞玄六重修为。
<br/>　　他们的族长，是一位洞玄九重的强大武者。
<br/>　　至于家族中的天才，更是修为过人，像这秦飞扬，本身的修炼天赋，加上金翼雕特效，年纪轻轻，已是先天八重修为。
<br/>　　故此，秦家乃是北山郡拥有洞玄境武者最多的家族，不下百个。
<br/>　　加之他们的族人十分团结，一人有难，全族出动。
<br/>　　所以，他们家族中虽然没有破玄境武者，却没人敢招惹。
<br/>　　只是，裂天剑派位于北山郡最北方，和秦家一南一北，相隔极远。
<br/>　　而那顾长风，本身就是百武城一个大家族中的少家主。其父顾家家主也是一位洞玄九重武者。
<br/>　　所以他敢毫无顾忌地对付秦飞扬。
<br/>　　战斗那边，不多时，秦飞扬已经遍体鳞伤。
<br/>　　顾长风甚至不用出手，单单手下的数百武者，耗也能耗死秦飞扬。
<br/>　　“秦飞扬，莫要冥顽不灵，本公子看得起你，才主动让你效命。”顾长风得意地说着。
<br/>　　“像你这种在南方偏隅之地的土包子，能为本公子效命，是你的荣幸。”
<br/>　　“若本公子看你不爽了，你也不过是废物一个。”
<br/>　　秦飞扬没有说话，疯了似地战斗，将围攻他的武者一个个击杀。
<br/>　　后方，藏身的四人。
<br/>　　萧逸眉头一皱，道，“那顾长风，不是好鸟。反倒是那秦飞扬，傲骨铮铮，不错。”
<br/>　　旁边的林劲说道，“哦？萧逸你看那长风公子不顺眼吗？”
<br/>　　铁牛则瓮声道，“我们四人是一个团队，萧逸兄弟你人很好，你看不顺眼的人，我铁牛也不惯他，要不要一起上去怼他？”
<br/>　　柳嫣然也美眸闪烁，柔声说道，“萧逸你想对付顾长风吗？我听你的。”
<br/>　　萧逸白了他们一眼，道，“刚才不是还说不是对手吗？”
<br/>　　“打不过，大不了跑。”林劲说道。
<br/>　　铁牛也道，“秦飞扬只要化作金翼雕，应该能驼上我们几人轻松离去的。”
<br/>　　萧逸点点头，道，“好，我们上。”
<br/>　　“要是有危险，我会保护你们的。”林劲、铁牛、柳嫣然，三人忽然同时说了一声。
<br/>　　“不用你保护。”三人再次同时说了一声。
<br/>　　“呵呵。”而后四人相视一笑。
<br/>　　萧逸也是暗暗一笑，自己倒是看出了这三人都在隐藏实力，但这三人，似乎都没互相看出来吧。
<br/>　　萧逸不同于他们三人，他连洞玄境武者都对战过，对武者的真正实力感觉很准。
<br/>　　单纯的修为，并不能代表什么，毕竟对于天才来说，越级战斗是家常便饭。
<br/>　　另一边，秦飞扬苦苦支撑，身上的伤势越发严重；而顾长风，则一副胜券在握的小人得志模样。
<br/>　　恰在此时，四道身影快速跃出。
<br/>　　秦飞扬先是一惊，以为有更强的敌人攻击，然而当他看到自己身旁的敌人一瞬间被秒杀后，立刻反应了过来。
<br/>　　“秦飞扬，我们几个是来救你的。”林劲喝道。
<br/>　　“要是我们打不过，赶紧变雕，驮我们走。”铁牛嗡声道。
<br/>　　秦飞扬一愣，还未等他说什么，林劲与铁牛已经和敌人厮打起来。
<br/>　　林劲每一刀挥出，那重若千斤的玄铁重刀，都是将敌人轰得重伤吐血。
<br/>　　铁牛也是，一拳打出，谁都接不住。
<br/>　　柳嫣然千条柳絮凭空而现，将敌人团团束缚住。
<br/>　　萧逸一道道清风剑罡打出，便是数十个敌人身体被穿刺而重伤。
<br/>　　秦飞扬其实并不比林劲等人实力差，只不过是独自一人应敌，很吃力而已。
<br/>　　如今有萧逸四人加入，形势急转。
<br/>　　一个人对付数百人，和五人对付数百人，自然是不同的。
<br/>　　一旁的顾长风见状，顿时一怒，特别是萧逸那边，动辄打伤数十人，让他心头更为愤怒。
<br/>　　“我的事你们也敢管？哼。”顾长风冷哼一声，率先朝萧逸攻去。
<br/>　　在他看来，萧逸杀伤他的手下最多，自然要先杀萧逸。
<br/>　　而且，在他的感知中，萧逸不过先天三重，轻易就能秒杀掉。
<br/>　　“小心。”柳嫣然大喝一声，柳絮飞出，准备捆绑住顾长风。
<br/>　　“破。”顾长风轻喝一声，轻松挣破柳絮，速度不减地朝萧逸攻去。
<br/>　　“来得好。”萧逸怡然不惧，正要对付他时，一道身影抢先一步赶来。
<br/>　　“萧逸兄弟，他交给我吧，你虽然战力过人，但始终是先天三重。以我的防御力，他奈何不了我。”铁牛瓮声说道。
<br/>　　“区区先天六重，也敢拦本公子，不知死活。”顾长风全力一掌轰出。
<br/>　　“轰”的一声。
<br/>　　铁牛身上一头撼地神牛虚影出现，轻松化去了顾长风的攻击。
<br/>　　反倒是顾长风，被震退了数步。
<br/>　　“嗯？好强的防御。”顾长风一惊。
<br/>　　而此时，萧逸和林劲已经再次击杀数十人。
<br/>　　“混蛋，我要你们死。”顾长风顿时怒不可遏，再次全力一掌轰出，手上红光涌动。
<br/>　　“嘻嘻，你伤不了我的。”铁牛自信无比。
<br/>　　然而，萧逸和林劲却是一惊，大喝道，“铁牛，快躲开，那家伙用武技了，观这气势，肯定是高级武技。”
<br/>　　话音刚落，顾长风已经一掌打向铁牛。
<br/>　　铁牛身上的撼地神牛虚影瞬间告破，口吐鲜血，直接被击飞数米。
<br/>　　顾长风趁势直上，意图取铁牛性命。
<br/>　　萧逸、林劲、柳嫣然大惊，意欲施以援手。
<br/>　　顾长风大喝一声，“给我拦住他们。”
<br/>　　百多个武者立刻将他们三人团团围住。
<br/>　　“滚开。”萧逸再无留手，数十道清风剑罡全力打出，秒杀数十人。
<br/>　　前方数十武者鲜血飞溅，清出一条血路。
<br/>　　林劲连忙奔袭，往铁牛身边赶。
<br/>　　柳嫣然千条柳絮挥出，恰恰挡了顾长风一秒时间。
<br/>　　“一群蝼蚁，妄图挡本公子，哼。”顾长风挣脱柳絮，蕴含武技的一掌轰向林劲和铁牛。
<br/>　　“杀神一刀。”林劲身上忽然气势大增，一把滔天巨刃凭空而现。
<br/>　　然而，顾长风也非常人，一掌碎了巨刃，余威不减地打响两人。
<br/>　　“噗。”林劲和铁牛顿时猛吐鲜血，面如金纸，直接被轰成重伤。
<br/>　　林劲那一刀，还有顾长风的血掌，都是玄阶巅峰武技，不相伯仲。
<br/>　　只是，他们一个先天七重，一个先天六重，和顾长风这个先天九重确实差距太大，自然打不过。
<br/>　　若是同为先天九重，顾长风绝对不是他们两人的对手。
<br/>　　“给本公子死。”顾长风冷笑一声，准备彻底击杀林劲和铁牛。
<br/>　　林劲和铁牛双目一冷，手上光芒涌动，似乎有什么底牌要使用。
<br/>　　但，有一人，比他们的动作更快，而且快得多。
<br/>　　“混账，你找死。”萧逸实力全开，升龙，武魂力量结合形意五绝之豹形，速度飙升到极致。
<br/>　　转瞬间已经来到顾长风面前，一剑挥出。
<br/>　　“好快。”顾长风瞳孔一缩，立刻后退。
<br/>　　“没事吧？”萧逸剑指顾长风，微微转头问道。
<br/>　　“没事，那鸟人还杀不了我们。”林劲和铁牛自信一笑。
<br/>　　但他们苍白的脸色，还有嘴角溢着的鲜血，无不表明他们受了不轻的伤势。
<br/>　　“一个接一个，你们还是一并上吧。“顾长风不屑道，“几只蝼蚁，本公子挥手可杀。”
<br/>　　萧逸面露杀意，冷冷道，“林劲，铁牛，那数百个混账尽皆杀了，一个不留；这顾长风，我亲自取他性命。”
<br/>　　话音刚落，萧逸化作一道疾风，直取顾长风而去。
<br/>　　那数百武者刚才的阻拦，还有顾长风，显然都让他杀意凛然。
<br/>　　区区一个先天九重，他还不放在眼里。
