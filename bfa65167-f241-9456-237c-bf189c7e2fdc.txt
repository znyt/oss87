        男人紧紧地抱着她，用尽自己的温度、自己的气息、自己的热情，去拥吻她。
<br/>
<br/>        诺筱颖使出浑身解数，双手乱捶，想要推开面前这个无礼的男人。
<br/>
<br/>        “嘶——”
<br/>
<br/>        男人忽然放开了她，吃痛地捂住了腹部。
<br/>
<br/>        诺筱颖怔愣了一下，猛然发现，他就是昨天晚上那个受了伤爬进她屋内的男人！
<br/>
<br/>        他夺走了她的第一次，竟然还敢出现在她面前！
<br/>
<br/>        “你……”诺筱颖本想破口大骂这男人，但见他痛得脸色惨白，心里又有些愧疚，只好改了口，“哎，算了！对不起啊！我是不小心的！要不，我帮你看看，你腹部上的伤口……”
<br/>
<br/>        “我没事，只是，我逗留的时间不多。本来打算回别苑去看看你，但是管家说，你出门了不在家中。我刚去医院换了药，没想到在这儿遇见了你。所以，我们之间，一定是命中注定的缘分。宝贝，你要等我，等我忙完这段日子，我就能天天陪着你了。”傅夜沉抬起手来，揉了揉诺筱颖的头顶，深情款款地说。
<br/>
<br/>        该怎么办是好？
<br/>
<br/>        他好像，真的已经爱上这个救了他的女人了。
<br/>
<br/>        一见钟情，再见倾心。
<br/>
<br/>        傅夜沉从来没想过，自己会一夜沉沦，陷入到这爱情的俗套之中，无法自拔。
<br/>
<br/>        明明他跟她刚认识不过二十四小时，却恍若认识了很久很久一样。
<br/>
<br/>        或许，他内心深处，对她有种似曾相识的感情。
<br/>
<br/>        正因为这样的感情，让他一直在等待着她的出现。
<br/>
<br/>        诺筱颖却只是目瞪口呆地看着傅夜沉，完全听不懂这个男人到底在说什么。
<br/>
<br/>        “宝贝，我该走了。晚安，要司机早点送你回家，不要让我担心。”傅夜沉随之俯身，温柔地吻了吻诺筱颖的眉心。
<br/>
<br/>        诺筱颖一脸茫然，刚想问这男人说的是什么意思的时候，只见这男人敏捷地转过身去，跃过路边的花坛，利索地跳上了一辆刚好停在路边，里面有人打开了后座的车门的军绿色越野车内。
<br/>
<br/>        “乓”地一声，车门刚关上，那辆军绿色的越野车便扬长而去。
<br/>
<br/>        他赶时间，赶到完全不给她问话的余地。
<br/>
<br/>        诺筱颖抬起手来，拍了拍自己的额头，会痛，所以，刚刚所发生的一切并不是梦。
<br/>
<br/>        还有，这个男人到底是怎么一回事啊？
<br/>
<br/>        一上来连句解释道歉都没有，就强吻她，还对她说了一些莫名其妙的话。
<br/>
<br/>        既然他平安无事，那么就不要再回来找她了啊！
<br/>
<br/>        刚刚看那辆军绿色的越野车，是白色的车牌，也就是说，是军用车。
<br/>
<br/>        难道，他是部队里的人？！
<br/>
<br/>        傅夜沉回到车上后，指腹摩挲着自己的唇瓣，像是在为某件事情回味无穷。

<br/>        战友韩剑锋拍了拍傅夜沉的肩头，噘着嘴，打趣着问：“我说，四少。别以为，刚刚我在车里就什么都没看到啊！怎么样？那女孩的唇，甜不甜？”

<br/>        “……”顿时，傅夜沉紧绷着一张脸，犀利的黑眸，瞪了韩剑锋一眼。

<br/>        韩剑锋忍俊不禁，“嘶溜”一声，抬起手来，在嘴上比了一个拉拉链的手势，示意自己闭嘴就是。

<br/>        没过多久，傅夜沉在进行今天的第二次任务之前，给依山别苑的刘管家发了一条信息，询问苏漫雪有没有回家。

<br/>        嗯，只要她喜欢就好。
