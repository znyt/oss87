        童筱颖就是这样，只要是接到陌生人的电话，她都会这样彬彬有礼地问候。
<br/>
<br/>        大学同窗了三年，她还真是一点变化也没有。
<br/>
<br/>        “筱颖，是我漫雪。”苏漫雪声音哽咽地说道。
<br/>
<br/>        童筱颖顿时就没好语气地问：“你怎么会有我电话？”
<br/>
<br/>        “我问了傅昼景。”苏漫雪黯然神伤地回答。
<br/>
<br/>        “好吧！你找我有事吗？”童筱颖仍旧没好气地问。
<br/>
<br/>        苏漫雪忧伤地回答：“筱颖，你来看看我，好吗？我真的好孤独……真的真的好孤独……”
<br/>
<br/>        “你孤独？什么意思？”童筱颖疑惑了。
<br/>
<br/>        苏漫雪接着解释道：“筱颖，上次在夜沉私宅里遇见那两个歹徒，逼我两交出金钥匙的事情，其实是冷爷爷他一手策划，是冷爷爷想要试探你。”
<br/>
<br/>        “我知道。”童筱颖淡漠道。
<br/>
<br/>        苏漫雪接着说道：“那些伤害你的话，其实是冷爷爷让我这么说的。他是为了让我试探你而已！”
<br/>
<br/>        “爷爷是在试探我们两吧！”童筱颖不以为然地回答，她真不明白，苏漫雪到此刻还在狡辩是为了什么。
<br/>
<br/>        “筱颖，那些伤害你的话，真的不是我的本意。而且，如今，冷爷爷把我囚禁了。不许我跟外界来往，也不许外界的人联系我，还不许我跟夜沉再见面……我、我连跟你打个电话都、都要……呜呜——”
<br/>
<br/>        苏漫雪说着说着，便抽泣了起来。
<br/>
<br/>        童筱颖不禁皱起了眉头。
<br/>
<br/>        “筱颖，我就你一个朋友……我的朋友中，也就只有你是真心待我。我俩可是最好的闺蜜呀！之前，是我对不起你。但是现在……我也尝到了报应……筱颖，你能来陪陪我说说话吗？我真的快要被关疯了……呜呜——筱颖……求你了……我求你了……”
<br/>
<br/>        苏漫雪抱着电话痛哭，一旁的刘管家实在是看不下去了，只好拿来纸巾，朝她递了过去。
<br/>
<br/>        “苏漫雪，你又想玩什么把戏？”童筱颖极其不耐烦地问道。
<br/>
<br/>        苏漫雪吸了吸鼻子，回答道：“筱颖，我没有想玩什么把戏。我就想你过来陪陪我，好不好？我被关在这里快两个月了，不能出门，也没有人愿意来看我。而且，我想和你聊聊傅昼景的事情。就是傅昼景和冷爷爷的事情。”
<br/>
<br/>        一提及傅昼景的事情，电话这边，童筱颖咬了咬唇，心里犹豫了许久，才问道：“你现在在哪？”
<br/>
<br/>        “临海城依山别苑。”苏漫雪哽咽地回答，“你到临海城市中心去，我派人来接你。事关傅昼景，这件事情你一定要知道。”
<br/>
<br/>        “电话里不能说吗？”童筱颖微微皱了皱眉。
<br/>
<br/>        苏漫雪弱弱地回答道：“电话里不方便。”
<br/>
<br/>        童筱颖犹豫了许久，才答应了苏漫雪去见她的要求。
<br/>
<br/>        她如约地赶到了市中心，被苏漫雪派来的人送到了她口中所说的那幢依山别苑院门口，按响院门旁的门铃后，出门来给她开门的是一个发上染霜的中年男子。
<br/>
<br/>        “你……”中年男子看到童筱颖后欲言又止。
<br/>
<br/>        此时，苏漫雪马不停蹄般地从屋内冲出来给童筱颖开门。
<br/>
<br/>        “刘叔，她是我闺蜜，也就是傅家的二少奶奶。”苏漫雪介绍道。
<br/>
<br/>        刘叔立即向童筱颖微微颔首，问候：“二少奶奶，您好！”
<br/>
<br/>        “你好。”童筱颖礼貌性地点了点头。
<br/>
<br/>        苏漫雪忙拉着童筱颖往屋内走。
<br/>
<br/>        刘管家并未多疑，只是好奇，为什么苏漫雪整容后跟傅家的二少奶奶长得一模一样？！
<br/>
<br/>        他只知道自己替傅大少爷接错了人，却并不知道后来所发生的事情。
<br/>
<br/>        进屋后，童筱颖在墙壁上看到了许多军功勋章，当她还未看完，就被苏漫雪强行牵着拉到了二楼的卧室里。
<br/>
<br/>        “筱颖，你能来看我，我真的很高兴……”苏漫雪突然抱住童筱颖，感激涕零地说。
<br/>
<br/>        而卧室门口，突然出现的一个身着黑色西装的女人，将童筱颖吓了一跳。
<br/>
<br/>        “对不起，这位女士。冒犯了，我得搜一下你身上。”女人冷言冷语地说。
<br/>
<br/>        童筱颖不解地看向苏漫雪。
<br/>
<br/>        苏漫雪哭红了眼，委屈地说道：“这是冷爷爷安排的，只要是靠近我的人，都必须搜身，看看有没有携带违禁物品。”
<br/>
<br/>        “好吧！”童筱颖只好张开双手，落落大方地让那女人搜身。
<br/>
<br/>        女人确认童筱颖身上没有携带什么危险物件后，便转身离去。
<br/>
<br/>        苏漫雪瘪嘴道：“瞧吧！你也看到了，我真的是被囚禁了……”
<br/>
<br/>        “爷爷，为什么要囚禁你？”童筱颖难以理解地问。
