那时的叶翎在外地孤身一人，谁都不认识，她躺在床上，看着整个床单都变得湿哒哒的了，那晚的房间静悄悄的，一点声音都没有，静的让人心里发慌。
<br/>
<br/>        她就那样躺在床上，身体痛到麻木了，慢慢的“滴答滴答”的声音就传来了，原来是床单上的血一点一点的滴落到了地板上。
<br/>
<br/>        她听着自己鲜血流出的声音，感觉到什么东西在剥离她的身体，可是这点疼痛却及不上心里的万分之一，她伸手捂住自己全是冷汗的小脸痛哭出声，因为那么一刻她突然明白了，这一场报复，顾夜瑾用了整整十年成为了她的全世界，然后，他亲手毁了她的全世界。
<br/>
<br/>        ……
<br/>
<br/>        顾夜瑾去沐浴间里冲了澡然后出来，他手里拿着一个温热的毛巾伫立在床边，开始帮她擦拭身体。
<br/>
<br/>        叶翎一直是那个姿势闭着眼，让人觉得她好像睡着了。
<br/>
<br/>        顾夜瑾大手捧着她的刘海温柔的往后梳了梳，薄唇落在她美人尖的额头上，“你休息一会儿，我去给你做饭。”
<br/>
<br/>        顾夜瑾离开了房间下楼，进了厨房继续做饭。
<br/>
<br/>        很快身后就传来了一道虚弱的嗓音，“我要吃避孕药。”
<br/>
<br/>        顾夜瑾转身，叶翎不知道什么时候已经下了楼，现在站在门边，她明艳的小脸很苍白，看着像大病一场的样子。
<br/>
<br/>        “吃什么药，我刚才都没有动你，最后的时候…我很小心，不可能怀孕的。”顾夜瑾不悦的掀动薄唇。
<br/>
<br/>        叶翎看着他，苍白的眼神格外的执拗，“你是不是听不懂我的话，现在去给我买药，我要吃药！”
<br/>
<br/>        顾夜瑾俊美的眉心一沉，一双黑眸如刀锋般锋利，“叶翎，闹够了没，你僵硬的身体实在让我索然无味，所以我自己解决的，我还没有说什么你现在脸色苍白的像个鬼摆出一副被强了的模样给谁看，你还要闹吃药，是不是以为我还会像以前那样纵容着你？”
<br/>
<br/>        叶翎收回了目光，“我可以走了吧，我自己去买。”
<br/>
<br/>        她转身离开。
<br/>
<br/>        顾夜瑾健步上前，一把扣住了她纤细的皓腕，掌心里的肌肤滑的跟绸缎似的，但是冰冷刺骨，胸膛里的火气一路往上窜，他倒宁愿看她颠倒众生的妖精模样，而不是被他碰了一下就这样不正常的样子。
<br/>
<br/>        他压抑着心头的火气，“这么晚了你跑到药店里去买避孕药，是想被记者拍，还是去招惹外面那些男人不三不四的目光，你想吃药，好，我现在就去给你买！”
<br/>
<br/>        顾夜瑾丢下她，拿了车钥匙“砰”一声甩上门离开了。
<br/>
<br/>        ……
<br/>
<br/>        顾夜瑾将豪车停在了药店门口，大步走了进去，药店里没有人，收银员在刷手机。
<br/>
<br/>        顾夜瑾单手抄裤兜里，拎着车钥匙的手指蜷起来在桌面上“咚咚”的敲了两下，凌厉有力。
<br/>
<br/>        收银员抬头，双眼直接一亮，哇，好俊俏的男人啊，“先…先生，请问你想要买什么？”
<br/>
<br/>        顾夜瑾看了一下货架上那一排英文名的瓶子，然后伸手一指，“那瓶维C片。”
<br/>
<br/>        ……
