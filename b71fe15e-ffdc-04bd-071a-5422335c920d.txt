        当她刚准备填写时，放在桌上的手机不逢其时地响了起来。
<br/>
<br/>        “筱颍，爷爷跟我说，今天第一批填报高考志愿，你得填临海大学。”
<br/>
<br/>        电话刚一接通，不等童筱颖先开口说话，傅昼景便抢先说道。
<br/>
<br/>        “为什么？我想读滨江大学，这样离家里近。”童筱颖瘪嘴道，一脸不乐意。
<br/>
<br/>        她才不要去临海大学，离家里远不说，而且前男友何明旭也在那所大学里。
<br/>
<br/>        这一辈子，童筱颖都不想再见到何明旭。
<br/>
<br/>        他背着她劈腿、脚踏两条船不说，而且还各种嫌弃她的那副嘴脸，当时真的是伤透了她的心。
<br/>
<br/>        傅昼景无奈地回答道：“没办法，爷爷的话，我们得听。就算你填了滨江大学，到时候爷爷肯定会通过关系把你调到临海大学去。”
<br/>
<br/>        “好吧……我明白了！”童筱颖懈气地应了声。
<br/>
<br/>        “我听说，大嫂也进了临海大学。”傅昼景接着说道，顿了顿后，补充道，“大嫂的分数线还差了点，是爷爷的关系进去的。”
<br/>
<br/>        “……”童筱颖顿时就不说话了。
<br/>
<br/>        “筱颍，你是不是不开心？”傅昼景轻语道。
<br/>
<br/>        童筱颖弱弱地应了声：“嗯。”
<br/>
<br/>        傅昼景还误以为是苏漫雪的缘故，让童筱颖不开心了，于是耐着性子，哄道：“筱颍放心吧！大嫂学的是工商管理专业，筱颍一定对这个专业不感兴趣。所以，就算你两在同一个大学，如果不在同一个系里面，平日里是不会碰见的。”
<br/>
<br/>        “那我学建筑学，好不好？以后等我毕业了，我还可以成为你的贤内助！”童筱颖微笑道。
<br/>
<br/>        “噗嗤”一声，傅昼景忍俊不禁道：“傻筱颍，女孩子学建筑学很累，经常熬夜不说，还得经常坐办公室。真的很累。家里有我一个学建筑的就足够了，你可以考虑别的专业。”
<br/>
<br/>        “你都说了累，那我就更应该去学了。这样，以后，我就可以帮你分担工作，让你不再那么累。”童筱颖异想天开道。
<br/>
<br/>        傅昼景微笑着劝说：“筱颍还是挑个轻松点的专业吧！将来，我不需要你替我分担工作，我只需要筱颍替我照顾好家。”
<br/>
<br/>        “不要，我还是想学建筑学！更何况，建筑学的专业里涵盖了室内设计这门课程。”童筱颖执意道。
<br/>
<br/>        傅昼景也不再勉强，而是打趣地说：“筱颍到时候累到哭鼻子的话，可以来我的怀里躺一躺。”
<br/>
<br/>        “我很吃苦耐劳的，所以才不会哭鼻子！”童筱颖笑着说道。
<br/>
<br/>        傅昼景妥协了：“那你填完高考志愿，把行李收拾了，我们就下楼来吧！大约半个小时后，我在酒店前坪的停车场里等你，我们一起回临海城。”
<br/>
<br/>        “为什么突然要回临海城？”童筱颖好奇地问。
<br/>
<br/>        傅昼景语气里有些无奈：“我小叔公，也就是我爷爷的那个最小的弟弟，今天六十岁大寿。我们得赶回去，给他祝寿，吃他的生日晚宴。”
<br/>
<br/>        这么说来，昼景对她是认真的喽！
<br/>
<br/>        他愿意，把她带回家去见其他亲戚了，也就代表他已经完全认可了她是他的妻子。
<br/>
<br/>        童筱颖不禁喜上眉梢地应了声：“嗯，好！”
<br/>
<br/>        这件事情愉快地决定后，童筱颖很快就填好了自己的高考志愿，并将自己所填的学校和专业发给了季雅姗。
<br/>
<br/>        她没有忘记自己与季雅姗的约定，而她也明明有机会重新选择学医，却为了傅昼景把外婆对她的期望给忘得一干二净。
<br/>
<br/>        其实，童筱颖的外婆，才是傅昼景的亲外婆。
<br/>
<br/>        傅昼景的外婆是赤脚医生，傅昼景的母亲是军医，但他傅昼景学的却是建筑学。
<br/>
<br/>        在傅昼景的心里，如果母亲不是军医，该多好……
<br/>
<br/>        这样他的母亲就不会遇见他的父亲，更不会被大哥的母亲所迫害，让他连一个完整的家都没有。
<br/>
<br/>        滨江城，傅夜沉的私宅。
<br/>
<br/>        傅夜沉本打算一个人回临海城去参加小叔公的生日寿宴，但是当他给爷爷傅炎廷打电话时，无意间听到听筒那边爷爷傅炎廷的助理林贵叔说二少爷和二少奶奶也会回来，于是才决定带上苏漫雪。
<br/>
<br/>        经过昨天的那件事后，苏漫雪彻底学乖了，在傅夜沉的面前，她开始装出一副非常温顺的样子。
<br/>
<br/>        今天一早上起来，苏漫雪不是给傅夜沉做早餐，就是给傅夜沉提鞋，只要是他傅夜沉的生活琐事，她都立马当先地抢着去做。
<br/>
<br/>        虽然傅夜沉一直对她视而不见，甚至充耳不闻，她依旧厚着脸皮倒贴自己。
<br/>
<br/>        只要傅夜沉不赶她走，让她做什么，她都愿意。
